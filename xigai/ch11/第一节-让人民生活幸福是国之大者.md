# 第一节 让人民生活幸福是"国之大者"

> 教材第 225–229 页 · 民生是人民幸福之基。

---

## 一、核心论断（必背金句）

> **让人民生活幸福是"国之大者"**。

> **民生是人民幸福之基**。

> 治国有常，**利民为本**。

### 增进民生福祉的"三个是"（必背）

| 定位 | 内容 |
| --- | --- |
| **本质要求** | 是坚持立党为公、执政为民的**本质要求** |
| **根本目的** | 是社会主义生产的**根本目的** |
| **应有之义** | 是全面建设社会主义现代化国家的**应有之义** |

---

## 二、"三个感"（必背，极高频考点）

::: danger 必背：三个感
不断增强人民：
- **获得感**
- **幸福感**
- **安全感**

——更加充实、更有保障、更可持续。
:::

---

## 三、脱贫攻坚（重大历史成就）

### 脱贫攻坚的成就（数据）

| 指标 | 成就 |
| --- | --- |
| 贫困县 | **832 个**全部摘帽 |
| 贫困村 | **12.8 万个**全部出列 |
| 脱贫人口 | 现行标准下**近 1 亿**农村贫困人口实现脱贫 |
| 历史意义 | **历史性地解决了绝对贫困问题** |

### 精准扶贫"六个精准"（高频考点）

::: danger 必背：六个精准
1. 扶持对象精准
2. 项目安排精准
3. 资金使用精准
4. 措施到户精准
5. 因村派人精准
6. 脱贫成效精准
:::

### "五个一批"

发展生产脱贫一批、易地搬迁脱贫一批、生态补偿脱贫一批、发展教育脱贫一批、社会保障兜底一批。

### "两不愁三保障"（必背）

::: danger 必背：两不愁三保障
- **两不愁**：不愁吃、不愁穿
- **三保障**：义务教育、基本医疗、住房安全有保障
:::

### 预测题一（单选）

**让人民生活幸福是（　）**
A. 党的根本宗旨
B. 国之大者
C. 社会主义的本质要求
D. 共同富裕

> **答案：B**
> **解析**：教材原文——习近平指出，**让人民生活幸福是"国之大者"**。A 是全心全意为人民服务；C 是共同富裕；D 是中特本质要求。

### 预测题二（简答）

**简述脱贫攻坚"两不愁三保障"的内容。**

> 脱贫攻坚战稳定实现贫困人口"**两不愁三保障**"：
> - **两不愁**：不愁吃、不愁穿
> - **三保障**：义务教育有保障、基本医疗有保障、住房安全有保障
>
> 在此基础上，近 1 亿农村贫困人口实现脱贫，832 个贫困县全部摘帽，12.8 万个贫困村全部出列，历史性地解决了绝对贫困问题。

### 易错点提醒

::: warning 本节三大易错点
1. **"两不愁三保障"内容**：吃、穿 + 教育、医疗、住房——不能错。
2. **"总钥匙"vs"指南针"**：发展是总钥匙；民生是指南针。
3. **四句话思路**：坚守底线、突出重点、完善制度、引导预期——顺序固定。
:::
