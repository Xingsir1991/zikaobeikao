# 第一节 文化是民族生存和发展的重要力量

> 教材第 201–206 页 · 文化是民族的灵魂，坚定文化自信。

---

## 一、核心论断（必背金句）

> **文化是一个国家、一个民族的灵魂**。

> 文化兴则国运兴，**文化强则民族强**。

> 没有高度的文化自信，没有文化的繁荣兴盛，就**没有中华民族伟大复兴**。

### 文化繁荣兴盛的"四个必然要求"

| 维度 | 内容 |
| --- | --- |
| **精神支撑** | 是实现中华民族伟大复兴的精神支撑 |
| **应有之义** | 是建设社会主义现代化强国的应有之义 |
| **内在要求** | 是满足人民日益增长的美好生活需要的内在要求 |
| **前提基础** | 是在世界文化激荡中站稳脚跟的前提基础 |

---

## 二、坚定中国特色社会主义文化自信（核心考点）

### 文化自信的地位（必背，回顾）

> 文化自信是**更基础、更广泛、更深厚**的自信，是一个国家、一个民族发展中**最基本、最深沉、最持久**的力量。

::: tip 复习：与第一章呼应
这是第一章"四个自信"部分的深化。文化自信的"三个更"和"三个最"在这里再次得到强调。
:::

### 文化自信的"三个基础"（必背）

| 基础 | 内容 |
| --- | --- |
| **深厚基础** | **中华优秀传统文化**（根脉） |
| **坚强基石** | **革命文化和社会主义先进文化** |
| **现实基础** | **中国特色社会主义伟大实践** |

### "三大文化"的统一（必背）

::: danger 必背：三大文化
中国特色社会主义文化发展，要：
- 发展**社会主义先进文化**
- 弘扬**革命文化**
- 传承**中华优秀传统文化**
:::

---

## 三、坚持中国特色社会主义文化发展道路

### 核心论断

> 中国特色社会主义文化发展道路，是推动社会主义文化繁荣兴盛的**唯一正确道路**。

### "五个方面"文化使命任务（必背）

::: danger 必背：举旗帜、聚民心、育新人、兴文化、展形象
建设社会主义文化强国的五大使命任务：
1. **举旗帜**——高举马克思主义、中国特色社会主义旗帜
2. **聚民心**——把握正确舆论导向，唱响主旋律
3. **育新人**——立德树人，培育时代新人
4. **兴文化**——推动文化创新创造，建设文化强国
5. **展形象**——讲好中国故事，提高国家文化软实力
:::

### 文化工作的"二为"方向和"双百"方针（必背）

| 方针 | 内容 |
| --- | --- |
| **"二为"方向** | 为**人民**服务、为**社会主义**服务 |
| **"双百"方针** | **百花齐放、百家争鸣** |

### "三个面向"的文化性质

> 发展**面向现代化、面向世界、面向未来**的，**民族的、科学的、大众的**社会主义文化。

### "三个不"的创新创造原则

> **不忘本来、吸收外来、面向未来**。
