# 第三节 新时代坚持和发展中国特色社会主义要一以贯之

> 教材第 29–34 页 · 回答"新时代怎么干"——基本理论路线方略与战略布局。

如果说第一节回答"从哪里来"、第二节回答"是什么"，那么第三节回答"**怎么办**"——明确新时代的行动指南和战略部署。

## 二、全面贯彻党的基本理论、基本路线、基本方略（p29-30）

### "三个基本"各自定位（必背）

| 名称 | 定位 | 内容要点 |
| --- | --- | --- |
| **党的基本理论** | 行动指南 | 马克思主义及其中囯化时代化成果（毛、邓、三、科、习） |
| **党的基本路线** | 国家的**生命线**、人民的**幸福线** | 党在社会主义初级阶段的基本路线 |
| **党的基本方略** | "十四个坚持" | 党的十九大概括，涵盖各领域 |

::: tip 重要论断（必背金句）
> **中国共产党为什么能，中国特色社会主义为什么好，归根到底是马克思主义行，是中国化时代化的马克思主义行。**

这"两个行"是新时代的重大理论论断，高频考点。
:::

### 党在社会主义初级阶段的基本路线（必背全文）

> **领导和团结全国各族人民，以经济建设为中心，坚持四项基本原则，坚持改革开放，自力更生，艰苦创业，为把我国建设成为富强民主文明和谐美丽的社会主义现代化强国而奋斗。**

这条基本路线可拆解为：

| 要素 | 内容 |
| --- | --- |
| **一个中心** | 以经济建设为中心（兴国之要） |
| **两个基本点** | 坚持四项基本原则（立国之本）、坚持改革开放（强国之路） |
| **奋斗目标** | 建设富强民主文明和谐美丽的社会主义现代化强国 |
| **主体力量** | 全国各族人民 |
| **精神状态** | 自力更生、艰苦创业 |

::: warning 易错点
奋斗目标中"富强民主文明和谐美丽"是**七个字**（对应"五位一体"），注意党的十九大加了"美丽"二字。旧版只有"富强民主文明和谐"（四个字），不要记成旧版。
:::

### 党的基本方略——"十四个坚持"

党的十九大概括的"十四个坚持"（基本方略），涵盖：坚持党的全面领导和全面从严治党、涵盖"五位一体"和"四个全面"、涵盖国防军队、国家安全、一国两制、祖国统一、对外战略等。

::: tip 与导论"十四个坚持"的关系
导论讲科学体系时提到"**十个明确、十四个坚持、十三个方面成就、六个必须坚持**"。本节的"十四个坚持"就是其中之一，定位是**新时代坚持和发展中国特色社会主义的基本方略**。
:::

## 三、"五位一体"与"四个全面"（核心考点，必背）

### "五位一体"总体布局

| 组成 | 内容 |
| --- | --- |
| **五大建设** | 经济建设、政治建设、文化建设、社会建设、生态文明建设 |
| **五大文明** | 物质文明、政治文明、精神文明、社会文明、生态文明 |

#### "五位一体"的演进过程（高频考点）

教材梳理了总体布局的演进脉络：

| 阶段 | 布局 | 时间 |
| --- | --- | --- |
| 改革开放初期 | "**两个文明**"（物质文明 + 精神文明） | — |
| 后续发展 | "**三位一体**"（经济 + 政治 + 文化） | — |
| 再发展 | "**四位一体**"（加入社会建设） | — |
| **党的十八大** | "**五位一体**"（加入生态文明建设） | 2012 |

::: tip 重大创新
将**生态文明建设**纳入总体布局形成"五位一体"，是重大理论和实践创新，带来了发展理念和发展方式的深刻转变。
:::

### "四个全面"战略布局（必背 + 注意新内涵）

#### "四个全面"的内容

| "全面" | 性质 |
| --- | --- |
| **全面建设社会主义现代化国家** | **战略目标**（引领地位） |
| **全面深化改革** | 战略举措 |
| **全面依法治国** | 战略举措 |
| **全面从严治党** | 战略举措 |

::: danger 高频易错：新内涵替换
"四个全面"的**第一个"全面"已经改变**：
- **旧版**（2020 年前）：**全面建成小康社会**
- **新版**（决胜全面建成小康社会后）：**全面建设社会主义现代化国家**

这是极易考的陷阱！现在的"四个全面"是：**全面建设社会主义现代化国家 + 全面深化改革 + 全面依法治国 + 全面从严治党**。
:::

#### "四个全面"的内在逻辑

> "四个全面"既有**战略目标**又有**战略举措**：
> - 战略目标：全面建设社会主义现代化国家（引领地位）
> - 三大战略举措：全面深化改革、全面依法治国、全面从严治党（提供重要保障）

四者相辅相成、相互促进、相得益彰，是统一体。

## 四、推动中国特色社会主义不断开拓前进（p32-34）

### 核心论断：坚持和发展中国特色社会主义是一篇"大文章"

习近平强调：

> "坚持和发展中国特色社会主义是一篇大文章"，"现在，我们这一代共产党人的任务，就是**继续把这篇大文章写下去**"。

### 两个"不容置疑"（辩证理解）

| 一方面 | 另一方面 |
| --- | --- |
| 对社会主义的认识、对建设规律的把握已达到**前所未有的新高度**（不容置疑） | 我国社会主义还处在初级阶段，面临很多没弄清的问题（也不容置疑） |

这体现了**认识过程**的辩证法：既肯定成就，又承认局限，要在实践中不断深化。

### 伟大斗争——"五个凡是"（必背）

新时代坚持和发展中国特色社会主义，必须进行具有许多新的历史特点的**伟大斗争**。习近平提出"五个凡是"（高频考点）：

> **凡是危害**以下方面的风险挑战，只要来了，就必须进行坚决斗争：
> 1. 危害**中国共产党领导和我国社会主义制度**的各种风险挑战
> 2. 危害我国**主权、安全、发展利益**的各种风险挑战
> 3. 危害我国**核心利益和重大原则**的各种风险挑战
> 4. 危害我国**人民根本利益**的各种风险挑战
> 5. 危害我国**实现第二个百年奋斗目标、实现中华民族伟大复兴**的各种风险挑战

::: tip 斗争的本质
教材特别强调：斗争"**不是为了斗争而斗争，也不是为了一己私利而斗争**"，而是为了实现人民对美好生活的向往、实现中华民族伟大复兴。担当和斗争是一种**精神、一种责任、一种格局**。
:::

## 六、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | "四个全面"的新内涵、基本路线的"一个中心两个基本点"、"五位一体"的内容、基本路线奋斗目标 |
| **多选** | "五位一体"五大建设、"四个全面"、党的基本理论/路线/方略 |
| **简答** | 什么是党在社会主义初级阶段的基本路线？"四位一体"到"五位一体"的重大意义？ |
| **论述** | 如何理解统筹推进"五位一体"总体布局和协调推进"四个全面"战略布局？ |

### 预测题一（单选）

**决胜全面建成小康社会后，"四个全面"战略布局被赋予了新的时代内涵。当前"四个全面"战略布局是指（　）**
A. 全面建成小康社会、全面深化改革、全面依法治国、全面从严治党
B. 全面建设社会主义现代化国家、全面深化改革、全面依法治国、全面从严治党
C. 全面建设社会主义现代化强国、全面改革开放、全面依法治国、全面从严治党
D. 全面建设社会主义现代化国家、全面深化改革、全面依法治国、全面依规治党

> **答案：B**
> **解析**：A 是**旧版**（2020 年前），第一个"全面"是"全面建成小康社会"；决胜全面建成小康社会后，第一个"全面"改为"**全面建设社会主义现代化国家**"，其余三个不变。C 错在"现代化强国"（应为"国家"）和"全面改革开放"（应为"全面深化改革"）；D 错在"依规治党"（应为"从严治党"）。

### 预测题二（简答）

**简述党在社会主义初级阶段的基本路线及其核心内容。**

> **【基本路线】** 领导和团结全国各族人民，以经济建设为中心，坚持四项基本原则，坚持改革开放，自力更生，艰苦创业，为把我国建设成为富强民主文明和谐美丽的社会主义现代化强国而奋斗。
>
> **【核心内容——"一个中心、两个基本点"】**
> - **一个中心**：以经济建设为中心（兴国之要）
> - **两个基本点**：坚持四项基本原则（立国之本）、坚持改革开放（强国之路）
>
> **【地位】** 党的基本路线是国家的**生命线**、人民的**幸福线**，必须牢牢坚持、毫不动摇。

### 易错点提醒

::: warning 本节三大易错点
1. **"四个全面"新旧内涵**：旧的第一个"全面"=全面建成小康社会；新的第一个"全面"=**全面建设社会主义现代化国家**。这是最高频的陷阱。
2. **基本路线奋斗目标的字数**：是"富强民主文明和谐美丽"（**七个字**，对应五位一体），党的十九大加了"美丽"，不要写成旧的四个字。
3. **"五位一体"演进顺序**：两个文明→三位一体→四位一体→五位一体（十八大加入生态文明）。不要记错顺序。
:::
