# 第三节 加快建设科技强国

> 教材第 146–150 页 · 科技自立自强是国家强盛之基、安全之要。

---

## 一、核心论断（必背金句）

> **科技自立自强是国家强盛之基、安全之要。**

> 科技兴则民族兴，**科技强则国家强**。

---

## 二、为什么要实现高水平科技自立自强（三个必然）

| 必然性 | 内容 |
| --- | --- |
| **① 战略基石** | 是国家强盛和民族复兴的战略基石；全面建成社会主义现代化强国关键看科技自立自强 |
| **② 应对风险** | 是应对风险挑战和维护国家利益的必然选择（掌握发展主动权） |
| **③ 内在要求** | 是构建新发展格局、推动高质量发展、满足人民美好生活需要的内在要求 |

### 关键金句

> 全面建成社会主义现代化强国**关键看科技自立自强**。

> 必须把科技的**命脉**牢牢掌握在自己手中。

---

## 三、打赢关键核心技术攻坚战（高频考点）

### 核心论断（必背）

> **关键核心技术是国之重器**。

> 关键核心技术是**要不来、买不来、讨不来**的。

::: danger 必背：三个"不来"
关键核心技术**要不来、买不来、讨不来**——必须自主创新。

由此引出："核心技术受制于人是我们最大的隐患"，必须打赢关键核心技术攻坚战。
:::

### 新型举国体制

| 内容 | 要点 |
| --- | --- |
| **性质** | 打赢关键核心技术攻坚战的体制优势 |
| **做法** | 把政府、市场、社会各方面力量拧成一股绳 |
| **原则** | 推动有效市场和有为政府更好结合 |
| **机制** | "揭榜挂帅"等制度 |

### 重大科技成就（举例）

"神舟"飞天、"蛟龙"入海、"嫦娥"奔月、"墨子"传信、"北斗"组网、"天眼"巡空、"天问"探火。

## 四、增强自主创新能力

### 自主创新的三种类型（必背）

| 类型 | 内容 |
| --- | --- |
| **原始创新** | 从 0 到 1 的突破（基础研究） |
| **集成创新** | 多技术融合 |
| **开放创新** | 国际合作 |

> 坚持**自主创新、重点跨越、支撑发展、引领未来**的方针。

### 基础研究（高频考点）

> **基础研究是科技创新的源头**。

- 地基打得牢，科技事业大厦才能建得高
- 我国"卡脖子"技术问题的根子是**基础理论研究跟不上**
- 坚持**目标导向和自由探索"两条腿走路"**

### 国家战略科技力量（必背四类）

| 力量 | 作用 |
| --- | --- |
| **国家实验室** | 发挥引领作用 |
| **国家科研机构** | 建制化组织作用、原始创新策源地 |
| **高水平研究型大学** | 基础研究主力军和重大科技突破生力军 |
| **科技领军企业** | 市场需求、集成创新、组织平台优势 |

> **企业是科技创新的主体**。

### 预测题一（单选）

**关于关键核心技术，下列表述正确的是（　）**
A. 关键核心技术可以通过购买获得
B. 关键核心技术是要不来、买不来、讨不来的
C. 关键核心技术可以依赖引进
D. 关键核心技术不重要

> **答案：B**
> **解析**：教材原文——关键核心技术是**要不来、买不来、讨不来**的，是国之重器。核心技术受制于人是最大隐患，必须自主创新，打赢关键核心技术攻坚战。

### 预测题二（简答）

**简述国家战略科技力量的组成部分。**

> 国家战略科技力量包括**四个方面**：
> 1. **国家实验室**——发挥引领作用，多出战略性、关键性重大科技成果
> 2. **国家科研机构**——发挥建制化组织作用，加快建设原始创新策源地
> 3. **高水平研究型大学**——基础研究的主力军和重大科技突破的生力军
> 4. **科技领军企业**——发挥市场需求、集成创新、组织平台优势
>
> 四者优势互补，增强体系化创新能力。同时，**企业是科技创新的主体**。

### 易错点提醒

::: warning 本节三大易错点
1. **三个"不来"**：要不来、买不来、讨不来（关键核心技术的不可依赖性）。
2. **"基础研究是源头"**：地基、源头——卡脖子问题根子在基础理论研究。
3. **"企业是科技创新主体"**：注意是"主体"，不是"主导"或"补充"。
:::
