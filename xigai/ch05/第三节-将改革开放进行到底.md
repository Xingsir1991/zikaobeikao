# 第三节 将改革开放进行到底

> 教材第 105–111 页 · 改革开放的时间观和开放观。

---

## 一、核心论断（必背金句）

> **"实践发展永无止境，解放思想永无止境，改革开放也永无止境，改革开放只有进行时、没有完成时。"**

---

## 二、小节一：改革开放永无止境（"三个永无止境"的依据）

### 为什么改革开放永无止境（三个依据）

| 依据 | 内容 |
| --- | --- |
| **① 社会基本矛盾运动规律** | 生产力不断发展→客观要求生产关系和上层建筑相适应→改革开放永无止境（历史唯物主义） |
| **② 世界社会主义经验** | 苏联东欧教训：固守旧体制死路一条；搞否定方向的"改革开放"也是死路一条 |
| **③ 党和人民事业发展** | 每一次重大改革都给事业注入新活力，新征程仍需通过改革解决问题 |

### 历史唯物主义的解释（理论深度）

> 改革开放说到底，就是要不断破除制约生产力发展、制约社会进步的障碍，推动经济社会更好更快发展。

- 生产力是最活跃、最革命的因素
- 生产力发展 → 要求生产关系适应 → 要求上层建筑适应
- 经济社会发展不停顿，改革开放就不止步

---

## 三、小节二：坚定不移把全面深化改革引向深入

### 核心论断

> "改革道路上仍面临着很多复杂的矛盾和问题，我们已经啃下了不少硬骨头但还有许多硬骨头要啃。"

### 新征程的根本动力

> **把全面深化改革作为推进中国式现代化的根本动力**，作为稳大局、应变局、开新局的重要抓手。

### 深化各领域改革的方向（六个"形成有利于"）

围绕全面建设社会主义现代化国家：

| 领域 | 形成有利于 |
| --- | --- |
| 经济 | 贯彻新发展理念、构建新发展格局、推动高质量发展 |
| 政治 | 发展全过程人民民主、建设法治体系 |
| 文化 | 社会主义意识形态、践行核心价值观 |
| 社会 | 完善分配、促进就业、健全社保、健康中国 |
| 生态 | 绿色转型、污染防治、生态系统多样性稳定性持续性 |
| 党建 | 党的全面领导、自我革命制度规范体系 |

### 重要金句（必背）

> **"改革推进到今天，比认识更重要的是决心，比方法更关键的是担当。"**

## 四、小节三：坚定不移扩大高水平对外开放

### 核心论断（必背金句）

> **"中国开放的大门只会越开越大，永远不会关上！"**

> **"开放也是改革。以开放促改革、促发展"**是我国发展不断取得新成就的宝贵经验。

### 对外开放的历史大势

- 经济全球化遭遇逆流，但**开放合作仍然是历史潮流**
- 经济全球化的历史大势**不可逆转**
- 世界决不会退回到相互封闭、彼此分割的状态

### 中国的开放战略

| 方面 | 内容 |
| --- | --- |
| **三个"决不会变"** | 扩大高水平开放的决心不会变；同世界分享发展机遇的决心不会变；推动经济全球化的决心不会变 |
| **开放导向** | 更加开放、包容、普惠、平衡、共赢 |
| **引进来+走出去** | 相互促进、更好结合 |
| **三大共建** | 共建"一带一路"成为深受欢迎的国际公共产品和国际合作平台 |

### 高水平对外开放的方向（"三个更"）

> 坚持实施**更大范围、更宽领域、更深层次**对外开放。

- 依托超大规模市场优势
- 稳步扩大**规则、规制、管理、标准**等制度型开放
- 建设更高水平开放型经济新体制

::: tip 制度型开放（新概念）
新时代开放的升级方向——从商品和要素流动型开放，转向**制度型开放**（规则、规制、管理、标准）。这是高水平开放的核心特征。
:::

## 六、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 永无止境、根本动力、开放的大门、三个"决不会变" |
| **多选** | 改革开放的三个永无止境依据、对外开放的战略 |
| **简答** | 为什么说改革开放永无止境？ |
| **论述** | 如何理解"中国开放的大门只会越开越大"？ |

### 预测题一（单选）

**关于改革开放，下列表述正确的是（　）**
A. 改革开放有完成时
B. 改革开放只有进行时、没有完成时
C. 改革开放已经基本结束
D. 改革开放可以暂时停滞

> **答案：B**
> **解析**：习近平原话——"实践发展永无止境，解放思想永无止境，改革开放也永无止境，**改革开放只有进行时、没有完成时**。"这是历史唯物主义态度。停顿和倒退没有出路。

### 预测题二（简答）

**为什么说改革开放永无止境？**

> "改革开放永无止境"是历史唯物主义态度，依据有三：
> 1. **社会基本矛盾运动规律的深刻反映**——生产力是最活跃最革命的因素，生产力发展要求生产关系和上层建筑相适应。经济社会发展不停顿，改革开放就不止步。
> 2. **总结世界社会主义经验的重要结论**——苏联东欧教训表明，固守旧体制死路一条，搞否定方向的"改革开放"也是死路一条。
> 3. **推进党和人民事业发展的必然要求**——完成中华民族伟大复兴的历史使命，需要不断通过改革开放解放和发展生产力、完善社会主义制度。
>
> 生产力发展永无止境，调整生产关系、完善上层建筑的改革开放也永无止境。

### 易错点提醒

::: warning 本节三大易错点
1. **"只有进行时没有完成时"**：这是固定表述，不要写成"有完成时"。
2. **"开放也是改革"**：开放和改革一体两面，**以开放促改革、促发展**。
3. **"制度型开放"**：新时代开放升级方向——规则、规制、管理、标准（不是商品要素开放）。
:::

1. **改革开放的地位**——关键一招、深刻革命、有方向有立场有原则（第一节）
