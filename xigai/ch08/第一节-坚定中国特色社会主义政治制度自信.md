# 第一节 坚定中国特色社会主义政治制度自信

> 教材第 157–163 页 · 人民民主是社会主义的生命。

---

## 一、核心论断（必背金句）

> **人民民主是社会主义的生命**。没有民主就没有社会主义，就没有社会主义现代化，就没有中华民族伟大复兴。

::: danger 必背：人民民主的地位
"人民民主是社会主义的生命"——这是关于民主地位的最重要论断。与之相关的还有"全过程人民民主是社会主义民主政治的本质属性"（第二节）。
:::

### 民主的"三个属性"（理论定性）

教材对民主的三个判断：
- 民主是**全人类共同价值**
- 民主是**历史的**（植根于本国历史文化传统）
- 民主是**具体的**（与国情相适应）

> 民主没有最好，只有更好。

---

## 二、中国特色社会主义政治制度的构成（必背）

### 政治制度的"四梁八柱"

| 层级 | 内容 |
| --- | --- |
| **根本政治制度** | 人民代表大会制度 |
| **基本政治制度**（三项） | ①中国共产党领导的多党合作和政治协商制度 ②民族区域自治制度 ③基层群众自治制度 |
| **重要政治制度** | 在根本和基本政治制度基础上派生的各领域政治制度 |

::: danger 必背：根本政治制度
**人民代表大会制度**是我国的**根本政治制度**——这是高频考点，不能混淆为"根本领导制度"（那是党的领导制度）或"根本政治制度"以外的东西。
:::

### 政治制度的"三个特色"

中国特色社会主义政治制度具有鲜明的**中国特色、民族特色、时代特色**。

---

## 三、坚定不移走中国特色社会主义政治发展道路

### "三个有机统一"（必背，极高频考点）

::: danger 必背：三个有机统一
走中国特色社会主义政治发展道路，必须坚持：
1. **党的领导**
2. **人民当家作主**
3. **依法治国**

**有机统一**。
:::

三者的关系：

| 内容 | 定位 |
| --- | --- |
| **党的领导** | 人民当家作主和依法治国的**根本保证** |
| **人民当家作主** | 社会主义民主政治的**本质特征** |
| **依法治国** | 党领导人民治理国家的**基本方式** |

> 三者统一于我国社会主义民主政治伟大实践。**最根本的是坚持党的领导**。

### 不能照搬西方政治制度

> 世界上不存在完全相同的政治制度，也不存在适用于一切国家的政治制度模式。

- 不能用单一标尺衡量政治制度
- 不能割断历史，搬来一座政治制度上的"**飞来峰**"
- 必须保持**政治定力**

### 预测题一（单选）

**我国的根本政治制度是（　）**
A. 中国共产党领导制度
B. 人民代表大会制度
C. 民主集中制
D. 中国特色社会主义制度

> **答案：B**
> **解析**：教材原文——**人民代表大会制度**是我国的**根本政治制度**。A 党的领导制度是根本**领导**制度；C 民主集中制是根本**组织原则**；D 中特制度是宏观制度体系。

### 预测题二（简答）

**简述党的领导、人民当家作主、依法治国的关系。**

> 党的领导、人民当家作主、依法治国**有机统一**，三者统一于我国社会主义民主政治伟大实践：
> - **党的领导**是人民当家作主和依法治国的**根本保证**
> - **人民当家作主**是社会主义民主政治的**本质特征**
> - **依法治国**是党领导人民治理国家的**基本方式**
>
> 三者有机统一，最根本的是坚持党的领导。

### 易错点提醒

::: warning 本节三大易错点
1. **根本政治制度=人民代表大会制度**：不是党的领导制度。
2. **三个有机统一的定位**：党的领导（保证）、人民当家作主（本质特征）、依法治国（基本方式）。
3. **基本政治制度三项**：多党合作和政治协商 + 民族区域自治 + 基层群众自治。
:::
