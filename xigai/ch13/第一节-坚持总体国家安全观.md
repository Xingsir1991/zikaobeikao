# 第一节 坚持总体国家安全观

> 教材第 260–264 页 · 总体国家安全观是新时代国家安全工作的基本遵循。

---

## 一、核心论断（必背金句）

> **国家安全是民族复兴的根基，社会稳定是国家强盛的前提**。

> 实现中华民族伟大复兴的中国梦，保证人民安居乐业，**国家安全是头等大事**。

---

## 二、总体国家安全观的"五个以"（极高频考点，必背）

::: danger 必背：五个以
总体国家安全观：
1. 以**人民安全**为**宗旨**
2. 以**政治安全**为**根本**
3. 以**经济安全**为**基础**
4. 以**军事、科技、文化、社会安全**为**保障**
5. 以**促进国际安全**为**依托**
:::

### "五个以"的内涵

| "以...为" | 内涵 |
| --- | --- |
| **人民安全为宗旨** | 国家安全一切为了人民、一切依靠人民 |
| **政治安全为根本** | 把政治安全放在首要位置 |
| **经济安全为基础** | 运用发展成果夯实安全基础 |
| **军事科技文化社会安全为保障** | 提供硬实力和软实力保障 |
| **促进国际安全为依托** | 共同构建普遍安全的人类命运共同体 |

### "三个有机统一"（必背）

::: danger 必背：三个有机统一
> **政治安全、人民安全、国家利益至上**有机统一。
:::

---

## 三、总体国家安全观的"关键是总体"（必背）

### 总体国家安全观的关键（必背）

> 总体国家安全观的关键是"**总体**"。

### "五个统筹"（必背）

::: danger 必背：五个统筹
1. 统筹**外部安全和内部安全**
2. 统筹**国土安全和国民安全**
3. 统筹**传统安全和非传统安全**
4. 统筹**自身安全和共同安全**
5. 统筹**维护国家安全和塑造国家安全**
:::

### "大安全"理念

总体国家安全观强调"大安全"理念，涵盖**20个领域**：政治、军事、国土、经济、金融、文化、社会、科技、网络、粮食、生态、资源、核、海外利益、太空、深海、极地、生物、人工智能、数据安全等。

## 四、习近平关于国家安全"十个坚持"（了解）

1. 坚持党对国家安全工作的**绝对领导**
2. 坚持中国特色**国家安全道路**
3. 坚持以**人民安全**为宗旨
4. 坚持**统筹发展和安全**
5. 坚持把**政治安全**放在首要位置
6. 坚持统筹推进**各领域安全**
7. 坚持把**防范化解国家安全风险**摆在突出位置
8. 坚持推进**国际共同安全**
9. 坚持推进**国家安全体系和能力现代化**
10. 坚持加强**国家安全干部队伍建设**

### 预测题一（单选）

**总体国家安全观以（　）为宗旨**
A. 政治安全
B. 经济安全
C. 人民安全
D. 国际安全

> **答案：C**
> **解析**：总体国家安全观"五个以"——以**人民安全为宗旨**，以政治安全为根本，以经济安全为基础，以军事科技文化社会安全为保障，以促进国际安全为依托。

### 预测题二（简答）

**简述总体国家安全观的"五个以"。**

> 总体国家安全观的"五个以"：
> 1. 以**人民安全**为**宗旨**——国家安全一切为了人民、依靠人民
> 2. 以**政治安全**为**根本**——把政治安全放在首要位置
> 3. 以**经济安全**为**基础**——运用发展成果夯实安全基础
> 4. 以**军事、科技、文化、社会安全**为**保障**——硬实力和软实力保障
> 5. 以**促进国际安全**为**依托**——构建普遍安全的人类命运共同体

### 易错点提醒

::: warning 本节三大易错点
1. **五个以的对应**：宗旨=人民；根本=政治；基础=经济；保障=军事科技文化社会；依托=国际安全。
2. **政治安全是"首要位置"**：与"根本"的区别（政治安全既是根本，又要放在首要位置）。
3. **全民国家安全教育日=4月15日**：不是别的日期。
:::
