# 第三节 全面落实以人民为中心的发展思想

> 教材第 84–90 页 · 以人民为中心的实践落实——群众路线 + 为民造福 + 共同富裕。

坚持以人民为中心的发展思想是**具体的、现实的**，必须落实到经济社会发展各个环节。本节讲三个层面的落实。

| 层面 | 内容 |
| --- | --- |

---

## 二、小节一：坚持和贯彻党的群众路线

### 核心论断（必背）

> **群众路线是我们党的生命线和根本工作路线**，是我们党永葆青春活力和战斗力的重要传家宝。

### 群众路线的完整内容（必背）

> **一切为了群众，一切依靠群众，从群众中来，到群众中去**，把党的正确主张变为群众的自觉行动。

::: danger 群众路线四句话必背
"一切为了群众、一切依靠群众、从群众中来、到群众中去"——这是党的根本工作路线，考试常考。两个"一切"+ 两个"群众"。
:::

### "知"与"行"的关系

贯彻群众路线：
- "**知**"是基础、是前提
- "**行**"是重点、是关键
- 必须**以知促行、以行促知，做到知行合一**

### 调查研究就是走群众路线

> 习近平："开展调查研究就是走群众路线，**没有调查就没有发言权，就没有决策权**。"

要"扑下身子、沉到一线、深入基层"，到困难较多、情况复杂、矛盾尖锐的地方去。

## 三、小节二：把为人民造福的事情真正办好办实

### 核心论断

> 以人民为中心的发展思想，**不是一句空洞口号**，不能只停留在口头上、止步于思想环节。

### 三个落实要求

#### ① 落实到各项事业全部工作

党在"五位一体"各领域都要体现为人民造福：

| 领域 | 为民要求 |
| --- | --- |
| 经济 | 满足个性化、多样化需求 |
| 政治 | 发展全过程人民民主 |
| 法治 | 让人民群众在**每一个司法案件**中感受到公平正义 |
| 文化 | 满足精神文化需求 |
| 民生 | 把人民安危冷暖放在心上 |
| 生态 | 创造天更蓝、山更绿、水更清的环境 |

#### ② 着力解决最关心最直接最现实的利益问题

> "要始终把人民安居乐业、安危冷暖放在心上，用心用情用力解决群众关心的就业、教育、社保、医疗、住房、养老、食品安全、社会治安等实际问题。"

#### ③ 久久为功

- 发扬**钉钉子精神**，一锤一锤接着敲
- 制度建设是持续为群众办实事的有力保障

## 四、小节三：推动全体人民共同富裕（极高频考点）

### 共同富裕的性质（必背）

> **共同富裕是中国特色社会主义的本质要求，是中国式现代化的重要特征。**

### "做大蛋糕"和"分好蛋糕"（必背金句）

> 实现共同富裕，首先要通过全国人民共同奋斗把"**蛋糕**"做大做好，然后通过合理的制度安排正确处理增长和分配关系，把"**蛋糕**"切好分好。

### 共同富裕的"四个不是"（高频考点，必背）

::: danger 必背：共同富裕的四个"不是"
1. **不是少数人的富裕**
2. **不是整齐划一的平均主义**
3. **不是所有人同时富裕**
4. **不是所有地区同时达到一个富裕水准**
:::

### 共同富裕的"四个是"（正面界定）

- 是**全体人民**共同富裕
- 是人民群众**物质生活和精神生活都富裕**
- 要处理好**先富和共富**的关系（允许一部分人先富，推动先富带后富）
- 是一个**长期历史过程**，不可能一蹴而就

### 推进共同富裕的原则和思路（必背）

#### 四条原则

| 原则 | 内容 |
| --- | --- |
| 1 | 鼓励**勤劳创新致富** |
| 2 | 坚持**基本经济制度** |
| 3 | **尽力而为、量力而行** |
| 4 | 坚持**循序渐进** |

#### 总思路

在高质量发展中促进共同富裕，正确处理**效率和公平**的关系。

#### 三次分配协调配套（必背）

| 分配 | 内容 |
| --- | --- |
| **初次分配** | 市场机制（劳动报酬等） |
| **再分配** | 政府（税收、社保、转移支付） |
| **三次分配** | 社会力量（慈善等） |

调节手段：加大税收、社保、转移支付调节力度；扩大中等收入群体；取缔非法收入。

#### 橄榄型分配结构（必背金句）

> 形成**中间大、两头小**的**橄榄型分配结构**。

## 六、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 群众路线（生命线）、共同富裕的性质、橄榄型分配、三次分配 |
| **多选** | 群众路线四句话、共同富裕四个"不是"、推进共同富裕四原则 |
| **简答** | 什么是党的群众路线？如何理解共同富裕？ |
| **论述** | 如何推动全体人民共同富裕取得实质性进展？ |

### 预测题一（单选）

**共同富裕是（　）**
A. 中国特色社会主义的根本任务
B. 中国特色社会主义的本质要求
C. 社会主义初级阶段的根本目标
D. 共产主义的最高理想

> **答案：B**
> **解析**：教材原文——**共同富裕是中国特色社会主义的本质要求**，是中国式现代化的重要特征。A"根本任务"是解放和发展生产力；C 不是规范表述；D 共产主义最高理想是远大目标。注意"本质要求"这个定位。

### 预测题二（论述）

**如何理解并扎实推进全体人民共同富裕？**

> **【破题】** 共同富裕是中特本质要求，是中国式现代化的重要特征。
>
> **【主体】**
> **第一**，要正确认识共同富裕的内涵。共同富裕是**全体人民**共同富裕，是物质和精神生活都富裕；**不是少数人富裕、不是平均主义、不是同时富裕、不是同时同等富裕**。实现共同富裕是长期历史过程，要处理好先富和共富关系。
>
> **第二**，要把握推进原则。坚持鼓励勤劳创新致富、坚持基本经济制度、尽力而为量力而行、坚持循序渐进。
>
> **第三**，要构建科学思路。在高质量发展中促进共同富裕，正确处理效率和公平关系，构建**初次分配、再分配、三次分配**协调配套的制度安排，形成**中间大、两头小的橄榄型分配结构**。
>
> **第四**，要"做大蛋糕"和"分好蛋糕"相结合。通过共同奋斗把蛋糕做大做好，通过合理制度安排切好分好。
>
> **【总结】** 推动共同富裕与促进人的全面发展高度统一，是中国式现代化的鲜明特征。

### 易错点提醒

::: warning 本节三大易错点
1. **共同富裕四个"不是"**：少数人富裕 / 平均主义 / 同时富裕 / 同时同等富裕——这是高频陷阱。
2. **橄榄型分配**：**中间大、两头小**（中等收入群体大），不要记反。
3. **群众路线是"生命线和根本工作路线"**——两个定位缺一不可。
:::
