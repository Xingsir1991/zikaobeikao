# 第一节 江山就是人民，人民就是江山

> 教材第 74–78 页 · 党与人民的关系——"打江山、守江山，守的是人民的心"。

---

## 一、核心金句（必背）

> **"江山就是人民，人民就是江山。中国共产党领导人民打江山、守江山，守的是人民的心。"**

这句话是习近平关于党群关系的**经典论断**，揭示了中国共产党的执政逻辑：**政权的人民性**。

---

## 二、人民是历史的创造者，是真正的英雄（理论根基）

### 历史唯物主义群众史观（理论基础）

坚持以人民中心，**体现了历史唯物主义基本原理**。历史唯物主义群众史观的三个核心观点：

| 维度 | 内容 |
| --- | --- |
| **物质财富** | 人民群众是社会物质财富的创造者（创造生活资料和生产资料） |
| **精神财富** | 人民群众是社会精神财富的创造者（精神财富的最终源泉植根于人民实践） |
| **社会变革** | 人民群众是社会变革的决定力量（生产关系变革、社会制度兴亡更替） |

::: tip 必背金句
> "人民是历史的创造者，是真正的英雄。"——一切成就都归功于人民，一切荣耀都归属于人民。
:::

### 党百年奋斗的历史经验

教材梳理了党依靠人民创造历史的脉络：

| 时期 | 人民的作用 |
| --- | --- |
| 红军时期 | 人民群众是党和人民军队的**铜墙铁壁** |
| 抗日战争 | 发动群众，使日本侵略者陷入人民战争的**汪洋大海** |
| 淮海战役 | 胜利是靠老百姓用**小车推出来的** |
| 渡江战役 | 胜利是靠老百姓用**小船划出来的** |
| 社会主义革命建设 | 成就是人民群众**干出来的** |
| 改革开放 | 历史伟剧是亿万人民群众**主演的** |
| 新时代 | 伟大变革离不开人民群众的**不懈奋斗** |

**根本原因**：党始终坚守了"为中国人民谋幸福、为中华民族谋复兴"的**初心和使命**。

## 三、打江山、守江山，守的是人民的心（民心政治）

### 核心论断

> **民心是最大的政治**，决定事业兴衰成败。

### 人民地位与国家政权性质的关系

- 人民在一个国家的地位由**国家政权性质**决定
- 一切存在阶级压迫和剥削的社会中，人民处于无权和被压迫地位
- **社会主义国家**第一次真正实现了**人民当家作主**（恩格斯："国家机关必须由社会主人变为社会公仆"）
- 我国是人民民主专政的社会主义国家，**国家一切权力属于人民**

### "江山就是人民"由党的性质宗旨决定

::: info 关键论断
中国共产党**除了国家、民族、人民的利益，没有自己特殊的利益**，从来不代表任何利益集团、任何权势团体、任何特权阶层的利益。党代表最广大人民的根本利益。
:::

马克思、恩格斯《共产党宣言》：

> "过去的一切运动都是少数人的，或者为少数人谋利益的运动。**无产阶级的运动是绝大多数人的，为绝大多数人谋利益的独立的运动。**"

**核心要点**："不谋私利才能谋根本、谋大利"——这是党全心全意为人民服务的理论基础。

### 党执政的根基

| 关键论断 | 内容 |
| --- | --- |
| **最大底气** | 人民是党执政的最大底气 |
| **最深厚根基** | 人民是党执政最深厚的根基 |
| **三个在人民** | 党的**根基**在人民、**血脉**在人民、**力量**在人民 |

## 四、人民立场是中国共产党的根本政治立场

### 核心论断（必背）

> **人民立场是中国共产党的根本政治立场**，是我们党区别于其他政党的显著标志。

### 坚持人民立场的两个要求

#### ① 始终牢记党的初心和使命

**初心和使命**：为中国人民谋幸福，为中华民族谋复兴。

- 是激励一代代中国共产党人前仆后继、英勇奋斗的**根本动力**
- 忘记这个初心使命，党就会改变性质、改变颜色、失去人民、失去未来

#### ② 始终保持党同人民群众的血肉联系

- 党的**根基、血脉、力量**都在人民
- 始终同人民群众**想在一起、干在一起**，风雨同舟、同甘共苦

### 预测题一（单选）

**中国共产党领导人民打江山、守江山，守的是（　）**
A. 国家政权
B. 人民的心
C. 社会主义制度
D. 民族复兴伟业

> **答案：B**
> **解析**：习近平原文——"中国共产党领导人民打江山、守江山，**守的是人民的心**"。这一论断揭示了党的执政根基在人民，民心是最大的政治。其他选项虽然重要，但都不是"守"的对象。

### 预测题二（简答）

**如何理解"江山就是人民，人民就是江山"？**

> **【破题】** 这句话深刻揭示了党的执政根基和政权的人民性。
>
> **【主体】**
> **第一**，从理论根基看，体现了历史唯物主义群众史观。人民是历史的创造者，是真正的英雄，是物质财富、精神财富的创造者和社会变革的决定力量。
>
> **第二**，从政治逻辑看，民心是最大的政治。我国是人民民主专政的社会主义国家，国家一切权力属于人民，党执政的最大底气在人民。
>
> **第三**，从党的性质看，党除了国家、民族、人民的利益，没有自己特殊的利益。党为人民夺取政权打江山，为人民巩固政权守江山。
>
> **【总结】** 因此，"江山就是人民，人民就是江山"揭示了党的根本立场——人民立场，是党区别于其他政党的显著标志。

### 易错点提醒

::: warning 本节三大易错点
1. **"守的是人民的心"**：不是守政权、守制度，而是守**人民的心**（民心政治）。
2. **"人民立场"的定位**：是**根本政治立场**（不是"根本宗旨"或"根本保证"）。
3. **党的利益的表述**：党**没有自己特殊的利益**（不代表利益集团、权势团体、特权阶层）。
:::
