# 第四章 坚持以人民为中心

> 教材第 74–90 页 · 新时代坚持和发展中国特色社会主义的**根本立场**。

## 本章框架

人民性是马克思主义的本质属性和鲜明品格。坚持以人民为中心，是新时代坚持和发展中国特色社会主义的**根本立场**，是贯穿党治国理政全部活动的一条**红线**。

## 学习要点（教材要求）

1. "江山就是人民，人民就是江山"的深刻内涵
2. 坚持人民至上的实践要求
3. 扎实推动全体人民共同富裕的原则和思路

## 本章的特殊地位

::: tip 第四章是"根本立场章"
第三章讲党的领导是**根本保证**，第四章讲以人民为中心是**根本立场**——回答"为了谁、依靠谁、我是谁"这个根本问题。"人民"二字贯穿整个习概，本章是其理论原点。
:::
