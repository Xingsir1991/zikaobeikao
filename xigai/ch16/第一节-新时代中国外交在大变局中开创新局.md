# 第一节 新时代中国外交在大变局中开创新局

> 教材第 316–321 页 · 当今世界正经历百年未有之大变局。

---

## 一、核心论断（必背金句）

> **当今世界正经历百年未有之大变局**。

> 世界之变、时代之变、历史之变正以前所未有的方式展开。

### 大变局的"三个方面"

| 方面 | 内容 |
| --- | --- |
| **国际力量对比深刻变化** | 新兴市场国家和发展中国家群体性崛起；"东升西降""中治西乱" |
| **新一轮科技革命和产业变革深入发展** | 人工智能、大数据、量子技术等前沿科技突破 |
| **国际体系和国际秩序深度调整** | 国际关系民主化不可阻挡 |

### "四个赤字"（必背）

::: danger 必背：四个赤字
世界面临：
1. **和平赤字**
2. **发展赤字**
3. **安全赤字**
4. **治理赤字**
:::

### "时代之问"（四个选择）

> 世界向何处去？**和平还是战争？发展还是衰退？开放还是封闭？合作还是对抗？**

## 二、中国必须有自己特色的大国外交

### 核心论断

> **中国必须有自己特色的大国外交**。

### "三个观"（必背）

::: danger 必背：把握国际形势的三个观
> 把握国际形势要树立**正确的历史观、大局观、角色观**。
:::

| 观 | 要求 |
| --- | --- |
| **历史观** | 端起历史望远镜回顾过去、展望未来 |
| **大局观** | 把握本质和全局，抓住主要矛盾 |
| **角色观** | 把自己摆进去，弄清中国的地位和作用 |

### 中国特色大国外交的"一条主线"

> 牢牢把握**服务民族复兴、促进人类进步**这条主线。

### "三个建设者"（必背，回顾）

> 始终做：
> - **世界和平的建设者**
> - **全球发展的贡献者**
> - **国际秩序的维护者**

**简述"百年未有之大变局"的主要表现。**

> 当今世界正经历百年未有之大变局，主要表现在三个方面：
> 1. **国际力量对比深刻变化**——新兴市场国家和发展中国家群体性崛起，"东升西降""中治西乱"
> 2. **新一轮科技革命和产业变革深入发展**——人工智能、大数据、量子技术等前沿科技突破
> 3. **国际体系和国际秩序深度调整**——国际关系民主化不可阻挡
>
> 同时世界面临和平赤字、发展赤字、安全赤字、治理赤字"四个赤字"。
