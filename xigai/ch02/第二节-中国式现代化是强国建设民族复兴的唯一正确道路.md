# 第二节 中国式现代化是强国建设、民族复兴的唯一正确道路

> 教材第 41–48 页 · **中国式现代化**核心概念章——五大中国特色 + 本质要求。

本节是整个习概**最核心的概念章**。"中国式现代化"是新时代党的创新理论的标志性概念，几乎每年必考。

---

## 一、核心概念：什么是中国式现代化

### 定性（必背金句）

> **中国式现代化，是中国共产党领导的社会主义现代化，既有各国现代化的共同特征，更有基于自己国情的中国特色。**

这句话包含三层：
- **领导力量**：中国共产党领导
- **性质**：社会主义现代化（不是别的什么现代化）
- **特征**：共同特征 + 中国特色

---

## 二、中国式现代化的"五个中国特色"（极高频考点，必背）

党的二十大集中概括了中国式现代化**五个方面**的中国特色：

### ① 人口规模巨大的现代化

- 我国 14 亿人口整体迈进现代化，**超过现有发达国家人口总和**
- 极大改变现代化的世界版图
- 艰巨性、复杂性前所未有，必须保持**历史耐心**，稳中求进、循序渐进

### ② 全体人民共同富裕的现代化

- 这是中国式现代化**区别于西方现代化的显著标志**
- 西方弊端：以资本为中心、两极分化严重
- 中国做法：以人民为中心，**坚决防止两极分化**

### ③ 物质文明和精神文明相协调的现代化

- **物质贫困不是社会主义，精神贫乏也不是社会主义**
- 习近平金句：

  > "当高楼大厦在我国大地上遍地林立时，中华民族精神的大厦也应该巍然耸立。"

- 要求：**两手抓、两手硬**，物的全面丰富 + 人的全面发展

### ④ 人与自然和谐共生的现代化

- 西方现代化经历过肆意掠夺、生态破坏阶段
- 中国做法：**绿水青山就是金山银山**，走生产发展、生活富裕、生态良好的文明发展道路

### ⑤ 走和平发展道路的现代化

- 西方老路：战争、殖民、掠夺
- 中华民族文化基因：崇尚和平，"强不执弱""富不侮贫""协和万邦"
- 中国做法：**独立自主、自力更生**，高举和平、发展、合作、共赢旗帜

::: tip 五个特色的记忆法
**口诀："大-富-双-绿-和"**：人口规模**大**、共同**富**裕、**双**文明（物质+精神）、**绿**水青山、**和**平发展。

考试最常考的判断点：
- 区别于西方的**显著标志** = 共同富裕
- 区别于西方掠夺的 = 和平发展
- 体现社会主义根本要求的 = 物质文明+精神文明协调
:::

## 三、中国式现代化的本质要求（必背九句）

### 本质要求全文（极高频考点，建议整段背）

> 中国式现代化的本质要求是：
> 1. **坚持中国共产党领导**
> 2. **坚持中国特色社会主义**
> 3. **实现高质量发展**
> 4. **发展全过程人民民主**
> 5. **丰富人民精神世界**
> 6. **实现全体人民共同富裕**
> 7. **促进人与自然和谐共生**
> 8. **推动构建人类命运共同体**
> 9. **创造人类文明新形态**

::: danger 本质要求结构解析（便于记忆）
九条可分三组：
- **领导与道路**（1-2）：党的领导 + 中国特色社会主义
- **国内建设**（3-7）：高质量发展 + 全过程人民民主 + 精神世界 + 共同富裕 + 人与自然
- **国际贡献**（8-9）：人类命运共同体 + 人类文明新形态

记忆口诀：**"党、路、经、政、文、富、生、共、态"**（领导、道路、经济、政治、文化、富裕、生态、共同体、新形态）。
:::

### "党的领导"在本质要求中的根本地位

习近平强调党的领导"**直接关系中国式现代化的根本方向、前途命运、最终成败**"，体现在四个方面：

| 作用 | 内容 |
| --- | --- |
| **决定根本性质** | 党的性质宗旨决定了中国式现代化是社会主义现代化 |
| **锚定奋斗目标** | 保证奋斗目标一以贯之、接力推进 |
| **激发强劲动力** | 勇于改革创新，破除体制机制弊端 |
| **凝聚磅礴力量** | 依靠人民，汇集全体人民智慧和力量 |

## 四、中国式现代化创造了人类文明新形态

### 核心论断（必背）

> 中国式现代化，**打破了"现代化 = 西方化"的迷思**，展现了不同于西方现代化的新图景，是一种**全新的人类文明形态**。

### 三个层次的重大意义

| 层次 | 意义 |
| --- | --- |
| **提供新模式** | 提供了一种全新的现代化模式，现代化不是少数国家的"专利品"、不是"单选题" |
| **重大超越** | 是对西方式现代化理论和实践的重大超越（摒弃以资本为中心、两极分化、物质主义膨胀、对外扩张掠夺的老路） |
| **全新选择** | 为广大发展中国家独立自主迈向现代化提供了**全新选择** |

::: tip "东升西降""中治西乱"
教材用"东升西降""中治西乱"概括新时代中国与西方发展的鲜明对比，论证中国式现代化的世界意义。
:::

### 预测题一（单选）

**中国式现代化区别于西方资本主义现代化的显著标志是（　）**
A. 人口规模巨大
B. 全体人民共同富裕
C. 物质文明和精神文明相协调
D. 走和平发展道路

> **答案：B**
> **解析**："全体人民共同富裕"是中国式现代化**区别于西方现代化的显著标志**（教材原话）。西方最大弊端是以资本为中心、两极分化。A 是中国式现代化的国情特点（非"区别标志"）；C、D 都是中国特色，但教材明确把"显著标志"赋予了 B（共同富裕）。

### 预测题二（简答）

**简述中国式现代化的中国特色。**

> 中国式现代化是**中国共产党领导的社会主义现代化**，既有各国现代化的共同特征，更有基于自己国情的**五个方面中国特色**：
> 1. **人口规模巨大**的现代化（14 亿人口整体迈进现代化）
> 2. **全体人民共同富裕**的现代化（区别于西方的显著标志）
> 3. **物质文明和精神文明相协调**的现代化（两手抓、两手硬）
> 4. **人与自然和谐共生**的现代化（绿水青山就是金山银山）
> 5. **走和平发展道路**的现代化（不走战争、殖民、掠夺老路）

### 易错点提醒

::: warning 本节三大易错点
1. **五个特色不要漏/串**：人口规模巨大、共同富裕、物质精神协调、人与自然和谐、和平发展。这是二十大原文，表述必须准确。
2. **本质要求九条**：党的领导是第一条，不能漏；"创造人类文明新形态"是最后一条。
3. **"显著标志"vs"中国特色"**：共同富裕是**区别于西方的显著标志**；五个方面都是**中国特色**。注意题干问的是哪个。
:::
