# 第六章 推动高质量发展

## 学习要点（教材要求）

1. 新发展理念的科学内涵和实践要求
2. 高质量发展的深刻内涵和重大意义
3. 坚持和完善社会主义基本经济制度
4. 构建以国内大循环为主体、国内国际双循环相互促进的新发展格局

## 本章的特殊地位

::: tip 第六章是"经济思想章"
本章是**习近平经济思想**的集中体现。高质量发展是全面建设社会主义现代化国家的**首要任务**（注意：是"首要任务"，不是"根本任务"）。"三新"——新发展阶段、新发展理念、新发展格局是本章的主线。
:::
