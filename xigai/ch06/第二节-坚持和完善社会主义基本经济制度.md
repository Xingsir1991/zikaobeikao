# 第二节 坚持和完善社会主义基本经济制度

> 教材第 119–124 页 · 高质量发展的制度基础。

社会主义基本经济制度是中特制度的重要支柱，是新时代推动实现高质量发展的**制度基础**。

---

## 一、核心概念：社会主义基本经济制度"三项内容"（必背）

### 基本经济制度的三项内容（极高频考点）

党的十八大以来，对社会主义基本经济制度作出新概括，包括**三个方面**：

| 序号 | 内容 |
| --- | --- |
| ① | **公有制为主体、多种所有制经济共同发展**（所有制） |
| ② | **按劳分配为主体、多种分配方式并存**（分配制度） |
| ③ | **社会主义市场经济体制**（资源配置体制） |

::: danger 必背：基本经济制度三项
这是高频考点，三项缺一不可。注意第三项"社会主义市场经济体制"是党的十八大后**新纳入**基本经济制度的（原来说的只是前两项）。
:::

### 三项的关系

| 基础 | 所有制结构（决定分配方式和资源配置方式） |
| --- | --- |
| 配套 | 合理的分配方式和资源配置方式有利于完善所有制结构 |

三者是**相互联系、相互支持、相互促进的内在统一整体**。

---

## 二、"两个毫不动摇"（必背）

### 核心论断（必背金句）

> **毫不动摇巩固和发展公有制经济，毫不动摇鼓励、支持、引导非公有制经济发展。**

### 公有制经济

| 内容 | 要点 |
| --- | --- |
| **范围** | 国有经济 + 集体经济 + 混合所有制中的国有/集体成分 |
| **主体地位** | 公有制主体地位不能动摇、国有经济主导作用不能动摇 |
| **国有企业** | 是中国特色社会主义的重要**物质基础和政治基础** |
| **国企改革** | "三个有利于"：有利于国有资产保值增值、有利于提高国有经济竞争力、有利于放大国有资本功能 |

### 非公有制经济

| 内容 | 要点 |
| --- | --- |
| **范围** | 个体、私营、港澳台投资、外商投资经济 + 混合所有制中的非公有成分 |
| **地位** | 社会主义市场经济的**重要组成部分** |
| **作用** | 稳定经济、税收来源、就业创业、技术创新、金融发展的重要依托 |
| **民营经济** | 非公有制经济的主要组织形式，推进中国式现代化的**生力军** |

### 两个"没有变"（高频考点）

> 习近平："非公有制经济在我国经济社会发展中的地位和作用**没有变**，我们毫不动摇鼓励、支持、引导非公有制经济发展的方针政策**没有变**。"

### 公有制与非公有制的关系（辩证统一）

::: warning 关键论断
- 公有制经济与非公有制经济**相辅相成、相得益彰**
- 任何想把公有制否定掉或把非公有制否定掉的观点，都是**错误的**
- 坚持**两点论和重点论的统一**
:::

## 三、坚持按劳分配为主体、多种分配方式并存

### 基本分配制度的依据

- 由我国**社会主义性质和基本国情**决定
- 所有制决定分配方式：公有制为主体 → 按劳分配为主体

### 分配制度的两个"坚持"

| 坚持项 | 内容 |
| --- | --- |
| **坚持多劳多得** | 着重保护劳动所得，鼓励勤劳致富 |
| **允许生产要素参与分配** | 资本、土地、知识、技术、管理、**数据**等生产要素参与分配 |

::: tip "数据"作为生产要素
注意教材把**数据**列入生产要素——这是新时代的新表述，体现数字经济特征。数据权属、共享、交易规则正在健全。
:::

### 两个"同步"（必背）

> 努力推动**居民收入增长和经济增长同步**、**劳动报酬提高和劳动生产率提高同步**。

## 四、构建高水平社会主义市场经济体制

### 核心论断：处理政府和市场的关系

> **使市场在资源配置中起决定性作用，更好发挥政府作用。**

| 手段 | 作用 |
| --- | --- |
| **市场**（看不见的手） | 在资源配置中起**决定性作用** |
| **政府**（看得见的手） | **更好发挥政府作用**（管市场管不了的事） |

::: danger 必背：市场与政府的关系
- **市场起决定性作用**（不是"基础性作用"——党的十八届三中全会从"基础性"改为"决定性"）
- **更好发挥政府作用**（不是"不要政府"）
- 两者**有机统一**，不是相互否定
- 推动有效市场和有为政府**更好结合**
:::

### 高标准市场体系

- 最稀缺的资源是**市场**
- 建设**统一开放、竞争有序、制度完备、治理完善**的高标准市场体系

### 资本的健康发展（高频新概念）

> 为资本设立**"红绿灯"**。

- 资本是带动各类生产要素集聚配置的重要纽带
- 资本具有**逐利本性**，须依法规范和引导
- 健全事前引导、事中防范、事后监管的全链条治理体系

## 六、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 基本经济制度三项、两个毫不动摇、市场决定性作用、红绿灯 |
| **多选** | 基本经济制度三项、生产要素范围（含数据）、两个同步 |
| **简答** | 什么是社会主义基本经济制度？什么是"两个毫不动摇"？ |
| **论述** | 如何处理政府和市场的关系？ |

### 预测题一（单选）

**我国社会主义基本经济制度包括（　）**
A. 公有制为主体、多种所有制经济共同发展
B. 按劳分配为主体、多种分配方式并存
C. 社会主义市场经济体制
D. 以上三项都是

> **答案：D**
> **解析**：社会主义基本经济制度包括**三项**：①公有制为主体、多种所有制经济共同发展 ②按劳分配为主体、多种分配方式并存 ③社会主义市场经济体制。三项缺一不可，D 正确。这是党的十八大后的新概括，第三项是新纳入的。

### 预测题二（简答）

**什么是"两个毫不动摇"？**

> "两个毫不动摇"是坚持和完善基本经济制度的重要方针：
> 1. **毫不动摇巩固和发展公有制经济**——坚持公有制主体地位、国有经济主导作用，做强做优做大国有资本和国有企业
> 2. **毫不动摇鼓励、支持、引导非公有制经济发展**——民营经济是推进中国式现代化的生力军，只能壮大、不能弱化
>
> 公有制经济和非公有制经济相辅相成、相得益彰，统一于社会主义基本经济制度之中。

### 易错点提醒

::: warning 本节三大易错点
1. **基本经济制度三项必须齐全**：所有制 + 分配 + 市场经济体制。
2. **市场"决定性作用"**（不是"基础性"）：党的十八届三中全会的重大突破。
3. **生产要素的范围**：劳动、资本、土地、知识、技术、管理、**数据**——注意"数据"是新要素。
:::
