# 第五章 中国革命的新道路

> 教材第 120–137 页 · **从城市到乡村的革命道路**——以乡村为中心的中国革命新道路的开辟。

## 本章框架

大革命失败后，国民党建立代表大地主大资产阶级利益的政权，对共产党人进行血腥屠杀。中国共产党举行南昌起义，开辟井冈山革命根据地，找到了"农村包围城市、武装夺取政权"的革命新道路。土地革命在曲折中发展，长征胜利宣告了革命力量的复苏。

## 本章的特殊地位

::: tip 第五章是"中国特色革命道路的开辟"
本章回答了"大革命失败后，中国革命走什么道路"：
- **南昌起义**——打响武装反抗国民党反动派第一枪，人民军队诞生
- **井冈山革命根据地**——第一个农村革命根据地
- **农村包围城市、武装夺取政权**——毛泽东开辟的中国革命新道路，是马克思主义中国化的重大成果
- **遵义会议**——党的历史上生死攸关的转折点
- **长征胜利**——挽救了党和红军，保存了革命火种

本章揭示了一条根本规律：**中国革命必须走自己的路**。
:::
