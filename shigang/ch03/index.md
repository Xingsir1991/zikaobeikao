# 第三章 辛亥革命与君主专制制度的终结

> 教材第 68–90 页 · **资产阶级革命派的救国探索**——20 世纪中国第一次历史性巨变。

## 本章框架

辛亥革命是中国完全意义上的近代民族民主革命，推翻了统治中国几千年的君主专制制度，建立了中华民国，使民主共和观念深入人心，但没有改变中国半殖民地半封建社会性质，旧民主主义革命最终失败。

## 本章的特殊地位

::: tip 第三章是"旧民主主义革命的高潮与终结"
本章回答了"资产阶级革命派的探索能达到什么高度，为什么最终失败"：
- **辛亥革命=20 世纪中国第一次历史性巨变**——推翻帝制、建立共和
- **但辛亥革命失败了**——没有改变中国半殖民地半封建社会性质
- **失败根本原因**——民族资产阶级的软弱性和妥协性

辛亥革命的失败说明：**资产阶级不能领导中国民主革命走向胜利，呼唤新的领导阶级和新的革命道路**——为第四章（中国共产党成立）作了铺垫。
:::
