# 🟢 史纲（中国近现代史纲要）

> 1840 年至今的重大事件、人物与历史脉络。

## 使用说明

- 左侧目录展示本科目所有章节
- 每篇笔记底部可点击 **「标记为已掌握」**，在 [进度看板](/progress) 汇总
- 顶部搜索框支持中文全文检索

## 添加新资料

把整理好的 md 文件直接放进 `shigang/` 目录，例如：

```
shigang/
├── index.md                    # 本页
├── 旧民主主义革命.md
├── 新民主主义革命.md
└── 社会主义建设.md
```

并在 `.vitepress/config.mts` 的 `sidebarGroup` 中补充对应条目。

## 重点提示（示例）

::: tip 备考要点
- 以时间轴为主线，重大事件、时间、人物三要素对齐记忆
- 旧民主主义革命 → 新民主主义革命 → 社会主义建设 → 改革开放
- 注意历史意义、经验总结类论述题的标准表述
:::
