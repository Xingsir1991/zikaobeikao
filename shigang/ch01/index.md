# 第一章 进入近代后中华民族的磨难与抗争

## 本章框架

1840 年鸦片战争是中国近代史的起点。本章阐述近代中国社会的性质（半殖民地半封建）、主要矛盾、两大历史任务，以及列强侵略的四个方面、中国人民的反抗斗争、反侵略战争失败的原因和民族意识的觉醒。

## 本章的特殊地位

::: tip 第一章是"近代史总论"
第一章是整个中国近代史的**总论章**，回答了三个根本问题：
- **近代中国是什么性质的社会？**（半殖民地半封建）
- **近代中国的主要矛盾和两大历史任务是什么？**（帝国主义与中华民族、封建主义与人民大众；民族独立人民解放、国家富强人民幸福）
- **为什么中国会落后挨打？**（社会制度腐败、经济技术落后）

这三个问题贯穿整个近代史——后续章节（太平天国、洋务运动、辛亥革命、新民主主义革命等）都是对"两大历史任务"的接续探索。
:::
