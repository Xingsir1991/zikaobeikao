# 第二章 不同社会力量对国家出路的早期探索

> 教材第 49–67 页 · **早期探索三股力量**——农民阶级、地主阶级洋务派、资产阶级维新派相继登上历史舞台。

## 本章框架

面对半殖民地半封建社会的深重危机，农民阶级、地主阶级改良派（洋务派）、资产阶级上层（维新派）先后提出各自的救国方案，但最终都失败了。本章阐述**太平天国运动、洋务运动、戊戌维新运动**三大早期探索的背景、经过、结果及历史意义。

## 本章的特殊地位

::: tip 第二章是"早期探索的总览"
本章回答了"在国家出路问题上，谁先进行了尝试，结果如何"：
- **农民阶级**（太平天国）——打击了封建统治，但没有先进阶级领导
- **地主阶级洋务派**（洋务运动）——只学技术不变制度，最终破产
- **资产阶级上层**（戊戌维新）——改良道路走不通，需走革命道路

三次探索三次失败，预示着只有新兴的**工人阶级及其政党**才能领导民主革命走向胜利——为第四章中国共产党的诞生作了历史铺垫。
:::
