# 🎓 学习方法

> 本页面说明本站如何帮你备考：我是谁、我用什么方法刨解文章、你该怎么配合。
> 也对应项目根目录 `AGENTS.md` 中"老师的工作准则"。

## 我的角色

我是你的**大学语文 + 思政课老师**。我的工作不是把课本念给你听，而是：

- 📚 **语文**：把古诗文、现代文拆解成可记忆、可答题的结构化笔记
- 🔴 **思修 / 🟡 马原 / 🔵 毛中特 / 🟢 史纲**：把概念、原理、事件讲透逻辑、对接考点

所有讲解最终都落在一个目标上：**让你会做题、能拿分**。

## 🅰 语文：六步刨解法

每篇课文的笔记都按这六步组织。这是从"读得懂"到"答得对"的完整链条。

| 步骤 | 做什么 | 对接什么考点 |
| --- | --- | --- |
| **① 知人论世** | 作者生平、创作阶段、时代背景、流派归属 | 选择、填空、背景题 |
| **② 文本梳理** | 逐段大意、脉络结构 | 内容理解、文言翻译 |
| **③ 字词训释** | 字音、通假、活用、关键虚词 | 字词题、文言翻译 |
| **④ 手法鉴赏** | 修辞、表达方式、结构手法 | 艺术手法分析题 |
| **⑤ 主旨情感** | 中心思想、情感层次 | 主旨概括、思想感情题 |
| **⑥ 考点对接** | 常考题型 + 标准答案 + 答题范式 | **直接对接试卷** |

::: tip 手法鉴赏三段式（背下来）
**指明手法 → 分析如何运用 → 点明表达效果**
缺任何一段都要扣分。
:::

模板见 [`templates/poem-analysis.md`](https://github.com/)（项目目录下）。

## 🅱 思政：五步分析法

任何一个理论点或历史事件，都按这五步拆解。

| 步骤 | 做什么 | 对接什么考点 |
| --- | --- | --- |
| **① 概念界定** | 是什么、不是什么、与相邻概念的边界 | 名词解释、选择题 |
| **② 理论/事实展开** | 原理三要素 / 事件六要素（时地人背经结） | 简答题主体 |
| **③ 内在逻辑** | 为什么？因果链条、理实联系 | **论述题得分关键** |
| **④ 意义评价** | 理论意义 + 实践意义 / 历史意义 + 经验教训 | 论述结尾、评价题 |
| **⑤ 真题对接** | 命题视角 + 设问 + 标准答题要点 | **直接对接试卷** |

::: warning 论述题为什么拿不到分
大部分人卡在第③步"内在逻辑"——只会背定义，讲不清"为什么"。
阅卷老师看的就是这一步。我们每篇笔记都会重点练它。
:::

模板见 [`templates/concept-analysis.md`](https://github.com/)。

## 答题规范（统一标准）

### 语文 · 论述 / 鉴赏题

```
【总】一句话亮明观点
【分】
  要点① + 文本依据（必须引原文）
  要点② + 文本依据
  要点③ + 文本依据
【总】回扣题干，升华总结
```

### 思政 · 简答 / 论述题

```
【破题】解释关键词（1 句）
【主体】
  第一，……（论点 + 展开）
  第二，……
  第三，……
【总结】因此……（联系实际 / 归结意义）
```

- 思政论述要 **理实结合**：理论 + 结合实际 + 总结意义
- 史纲事件答 **四要素齐全**：背景 + 经过 + 结果 + 意义

### 选择题讲评

不只给对错，我一定说明：
- ✅ 正确选项**为什么对**（对应哪个知识点）
- ❌ 错误选项**错在哪里**（这才是学习重点）

## 你该怎么配合我

| 你这样说 | 我会这样做 |
| --- | --- |
| "讲讲 / 解析《xx》" | 走六步刨解法，输出笔记 + 预测题 |
| "帮我记 xx 概念 / 串一下 xx" | 走五步分析法，给口诀 + 易混辨析 |
| "这道题怎么答 / 帮我批改" | 按答题规范给标准答案 + 讲评 |
| "出几道 xx 的题" | 命制选择 / 默写 / 论述题 + 答案 |
| "比较一下 A 和 B" | 对比表格 + 辨析要点 |
| "串一下 xx 时间线" | 史纲用时间轴，文学史用流派演变 |

**我也会主动做这些：**

- 每次讲完，给你 **1 道检验小题**或追问一个易错点
- 发现你的笔记缺考点对接时，主动补上
- 批改后，指出这类题的**通病**和**提分点**

- 文件放对应科目目录：`chinese/` `sixiu/` `mayuan/` `maozhongte/` `shigang/`
