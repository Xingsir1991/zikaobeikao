# 🔴 15042 思想道德与法治

> 自考课程代码 **15042** · 思想道德与法治（2025 版新教材）

## 课程概览

本课程是高等教育自学考试**思想政治理论课**必修课，原课程代码 03706（旧名"思想道德修养与法律基础"），现为 **15042 思想道德与法治**。

内容分**三大板块**：思想道德修养（第 1-5 章）+ 法治素养（第 6 章）。

| 板块 | 章节 | 核心 |
| --- | --- | --- |
| 时代背景 | 绪论 / 第一章 | 新时代、新使命、新要求 |
| **思想篇** | 第二章 | 理想信念（马克思主义信仰、中国特色社会主义） |
| **精神篇** | 第三章 | 中国精神（民族精神、时代精神、爱国主义） |
| **价值篇** | 第四章 | 社会主义核心价值观 |
| **道德篇** | 第五章 | 道德观（社会公德、职业道德、家庭美德、个人品德） |
| **法治篇** | 第六章 | 法治观（社会主义法治、宪法、法律体系） |

## 使用说明

- 左侧目录按章节组织，点击进入
- 每篇笔记底部可点击 **「标记为已掌握」**，在 [进度看板](/progress) 汇总
- 顶部搜索框支持中文全文检索

## 备考策略

::: tip 备考要点
- **选择题**：抓"核心表述"（如社会主义核心价值观 24 字、宪法基本原则）
- **简答题**：按"是什么 + 为什么 + 怎么做"三段式
- **论述题**：理论 + 结合实际 + 总结意义（理实结合）
- **材料分析题**：先判断材料涉及哪个知识点，再分点分析
:::
