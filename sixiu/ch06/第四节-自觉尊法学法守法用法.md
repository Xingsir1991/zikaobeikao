# 第四节 自觉尊法学法守法用法

> 教材第 232–251 页 · 实践层——法治思维、权利义务。

本节是法治篇的实践落脚点：在阐述法律基础、依法治国、宪法权威之后，进一步回答大学生应当具备什么样的法治思维、如何处理法律权利与法律义务的关系。

---

## 一、核心论断（必背）

> 大学生要自觉**尊法、学法、守法、用法**——不断提升法治素养。

> **法治思维**是以法治理念为基础、以法律规则为依据，运用法律原则、法律精神、法律逻辑思考和解决问题的思维方式。

> **法律权利**与**法律义务**是**辩证统一**的关系。

---

## 二、尊法学法守法用法（必背）

::: danger 必背：尊学守用四环节
大学生提升法治素养要做到：
1. **尊法**——内心敬畏法律、信仰法治
2. **学法**——学习法律知识，了解法律体系
3. **守法**——自觉遵守法律，依法办事
4. **用法**——运用法律维护权益、解决纠纷
:::

### 四个环节的递进关系

| 环节 | 内涵 | 层次 |
| --- | --- | --- |
| **尊法** | 内心信仰、敬畏法治 | 精神层 |
| **学法** | 学习法律、知法懂法 | 知识层 |
| **守法** | 遵守法律、依法办事 | 行为层 |
| **用法** | 运用法律、维护权益 | 实践层 |

::: tip 记忆口诀
**"尊—学—守—用"**——四者由内而外、由认知到实践，构成法治素养的完整链路。
:::

::: warning 易错点
**"尊学守用"的顺序不能颠倒**——尊法（内心）→ 学法（认知）→ 守法（行为）→ 用法（实践）——这是循序渐进的法治素养培养路径。
:::

---

## 三、法治思维（极高频考点）

### 1. 法治思维的含义（必背）

::: danger 必背：法治思维的含义
**法治思维**是以**法治理念为基础、法律规则为依据**，运用法律原则、法律精神、法律逻辑来**思考和解决问题**的思维方式。

它区别于人治思维、权力思维、关系思维。
:::

### 2. 法治思维与人治思维的对比

| 思维方式 | 法治思维 | 人治思维 |
| --- | --- | --- |
| **依据** | 法律 | 个人意志 |
| **标准** | 规则 | 权力、关系 |
| **方式** | 程序化、规范化 | 任意、随意 |
| **结果** | 公平可预期 | 因人而异 |

### 3. 法治思维的基本内容（必背）

::: danger 必背：法治思维的基本内容
1. **法律至上**——宪法法律具有最高权威
2. **权力制约**——权力必须受到制约和监督
3. **公平正义**——追求公平正义是法治的价值追求
4. **权利保障**——保障人权是法治的根本目的
5. **正当程序**——程序正当是法治的基本原则
:::

::: tip 记忆口诀
**"法律至上—权力制约—公平正义—权利保障—正当程序"**——法治思维的五个基本内容。
:::

::: warning 易错点
**法治思维的五个基本内容要分清**：法律至上、权力制约、公平正义、权利保障、正当程序——常考多选，不要遗漏。
:::

## 四、法律权利与法律义务（必背）

### 1. 法律权利的含义

::: danger 必背：法律权利
**法律权利**是**国家通过法律规定**的对法律关系主体可以自主决定作出某种行为的**许可和保障**。
:::

### 2. 法律义务的含义

::: danger 必背：法律义务
**法律义务**是**国家通过法律规定**的对法律关系主体必须作出一定行为或不得作出一定行为的**约束**。
:::

### 3. 法律权利与法律义务的关系（极高频考点）

::: danger 必背：法律权利与法律义务的关系
1. **辩证统一**——权利与义务相互依存、相互促进
2. **一律平等**——所有公民在法律面前一律平等享有权利、履行义务
3. **互利共赢**——权利义务的设定追求互利共赢
:::

### 关系对照

| 维度 | 法律权利 | 法律义务 |
| --- | --- | --- |
| **性质** | 可以做（许可保障） | 必须做/不得做（约束） |
| **来源** | 国家法律规定 | 国家法律规定 |
| **关系** | 与义务辩证统一 | 与权利辩证统一 |

::: warning 易错点
**法律权利和法律义务是辩证统一的**——没有无权利的义务，也没有无义务的权利。两者相互依存，不能只讲权利不讲义务，也不能只讲义务不讲权利。
:::

## 五、我国宪法规定的公民基本权利和义务（理解）

### 1. 公民的基本权利

::: danger 必背：公民的基本权利
1. **政治权利和自由**（选举权、被选举权、政治自由）
2. **人身自由**
3. **宗教信仰自由**
4. **社会经济权利**（劳动权、受教育权等）
5. **文化教育权利**
6. **特定主体的权利保障**
:::

### 2. 公民的基本义务

::: danger 必背：公民的基本义务
1. **维护国家统一和民族团结**
2. **遵守宪法和法律**
3. **维护国家安全、荣誉和利益**
4. **依法服兵役**
5. **依法纳税**
6. 其他义务（如劳动、受教育等既是权利又是义务）
:::

### 预测题一（单选）

**下列各项不属于法治思维基本内容的是（　）**
A. 法律至上
B. 权力制约
C. 关系至上
D. 公平正义

> **答案：C**
> **解析**：法治思维的基本内容包括**法律至上、权力制约、公平正义、权利保障、正当程序**五项。"关系至上"是人治思维、关系思维的典型特征，与法治思维对立。

### 预测题二（简答）

**简述法律权利与法律义务的关系。**

> **【主体】** 法律权利与法律义务的关系是辩证的：
>
> **第一，二者辩证统一。** 法律权利和法律义务相互依存、相互促进——没有无权利的义务，也没有无义务的权利。权利的实现需要义务的履行，义务的履行保障了权利的实现。
>
> **第二，权利义务一律平等。** 所有公民在法律面前一律平等享有权利、一律平等履行义务，没有超越法律的特权。
>
> **第三，权利义务互利共赢。** 法律设定权利义务的根本目的是保障人民利益，实现互利共赢。
>
> **【总结】** 因此，公民既要依法行使权利，也要依法履行义务——两者辩证统一、不可割裂。

### 易错点提醒

::: warning 本节三大易错点
1. **"尊学守用"的顺序固定**：尊法→学法→守法→用法——不能颠倒。
2. **法治思维五项基本内容**：法律至上、权力制约、公平正义、权利保障、正当程序——五项不能漏。
3. **法律权利与法律义务辩证统一**——不是对立、不是割裂，是相互依存、互利共赢。
:::
