# 第一节 社会主义法律的特征和运行

> 教材第 189–201 页 · 概念界定——法律是什么、社会主义法律的本质特征。

本节是法治篇的概念基础。它先界定法律的含义和特征，再阐明社会主义法律的本质特征，最后说明法律的运行（制定、执行、适用、遵守）。

---

## 一、核心论断（必背）

> 法律是由**国家制定或认可**、以**国家强制力保证实施**的、具有**普遍约束力**的特殊行为规范体系。

> 我国社会主义法律体现了**党的主张和人民共同意志**的统一，是**阶级性和人民性的统一**。

---

## 二、法律的含义（必背）

### 1. 法律的定义

::: danger 必背：法律的含义
**法律**是由国家制定或认可、由国家强制力保证实施的、对全体社会成员具有普遍约束力的特殊行为规范体系。
:::

### 2. 法律的三个特征（极高频考点）

::: danger 必背：法律的三个特征
1. **国家意志性**——由国家制定或认可，体现国家意志
2. **国家强制性**——由国家强制力保证实施
3. **普遍约束性**——对全体社会成员具有普遍约束力
:::

::: tip 记忆口诀
**"国家意志—国家强制—普遍约束"**——三个特征都围绕"国家"展开。
:::

### 易混辨析：法律 vs 道德 vs 纪律

| 规范 | 来源 | 强制力 | 范围 |
| --- | --- | --- | --- |
| **法律** | 国家制定认可 | **国家强制力** | 全体社会成员 |
| **道德** | 社会舆论、传统习俗 | 内心信念、舆论 | 道德共同体 |
| **纪律** | 组织规定 | 组织纪律 | 组织成员 |

## 三、法律的历史发展（理解）

| 类型 | 基础 |
| --- | --- |
| **奴隶制法律** | 奴隶主所有制 |
| **封建制法律** | 封建地主土地所有制 |
| **资本主义法律** | 资本主义私有制 |
| **社会主义法律** | 社会主义公有制（最先进） |

---

## 四、社会主义法律的本质特征（极高频考点）

::: danger 必背：我国社会主义法律的本质特征
1. **体现了党的主张和人民共同意志的统一**——党的领导、人民当家作主、依法治国有机统一
2. **体现了阶级性与人民性的统一**——既有工人阶级先锋队的阶级性，又有最广大人民的人民性
3. **体现了科学性与先进性的统一**——体现社会发展规律，是最先进的法律
:::

::: tip 记忆口诀
**"两统一"**——党的主张与人民意志的统一；阶级性与人民性的统一。再加科学先进性的统一——构成三大本质特征。
:::

::: warning 易错点
**社会主义法律的本质特征**不是单一属性，而是"几个统一"——这是马克思主义法学观点，区别于资产阶级"法律超阶级超党派"的错误观点。
:::

---

## 五、我国社会主义法律的运行（必背）

::: danger 必背：法律运行的四个环节
法律的运行包括四个环节：
1. **法律制定**（立法）——全国人大及其常委会
2. **法律执行**（执法）——行政机关（政府）
3. **法律适用**（司法）——司法机关（法院、检察院）
4. **法律遵守**（守法）——所有公民和组织
:::

### 四个环节对照

| 环节 | 主体 | 内容 |
| --- | --- | --- |
| **立法**（制定） | 全国人大及其常委会 | 制定法律 |
| **执法**（执行） | 行政机关（政府） | 执行法律 |
| **司法**（适用） | 法院、检察院 | 适用法律处理案件 |
| **守法**（遵守） | 全体公民、组织 | 遵守法律 |

::: tip 记忆口诀
**"立法—执法—司法—守法"**——四个环节对应四类主体：人大、政府、司法、全民。
:::

### 预测题一（单选）

**法律区别于道德、纪律等行为规范的最主要特征是（　）**
A. 由社会舆论保证实施
B. 由国家强制力保证实施
C. 由组织纪律保证实施
D. 由道德良心保证实施

> **答案：B**
> **解析**：法律的最主要特征是**由国家强制力保证实施**——这是法律区别于道德（舆论）、纪律（组织）的根本特征。A、C、D 都不是法律的特征。

### 预测题二（简答）

**简述我国社会主义法律的本质特征。**

> **【主体】** 我国社会主义法律的本质特征体现在三个方面：
>
> **第一，体现了党的主张和人民共同意志的统一。** 社会主义法律是党的领导、人民当家作主、依法治国有机统一的体现，反映了党对国家生活和社会生活的领导与最广大人民共同意志。
>
> **第二，体现了阶级性与人民性的统一。** 社会主义法律既有工人阶级先锋队的阶级性，又有最广大人民的人民性——阶级性与人民性在社会主义法律中得到统一。
>
> **第三，体现了科学性与先进性的统一。** 社会主义法律体现社会发展规律，是人类历史上最先进、最科学的法律。
>
> **【总结】** 这三大本质特征决定我国社会主义法律是真正反映人民意志、维护人民利益的先进法律。

### 易错点提醒

::: warning 本节三大易错点
1. **法律的最主要特征=国家强制力**——不是"舆论""良心""纪律"。
2. **社会主义法律本质特征是"几个统一"**——党的意志与人民意志、阶级性与人民性、科学性与先进性。
3. **法律运行四环节主体不同**：立法（人大）、执法（政府）、司法（法院检察院）、守法（全民）——主体易混。
:::
