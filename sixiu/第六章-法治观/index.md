## 第一节 社会主义法律的特征和运行

### 一、法律的含义与特征 ★

::: warning 法律的含义（必背）
法律是由**国家制定或认可**、由国家**强制力保证实施**、具有**普遍约束力**的行为规范体系。
:::

**法律的三大特征**：★★（必考多选）

| 特征 | 含义 |
| --- | --- |
| **国家制定或认可** | 法律由国家专门机关依职权制定，或认可习惯 |
| **国家强制力保证实施** | 这是法律区别于道德、宗教等规范的最重要特征 |
| **普遍约束力** | 对全体社会成员有效（法律面前人人平等） |

### 二、社会主义法律的特征 ★

- **党的主张和人民意愿的统一**（本质特征）
- **科学性与先进性**（反映社会发展规律）

### 三、法律的运行 ★

法律的运行是一个从**创制、实施到实现**的过程：

| 环节 | 含义 | 主体 |
| --- | --- | --- |
| **法律制定**（立法） | 创制法律 | 国家立法机关 |
| **法律执行**（执法） | 行政机关执行法律 | 政府（行政机关） |
| **法律适用**（司法） | 司法机关应用法律处理案件 | 法院、检察院 |
| **法律遵守**（守法） | 全体社会成员遵守法律 | 全体公民 |

::: warning 易混辨析（高频考点）
- **立法** → 人民代表大会
- **执法** → 政府（行政机关）
- **司法** → 法院、检察院
- **守法** → 全体公民
**不要把"执法"和"司法"搞混！执法=行政，司法=法院检察院。**
:::

## 第二节 坚持全面依法治国

### 一、全面依法治国的总体目标 ★★

::: warning 必背
全面依法治国的总目标：建设**中国特色社会主义法治体系**，建设**社会主义法治国家**。
:::

### 二、全面依法治国的工作格局 ★

| 环节 | 要求 |
| --- | --- |
| **科学立法** | 完善法律体系（前提） |
| **严格执法** | 政府依法行政（关键） |
| **公正司法** | 维护公平正义（防线） |
| **全民守法** | 全社会法治意识（基础） |

### 三、三个"依法" ★★（高频考点，必辨析）

| 概念 | 主体 | 含义 |
| --- | --- | --- |
| **依法治国** | **人民**是主体、力量源泉 | 全体人民在党的领导下依法治理国家 |
| **依法执政** | **党**（中国共产党） | 党领导立法、保证执法、支持司法、带头守法 |
| **依法行政** | **政府**（行政机关） | 各级政府在法定权限内行使权力 |

::: warning 易错点
- 依法治国的主体 = **人民**（不是党、不是政府）
- 依法执政的主体 = **党**
- 依法行政的主体 = **政府**
**三者的主体不同，是高频考点！**
:::

### 四、法治与德治 ★

- 法治是**基本方式**
- 德治是**重要方式**
- 坚持依法治国和以德治国**相结合**

## 第三节 维护宪法权威

### 一、宪法的特征 ★★（必考）

**宪法是国家的根本法**，与普通法律相比，有以下特征：

| 特征 | 内容 |
| --- | --- |
| **内容上** | 规定国家最根本、最重要的问题（国体政体、公民基本权利义务） |
| **效力上** | 最高法律效力（一切法律不得与宪法相抵触） |
| **制定和修改程序上** | 比普通法律更严格（全国人大 2/3 以上多数通过） |

### 二、宪法的基本原则 ★★（必考）

| 原则 | 含义 |
| --- | --- |
| **党的领导原则** | 中国共产党领导是中国特色社会主义最本质的特征 |
| **人民主权原则** | 国家一切权力属于人民 |
| **人权保障原则** | 尊重和保障人权 |
| **法治原则** | 依法治国，建设社会主义法治国家 |
| **民主集中制原则** | 我国国家机构的组织和活动原则 |

### 三、宪法规定的公民基本权利 ★★（高频）

| 类别 | 具体权利 |
| --- | --- |
| **平等权** | 法律面前一律平等 |
| **政治权利与自由** | 选举权和被选举权、言论/出版/集会/结社/游行/示威自由 |
| **人身自由** | 人身自由不受侵犯、人格尊严、住宅不受侵犯、通信自由 |
| **宗教信仰自由** | — |
| **监督权** | 批评、建议、申诉、控告、检举权 |
| **社会经济权利** | 劳动权、休息权、社会保障权、物质帮助权 |
| **文化教育权利** | 受教育权、文化活动自由 |

::: warning 易混
- **选举权和被选举权**的条件：年满 18 周岁、中国公民、未被剥夺政治权利
- **劳动权**和**受教育权**既是权利也是**义务**
:::

### 四、国家宪法日 ★

- 每年 **12 月 4 日**是国家宪法日（现行宪法 1982 年 12 月 4 日颁布）

## 第四节 自觉尊法学法守法用法

### 一、培养社会主义法治思维 ★★（必考）

#### ① 法治思维的五要素

::: warning 法治思维的五要素（必背）
1. **法律至上**：法律具有最高权威
2. **权力制约**：有权必有责、用权受监督
3. **公平正义**：法律的核心价值
4. **权利保障**：保障人权
5. **正当程序**：程序正当
:::

#### ② 维护法律权威

- 法律权威源于**人民的内心拥护和真诚信仰**
- 维护法律权威：信仰法律、遵守法律、服从法律、维护法律

### 二、依法行使权利与履行义务 ★

| | 权利 | 义务 |
| --- | --- | --- |
| 关系 | **密不可分、相互依存** | — |
| 特点 | 权利可以放弃 | 义务必须履行（不可放弃） |

**既是权利又是义务**：劳动、受教育。

### 三、不断提升法治素养 ★

#### ① 法治素养的内容

- **法治知识**：了解法律常识
- **法治意识**：规则意识、程序意识、权利义务意识
- **法治行为**：依法办事、遇事找法

#### ② "尊学守用"四字诀

大学生提升法治素养，要做到**尊法、学法、守法、用法**：

| 要求 | 内涵 |
| --- | --- |
| **尊法** | 尊重法律权威，树立法律信仰 |
| **学法** | 学习法律知识，掌握法律常识 |
| **守法** | 自觉遵守法律，依法规范行为 |
| **用法** | 依法维护权益，遇事找法、解决问题靠法 |

## 易错点提醒

::: warning 易错辨析（本章最多）
1. **执法 ≠ 司法**：执法=政府（行政），司法=法院检察院
2. **三个"依法"的主体**：治国→人民，执政→党，行政→政府
3. **宪法修改**：全国人大 **2/3 以上**多数（不是过半数）
4. **既是权利又是义务**：劳动、受教育（不是"休息"，休息只是权利）
5. **选举权年龄**：18 周岁（不是 16 周岁）
6. **法治 vs 法制**：法治=依法治理（动态），法制=法律制度（静态）
7. **国家宪法日**：12 月 4 日（不是 10 月 1 日或 12 月 1 日）
:::

## 模拟题

### 【单选 1】法律区别于道德、宗教等规范的最重要特征是（ ）
- A. 国家制定或认可
- B. 国家强制力保证实施 ✔
- C. 普遍约束力
- D. 阶级性

> **答案：B**
> **解析**：法律三特征中，"**国家强制力保证实施**"是法律区别于其他社会规范的最重要特征——道德靠内心信念、宗教靠信仰，只有法律靠国家强制力。

### 【单选 2】依法治国的主体是（ ）
- A. 中国共产党
- B. 人民政府
- C. 人民 ✔
- D. 人民法院

> **答案：C**
> **解析**：依法治国的主体是**人民**（人民当家作主）；依法执政的主体是**党**；依法行政的主体是**政府**。三者主体不同，高频考点。

### 【单选 3】我国宪法规定，修改宪法必须由全国人大（ ）
- A. 过半数通过
- B. 三分之二以上多数通过 ✔
- C. 四分之三以上多数通过
- D. 一致同意

> **答案：B**
> **解析**：宪法的制定和修改程序比普通法律更严格，修改须全国人大 **2/3 以上**多数通过。普通法律过半数即可。

### 【简答】简述法治思维的基本内容。

> **标准答案：**
> 法治思维是以法治理念为基础、以法律为判断是非曲直标准的思维方式，包含五要素：
> ① **法律至上**：法律具有最高权威，任何组织和个人不得凌驾于法律之上。
> ② **权力制约**：有权必有责、用权受监督、滥权受追究。
> ③ **公平正义**：法律的核心价值追求。
> ④ **权利保障**：尊重和保障人权。
> ⑤ **正当程序**：做决定、办事情必须遵循法定程序。

### 【论述】结合实际，谈谈大学生如何自觉尊法学法守法用法，提升法治素养。

> **标准答案：**
> 大学生提升法治素养，要做到**尊法、学法、守法、用法**四字诀：
> ① **尊法**：尊重法律权威，树立法律信仰——法律不是束缚，而是保护。
> ② **学法**：学习法律知识（宪法、民法、刑法等基本常识）。
> ③ **守法**：自觉遵守法律法规，依法规范自身行为。
> ④ **用法**：遇事找法、解决问题靠法——用法律手段维护自身合法权益。
> ⑤ **培养社会主义法治思维**：养成法律至上、程序正当的思维方式。
>
> **结合实际**：如在学校遵守校纪校规（守法的日常体现）、遇到消费纠纷时依法维权（用法的实践）、不参与网络暴力（守法+尊重他人权利）。
