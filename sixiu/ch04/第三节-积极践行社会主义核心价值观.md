# 第三节 积极践行社会主义核心价值观

> 教材第 129–137 页 · 实践层面——大学生如何践行社会主义核心价值观。

本节是第四章的落脚点：在明确社会主义核心价值观的内涵和特征之后，进一步回答大学生为什么要践行、如何践行社会主义核心价值观。

---

## 一、核心论断（必背）

> 青年的价值取向决定了未来整个社会的价值取向。**人生的扣子从一开始就要扣好**。

> 大学生要切实做到**勤学、修德、明辨、笃实**。

---

## 二、为什么要扣好人生的扣子（高频考点）

### 1. 习近平的"扣子"比喻（必背金句）

::: danger 必背：扣好人生的扣子
习近平比喻："**青年的价值取向决定了未来整个社会的价值取向，而青年又处在价值观形成和确立的时期，抓好这一时期的价值观养成十分重要。这就像穿衣服扣扣子一样，如果第一粒扣子扣错了，剩余的扣子都会扣错。人生的扣子从一开始就要扣好。**"
:::

### 2. 为什么是"第一粒扣子"

| 理由 | 说明 |
| --- | --- |
| **青年是价值观形成的关键期** | 大学是价值观确立的关键阶段 |
| **青年价值取向决定社会价值取向** | 青年决定未来社会走向 |
| **错误价值观影响终生** | "第一粒扣子"错，后面都错 |

::: tip 论述角度
"为什么扣好人生扣子"的论述，可从"价值观形成规律 + 青年地位 + 错误影响终生"三方面展开。
:::

---

## 三、大学生践行社会主义核心价值观的要求（必背）

::: danger 必背：勤学、修德、明辨、笃实
大学生践行社会主义核心价值观，要切实做到：
1. **勤学**——下得苦功夫，求得真学问
2. **修德**——加强道德修养，注重道德实践
3. **明辨**——善于明辨是非，善于决断选择
4. **笃实**——扎扎实实干事，踏踏实实做人
:::

### 四项要求的逐条解读

| 要求 | 内涵 |
| --- | --- |
| **勤学** | 知识是树立核心价值观的重要基础；要下苦功夫求真学问 |
| **修德** | 道德是做人做事的根本；要德才兼备、以德为先 |
| **明辨** | 善于辨别是非、善恶、美丑；不受错误思潮影响 |
| **笃实** | 道不可坐论，德不能空谈；要知行合一、笃实做人 |

::: tip 记忆口诀
**"勤、修、明、笃"**——四个字浓缩四要求：勤学、修德、明辨、笃实。
:::

::: warning 易错点
**"勤学、修德、明辨、笃实"是固定表述，不能换**。常见干扰项如"勤学、修身、明辨、笃行"或"勤学、修德、慎思、笃实"——均与教材原文不符。
:::

## 四、践行社会主义核心价值观的方法（理解）

### 1. 内化于心、外化于行

| 层次 | 要求 |
| --- | --- |
| **认知层（内化）** | 学习理解社会主义核心价值观 |
| **认同层（信念）** | 坚定对社会主义核心价值观的信念 |
| **实践层（外化）** | 自觉在日常学习生活中践行 |

### 2. 落细落小落实

- **落细**——从细节入手
- **落小**——从小事做起
- **落实**——落到实处、求得实效

## 五、在日常生活中践行（理解）

大学生在日常生活中践行社会主义核心价值观要做到：

| 场景 | 实践要求 |
| --- | --- |

| **生活** | 艰苦朴素，勤俭节约 |

### 预测题一（单选）

**习近平比喻，青年时期价值观养成"就像穿衣服扣扣子一样"——这强调的是（　）**
A. 人生的扣子从一开始就要扣好
B. 价值观是后天形成的，可以任意改变
C. 青年时期价值观养成不重要
D. 价值观与社会实践无关

> **答案：A**
> **解析**：习近平比喻——"**人生的扣子从一开始就要扣好**"。青年处于价值观形成和确立的关键时期，要抓好这一时期的价值观养成。B、C、D 都偏离了原文含义。

### 预测题二（简答）

**大学生应当如何践行社会主义核心价值观？**

> **【主体】** 大学生践行社会主义核心价值观，要切实做到四点：
>
> **第一，勤学。** 知识是树立核心价值观的重要基础。要下得苦功夫，求得真学问，扎扎实实打牢知识根基。
>
> **第二，修德。** 道德是做人做事的根本。要加强道德修养、注重道德实践，做到德才兼备、以德为先。
>
> **第三，明辨。** 善于明辨是非、善恶、美丑，善于决断选择，自觉抵制错误思潮。
>
> **第四，笃实。** 道不可坐论，德不能空谈。要扎扎实实干事、踏踏实实做人，做到知行合一。
>
> **【总结】** 青年兴则国家兴。大学生要扣好人生的第一粒扣子，在勤学、修德、明辨、笃实中把社会主义核心价值观内化于心、外化于行。

### 易错点提醒

::: warning 本节三大易错点
1. **"扣好人生的扣子"是习近平对青年价值观养成的比喻**——常考出处。
2. **"勤学、修德、明辨、笃实"四项要求用词固定**——不能写成"勤学、修身、明辨、笃行"。
3. **青年的价值取向决定未来整个社会的价值取向**——这是青年地位的核心论断。
:::
