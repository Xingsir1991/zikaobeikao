## 第一节 社会主义道德的核心与原则

### 一、坚持马克思主义道德观 ★★★★

#### ① 道德的含义与起源

1. **道德的含义**：道德是以善恶为评价方式，主要依靠社会舆论、传统习俗和内心信念来发挥作用的行为规范的总和。它调节**人与人、人与社会、人与自然**之间的关系。
2. **道德的起源**：道德产生于人类社会的**生产生活实践**，是社会历史发展的产物。劳动实践是道德起源的**客观前提**，社会关系的形成是道德产生的**客观条件**，人类的自我意识是道德产生的**主观条件**。

#### ② 道德的本质（高频考点）

::: tip 必背
道德是一种**由一定社会的经济基础所决定的特殊意识形态**，是上层建筑的重要组成部分。
:::

要点拆解：
- **根源**：道德归根到底由**经济基础**决定，反映社会物质生活条件。
- **属性**：道德属于**上层建筑**中的**意识形态**，具有鲜明的阶级性（在阶级社会中）。
- **特征**：道德是**特殊的**意识形态——它以善恶为标准，靠舆论、习俗、信念维持，区别于靠强制力维持的法律。

#### ③ 道德的功能

| 功能 | 含义 | 地位 |
| --- | --- | --- |
| **认识功能** | 帮助人们认识自己对他人、社会应负的责任 | 基础功能 |
| **调节功能** | 调节人与人、人与社会、人与自然的关系 | **最突出、最主要、最基本** |
| **教育功能** | 通过舆论、榜样等培养人的道德品质 | 重要功能 |
| **评价功能** | 以善恶为标准评价人们的行为 | 常见功能 |

::: tip 高频单选
问"道德最基本/最突出/最主要的功能是什么"——一律答 **调节功能**。
:::

#### ④ 马克思主义道德观的基本立场

马克思主义道德观认为：
- 道德是**历史的、具体的**——没有超历史的永恒道德
- 道德由**经济基础决定**，具有阶级性
- 道德随着社会实践的发展而**发展变化**

### 二、坚持以为人民服务为核心 ★★★★★（必考）

道德的核心和原则，本质上回答两个根本问题：

| 问题 | 回答 | 体现 |
| --- | --- | --- |
| **为什么人服务**（核心问题） | **为人民服务** | 社会主义道德的**核心** |
| **如何处理利益关系**（原则问题） | **集体主义** | 社会主义道德的**原则** |

#### ① 为什么是核心

1. 为人民服务是社会主义经济基础的**客观要求**（公有制为主体）；
2. 为人民服务是社会主义政治制度的**本质要求**（人民当家作主）；
3. 为人民服务是马克思主义的**根本立场**，是中国共产党人的**初心使命**。

#### ② 基本含义

把人民利益放在第一位，一切从人民利益出发，一切为了群众、一切依靠群众。

#### ③ 层次性

既包括对**先进分子**的无私奉献要求，也包括对**广大群众**的"我为人人、人人为我"的基本要求（既有先进性，又有广泛性）。

::: warning 易混
"为人民服务"是**核心**，不是原则；它回答"为什么人"的问题，不直接回答"如何处理利益关系"。
:::

### 三、坚持以集体主义为原则 ★★★★★（必考）

#### ① 基本内涵

强调国家、社会整体利益与个人利益的**辩证统一**；当二者发生冲突时，**国家、集体利益高于个人利益**；同时**保障个人的正当利益**。

#### ② 三个层次的要求（必背，按境界由高到低）

| 层次 | 要求 | 适用对象 |
| --- | --- | --- |
| **最高层次** | 无私奉献，一心为公 | 先进分子（党员、先进人物） |
| **基本层次** | 先公后私，顾全大局 | 一般群众的基本要求 |
| **底线层次** | 顾全大局，不损人利己 | 所有公民的最低要求 |

::: tip 必考
集体主义**不否认**个人利益，而是**保障个人的正当利益**；它反对的是**个人主义**（把个人利益凌驾于集体利益之上）。
:::

## 第二节 吸收借鉴优秀道德成果

### 一、传承中华传统美德 ★★★★★

1. **基本精神**（5 条，高频）：
   - 注重**整体利益、国家利益和民族利益**，强调对社会、民族的奉献（"公义胜私欲"）；
   - 推崇**"仁爱"原则**，主张"己所不欲，勿施于人""老吾老以及人之老"；
   - 提倡**人伦价值**，强调"五伦"关系（父子、君臣、夫妇、长幼、朋友）；
   - 追求**精神境界和理想人格**（"修身齐家治国平天下"）；
   - 强调**道德修养**，注重"慎独""克己""内省"。
2. **对待传统美德的正确态度**（简答、论述常考）：
   - 坚持**马克思主义道德观**，进行**创造性转化**和**创新性发展**；
   - 反对**复古守旧**（全盘照搬）和**历史虚无主义**（全盘否定）两种错误倾向；
   - 与**社会主义核心价值观**相结合，赋予新的时代内涵。

### 二、发扬中国革命道德 ★★★★

1. **含义**：中国共产党人在长期革命、建设、改革实践中形成的**优良道德传统**，是马克思主义与中国革命实践相结合的产物。
2. **主要内容**（5 条）：
   - 为实现**社会主义和共产主义**而奋斗的理想信念；
   - 始终把**革命利益**放在首位；
   - 全心全意为**人民服务**；
   - 树立社会**新风尚、新道德**；
   - **修身自律、保持节操**，艰苦朴素、廉洁奉公。
3. **现实意义**（简答常考）：
   - 是坚定**理想信念**、实现中华民族伟大复兴的精神动力；
   - 是培育和践行**社会主义核心价值观**的重要资源；
   - 是加强**党风廉政建设**、密切党同人民群众联系的内在要求；
   - 是培养时代新人、提升全民族**道德素质**的现实需要。

::: tip 联系
革命道德的核心同样是 **为人民服务**、**集体主义**，与社会主义道德一脉相承。
:::

### 三、借鉴人类文明优秀道德成果 ★★★

- 对待态度：坚持**以我为主、为我所用、辩证取舍、择善而从**的原则；
- 既反对**全盘西化、照搬照抄**，也反对**封闭保守、盲目排外**；
- 注意鉴别其中符合**人类共同价值**的合理因素（如诚信、友善、正义等）。

## 第三节 投身崇德向善的道德实践

本节对应社会生活的**四大领域**，每类规范**各 5 条**，是全章**必背**考点。

### 一、遵守社会公德 ★★★★★

| 规范 | 要点 |
| --- | --- |
| **文明礼貌** | 言谈举止得体，尊重他人 |
| **助人为乐** | 关心帮助他人，"赠人玫瑰，手有余香" |
| **爱护公物** | 珍惜公共财物，不损坏公物 |
| **保护环境** | 节约资源，绿色低碳 |
| **遵纪守法** | 遵守法律法规（**底线**） |

> 记忆口诀：**文明助人，护物护境，遵纪守法**。

### 二、恪守职业道德 ★★★★★

| 规范 | 要点 |
| --- | --- |
| **爱岗敬业** | 热爱本职工作，尽职尽责（**基础**） |
| **诚实守信** | 真诚待人，信守承诺（**核心、根本**） |
| **办事公道** | 处事公正，一视同仁 |
| **热情服务** | 服务群众，态度热情 |
| **奉献社会** | 为社会作贡献（**最高境界**） |

::: tip 考点
职业道德的**基础 = 爱岗敬业**；**最高境界 = 奉献社会**；**核心/根本 = 诚实守信**（与市场经济直接挂钩）。
:::

### 三、弘扬家庭美德 ★★★★★

| 规范 | 要点 |
| --- | --- |
| **尊老爱幼** | 孝敬父母，爱护子女（**核心**） |
| **男女平等** | 反对性别歧视 |
| **夫妻和睦** | 互相忠诚、互相尊重（家庭关系的**核心**） |
| **勤俭持家** | 量入为出，反对铺张浪费 |
| **邻里团结** | 互谅互让，守望相助 |

> 记忆口诀：**尊老爱幼，男女平等，夫妻和睦，勤俭持家，邻里团结**。

### 四、锤炼个人品德 ★★★★★

1. **含义**：个人品德是个人的**道德品质**和**道德境界**，是个人在社会生活中表现出来的**稳定的**心理特征和行为倾向。
2. **特点**：**实践性、稳定性、综合性**。
3. **五条规范**：

| 规范 | 要点 |
| --- | --- |
| **爱国奉献** | 把个人理想融入国家发展 |
| **明礼遵规** | 讲文明、守规矩 |
| **勤劳善良** | 勤奋劳动、与人为善 |
| **宽厚正直** | 宽容大度、为人正派 |
| **自尊自强** | 自尊自爱、自立自强 |

4. **个人品德修养的方法**（简答高频）：**学思并重、省察克治、慎独自律、知行合一、积善成德**。
   - **慎独**：在无人监督时仍能自觉按道德准则行事，是中华传统美德中重要的修养方法。

::: warning 四类规范易混
- **遵纪守法、保护环境** → 社会公德
- **诚实守信、办事公道** → 职业道德
- **勤俭持家、邻里团结** → 家庭美德
- **自尊自强、宽厚正直** → 个人品德

考试常在选项中"移花接木"，务必逐条对位。
:::

## 易错点

::: warning 易错警示
1. **核心 vs 原则**：核心=为人民服务（为什么人），原则=集体主义（如何处理利益），**二者不可互换**。出题人最爱设的陷阱就是把两者对调。
2. **调节功能 vs 认识功能**：问"最突出/最主要/最基本"——一律答**调节功能**；"认识功能"是基础，但不是"最主要"。
3. **集体主义不否认个人利益**：集体主义**保障个人的正当利益**，反对的是把个人利益凌驾于集体之上的**个人主义**。判断题"集体主义要求取消个人利益"是**错误**表述。
4. **三类规范"五条"易串台**：
   - 社会公德里有 **遵纪守法、保护环境**；
   - 职业道德里有 **诚实守信、办事公道、热情服务**；
   - 家庭美德里有 **勤俭持家、邻里团结**；
   - 不要把"诚信"放到家庭美德，也不要把"勤俭"放到职业道德。
5. **"五爱"≠"五条规范"**："五爱"（爱祖国、爱人民、爱劳动、爱科学、爱社会主义）是社会主义道德的**基本要求**，不是某一类领域的五条规范。
6. **慎独 ≠ 慎微**：慎独指**无人监督**时仍守道德准则；不要与"慎微（防微杜渐）"混淆。
7. **创造性转化 vs 创新性发展**：转化=改造旧有形式以适应新时代；发展=赋予新的内涵、推出新的形态。两者并列，不要混为一谈。
8. **个人品德"稳定性"**：品德是**长期、反复实践**形成的稳定特征，不是一时一事的表现。
:::

## 模拟题

### （一）单项选择题

**1. 道德最基本、最突出的功能是（　）**
A. 认识功能　B. 调节功能　C. 教育功能　D. 评价功能

> **答案：B**。道德最主要、最突出的功能是调节功能，调节人与人、人与社会、人与自然的关系。

**2. 社会主义道德的核心是（　）**
A. 集体主义　B. 为人民服务　C. 共同富裕　D. 爱国主义

> **答案：B**。核心=为人民服务（为什么人）；原则=集体主义（如何处理利益关系），二者易混。

**3. 下列各项中，属于职业道德规范的是（　）**
A. 遵纪守法　B. 勤俭持家　C. 办事公道　D. 邻里团结

> **答案：C**。职业道德五条：爱岗敬业、诚实守信、**办事公道**、热情服务、奉献社会。A 属社会公德，B、D 属家庭美德。

### （二）简答题

**4. 简述集体主义原则的基本内涵及三个层次要求。**

**参考答案：**
（1）基本内涵：集体主义强调国家、社会整体利益与个人利益的**辩证统一**；当二者发生冲突时，**国家、集体利益高于个人利益**；同时**保障个人的正当利益**。
（2）三个层次要求：
- **最高层次**——无私奉献、一心为公，是对先进分子的要求；
- **基本层次**——先公后私、顾全大局，是一般群众的基本要求；
- **底线层次**——不损人利己、自觉维护集体利益，是所有公民的最低要求。

### （三）论述题

**5. 联系实际，论述新时代大学生应如何践行职业道德，争做担当民族复兴大任的时代新人。**

**参考答案要点：**
（1）**爱岗敬业**——树立正确的职业观，干一行、爱一行、钻一行，刻苦学习专业知识，提升职业技能；
（2）**诚实守信**——以诚信立身、立业，恪守契约精神，杜绝学术不端、简历造假；
（3）**办事公道**——处事公正，坚持原则，在评优、就业、工作竞争中不徇私、不取巧；
（4）**热情服务**——树立服务人民、服务社会的意识，从基层岗位做起，主动担当；
（5）**奉献社会**——把个人理想融入国家发展，到祖国和人民最需要的地方去，在奉献中实现人生价值；
（6）**联系实际**：结合大学生在校学习、社会实践、志愿服务、毕业就业选择等具体事例，强调将职业道德内化于心、外化于行，争做有理想、敢担当、能吃苦、肯奋斗的新时代好青年。
