# 第三节 投身崇德向善的道德实践

> 教材第 169–188 页 · 实践层——社会公德、职业道德、家庭美德、个人品德。

本节是全章的实践落脚点。它把社会主义道德从理论层面落实到生活层面，给出**四德**的具体要求——社会公德、职业道德、家庭美德、个人品德。

---

## 一、核心论断（必背）

> 公民道德建设要以**社会公德、职业道德、家庭美德、个人品德**建设为着力点。

> **社会公德**是公民在**社会交往和公共生活中**应当遵循的道德准则。

> **职业道德**是从事一定职业的人在**职业生活中**应当遵循的道德准则。

> **家庭美德**是调节**家庭内部成员**之间关系的道德准则。

> **个人品德**是一定社会的**道德原则和规范**在个人思想和行为中的体现。

---

## 二、"四德"的总体框架（极高频考点）

::: danger 必背：四德的内容
| 道德类型 | 调节范围 | 关键词 |
| --- | --- | --- |
| **社会公德** | 社会交往、公共生活 | 文明礼貌、助人为乐、爱护公物、保护环境、遵纪守法 |
| **职业道德** | 职业生活 | 爱岗敬业、诚实守信、办事公道、热情服务、奉献社会 |
| **家庭美德** | 家庭生活 | 尊老爱幼、男女平等、夫妻和睦、勤俭持家、邻里团结 |
| **个人品德** | 个人修养 | 爱国奉献、明礼遵规、勤劳善良、宽厚正直、自尊自强 |
:::

::: tip 记忆口诀
**"社公—职职—家美—人德"**——四德对应不同生活领域：社会公共→职业→家庭→个人修养。
:::

---

## 三、社会公德（必背）

### 1. 社会公德的定义

::: danger 必背：社会公德的定义
**社会公德**是公民在**社会交往和公共生活中**应当遵循的**行为准则**，是维护社会公共生活正常进行、维护公共秩序稳定的基本条件。
:::

### 2. 社会公德的主要内容（必背）

::: danger 必背：社会公德的主要内容（五条）
1. **文明礼貌**
2. **助人为乐**
3. **爱护公物**
4. **保护环境**
5. **遵纪守法**
:::

### 3. 大学生践行社会公德的要求（理解）

| 场景 | 要求 |
| --- | --- |
| **公共生活** | 遵守公共秩序、爱护公共财物 |
| **网络生活** | 健康文明上网，遵守网络道德 |
| **人际交往** | 尊重他人，文明礼貌 |

## 四、职业道德（必背）

### 1. 职业道德的定义

::: danger 必背：职业道德的定义
**职业道德**是从事一定职业的人在**职业生活中**应当遵循的**具有职业特征的道德要求和行为准则**。
:::

### 2. 职业道德的主要内容（必背）

::: danger 必背：职业道德的主要内容（五条）
1. **爱岗敬业**
2. **诚实守信**
3. **办事公道**
4. **热情服务**
5. **奉献社会**
:::

### 3. 大学生的职业道德准备（理解）

| 方面 | 要求 |
| --- | --- |
| **树立正确的择业观** | 树立职业平等观、积极择业观 |
| **树立正确的就业观** | 把个人志愿与国家需要结合 |
| **培养职业道德** | 在学习生活中培养敬业诚信品质 |

::: tip 论述角度
论述"大学生如何践行职业道德"，可从择业观、就业观、培养职业品质三方面展开。
:::

## 六、个人品德（必背）

### 1. 个人品德的定义

::: danger 必背：个人品德的定义
**个人品德**是一定社会的**道德原则和规范**在个人思想和行为中的体现，是个体在道德行为中表现出来的**稳定的心理特征和价值取向**。
:::

### 2. 个人品德的主要内容（必背）

::: danger 必背：个人品德的主要内容（五条）
1. **爱国奉献**
2. **明礼遵规**
3. **勤劳善良**
4. **宽厚正直**
5. **自尊自强**
:::

### 3. 个人品德的特征（理解）

- **实践性**：在道德实践中形成
- **稳定性**：一旦形成不易改变
- **一致性**：言行一致、内外统一

### 4. 个人品德与道德的关系

| 维度 | 个人品德 | 公德、职德、美德 |
| --- | --- | --- |
| **地位** | 内在修养 | 外在表现 |
| **关系** | 个人品德是"四德"的内在基础 | "四德"是个人品德的具体体现 |

::: danger 必背：个人品德的地位
**个人品德是"四德"（社会公德、职业道德、家庭美德）的内在基础**。
:::

### 预测题一（单选）

**下列各项不属于社会公德主要内容的是（　）**
A. 文明礼貌
B. 助人为乐
C. 爱岗敬业
D. 保护环境

> **答案：C**
> **解析**：社会公德主要包括**文明礼貌、助人为乐、爱护公物、保护环境、遵纪守法**。"爱岗敬业"属于**职业道德**（爱岗敬业、诚实守信、办事公道、热情服务、奉献社会）。

### 预测题二（简答）

**简述公民道德建设的"四德"及其主要内容。**

> **【主体】** 公民道德建设以"四德"为着力点：
>
> **第一，社会公德。** 调节社会交往和公共生活。主要内容：**文明礼貌、助人为乐、爱护公物、保护环境、遵纪守法**。
>
> **第二，职业道德。** 调节职业生活。主要内容：**爱岗敬业、诚实守信、办事公道、热情服务、奉献社会**。
>
> **第三，家庭美德。** 调节家庭内部成员关系。主要内容：**尊老爱幼、男女平等、夫妻和睦、勤俭持家、邻里团结**。
>
> **第四，个人品德。** 是一定社会道德原则规范在个人身上的体现。主要内容：**爱国奉献、明礼遵规、勤劳善良、宽厚正直、自尊自强**。
>
> **【总结】** 四德相互联系、相互促进——个人品德是内在基础，社会公德、职业道德、家庭美德是外在表现，共同构成完整的公民道德体系。

### 易错点提醒

::: warning 本节三大易错点
1. **四德各自的"五条"内容要分清**：尤其要分清"爱岗敬业"（职业）vs"助人为乐"（社会）；"尊老爱幼"（家庭）vs"爱国奉献"（个人品德）。
2. **个人品德是四德的"内在基础"**——四德是个人品德的外在表现。
3. **职业道德中"奉献社会"是最高层次**——是职业道德的最高境界（不是"诚实守信"）。
:::
