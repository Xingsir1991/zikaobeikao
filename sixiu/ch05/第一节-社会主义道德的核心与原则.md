# 第一节 社会主义道德的核心与原则

> 教材第 138–151 页 · 必背——社会主义道德的核心和原则。

本节是全章的理论基石。它给出社会主义道德的两大支柱：**核心是为人民服务，原则是集体主义**。这两句话是思修最高频金句之一。

---

## 一、核心论断（必背）

> 社会主义道德的核心是**为人民服务**。

> 社会主义道德的原则是**集体主义**。

---

## 二、社会主义道德的核心——为人民服务（极高频考点）

### 1. 核心定义（必背）

::: danger 必背：社会主义道德的核心
**为人民服务**是社会主义道德的核心。
:::

### 2. 为什么"为人民服务"是核心（必背，三个理由）

::: danger 必背：为人民服务是核心的三个理由
1. **由社会主义社会的性质决定**——社会主义是人民当家作主的社会
2. **由社会主义公有制和经济基础决定**——公有制决定了必须为人民服务
3. **由人民群众的历史地位决定**——人民是历史的创造者
:::

### 3. 为人民服务的层次性（理解）

为人民服务有**两个层次**：

| 层次 | 内容 |
| --- | --- |
| **高层次（先进性）** | 毫不利己、专门利人，无私奉献（共产党员、先进分子） |
| **基本层次（广泛性）** | 顾全大局、诚实劳动、热心公益（普通公民） |

::: danger 必背：为人民服务的层次性
为人民服务不仅是**共产党人的宗旨**，也是**对全体社会成员的道德要求**。
- 高层次：无私奉献（先进性要求）
- 基本层次：热心公益、利人利己（广泛性要求）
:::

::: warning 易错点
**为人民服务不是只对共产党员的要求，而是对全体社会成员的要求**。它分高层次（无私奉献）和基本层次（热心公益）——这是常考易错点。
:::

---

## 三、社会主义道德的原则——集体主义（极高频考点）

### 1. 原则定义（必背）

::: danger 必背：社会主义道德的原则
**集体主义**是社会主义道德的原则。
:::

### 2. 集体主义原则的内涵（必背）

::: danger 必背：集体主义原则的三个内涵
集体主义原则要求：
1. 强调**集体利益和个人利益的辩证统一**
2. 强调**集体利益高于个人利益**
3. 强调**重视、保障、发展个人的正当利益**
:::

### 3. 集体主义原则的层次（理解）

| 层次 | 内容 |
| --- | --- |
| **无私奉献**（最高层次） | 毫不利己、专门利人 |
| **先公后私**（中间层次） | 集体利益为先，兼顾个人 |
| **公私兼顾**（基础层次） | 不损公肥私，不损人利己 |

### 4. 集体主义的反面——反对个人主义（必背）

::: danger 必背：集体主义反对个人主义
集体主义原则坚决反对**个人主义**——
- 反对把个人利益凌驾于集体利益之上
- 反对"个人本位""自我中心"
- 反对为了个人利益损害集体利益
:::

### 易混辨析：集体主义 vs 个人主义

| 类别 | 集体主义 | 个人主义 |
| --- | --- | --- |
| **核心** | 集体利益优先 | 个人利益至上 |
| **对待他人** | 顾全大局、互助合作 | 利己、竞争至上 |
| **对待集体** | 维护集体利益 | 利用集体、损害集体 |
| **道德境界** | 高尚 | 庸俗 |

::: warning 易错点
**集体主义不否认个人利益**——它"重视、保障、发展个人的正当利益"。集体主义反对的是**个人主义**（把个人凌驾于集体之上），不是反对个人正当利益。
:::

---

## 四、核心与原则的关系（理解）

| 项 | 为人民服务 | 集体主义 |
| --- | --- | --- |
| **地位** | 核心 | 原则 |
| **回答** | 为谁服务 | 怎样处理利益关系 |
| **关系** | 决定原则的方向 | 体现核心的要求 |

::: tip 记忆口诀
**"核心是为人民服务，原则是集体主义"**——两句话构成完整的社会主义道德架构。核心回答"为谁"，原则回答"怎样"。
:::

## 五、社会主义道德与中华传统美德、革命道德的关系（理解）

::: danger 必背：道德传承关系
- 社会主义道德**承接和弘扬了中华传统美德**
- 社会主义道德**继承和发扬了中国革命道德**
- 社会主义道德**借鉴和吸收了人类文明优秀道德成果**

### 预测题一（单选）

**社会主义道德的原则是（　）**
A. 为人民服务
B. 集体主义
C. 爱国主义
D. 个人主义

> **答案：B**
> **解析**：**集体主义**是社会主义道德的**原则**。A 是社会主义道德的**核心**（不是原则）。C 是民族精神的核心；D 是被反对的错误原则。

### 预测题二（简答）

**为什么说为人民服务是社会主义道德的核心？**

> **【主体】** 为人民服务是社会主义道德的核心，其理由有三：
>
> **第一，由社会主义社会的性质决定。** 社会主义社会是人民当家作主的社会，社会的性质决定了道德建设必须为人民服务。
>
> **第二，由社会主义公有制和经济基础决定。** 社会主义公有制为主体，决定了道德的核心必然是为人民服务。
>
> **第三，由人民群众的历史地位决定。** 人民群众是历史的创造者，是推动社会发展的决定力量，因此道德建设必须服务于人民。
>
> **【总结】** 为人民服务既是共产党人的根本宗旨，也是对全体社会成员的道德要求——它体现社会主义性质、经济基础和人民地位，是社会主义道德的核心。

### 易错点提醒

::: warning 本节三大易错点
1. **核心=为人民服务，原则=集体主义**——两句话常考、必背。
2. **集体主义不否认个人利益**——它"重视、保障、发展个人的正当利益"。集体主义反对的是**个人主义**。
3. **为人民服务不只是对党员的要求**——分高层次（无私奉献）和基本层次（热心公益），是对全体社会成员的道德要求。
:::
