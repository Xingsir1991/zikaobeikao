# 第二节 吸收借鉴优秀道德成果

> 教材第 152–168 页 · 传承层——传统美德、革命道德、人类文明成果。

本节阐述社会主义道德的三大来源：**中华传统美德**（民族根基）、**中国革命道德**（红色基因）、**人类文明优秀道德成果**（开放借鉴）。

---

## 一、核心论断（必背）

> 中华传统美德是中华民族历史流传下来的、**具有当代价值的道德遗产**。

> 中国革命道德是中国共产党人、人民军队、一切先进分子和人民群众在**新民主主义革命和社会主义革命、建设、改革**中形成的**优良道德**。

> 要推动中华优秀传统文化**创造性转化、创新性发展**。

---

## 二、中华传统美德（必背）

### 1. 传统美德的基本精神（极高频考点）

::: danger 必背：中华传统美德的基本精神（六条）
1. **注重整体利益、国家利益和民族利益**——强调对社会、民族、国家的责任意识和奉献精神（"天下兴亡，匹夫有责"）
2. **推崇"仁爱"原则，注重以和为贵**——提倡"己所不欲，勿施于人"
3. **提倡人伦价值，重视道德义务**——五伦关系
4. **追求精神境界，把道德理想的实现看作人生最高需要**——重义轻利
5. **强调道德修养，注重道德践履**——"修身齐家治国平天下"
6. **重视道德人格的养成**——注重道德理想人格的塑造
:::

::: tip 记忆口诀
**"整体—仁爱—人伦—境界—修养—人格"**——六个维度构成传统美德的基本精神。
:::

### 2. 传统美德的当代价值

| 维度 | 价值 |
| --- | --- |
| **文化根脉** | 是中华民族的精神基因 |
| **道德资源** | 是社会主义道德的源头活水 |
| **文化自信** | 是文化自信的重要基础 |

### 3. 创造性转化和创新性发展（必背）

::: danger 必背：双创方针
对待中华传统美德，要坚持：
- **创造性转化**——按时代特点和要求，对陈旧表现形式加以改造
- **创新性发展**——按时代的新进步新进展，对中华传统美德的内涵加以补充、拓展、完善
:::

::: warning 易错点
**对待传统美德的态度**：既不是全盘肯定（守旧主义），也不是全盘否定（民族虚无主义），而是**创造性转化、创新性发展**——这是新时代的正确态度。
:::

---

## 三、中国革命道德（必背）

### 1. 革命道德的内涵

::: danger 必背：中国革命道德
**中国革命道德**是指中国共产党人、人民军队、一切先进分子和人民群众在**新民主主义革命和社会主义革命、建设、改革**中形成的**优良道德**。

它是马克思主义与中国革命建设改革伟大实践相结合的产物。
:::

### 2. 革命道德的主要内容（理解）

::: danger 必背：中国革命道德的主要内容
1. **为实现社会主义和共产主义理想而奋斗**
2. **全心全意为人民服务**
3. **始终把革命利益放在首位**
4. **树立社会新风，建立新型人际关系**
5. **修身自律，保持节操**
:::

### 3. 革命道德的当代价值（高频考点）

::: danger 必背：革命道德的当代价值
1. 有利于**坚定理想信念**——为实现共产主义远大理想而奋斗
2. 有利于**培育和践行社会主义核心价值观**
3. 有利于**加强社会主义道德建设**
4. 有利于**凝聚实现中华民族伟大复兴的强大精神力量**
:::

### 易混辨析：传统美德 vs 革命道德

| 类别 | 中华传统美德 | 中国革命道德 |
| --- | --- | --- |
| **时间** | 历史流传（古代至近代） | 新民主主义革命以来 |
| **主体** | 中华民族世代创造 | 中国共产党领导下的人民 |
| **核心** | 仁爱、忠孝、整体利益 | 全心全意为人民服务 |
| **关系** | 是革命道德的文化根基 | 是传统美德的升华 |

## 四、人类文明优秀道德成果（理解）

::: danger 必背：吸收借鉴人类文明优秀道德成果
社会主义道德建设要**积极吸收借鉴人类文明优秀道德成果**：
- 坚持**以我为主、为我所用**的原则
- 反对**全盘西化**和**民族虚无主义**
- 取其精华、去其糟粕
:::

::: warning 易错点
**借鉴人类文明成果的原则是"以我为主、为我所用"**——不是全盘西化（盲目照搬），也不是封闭排外。
:::

---

## 五、三大道德来源的关系（理解）

::: danger 必背：社会主义道德的三大来源
社会主义道德形成和发展于中国特色社会主义实践，其来源是：
1. **中华传统美德**（民族根基）
2. **中国革命道德**（红色基因）
3. **人类文明优秀道德成果**（开放借鉴）

三者辩证统一于社会主义道德建设实践。
:::

### 预测题一（单选）

**对待中华传统美德，应当坚持的方针是（　）**
A. 全盘继承，原封不动
B. 全盘否定，民族虚无
C. 创造性转化、创新性发展
D. 全盘西化，外为中用

> **答案：C**
> **解析**：对待中华传统美德，要坚持**创造性转化、创新性发展**。A 是守旧主义，B 是民族虚无主义，D 是全盘西化——都是错误态度。

### 预测题二（简答）

**简述中国革命道德的当代价值。**

> **【主体】** 中国革命道德是新时代凝聚共识、激励奋进的强大精神力量，其当代价值体现在四个方面：
>
> **第一，有利于坚定理想信念。** 革命道德核心是为实现社会主义和共产主义理想而奋斗，有助于新时代共产党人坚定理想信念。
>
> **第二，有利于培育和践行社会主义核心价值观。** 革命道德是社会主义核心价值观的重要来源。
>
> **第三，有利于加强社会主义道德建设。** 革命道德是社会主义道德的红色基因。
>
> **第四，有利于凝聚实现中华民族伟大复兴的强大精神力量。** 革命道德激励全国人民投身民族复兴伟业。
>
> **【总结】** 中国革命道德是当代中国最宝贵的精神财富之一，弘扬革命道德对新时代道德建设和民族复兴具有重要意义。

### 易错点提醒

::: warning 本节三大易错点
1. **对待传统美德的方针=创造性转化、创新性发展**（不是"全盘继承"或"全盘否定"）。
2. **革命道德的当代价值有四方面**：坚定信念、践行核心价值观、加强道德建设、凝聚复兴力量——不要遗漏。
3. **借鉴人类文明的原则=以我为主、为我所用**（不是"全盘西化"或"封闭排外"）。
:::
