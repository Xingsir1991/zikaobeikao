## 第一节 中国精神是兴国强国之魂

### 一、崇尚精神是中华民族的优秀传统 ★★★★

中华民族在 5000 多年的历史长河中，不仅创造了光辉灿烂的文化，而且形成了崇尚精神的优秀传统。

**中华民族崇尚精神的传统主要表现**：

1. **重精神而轻物质**——先义后利、重义轻利，强调精神境界的高尚
2. **重理想而轻现实**——追求"天下为公"的大同理想，胸怀远大抱负
3. **重道德而轻功利**——推崇"修身齐家治国平天下"，重视人格完善
4. **对物质与精神关系的独到理解**——"不义而富且贵，于我如浮云"

::: tip 经典表述
中华民族是**注重精神生活、追求精神境界**的民族。中国传统文化强调"**自强不息、厚德载物**"，把精神追求看得比物质享受更重要。
:::

### 二、中国精神的丰富内涵 ★★★★★（必考）

**中国精神**＝以**爱国主义**为核心的**民族精神**＋以**改革创新**为核心的**时代精神**的统一。

#### ① 民族精神及其核心

- **民族精神**：一个民族在长期共同生活和社会实践中形成的，为本民族大多数成员所认同的**价值取向、思维方式、道德规范、精神气质**的总和，是一个民族赖以生存和发展的精神支柱。
- 民族精神的核心是**爱国主义**。

**民族精神的基本内涵——"四个伟大精神"**（习近平概括）：

1. **伟大创造精神**——创造了悠久历史、灿烂文化和"四大发明"
2. **伟大奋斗精神**——历经磨难而不衰，自强不息、艰苦奋斗
3. **伟大团结精神**——各民族相依共存、紧密团结，多元一体
4. **伟大梦想精神**——盘古开天、精卫填海等神话反映的追梦精神

#### ② 时代精神及其核心

- **时代精神**：在新的历史条件下形成和发展的，体现民族特质、顺应时代潮流的思想观念、行为方式、价值取向、精神风貌和社会风尚的总和。
- 时代精神的核心是**改革创新**。
- 时代精神还表现为：与时俱进、开放包容、求真务实。

::: warning 易混辨析（最高频陷阱）
- 民族精神的核心＝**爱国主义**
- 时代精神的核心＝**改革创新**

两者配对相反，是选择题最爱考的陷阱，必须记牢对应关系。
:::

### 三、中国共产党是中国精神的忠实继承者和坚定弘扬者 ★★★★

中国共产党成立以来，始终是**中国精神的忠实继承者和坚定弘扬者**。

#### ① 党在不同时期铸就的精神丰碑

| 时期 | 典型精神 |
| --- | --- |
| 革命时期 | 井冈山精神、长征精神、延安精神、西柏坡精神、红岩精神 |
| 建设时期 | 抗美援朝精神、"两弹一星"精神、雷锋精神、焦裕禄精神、铁人精神 |
| 改革开放新时期 | 特区精神、抗洪精神、载人航天精神、抗震救灾精神 |
| 新时代 | 脱贫攻坚精神、抗疫精神、丝路精神、北斗精神、科学家精神 |

::: tip 关键表述
**伟大建党精神**是中国共产党的**精神之源**——坚持真理、坚守理想，践行初心、担当使命，不怕牺牲、英勇斗争，对党忠诚、不负人民。
:::

#### ② 中国共产党人的精神谱系

党的伟大精神形成了一个**精神谱系**，凝结着中国共产党人的**初心和使命**，是党和人民取得一切成就的**强大精神力量**。

### 四、实现中国梦必须弘扬中国精神 ★★★★

中国精神是**兴国之魂、强国之魂**：

1. 中国精神是**凝聚中国力量的精神纽带**——把亿万人民团结在一起
2. 中国精神是**激发创新创造的精神动力**——推动社会生产力发展
3. 中国精神是**推进复兴伟业的精神支撑**——实现民族复兴的精神根基

::: tip 必背表述
**实现中国梦必须弘扬中国精神**。这就是以爱国主义为核心的民族精神，以改革创新为核心的时代精神。这种精神是凝心聚力的兴国之魂、强国之魂。
:::

## 第二节 做新时代的忠诚爱国者

爱国主义是中华民族精神的核心。在当代中国，做忠诚坚定的爱国者，必须做到"**四个统一**"。

### 一、坚持爱国爱党爱社会主义相统一 ★★★★★（必考）

#### ① 新时代爱国主义最鲜明的论断

::: warning 核心表述（必背）
**在当代中国，爱国和爱党、爱社会主义是高度统一的。**
这是新时代爱国主义最鲜明的论断，三者不可拆分、不可割裂。
:::

理由（"三个密不可分"）：
1. 在新中国，**祖国的命运与党的命运、社会主义的命运密不可分**
2. 中国共产党是领导中国特色社会主义事业的**核心力量**
3. 中国特色社会主义是实现中华民族伟大复兴的**必由之路**

> **关键理解**：爱国首先体现在热爱**中国共产党领导的社会主义中国**。脱离党的领导、脱离社会主义道路的"爱国"，在当代中国是空洞的、错误的。

#### ② 爱国主义的三层含义 ★★★★★

爱国主义体现了个人对祖国的依存关系，是归属感、认同感、尊严感与荣誉感的统一：

1. **爱祖国的大好河山**（国土——"禾苗离土即枯"）
2. **爱自己的骨肉同胞**（人民——血肉相连的同胞情）
3. **爱祖国的灿烂文化**（文化——文化与国家民族命运紧密相连）

> 爱国主义是**历史范畴**，在不同时代有不同内涵，但**核心**始终是**对祖国的忠诚和热爱**。

### 二、维护祖国统一和民族团结 ★★★★

#### ① 维护祖国统一

- 祖国统一是中华民族的**根本利益所在**，是实现民族复兴的**基本前提**
- 反对一切**分裂国家的言行**，旗帜鲜明地维护国家主权和领土完整
- 解决台湾问题、实现祖国完全统一，是全体中华儿女的共同愿望

#### ② 维护民族团结

- **民族团结**是国家统一、繁荣发展的重要保证
- 牢固树立**中华民族共同体意识**，促进各民族**交往交流交融**
- 反对**大汉族主义**和**地方民族主义**，旗帜鲜明地反对民族分裂活动

::: tip 必背表述
铸牢**中华民族共同体意识**，是新时代党的民族工作的主线。各民族要像石榴籽一样紧紧抱在一起。
:::

### 三、尊重和传承中华民族历史文化 ★★★★★

#### ① 中华优秀传统文化是民族的根和魂

- **中华传统美德**是中华优秀传统文化的**精髓**，主要内容以"**仁、义、礼、智、信**"为核心
- 中华文化是中华民族生生不息、发展壮大的**丰厚滋养**

中华传统美德的基本精神：

1. **注重整体利益、国家利益、民族利益**（"天下兴亡，匹夫有责""先天下之忧而忧"）
2. **推崇仁爱原则**（"己所不欲，勿施于人""仁者爱人"）
3. **提倡人伦价值**（"父慈子孝、兄友弟恭、朋友有信"）
4. **追求精神境界**（"修身齐家治国平天下"）
5. **强调道德修养**（重践履、慎独、自律）

#### ② 传统美德的创造性转化和创新性发展 ★★★★★

习近平总书记关于传承发展中华优秀传统文化的重要方针：

- **创造性转化**：按时代特点和要求，对传统美德中陈旧的表现形式加以改造，赋予其新的时代内涵和现代表达形式。
- **创新性发展**：按时代的新进步新进展，对中华传统美德进行补充、拓展、完善，增强其影响力和感召力。

::: warning 易混辨析
- **创造性转化**重在"**改**"——改造旧的形式，赋予新内涵
- **创新性发展**重在"**新**"——补充、拓展、完善，发展出新的内容
:::

基本态度："**取其精华、去其糟粕**"，"**古为今用、推陈出新**"，既要反对**历史虚无主义**（全盘否定传统），也要反对**守旧主义/复古主义**（全盘照搬）。

#### ③ 树立正确历史观

- 树立**正确历史观、民族观、国家观、文化观**
- 尊重中华民族历史，**反对历史虚无主义**
- 坚定**文化自信**，传承中华优秀传统文化

### 四、坚持立足中国又面向世界 ★★★★

#### ① 立足中国

- 把弘扬爱国主义精神与**扩大对外开放**结合
- 维护国家**主权、安全、发展利益**

#### ② 面向世界

- 树立**人类命运共同体**理念，关注人类前途命运
- 尊重各国的**历史文化、社会制度和发展道路**
- 倡导**和平、发展、公平、正义、民主、自由**的全人类共同价值

::: tip 关键表述
新时代的爱国者，既要**立足中国**、热爱祖国、维护国家利益；又要**面向世界**，树立世界眼光，把握时代大势，积极推动构建**人类命运共同体**。
:::

#### ③ 延伸考点：总体国家安全观 ★★★★★

::: warning 总体国家安全观"五个层次"（必背对应关系）
| 层次 | 内容 |
| --- | --- |
| **宗旨** | 以**人民安全**为宗旨 |
| **根本** | 以**政治安全**为根本 |
| **基础** | 以**经济安全**为基础 |
| **保障** | 以**军事、文化、社会安全**为保障 |
| **依托** | 以**促进国际安全**为依托 |
:::

大学生要求：维护国家安全是公民的**神圣义务**；要树立**国家安全底线思维**，自觉履行维护国家安全、荣誉和利益的义务，自觉同一切危害国家安全的行为作斗争。

## 第三节 让改革创新成为青春远航的动力

### 一、改革开放是当代中国的显著特征 ★★★★

#### ① 改革开放是决定当代中国命运的关键一招

- **改革开放**是党的一次**伟大觉醒**，是决定当代中国命运的**关键一招**
- 改革开放 40 多年的实践证明：**只有改革开放才能发展中国、发展社会主义、发展马克思主义**
- 改革开放是当代中国**最显著的特征、最壮丽的气象**

#### ② 改革开放是发展中国特色社会主义的必由之路

- 改革开放推动我国发生**历史性变化**、取得**历史性成就**
- 使中华民族迎来了从**站起来、富起来到强起来**的伟大飞跃
- 中国特色社会主义在改革开放中**不断焕发生机活力**

::: tip 关键表述
**改革开放是决定当代中国命运的关键一招，也是决定实现"两个一百年"奋斗目标、实现中华民族伟大复兴的关键一招。**
:::

### 二、改革创新是新时代的迫切要求 ★★★★★

#### ① 改革创新是时代精神的核心

- **改革创新**是时代精神的**核心**，是进一步解放和发展社会生产力的**必然要求**
- 创新是**民族进步的灵魂**，是一个国家兴旺发达的**不竭源泉**
- 创新是引领发展的**第一动力**

#### ② 改革创新是新时代的迫切要求

- **建设社会主义创新型国家**的迫切需要
- **全面深化改革、推动社会全面发展**的重要动力
- **实现中华民族伟大复兴**的必然要求

::: warning 必背表述
**实施创新驱动发展战略**，把创新摆在国家发展全局的**核心位置**。**科技是第一生产力、人才是第一资源、创新是第一动力**。
:::

#### ③ 大力弘扬改革创新精神

改革创新精神表现为：**与时俱进、锐意进取、勤于探索、勇于实践**。

### 三、做改革创新生力军 ★★★★★（论述题高频）

当代大学生要树立**改革创新的自觉意识**，做改革创新的**生力军**。

#### ① 树立突破陈规陋习的创新意识

- 培养**创新思维**，不盲从权威、不墨守成规
- 增强创新自信，敢于质疑、勇于探索
- 树立与时代发展相适应的**新观念**

#### ② 培养改革创新的责任感

- 把个人理想融入国家民族事业，把**创新创造**作为人生追求
- 以**国家需求**为导向，勇攀科技高峰、攻克核心关键技术
- 立志成为**创新型、复合型人才**

#### ③ 增强改革创新的能力本领

- **扎实系统学习**——打牢专业基础，掌握科学文化知识
- **培养创造性思维**——训练求异思维、发散思维、批判性思维
- **积极投身实践**——在实验、科研、竞赛、社会实践中锻炼创新能力

::: tip 大学生做创新生力军"三步走"
**意识（树立创新意识）→ 责任（培养创新责任感）→ 本领（增强创新能力）**

只有三方面齐备，才能真正成为改革创新的生力军，让改革创新成为青春远航的动力。
:::

## 易错点提醒

::: warning 易错雷区
1. **核心配对最易混**：民族精神的核心＝**爱国主义**；时代精神的核心＝**改革创新**。出题人常反着配对来考。
2. **"四个伟大精神"易漏易混**：是"创造、奋斗、团结、梦想"四种精神，不要写成"伟大建党精神/红船精神"等其他表述。
3. **爱国主义"三层含义"易漏**：要答全"爱大好河山＋爱骨肉同胞＋爱灿烂文化"，不要只写"爱祖国"。
4. **新时代爱国主义"三个相统一"**：**爱国、爱党、爱社会主义**三者不可拆分，是新时代最鲜明的论断。
5. **总体国家安全观"宗旨/根本/基础"三组对应是选择题重灾区**：
   - 宗旨＝人民安全（不是"政治"）
   - 根本＝政治安全（不是"经济"）
   - 基础＝经济安全（不是"军事"）
6. **创造性转化 vs 创新性发展**：转化重在"改"（改造旧形式、赋新内涵）；发展重在"新"（补充、拓展、完善）。
7. **传统美德的对待态度**：既反对**历史虚无主义**（全盘否定），也反对**复古主义/守旧主义**（全盘照搬）。
8. **"关键一招"的表述**：改革开放是决定当代中国命运的**关键一招**，也是决定实现"两个一百年"奋斗目标和民族复兴的**关键一招**。
9. **"三个第一"易混**：科技是第一**生产力**、人才是第一**资源**、创新是第一**动力**——三者的对应不要颠倒。
:::

## 模拟题（含标准答案）

### （一）单项选择题（3 题）

**1. 民族精神的核心是（　）**
A. 改革创新　　B. 爱国主义　　C. 艰苦奋斗　　D. 伟大梦想

> **标准答案：B**
> 解析：以爱国主义为核心的民族精神和以改革创新为核心的时代精神构成中国精神。

**2. 总体国家安全观以（　）为根本。**
A. 人民安全　　B. 经济安全　　C. 政治安全　　D. 军事安全

> **标准答案：C**
> 解析：以人民安全为宗旨，以政治安全为根本，以经济安全为基础。

**3. 时代精神的核心是（　）**
A. 爱国主义　　B. 改革创新　　C. 与时俱进　　D. 开放包容

> **标准答案：B**
> 解析：改革创新是时代精神的核心，是进一步解放和发展社会生产力的必然要求。

**4. 决定当代中国命运的"关键一招"是（　）**
A. 科技创新　　B. 改革开放　　C. 依法治国　　D. 文化自信

> **标准答案：B**
> 解析：改革开放是党的一次伟大觉醒，是决定当代中国命运的关键一招，也是决定实现"两个一百年"奋斗目标、实现中华民族伟大复兴的关键一招。

### （二）简答题（1 题）

**5. 为什么说"在当代中国，爱国和爱党、爱社会主义是高度统一的"？**

> **标准答案要点**：
> 在当代中国，爱国和爱党、爱社会主义是**高度统一**的，这是新时代爱国主义最鲜明的论断。理由如下：
> （1）在新中国，**祖国的命运与党的命运、社会主义的命运密不可分**。国家的前途、党的命运、社会主义的事业是内在一致的。
> （2）**中国共产党是领导中国特色社会主义事业的核心力量**。没有共产党就没有新中国，党的领导是中国特色社会主义最本质的特征。
> （3）**中国特色社会主义是实现中华民族伟大复兴的必由之路**。只有社会主义才能救中国，只有中国特色社会主义才能发展中国。
>
> 因此，在当代中国，爱国首先体现在热爱**中国共产党领导的社会主义中国**，爱国、爱党、爱社会主义三者不可拆分、不可割裂。

### （三）论述题（2 题）

**6. 论述新时代爱国主义的时代要求，并结合大学生实际，谈谈如何做忠诚坚定的爱国者。**

> **标准答案要点**：
> **（一）爱国主义的时代要求（理论部分）**
> 爱国主义是中华民族精神的核心，是历史的、具体的。在当代中国，做忠诚坚定的爱国者必须做到"四个统一"：
> 1. **坚持爱国和爱党、爱社会主义相统一**。这是新时代最鲜明的论断，在当代中国，爱国首先体现在热爱中国共产党领导的社会主义中国。
> 2. **维护祖国统一和民族团结**。这是爱国主义的重要体现，也是中华民族伟大复兴的必然要求。
> 3. **尊重和传承中华民族历史和文化**。要树立正确历史观，传承中华优秀传统文化，反对历史虚无主义。
> 4. **立足民族又面向世界**。把弘扬爱国主义精神与扩大对外开放结合起来，倡导构建人类命运共同体。
>
> **（二）大学生如何做忠诚坚定的爱国者（结合实际）**
> 1. **胸怀爱国之情**：树立国家荣誉感和民族自豪感，自觉维护国家主权、安全和发展利益。
> 2. **确立强国之志**：把个人理想融入中华民族伟大复兴的中国梦，立志成长为担当民族复兴大任的时代新人。
> 3. **实践报国之行**：刻苦学习专业知识，提升综合素养；弘扬中国精神，传承中华优秀传统文化；树立总体国家安全观，自觉履行维护国家安全的义务；到祖国和人民最需要的地方去建功立业。
>
> **（三）总结**：新时代大学生只有把爱国、爱党、爱社会主义统一起来，把个人前途与国家命运结合起来，才能成为忠诚坚定的爱国者，为实现中华民族伟大复兴贡献青春力量。

**7. 联系实际，论述大学生如何做改革创新的生力军，让改革创新成为青春远航的动力。**

> **标准答案要点**：
> **（一）理论：改革创新的时代意义**
> 改革开放是决定当代中国命运的**关键一招**，改革创新是时代精神的**核心**，是引领发展的**第一动力**。新时代大学生肩负民族复兴重任，必须做改革创新的生力军。
>
> **（二）大学生做改革创新生力军（三个方面）**
> 1. **树立突破陈规陋习的创新意识**：培养创新思维，不盲从权威、不墨守成规，增强创新自信，敢于质疑、勇于探索。
> 2. **培养改革创新的责任感**：把个人理想融入国家民族事业，以国家需求为导向，立志成为创新型、复合型人才。
> 3. **增强改革创新的能力本领**：扎实系统学习打牢基础；培养求异思维、发散思维、批判性思维；积极投身实验、科研、竞赛、社会实践。
>
> **（三）结合实际**：联系自身专业学习，如参与科研项目（增强能力）、关注"卡脖子"技术（培养责任感）、敢于提出新观点（树立意识），在知行合一中让改革创新成为青春远航的动力。
>
> **（四）总结**：青年是改革创新的生力军。当代大学生应把创新作为人生追求，在奋斗中成就出彩人生，为实现中华民族伟大复兴注入青春能量。
