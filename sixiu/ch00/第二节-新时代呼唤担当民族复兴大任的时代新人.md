# 第二节 新时代呼唤担当民族复兴大任的时代新人

> 教材第 7–9 页 · 交代全书的育人目标——"时代新人"。

本节在交代"新时代"方位之后，进一步回答"新时代需要什么样的人"。**大学生是新时代的生力军，是民族复兴大任的担当者**——这是思修教材对青年学生角色定位的总判断。

---

## 一、核心论断（必背）

> **青年一代有理想、有本领、有担当，国家就有前途，民族就有希望。**

> **时代新人**要有理想、有本领、有担当。大学生要努力成为**担当民族复兴大任的时代新人**。

---

## 二、青年是国家的未来、民族的希望（必背）

::: danger 必背：青年的历史使命
青年兴则国家兴，青年强则国家强。青年一代**有理想、有本领、有担当**，国家就有前途，民族就有希望。**中华民族伟大复兴的中国梦终将在一代代青年的接力奋斗中变为现实。**
:::

### 青年与时代的辩证关系

| 命题 | 含义 |
| --- | --- |
| **时代塑造青年** | 时代为青年提供施展才华的舞台（新时代是青年发展最好的时期） |
| **青年塑造时代** | 青年是推动历史前进的生力军（民族复兴要靠青年接力奋斗） |

::: tip 习近平给青年的寄语
"中国梦是历史的、现实的，也是未来的；是我们这一代的，更是青年一代的。**中华民族伟大复兴的中国梦终将在一代代青年的接力奋斗中变为现实。**"——这是论述题必备金句。
:::

---

## 三、时代新人的内涵——"四个要求"（极高频考点）

::: danger 必背：时代新人的"四个要求"
新时代青年要努力成为：
1. **有理想**的新时代青年
2. **敢担当**的新时代青年
3. **能吃苦**的新时代青年
4. **肯奋斗**的新时代青年
:::

### "四个要求"的逐条解读

| 要求 | 内涵 | 对应的青年姿态 |
| --- | --- | --- |
| **有理想** | 树立共产主义远大理想和中国特色社会主义共同理想 | 立志高远、信仰坚定 |
| **敢担当** | 把个人前途同国家命运、民族复兴紧密结合 | 责任在肩、知重负重 |
| **能吃苦** | 不怕艰苦、磨砺意志，主动到基层一线经受锻炼 | 艰苦奋斗、意志坚定 |
| **肯奋斗** | 把奋斗作为青春最亮丽的底色，知行合一 | 脚踏实地、自强不息 |

::: tip 记忆口诀
**"理想—担当—吃苦—奋斗"**——四个词的递进关系：先立志（有理想），再扛事（敢担当），扛事要耐住磨炼（能吃苦），最后落实到行动（肯奋斗）。
:::

### 易混辨析：时代新人 vs "四有"新人 vs "三有"青年

| 概念 | 内涵 | 来源 |
| --- | --- | --- |
| **"四有"新人** | 有理想、有道德、有文化、有纪律 | 邓小平 1985 年提出 |
| **"三有"青年** | 有理想、有本领、有担当 | 习近平在党的十九大报告中提出 |
| **时代新人"四要求"** | 有理想、敢担当、能吃苦、肯奋斗 | 新时代对青年的最新要求 |

::: warning 易错点
**不要把"时代新人四要求"与"四有新人"混为一谈**。"四有"（理想道德文化纪律）是 1985 年邓小平提出的老提法；"四要求"（理想担当吃苦奋斗）是新时代最新提法——两者的内容完全不同，考题常设干扰项。
:::

## 四、做时代新人的现实要求（理解）

教材对大学生做时代新人提出三个层次的要求：

| 层次 | 要求 |
| --- | --- |
| **认知层** | 深刻认识新时代，把握历史使命 |
| **价值层** | 树立正确的世界观、人生观、价值观 |
| **实践层** | 在实现中国梦的伟大实践中放飞青春梦想 |

::: tip 联系实际的论述角度
回答"如何做时代新人"的论述题，可以按"认知—价值—实践"三层展开：
1. 认知上：深刻理解新时代的内涵和青年的历史使命；
2. 价值上：坚定理想信念、树立正确人生观价值观；
3. 实践上：勤学、修德、明辨、笃实，把青春融入民族复兴大业。
:::

## 五、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 时代新人"四要求"、青年"三有"、"青年兴则国家兴" |
| **多选** | 时代新人的四个要求、青年与时代的辩证关系 |
| **简答** | 时代新人的内涵是什么？为什么说青年是国家的未来？ |
| **论述** | 联系实际谈大学生如何成为担当民族复兴大任的时代新人 |

### 预测题一（单选）

**新时代对青年的最新要求，即时代新人应当具备的素质是（　）**
A. 有理想、有道德、有文化、有纪律
B. 有理想、有本领、有担当
C. 有理想、敢担当、能吃苦、肯奋斗
D. 有道德、有文化、有纪律、有本领

> **答案：C**
> **解析**：**时代新人"四要求"**= 有理想、敢担当、能吃苦、肯奋斗（最新提法）。A 是邓小平 1985 年提出的"四有新人"；B 是习近平在党的十九大报告中提出的青年"三有"（有理想、有本领、有担当）。三者不要混淆。

### 预测题二（论述）

**联系实际，谈谈当代大学生如何成为担当民族复兴大任的时代新人。**

> **【破题】** 把握"时代新人"的四要求（理想—担当—吃苦—奋斗），结合大学生实际逐层展开。
>
> **【主体】**
> **第一，做"有理想"的新时代青年。** 树立共产主义远大理想和中国特色社会主义共同理想，把个人理想融入民族复兴的伟大事业，用理想之光指引人生方向。
>
> **第二，做"敢担当"的新时代青年。** 把个人前途同国家命运紧密结合，敢于承担民族复兴的历史责任，到祖国和人民最需要的地方去建功立业。
>
> **第三，做"能吃苦"的新时代青年。** 不怕艰难困苦，主动到基层一线经受锻炼，磨砺坚强意志，在艰苦环境中成长成才。
>
> **第四，做"肯奋斗"的新时代青年。** 把奋斗作为青春最亮丽的底色，勤学修德、明辨笃实，知行合一、脚踏实地，在实现中国梦的伟大实践中放飞青春梦想。
>
> **【总结】** 青年兴则国家兴，青年强则国家强。大学生要珍惜新时代、奋进新征程，以有理想、敢担当、能吃苦、肯奋斗的姿态，扛起民族复兴大任，让青春在奉献祖国中绽放绚丽光彩。

### 易错点提醒

::: warning 本节三大易错点
1. **"时代新人四要求"≠"四有新人"≠"三有青年"**：四要求=理想/担当/吃苦/奋斗；四有=理想/道德/文化/纪律；三有=理想/本领/担当。三套提法不同。
2. **青年"三有"中的"本领"**：有理想、有本领、有担当——容易漏掉"有本领"，或与"有道德"混淆。
3. **"敢担当"vs"肯奋斗"**：担当强调责任意识（敢扛事），奋斗强调行动实践（肯干事）——两者一个偏认识一个偏行动，不能合并。
:::
