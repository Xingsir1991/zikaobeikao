# 第三节 不断提升思想道德素质和法治素养

> 教材第 10–12 页 · 点明全书两条主线——**思想道德**与**法治**。

本节是绪论的收束，也是全书的总纲。它把"成为时代新人"的总目标分解为两条具体主线——**思想道德素质**和**法治素养**，分别对应全书第 1-5 章和第 6 章。

---

## 一、核心论断（必背）

> 思想道德素质和法治素养，是**时代新人担当民族复兴大任必须具备的思想基础**。

> 坚持依法治国和以德治国相结合，是**中国特色社会主义法治道路的鲜明特色**。

---

## 二、思想道德素质的内涵（必背）

::: danger 必背：思想道德素质的内涵
**思想道德素质**是人们的思想观念、政治立场、价值取向、道德情操等方面的基本品质。

它决定着人们在日常生活中**想什么、做什么、追求什么**，是个人成长发展的重要基础。
:::

### 思想道德素质的构成（理解）

| 维度 | 内容 | 对应章节 |
| --- | --- | --- |
| **思想素质** | 思想观念、政治立场、价值取向 | 绪论 + 第一、二、四章 |
| **道德素质** | 道德认知、道德情感、道德意志、道德行为 | 第五、三章 |
| **精神素质** | 理想信念、民族精神、时代精神 | 第二、三章 |

---

## 三、法治素养的内涵（必背）

::: danger 必背：法治素养的内涵
**法治素养**是人们对法律规范体系的认识、理解、运用和信奉程度。

它包括**法治思维、法律原则、法律知识、法律规则、法律程序、法律义务和法律责任**等。
:::

### 法治素养的四个要素（理解，对应第六章）

| 要素 | 内涵 |
| --- | --- |
| **尊法** | 内心敬畏法律，信仰法治 |
| **学法** | 学习法律知识，了解法律体系 |
| **守法** | 自觉遵守法律，依法办事 |
| **用法** | 运用法律维护权益、解决纠纷 |

::: tip 法治素养的"尊学守用"
**尊法、学法、守法、用法**——四者由内而外、由认知到实践，是法治素养的完整链路，也是第六章第四节的核心考点。
:::

---

## 四、思想道德素质与法治素养的关系（极高频考点）

::: danger 必背：德治与法治的关系
- **思想道德素质**为法治素养提供**价值基础**和**道义支撑**
- **法治素养**为思想道德素质提供**制度保障**和**法律支撑**
- 二者**相辅相成、相互促进**，统一于培育时代新人的全过程
:::

### 德治与法治的关系表

| 关系 | 道德（德治） | 法律（法治） |
| --- | --- | --- |
| **作用方式** | 内心信念、社会舆论 | 国家强制力 |
| **作用范围** | 调整范围更广（含法律管不到的领域） | 调整基本社会关系 |
| **特点** | 自律、柔性 | 他律、刚性 |
| **功能** | 为法治提供价值基础 | 为德治提供制度保障 |

::: warning 易错点
**法治和德治不能割裂、不能对立**。常见错误观点有两种：
1. "法治万能论"——认为有了法律就够了，忽视道德的基础作用；
2. "德治万能论"——认为有了道德就够了，忽视法律的强制保障。
正确观点：**依法治国与以德治国相结合**，二者相辅相成。
:::

## 七、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 思想道德素质/法治素养的内涵、法治素养的"尊学守用" |
| **多选** | 思想道德素质的构成、法治素养的要素、德治与法治的关系 |
| **简答** | 思想道德素质和法治素养的内涵？两者是什么关系？ |
| **论述** | 大学生为什么要提升思想道德素质和法治素养？ |

### 预测题一（单选）

**下列关于思想道德素质和法治素养关系的表述，正确的是（　）**
A. 法治素养决定思想道德素质，二者互不相关
B. 思想道德素质是法治素养的价值基础，二者相辅相成
C. 法治素养可替代思想道德素质，二者作用相同
D. 思想道德素质与法治素养相互对立，不能并存

> **答案：B**
> **解析**：思想道德素质和法治素养**相辅相成、相互促进**——思想道德素质为法治素养提供**价值基础和道义支撑**，法治素养为思想道德素质提供**制度保障和法律支撑**。A 错（互不相关），C 错（不可替代），D 错（不是对立）。

### 预测题二（简答）

**简述思想道德素质和法治素养的内涵，并说明二者关系。**

> **【思想道德素质】** 是人们的**思想观念、政治立场、价值取向、道德情操**等方面的基本品质，决定人们在日常生活中想什么、做什么、追求什么。
>
> **【法治素养】** 是人们对**法律规范体系**的认识、理解、运用和信奉程度，包括法治思维、法律原则、法律知识、法律义务和责任等。
>
> **【二者关系】** 二者相辅相成、相互促进，统一于培育时代新人的全过程：
> - 思想道德素质为法治素养提供**价值基础和道义支撑**；
> - 法治素养为思想道德素质提供**制度保障和法律支撑**；
> - 必须坚持**依法治国与以德治国相结合**。

### 易错点提醒

::: warning 本节三大易错点
1. **法治素养的内容不只是"法律知识"**：还包括法治思维、法律原则、规则、程序、义务、责任等，是一个综合体系。
2. **德治与法治"相互结合"而非"相互替代"**：两者各有功能，法治是刚性保障、德治是柔性基础。
3. **法治素养的"尊学守用"顺序**：尊法（内心）→ 学法（认知）→ 守法（行动）→ 用法（应用），不能颠倒。
:::
