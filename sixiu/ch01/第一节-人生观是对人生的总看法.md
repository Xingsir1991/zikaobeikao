# 第一节 人生观是对人生的总看法

> 教材第 13–22 页 · 思想道德修养的**起点**——人的本质与人生观。

本节是全章的理论基石。它先回答"人是什么"（人的本质），再回答"如何认识人生"（人生观及其三要素）。**理解人的本质是一切社会关系的总和，是树立正确人生观的前提。**

---

## 一、核心论断（必背）

> **人的本质不是单个人所固有的抽象物，在其现实性上，它是一切社会关系的总和。**——马克思

> 人生观是人们在实践中形成的对于**人生目的、人生态度、人生价值**等问题的根本看法。

---

## 二、正确认识人的本质（极高频考点）

### 1. 人的双重属性

| 属性 | 含义 |
| --- | --- |
| **自然属性** | 人作为生物体的生理本能（饮食、繁衍等） |
| **社会属性** | 人在社会关系中获得的规定性（人的本质所在） |

::: danger 必背：人的本质在于社会属性
人的**本质**不是自然属性，而是**社会属性**。人是社会的人，离开社会，人就无法成为真正的人（狼孩的例子）。
:::

### 2. 马克思的经典论断（必背，原话）

::: danger 必背：人的本质的定义
马克思在《关于费尔巴哈的提纲》中指出："**人的本质不是单个人所固有的抽象物，在其现实性上，它是一切社会关系的总和。**"
:::

### 3. "一切社会关系的总和"三层含义（理解）

| 层次 | 含义 |
| --- | --- |
| **不是抽象的** | 人的本质不能脱离具体社会历史条件 |
| **不是先天的** | 人的本质是在后天社会实践中形成的 |
| **不是单一的** | 人的本质由多重社会关系（经济、政治、法律、道德、家庭等）共同规定 |

::: tip 记忆口诀
**"非抽象、非先天、非单一"**——三个否定，划清与"人性抽象论""天赋人权论""单一人性论"的界限。
:::

### 易混辨析：马克思主义人性观 vs 资产阶级人性观

| 观点 | 马克思主义 | 资产阶级 |
| --- | --- | --- |
| **本质来源** | 一切社会关系的总和（社会性） | 单个人固有的抽象物（自然性） |
| **是否变化** | 随社会关系变化而变化 | 永恒不变、天赋的 |
| **共同人性** | 承认具体历史条件下的共同人性 | 主张超历史的普遍人性 |

::: warning 易错点
**人的本质 ≠ 人的本性**。"人的本质"是社会关系总和（马克思主义观点）；"人的本性"在很多语境下指自然本能。考题常设陷阱让人混淆两者。
:::

---

## 三、人生观的内涵（必背）

### 1. 人生观的定义

::: danger 必背：人生观的定义
**人生观**是人们在实践中形成的对于**人生目的、人生态度、人生价值**等问题的根本看法。
:::

### 2. 人生观三要素（极高频考点）

::: danger 必背：人生观三要素
人生观包括三个相互联系的内容：
1. **人生目的**——人为什么活着（回答"为什么"）
2. **人生态度**——人应当怎样对待生活（回答"怎么做"）
3. **人生价值**——什么样的人生才有意义（回答"值不值"）
:::

### 3. 三要素的地位（必背）

| 要素 | 地位 | 回答 |
| --- | --- | --- |
| **人生目的** | **核心**（决定方向） | 为什么活 |
| **人生态度** | 外在表现 | 怎么活 |
| **人生价值** | 评价尺度 | 活得怎样 |

::: danger 必背：人生目的的地位
**人生目的是人生观的核心**。它决定人生道路、人生态度、人生价值标准。
:::

::: warning 易错点
**"人生观的核心是人生目的"**——这是必考填空/选择。容易错选"人生价值"或"人生态度"。三者关系：人生目的（决定）→ 人生态度（表现）→ 人生价值（评价）。
:::

---

## 四、人生观与世界观的关系（理解）

| 概念 | 内涵 | 范围 |
| --- | --- | --- |
| **世界观** | 人们对整个世界的总的看法和根本观点 | 最广 |
| **人生观** | 人们对人生问题的根本看法 | 居中 |
| **价值观** | 人们对价值的根本看法和态度 | 具体 |

::: danger 必背：世界观、人生观、价值观的关系
- **世界观决定人生观**：世界观是人生观的理论基础
- **人生观是世界观的组成部分**：人生观体现世界观
- **世界观、人生观、价值观统一**于人们的社会实践
:::

---

## 五、人生观的形成和特征（理解）

### 人生观的形成

| 因素 | 作用 |
| --- | --- |
| **社会物质生活条件** | 决定人生观的根本内容 |
| **社会文化传统** | 影响人生观的表达形式 |
| **个人社会实践活动** | 形成和改变人生观的直接途径 |

### 人生观的特征

- **时代性**：不同时代有不同人生观
- **阶级性**：在阶级社会，人生观带有阶级烙印
- **实践性**：人生观在实践中形成、检验和发展

## 六、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 人的本质（社会关系总和）、人生观核心（人生目的）、人生观三要素 |
| **多选** | 人的本质属性、人生观三要素、人生观特征 |
| **简答** | 为什么说人的本质是一切社会关系的总和？人生观三要素及其关系？ |
| **论述** | 世界观、人生观、价值观的关系及其对大学生的启示 |

### 预测题一（单选）

**马克思主义认为，人的本质是（　）**
A. 单个人所固有的抽象物
B. 自私自利的本性
C. 一切社会关系的总和
D. 追求幸福和自由的天性

> **答案：C**
> **解析**：马克思指出"人的本质不是单个人所固有的抽象物，在其现实性上，它是**一切社会关系的总和**"。A 正是马克思批判的费尔巴哈观点；B、D 都是资产阶级抽象人性论。

### 预测题二（简答）

**简述人生观三要素及其相互关系。**

> **【三要素】** 人生观包括三个相互联系的内容：
> 1. **人生目的**——回答"人为什么活着"（决定人生方向）
> 2. **人生态度**——回答"人应当怎样对待生活"（外在表现）
> 3. **人生价值**——回答"什么样的人生才有意义"（评价尺度）
>
> **【相互关系】**
> - **人生目的是人生观的核心**，决定人生态度和人生价值标准；
> - **人生态度**是人生目的的外在表现，影响人生价值的实现；
> - **人生价值**是对人生目的和人生态度的评价尺度。
>
> 三者相互联系、相互制约，统一于人生实践。

### 易错点提醒

::: warning 本节三大易错点
1. **人的本质=一切社会关系的总和**（不是"自保""自利""抽象人性"）——这是马克思主义与资产阶级人性观的根本分野。
2. **人生观的核心=人生目的**（不是人生价值或人生态度）——人生目的回答"为什么活"，处于决定地位。
3. **人的本质≠人的本性**：本质是后天社会关系决定的；本性有时指自然本能。两者是不同概念。
:::
