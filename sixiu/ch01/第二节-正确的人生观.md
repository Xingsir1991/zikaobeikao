# 第二节 正确的人生观

> 教材第 23–32 页 · 回答"应当树立什么样的人生观"。

本节从人生观三要素展开，给出正确人生观的标准：**高尚的人生目的（服务人民奉献社会）+ 积极进取的人生态度 + 科学的人生价值评价标准**。

---

## 一、核心论断（必背）

> **高尚的人生追求**是服务人民、奉献社会。

> **积极进取的人生态度**——认真、务实、乐观、进取。

> 人生价值的标准，归根到底是看一个人**对社会的贡献**。

---

## 二、高尚的人生目的——服务人民、奉献社会（必背）

::: danger 必背：高尚的人生追求
**服务人民、奉献社会**是**最科学、最高尚的人生追求**，体现了马克思主义对人的本质的认识，是历史上一切进步人生观的共同特征。
:::

### 为什么"服务人民、奉献社会"是最高尚的追求

| 理由 | 说明 |
| --- | --- |
| **理论依据** | 人的本质是社会关系总和——个人价值只能在奉献社会中实现 |
| **历史依据** | 历史上一切进步思想家都强调为大众、为社会 |
| **实践依据** | 中国共产党人的根本宗旨就是全心全意为人民服务 |

::: tip 联系实际
"服务人民、奉献社会"在不同时代有不同表现：战争年代是抛头颅洒热血；和平年代是爱岗敬业、热心公益；新时代是投身民族复兴大业。这是论述题联系实际的常用角度。
:::

---

## 三、积极进取的人生态度——四个关键词（高频考点）

::: danger 必背：积极进取的人生态度四要素
1. **人生须认真**——严肃对待人生，对自己负责
2. **人生当务实**——脚踏实地，从客观实际出发
3. **人生应乐观**——积极向上，正确对待顺境逆境
4. **人生要进取**——开拓创新，迎难而上
:::

### 四要素的对照记忆

| 关键词 | 含义 | 反面 |
| --- | --- | --- |
| **认真** | 严正对待人生、对自己负责 | 敷衍了事 |
| **务实** | 客观看待人生、脚踏实地 | 空谈浮夸 |
| **乐观** | 用积极心态对待人生 | 悲观厌世 |
| **进取** | 奋发向上、改革创新 | 得过且过 |

::: warning 易错点
**"认真—务实—乐观—进取"四个词不能漏、不能换**。常见干扰项如"勤俭""谦逊""果断"——这些虽然是美德，但不是教材"积极进取人生态度"的原表述。
:::

### 正确对待顺境与逆境（理解）

| 顺境 | 逆境 |
| --- | --- |
| 顺利的环境，要珍惜、善用 | 困难的环境，要正视、磨砺 |
| 优点：便于发展 | 优点：锻炼意志、增长才干 |
| 警惕：骄傲自满、安于现状 | 警惕：消极沉沦、怨天尤人 |

::: tip 名言金句
**"宝剑锋从磨砺出，梅花香自苦寒来。"**——常用于论述"逆境锻炼人才"。孟子"生于忧患，死于安乐"也是同类金句。
:::

## 四、人生价值的评价标准（极高频考点）

### 1. 人生价值的双重内涵

::: danger 必背：人生价值的两个方面
- **社会价值**——个体对社会的**贡献**（主要标准、根本标准）
- **自我价值**——社会对个体的**尊重和满足**（基础）
:::

### 2. 人生价值的标准（必背）

::: danger 必背：人生价值的评价标准
**人生价值的标准，归根到底是看一个人对社会的贡献。**

劳动和贡献的多少是衡量人生价值的**根本尺度**。
:::

### 3. 评价人生价值的条件（理解）

| 条件 | 说明 |
| --- | --- |
| **坚持能力有大小与贡献须尽力相统一** | 不看绝对量，看是否尽力 |
| **坚持完善自己与贡献社会相统一** | 自我完善是为了更好贡献 |
| **坚持物质贡献与精神贡献相统一** | 精神贡献同样重要 |
| **坚持动机与效果相统一** | 既看动机也看效果 |

### 易混辨析：社会价值 vs 自我价值

| 类别 | 社会价值 | 自我价值 |
| --- | --- | --- |
| **方向** | 个人→社会 | 社会→个人 |
| **内容** | 个人对社会的贡献 | 社会对个人的满足和尊重 |
| **地位** | **主要标准、根本标准** | 基础 |

::: warning 易错点
**社会价值是主要标准**——不是自我价值。但社会价值与自我价值是辩证统一的：自我价值的实现离不开社会条件；社会价值的实现必然带来自我价值的提升。两者不是对立关系。
:::

## 六、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 高尚人生追求、积极人生态度四要素、人生价值的主要标准（社会价值） |
| **多选** | 积极进取人生态度四要素、人生价值评价的条件 |
| **简答** | 积极进取的人生态度是什么？为什么说人生价值的标准是对社会的贡献？ |
| **论述** | 如何评价人生价值？大学生应树立什么样的正确人生观？ |

### 预测题一（单选）

**衡量人生价值的根本尺度是（　）**
A. 个人拥有的财富多少
B. 个人获得的荣誉高低
C. 个体的劳动及其对社会的贡献
D. 个人职位的高低

> **答案：C**
> **解析**：人生价值的标准，归根到底是看一个人**对社会的贡献**。**劳动和贡献的多少是衡量人生价值的根本尺度**。A、B、D 都是从自我价值角度衡量，是错误的价值取向。

### 预测题二（论述）

**大学生应当如何评价人生价值？**

> **【破题】** 把握人生价值的双重内涵（社会价值 + 自我价值）及评价的根本标准（社会贡献）。
>
> **【主体】**
> **第一**，人生价值包括**社会价值**和**自我价值**两个方面。社会价值是个体对社会的贡献，自我价值是社会对个体的尊重和满足。其中**社会价值是人生价值的主要方面、根本标准**。
>
> **第二**，评价人生价值的**根本尺度**是个体的劳动及其对社会的贡献。一个人能力有大有小，但只要全心全意为人民服务、为社会作贡献，他的人生就有价值。
>
> **第三**，评价人生价值要做到"四个坚持"：
> - 坚持**能力有大小与贡献须尽力**相统一——不看绝对量，看是否尽力；
> - 坚持**完善自己与贡献社会**相统一——自我完善是为了更好贡献；
> - 坚持**物质贡献与精神贡献**相统一——精神贡献同样重要；
> - 坚持**动机与效果**相统一——既看动机也看效果。
>
> **【总结】** 大学生要树立科学的人生价值观，把个人的前途命运同国家和民族的前途命运紧密结合，在服务人民、奉献社会中实现自己的人生价值。

### 易错点提醒

::: warning 本节三大易错点
1. **人生价值的主要标准是"社会价值"**（社会贡献），不是自我价值。但两者辩证统一，不能割裂。
2. **积极人生态度四要素**：认真、务实、乐观、进取——用词固定，不能替换成"勤俭""谦虚"等。
3. **高尚人生追求**：服务人民、奉献社会——不是"出人头地""光宗耀祖""个人成名"。
:::
