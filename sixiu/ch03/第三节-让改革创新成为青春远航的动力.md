# 第三节 让改革创新成为青春远航的动力

> 教材第 95–106 页 · 实践层面——时代精神的核心：改革创新。

本节聚焦时代精神的核心——**改革创新**。在阐释爱国主义之后，教材进一步指出：新时代爱国者必须弘扬改革创新精神，做改革创新的实践者，让改革创新成为青春远航的动力。

---

## 一、核心论断（必背）

> **改革创新**是时代精神的核心，是进一步解放和发展生产力的必然要求。

> 大学生要做**改革创新的实践者**，让改革创新成为青春远航的动力。

---

## 二、改革创新的时代意义（必背）

::: danger 必背：改革创新的时代意义
1. **改革创新**是时代精神的核心
2. 改革创新是**进一步解放和发展生产力的必然要求**
3. 改革创新是**建设社会主义创新型国家的客观需要**
4. 改革创新是**全面深化改革、推动社会发展的强大动力**
:::

### 改革创新三大意义（理解）

| 维度 | 意义 |
| --- | --- |
| **生产力** | 进一步解放和发展生产力的必然要求 |
| **国家战略** | 建设创新型国家的客观需要 |
| **社会发展** | 全面深化改革、推动社会发展的强大动力 |

---

## 三、创新驱动发展战略（理解）

::: danger 必背：创新驱动发展战略
**创新是引领发展的第一动力**。

实施创新驱动发展战略，是把创新摆在国家发展全局的**核心位置**。
:::

### 创新驱动发展的内涵

| 维度 | 内容 |
| --- | --- |
| **科技** | 加强基础研究、应用研究，提高自主创新能力 |
| **人才** | 实施人才强国战略，培养创新型人才 |
| **制度** | 完善创新创业制度环境 |
| **产业** | 推动产业结构升级 |

::: tip 习近平金句
**"创新是引领发展的第一动力"**——常考金句，与"科技是第一生产力""人才是第一资源"并列。
:::

## 四、建设创新型国家（必背）

::: danger 必背：建设创新型国家的核心
**建设创新型国家**，核心是要把**增强自主创新能力**作为发展科学技术的战略基点，走出**中国特色自主创新道路**。
:::

### 创新型国家的特征

| 特征 | 内容 |
| --- | --- |
| **研发投入高** | R&D 经费占 GDP 比重较高 |
| **科技进步贡献率高** | 技术进步对经济增长贡献大 |
| **自主创新能力强** | 拥有大量自主知识产权 |
| **创新产出高** | 发明专利、创新成果丰富 |

### 预测题一（单选）

**时代精神的核心是（　）**
A. 爱国主义
B. 改革创新
C. 艰苦奋斗
D. 自强不息

> **答案：B**
> **解析**：**改革创新是时代精神的核心**。爱国主义是**民族精神**的核心（A 错）。C、D 都是中华民族传统美德，不是时代精神的核心。

### 预测题二（简答）

**大学生如何做改革创新的实践者？**

> **【主体】** 大学生要做改革创新的实践者，让改革创新成为青春远航的动力：
>
> **第一，树立改革创新的自觉意识。** 深刻认识改革创新是时代精神的核心、是解放和发展生产力的必然要求，增强改革创新的责任感和使命感。
>
> **第二，培养改革创新的责任感。** 把个人理想融入创新驱动发展战略和创新型国家建设，树立**创新报国**的志向。
>
> **第三，增强改革创新的能力本领。** 扎实掌握系统的专业知识，培养批判性思维和创造性思维，提高创新能力。
>
> **第四，积极投身实践。** 主动参加创新创业实践、科技竞赛、社会实践，在实践中磨砺创新品质、增长创新才干。
>
> **【总结】** 大学生要争做改革创新的实践者，让青春在创新创造中焕发光彩。

### 易错点提醒

::: warning 本节三大易错点
1. **时代精神核心=改革创新**；**民族精神核心=爱国主义**——两个核心要分清。
2. **创新是"第一动力"**（不是"第一生产力"——科技才是第一生产力）。
3. **建设创新型国家的核心=增强自主创新能力**——不要答成"科技人才""改革开放"。
:::
