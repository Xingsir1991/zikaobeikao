# 第二节 做新时代的忠诚爱国者

> 教材第 81–94 页 · 实践层面——新时代如何爱国。

本节是本章的高频核心。它在界定爱国主义内涵之后，提出新时代爱国主义的根本要求——**爱国、爱党、爱社会主义相统一**，并延伸到总体国家安全观、爱国统一战线等内容。

---

## 一、核心论断（必背）

> 爱国主义是中华民族精神的核心。

> 新时代爱国主义根本要求：**坚持爱国、爱党、爱社会主义相统一**。

> **总体国家安全观**：以人民安全为宗旨，以政治安全为根本，以经济安全为基础……

---

## 二、爱国主义的内涵与基本要求（必背）

### 1. 爱国主义的定义

::: danger 必背：爱国主义的定义
**爱国主义**体现了人们对自己祖国的深厚感情，揭示了**个人对祖国的依存关系**，是人们对自己家园以及民族和文化的**归属感、认同感、尊严感与荣誉感**的统一。
:::

### 2. 爱国主义的基本要求（必背）

::: danger 必背：爱国主义的四个基本要求
1. **爱祖国的大好河山**（爱乡土）
2. **爱自己的骨肉同胞**（爱人民）
3. **爱祖国的灿烂文化**（爱文化）
4. **爱自己的国家**（爱国家政权）
:::

::: tip 记忆口诀
**"河山—同胞—文化—国家"**——四要素由自然到人民、由文化到国家，构成爱国主义的完整内容。
:::

### 3. 爱国主义的特点

| 特点 | 含义 |
| --- | --- |
| **历史性** | 不同历史时期，爱国主义有不同内容 |
| **具体性** | 爱国主义是具体的、历史的，不是抽象的 |
| **阶级性** | 在阶级社会，爱国主义带有阶级烙印 |

---

## 三、新时代爱国主义的根本要求（极高频考点）

::: danger 必背：新时代爱国主义的根本要求
**坚持爱国、爱党、爱社会主义相统一**——这是新时代爱国主义的根本要求。

要把爱国情、强国志、报国行自觉融入**坚持和发展中国特色社会主义事业、建设社会主义现代化强国、实现中华民族伟大复兴**的奋斗之中。
:::

### 三个"爱"为什么必须统一（理解）

| 命题 | 内涵 |
| --- | --- |
| **爱国** | 爱中华人民共和国 |
| **爱党** | 爱中国共产党（中国特色社会主义最本质特征） |
| **爱社会主义** | 爱中国特色社会主义制度（中国取得一切成就的根本制度保障） |

::: danger 必背：为什么要"三个爱统一"
- 中国共产党是**中国特色社会主义事业的领导核心**
- 中国特色社会主义是**当代中国发展进步的根本方向**
- 祖国的命运与党的命运、社会主义的命运**密不可分**
- 因此**爱国必须爱党、爱社会主义**
:::

::: warning 易错点
**"爱国、爱党、爱社会主义"是新时代爱国主义的根本要求**——三者必须统一，不能割裂。这是新时代爱国主义区别于旧时代爱国主义的根本特征。
:::

---

## 四、总体国家安全观（高频考点）

### 1. 总体国家安全观的内涵（必背）

::: danger 必背：总体国家安全观的内涵
**总体国家安全观**：
- 以**人民安全**为宗旨
- 以**政治安全**为根本
- 以**经济安全**为基础
- 以**军事、文化、社会安全**为保障
- 以**促进国际安全**为依托
- 走出一条**中国特色国家安全道路**
:::

::: tip 记忆口诀
**"宗旨—根本—基础—保障—依托"**——五个层次：人民（宗旨）、政治（根本）、经济（基础）、军事文化社会（保障）、国际（依托）。
:::

### 2. 国家安全的范围（理解）

::: danger 必背：总体国家安全涵盖的领域
总体国家安全观涵盖**政治、国土、军事、经济、文化、社会、科技、网络、生态、资源、核、海外利益、太空、深海、极地、生物**等多个领域。
:::

### 3. 公民维护国家安全的义务

| 义务 | 内容 |
| --- | --- |
| **遵守宪法法律** | 保守国家秘密 |
| **及时报告线索** | 发现危害国家安全行为及时报告 |
| **如实提供证据** | 配合国家安全工作 |
| **为国家安全工作提供便利** | 协助国家安全机关 |

::: warning 易错点
**总体国家安全观的"五个以"表述要完整**：宗旨（人民）、根本（政治）、基础（经济）、保障（军事文化社会）、依托（国际）。常考"以什么为根本"（政治安全）。
:::

## 五、爱国统一战线（理解）

::: danger 必背：爱国统一战线的内容
爱国统一战线包括：
- 全体**社会主义劳动者**
- **社会主义事业的建设者**
- 拥护社会主义的**爱国者**
- 拥护祖国统一和致力于**中华民族伟大复兴的爱国者**
:::

### 预测题一（单选）

**总体国家安全观以（　）为根本**
A. 人民安全
B. 政治安全
C. 经济安全
D. 军事安全

> **答案：B**
> **解析**：总体国家安全观——以**人民安全为宗旨**，以**政治安全为根本**，以**经济安全为基础**，以军事、文化、社会安全为保障，以促进国际安全为依托。本题问"为根本"，对应政治安全。

### 预测题二（简答）

**简述新时代爱国主义的根本要求及其内涵。**

> **【根本要求】** 新时代爱国主义的根本要求是**坚持爱国、爱党、爱社会主义相统一**。
>
> **【为什么必须统一】**
> - 中国共产党是中国特色社会主义事业的**领导核心**
> - 中国特色社会主义是当代中国发展进步的**根本方向**
> - 祖国的命运与党的命运、社会主义的命运**密不可分**
>
> **【实践要求】** 要把**爱国情、强国志、报国行**自觉融入坚持和发展中国特色社会主义事业、建设社会主义现代化强国、实现中华民族伟大复兴的奋斗之中。
>
> **【总结】** 三个"爱"统一是新时代爱国主义区别于旧时代爱国主义的根本特征，是当代中国爱国主义的鲜明主题。

### 易错点提醒

::: warning 本节三大易错点
1. **新时代爱国主义根本要求=三个爱统一**（爱国、爱党、爱社会主义）——三者必须统一，不能割裂。
2. **总体国家安全观"五个以"**：宗旨=人民，根本=政治，基础=经济，保障=军事文化社会，依托=国际——对象要记牢。
3. **爱国主义基本要求四要素**：河山、同胞、文化、国家——不要漏"文化"或"同胞"。
:::
