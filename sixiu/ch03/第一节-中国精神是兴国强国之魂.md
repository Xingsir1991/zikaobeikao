# 第一节 中国精神是兴国强国之魂

> 教材第 70–80 页 · 概念界定——中国精神是什么。

本节是全章的理论基石，回答"什么是中国精神"。中国精神由**民族精神**和**时代精神**两部分构成，是凝心聚力的兴国之魂、强国之魂。

---

## 一、核心论断（必背）

> 中国精神是凝心聚力的**兴国之魂、强国之魂**。

> 中国精神 = **以爱国主义为核心的民族精神** + **以改革创新为核心的时代精神**。

---

## 二、中国精神的内涵（极高频考点）

::: danger 必背：中国精神的两大组成部分
**中国精神**包括两个方面：
1. **以爱国主义为核心的民族精神**——历史的积淀
2. **以改革创新为核心的时代精神**——时代的产物
:::

::: tip 记忆口诀
**"爱国主义核心的民族精神 + 改革创新核心的时代精神"**——两个核心、两个精神。
:::

---

## 三、民族精神的内涵——"四个伟大精神"（极高频考点）

::: danger 必背：民族精神的四个伟大精神
习近平在十三届全国人大一次会议上提出，中国人民的伟大民族精神：
1. **伟大创造精神**
2. **伟大奋斗精神**
3. **伟大团结精神**
4. **伟大梦想精神**
:::

### "四个伟大精神"逐条解读

| 精神 | 内涵 |
| --- | --- |
| **伟大创造精神** | 中国人民创造了悠久历史和灿烂文化，发明创造众多 |
| **伟大奋斗精神** | 中国人民革故鼎新、自强不息，始终不屈不挠 |
| **伟大团结精神** | 中国人民始终团结一心、同舟共济 |
| **伟大梦想精神** | 中国人民始终心怀梦想、不懈追求（盘古开天、女娲补天等） |

::: warning 易错点
**民族精神"四个伟大"=创造、奋斗、团结、梦想**——不能遗漏，不能替换为其他词（如"勇敢""勤劳"）。这是 2018 年习近平在十三届全国人大一次会议上的最新概括。
:::

### 民族精神的核心：爱国主义

::: danger 必背：民族精神的核心
**爱国主义**是中华民族精神的核心。
:::

---

## 四、时代精神的内涵——核心是改革创新（必背）

### 1. 时代精神的核心

::: danger 必背：时代精神的核心
**改革创新**是时代精神的核心。
:::

### 2. 时代精神的内容（理解）

时代精神是一个民族在新的历史条件下形成的精神风貌，包括：

| 内容 | 内涵 |
| --- | --- |
| **改革创新精神** | 核心，是推动时代发展的根本动力 |
| **解放思想、实事求是** | 思想路线 |
| **与时俱进、勇于创新** | 精神品格 |
| **知难而进、一往无前** | 奋斗精神 |
| **艰苦奋斗、务求实效** | 实践品格 |
| **淡泊名利、无私奉献** | 道德精神 |

---

## 五、民族精神与时代精神的关系（极高频考点）

::: danger 必背：民族精神与时代精神的关系
- 二者紧密联系、相辅相成，**共同构成中国精神**
- **民族精神**为时代精神提供**深厚底蕴**
- **时代精神**为民族精神注入**新的活力**
- 两者**统一于中国特色社会主义伟大实践**
:::

### 关系对照表

| 关系 | 民族精神 | 时代精神 |
| --- | --- | --- |
| **时间维度** | 历史积淀 | 时代产物 |
| **核心** | 爱国主义 | 改革创新 |
| **作用** | 提供深厚底蕴 | 注入新的活力 |
| **关系** | 互为根基、辩证统一 | 互为根基、辩证统一 |

::: warning 易错点
**民族精神核心=爱国主义**；**时代精神核心=改革创新**——两个核心要分清，常考易混。
:::

### 预测题一（单选）

**中华民族精神的核心是（　）**
A. 改革创新
B. 爱国主义
C. 团结统一
D. 自强不息

> **答案：B**
> **解析**：**爱国主义是民族精神的核心**。改革创新是**时代精神**的核心（A 错）。C、D 是中华民族传统美德，不是民族精神的核心。

### 预测题二（简答）

**简述中国精神的内涵及其两大组成部分的关系。**

> **【内涵】** 中国精神是凝心聚力的兴国之魂、强国之魂，由两大组成部分构成：
> - **以爱国主义为核心的民族精神**
> - **以改革创新为核心的时代精神**
>
> **【关系】** 民族精神与时代精神紧密联系、相辅相成：
> - **民族精神**为时代精神提供**深厚底蕴**
> - **时代精神**为民族精神注入**新的活力**
> - 二者**统一于中国特色社会主义伟大实践**
>
> **【总结】** 把握两个核心（爱国主义、改革创新），就是把握了中国精神的本质。

### 易错点提醒

::: warning 本节三大易错点
1. **两个核心要分清**：民族精神核心=爱国主义；时代精神核心=改革创新——常设干扰项让两者对调。
2. **民族精神"四个伟大"**：创造、奋斗、团结、梦想——不要漏、不要换成"勤劳、勇敢"等。
3. **中国精神是民族精神 + 时代精神**（两者并列，不是包含关系）。
:::
