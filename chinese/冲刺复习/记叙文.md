# 冲刺复习 · 记叙文（对接考点）

> 本课件针对五个核心考点精准设计：
> **考点 1：人物形象分析**（最高频）—— 性格 + 例证 + 典型意义

> **考点 3：细节描写作用**（高频）—— 具体细节 + 效果
> **考点 4：抒情方式**（重点）—— 直接 / 间接 / 克制

## 考点 1：人物形象分析（必考·最高频）

### 一、记叙文人物分析速查表

| 分析维度 | 切入点 | 本站典型人物 |
| --- | --- | --- |
| **言行** | 关键语言、典型动作 | 冯谖（弹剑）/项羽（嗔目辟易）/南霁云（断指射塔） |
| **细节** | 微小动作、表情、物件 | 母亲（灯火夜分）/傅雷（"怒"四现）/小弟（"我先洗"） |
| **侧面烘托** | 他人反应、环境反应 | "左右皆泣，莫能仰视"烘托项羽 |
| **欲扬先抑** | 先抑后扬的反差 | 冯谖 / 郭橐驼 / 牡丹 |
| **群像** | 多人各有个性 | 张巡 + 许远 + 南霁云 / 蹇叔 + 弦高 + 先轸 |

### 二、标准答题公式（三段式 · 背熟！）

```
总（定性）：XX 是一个"XX + XX"的 XX 形象（一句话定性，至少两个性格面）。
分（例证）：① 性格面 A——例："……"（言行细节原文）；
           ② 性格面 B——例："……"（言行细节原文）；
           ③ 性格面 C（可选）——例："……"。
结（典型意义）：矛盾统一 / 时代缩影 / 寄托作者 XX 之情，
              使人物形象立体丰满、富有张力（+ 典型意义）。
```

::: tip 评分要点
- **"一句话定性"必须含矛盾面**：项羽不是"神勇的将军"，而是"**神勇而刚愎**的悲剧英雄"；冯谖不是"有才能的食客"，而是"**深藏不露**的策士"。
- **例证必须引用原文细节**，不能只说"他很勇敢"。
- **结尾必须点典型意义**：是时代缩影？作者寄托？还是矛盾张力？——这是拉开分数的关键。
:::

### 三、本站七大典型人物案例（背诵版）

#### 案例 1：项羽（《垓下之围》）—— 矛盾统一的悲剧英雄 ⭐⭐⭐

> **总**：项羽是一个**神勇而刚愎、自尊而末路**的悲剧英雄。
> **分**：
> ① **神勇绝伦**——东城快战"斩将、刈旗、嗔目而叱，赤泉侯人马俱惊，辟易数里"，以二十八骑溃围斩将。
> ② **刚愎自用**——"**天亡我，非战之罪也**"，至死不悟失败因由，归咎于天。
> ③ **儿女情长与英雄自尊**——垓下悲歌别姬"力拔山兮气盖世，时不利兮骓不逝"；乌江拒渡"无颜见江东父老"，赠马亭长，自刎而死。
> **结**：神勇与刚愎、霸气与自尊交织，构成"矛盾统一的悲剧英雄"，富有戏剧张力。

#### 案例 2：冯谖（《冯谖客孟尝君》）—— 深藏不露的策士 ⭐⭐⭐

> **总**：冯谖是一个**看似贪得无厌、实则深谋远虑**的策士形象。
> **分**（欲扬先抑）：
> ① **先抑**——"无好无能"，三次弹剑歌"**长铗归来乎**！食无鱼 / 出无车 / 无以为家"，给人贪婪平庸印象。
> ② **后扬**——矫命焚券市义（"使君重爱"）；狡兔三窟，营就三窟。
> ③ **细节**：弹剑、揭剑——傲兀自信的"试探"姿态。
> **结**："无好无能"假象与"市义三窟"真才强烈反差，塑造"深藏不露"的策士典型。

#### 案例 3：南霁云（《张中丞传后叙》）—— 刚烈忠义的勇将 ⭐⭐⭐

> **总**：南霁云是一个**刚烈忠义、重诺轻生**的勇将。
> **分**：
> ① **刚烈**——求救贺兰被拒，**"拔刀断一指，血淋漓，以示贺兰"**；将出城，**抽矢射佛寺浮图**，立誓"**吾归破贼，必灭贺兰，此矢所以志也**！"
> ② **忠义**——"云虽欲独食，义不忍，虽食，且不下咽"（睢阳不食月余，他不能下咽）。
> ③ **重诺感恩**——"**公有言，云敢不死**"，张巡一呼即殉国。
> **结**：断指射塔的细节震撼人心，是忠义刚烈的典型；与张巡、许远构成睢阳殉国群像。

#### 案例 4：郭橐驼（《种树郭橐驼传》）—— 貌寝才高的智者 ⭐⭐

> **总**：郭橐驼是一个**貌寝而才高、朴素而哲思**的种树智者。
> **分**：
> ① **貌寝**（抑）——"病偻，隆然伏行，有类橐驼"，被人取绰号而不怒，自号"橐驼"。
> ② **才高**（扬）——种树"或迁徙，无不活，且硕茂早实以蕃"，他人"窥伺效慕"皆莫能如。
> ③ **哲思**——"**能顺木之天，以致其性焉尔**"，由种树之道悟治民之道。
> **结**：貌寝才高的反差，使其成为"托物言志"的载体——顺木之天即顺民之性。

#### 案例 5：母亲（《我与地坛》）—— 隐忍深爱的母亲 ⭐⭐⭐

> **总**：母亲是一个**隐忍克制、深爱无私**的形象。
> **分**：
> ① **深爱**——"**她不是那种光会疼爱儿子而不懂得理解儿子的母亲**"，明知儿子去地坛会痛苦沉思，仍让他去。
> ② **隐忍克制**——"她**悄悄地**找，**偷偷地**听"，发现了也不惊扰，把焦虑深埋。
> ③ **牺牲**——以早逝为代价，撑起儿子的"生路"；"**这园中不单是处处都有我的车辙，有过我的车辙的地方也都有过母亲的脚印**"。
> **结**：母亲的白描形象是"克制抒情"的典范，越平淡越深沉，作者借此领悟"怎样活"的生命意义。

#### 案例 6：傅雷（《纪念傅雷》）—— 刚直孤傲的知识分子 ⭐⭐

> **总**：傅雷是一个**刚直不阿、孤傲执着**的知识分子。
> **分**（以"怒"立骨）：
> ① **生活之怒**——对琐事的不苟且；
> ② **学术之怒**——对浮夸学风的不容；
> ③ **艺术之怒**——对庸俗艺术的不妥协；
> ④ **气节之怒**——"**一生生机勃勃，一生 stalwart（刚强）**"，最后以死殉其刚直。
> **结**：以一字"怒"立骨，层层升华，塑造中国现代知识分子"傲骨"的典型。

#### 案例 7：庄宗（《五代史伶官传序》——记叙为议论服务） ⭐

> **总**：庄宗是一个**忧劳兴国、逸豫亡身**的典型。
> **分**：
> ① **盛**——受矢负矢，"**及其成功，天下莫能与之争**"，意气之盛。
> ② **衰**——"**数十伶人呼于前，而身死国灭**"，一夫夜呼，泣下沾襟。
> **结**：庄宗是作为论据出现的，他的盛衰之变印证"忧劳兴国、逸豫亡身"——典型意义在于警示后人。

#### 【模拟 2 · 简答】《冯谖客孟尝君》是如何塑造冯谖形象的？

> 效果：细节具体可感，群像中独树一帜。

## 考点 2：叙事手法 / 结构（高频）

### 一、六大叙事手法速查表

| 手法 | 核心特征 | 答题效果词 | 本站典范 |
| --- | --- | --- | --- |
| **欲扬先抑** | 先压低，后抬高 | **反差鲜明、欲扬故抑** | 冯谖 / 郭橐驼 / 牡丹 |
| **多场面描写** | 几个经典场面结构全篇 | **层层递进、命运完整** | 垓下之围 |
| **分类描摹** | 将对象分类逐一描写 | **层次分明、寓褒贬** | 西湖七月半 |
| **今昔对比** | 过去 vs 现在 | **凸显变化、以小见大** | 香市 / 爱尔克的灯光 |
| **托物言志** | 借事物寄托道理 | **借物喻理、含蓄深刻** | 郭橐驼 / 牡丹 / 蚂蚁大战 |
| **叙议结合** | 叙事与议论交替 | **情理交融、说服感染** | 张中丞传后叙 / 都江堰 / 香市 |

### 二、六大手法的标准答题公式（三段式 · 背熟！）

> **通用公式：指明手法 → 分析如何运用 → 点明表达效果**

#### ① 欲扬先抑

```
本文运用了"欲扬先抑"手法，
先写 XX 的 XX（抑：无好无能/貌寝/迟不开），
后写 XX 的 XX（扬：市义三窟/种树无敌/坚守花期），
前后形成强烈反差，使 XX 形象更鲜明突出，富有戏剧性。
```

**本站案例：**
- 《冯谖客孟尝君》：无好无能 → 市义三窟（深藏不露）
- 《种树郭橐驼传》：病偻癃然 → 种树无敌 + 哲思（貌寝才高）
- 《牡丹的拒绝》：迟迟不开令人失望 → 坚守花期、不媚权贵（高贵品格）

#### ② 多场面描写

```
本文运用了多场面描写，
以 XX、XX、XX（三个/几个场面）结构全篇，
场面之间层层递进（或气氛转换），
完整呈现人物命运的 XX 变化，富有戏剧张力。
```

**本站案例**（《垓下之围》三场面）：

| 场面 | 内容 | 氛围 |
| --- | --- | --- |
| 垓下之围 | 四面楚歌，悲歌别姬 | 悲凉 |
| 东城快战 | 溃围斩将，嗔目辟易 | 壮烈 |
| 乌江自刎 | 拒渡赠马，自刎而亡 | 苍凉 |

#### ③ 分类描摹

```
本文运用了分类描摹手法，
将 XX 分为 X 类（具体类别），逐一描写其 XX 特征，
层次分明，雅俗（或高下）递进，寓褒贬于客观分类，
凸显作者 XX 的情怀与取向。
```

**本站案例**（《西湖七月半》五类看月之人）：
1. 不见月者（看人者）2. 欲见月者（附庸风雅）3. 亦看月而欲人看其看月者（矫情）4. 月亦看、人亦看而不欲看人者（雅俗之间）5. **不作意看月者**（真雅士，作者自况）。

#### ④ 今昔对比

```
本文运用了今昔对比手法，
先写昔日 XX（热闹/盛/纯真），后写今日 XX（冷清/衰/幻灭），
通过纵向对比凸显 XX 的变化，
以小见大，反映 XX 的社会（或人生）主题。
```

**本站案例：**
- 《香市》：昔日"五六十亩挤满"vs 今日"戏台坍塌、屋椽像瘦人的肋骨"——**反映农村破产**。
- 《爱尔克的灯光》：十八年后重访故居——"大门内的微光"vs"江上的灯光"——**告别旧家、走向新路**。
- 《马伶传》：马伶前败（丑化严嵩）vs 后胜（赴京学顾秉谦三年）——**艺术精益求精**。

#### ⑤ 托物言志

```
本文运用了托物言志手法，
借 XX（具体事物：树/牡丹/蚂蚁）寄托 XX 的道理（顺民之性/高贵人格/反战），
化抽象为具象，使道理含蓄深刻、形象可感。
```

**本站案例：**
- 《种树郭橐驼传》：借种树之道 → 喻治民之道（**顺木之天 = 顺民之性**）
- 《牡丹的拒绝》：借牡丹坚守花期 → 喻高贵人格（不媚权贵）
- 《蚂蚁大战》：借蚂蚁厮杀 → 反思人类战争的荒谬（以小见大）

#### ⑥ 叙议结合

```
本文运用了叙议结合手法，
前半 XX（议论：辩诬/感悟），后半 XX（叙事：补事迹/记游览），
议论为叙事张本，叙事印证议论，
情理交融，既有说服力又有感染力。
```

**本站案例：**
- 《张中丞传后叙》：前半议论（辩诬许远）+ 后半叙事（补南霁云、张巡事迹）——议论为叙事张本，叙事印证议论。
- 《都江堰》：夹叙夹议（游览都江堰 + 文化感悟）——赞美李冰精神。
- 《香市》：记叙昔日盛况 + 结尾议论（揭示农村破产）——以小见大。

### 三、考点 2 模拟练习（含答案）

#### 【模拟 1 · 简答】《西湖七月半》如何运用分类描摹？有何作用？

> **标准答案**：本文将看月之人分为**五类**，逐一描摹：① 不见月者（专看人）；② 欲见月而人看其看月者（附庸）；③ 亦看月而欲人看其看月者（矫情）；④ 月亦看而不欲看人者（半雅）；⑤ **不作意看月者**（真雅士，作者自况）。
> 前四类为俗众，第五类为雅士，**层次分明、雅俗递进**，寓褒贬于客观分类，凸显作者**清高雅脱**的情怀。

#### 【模拟 2 · 论述】以《冯谖客孟尝君》和《种树郭橐驼传》为例，分析"欲扬先抑"手法。

> **标准答案**：两篇都用欲扬先抑，但抑扬各有侧重：
> **《冯谖客孟尝君》**：先抑冯谖"无好无能"、三次弹剑索鱼索车索家（**人品抑**）；后扬其焚券市义、狡兔三窟（**才略扬**）——塑造"深藏不露"的策士。
> **《种树郭橐驼传》**：先抑其"病偻癃然，类橐驼"（**外貌抑**）；后扬其种树无敌、哲思深刻（**才能与思想扬**）——塑造"貌寝才高"的智者，并托物言志。
> **共同效果**：前后反差，使形象鲜明、富有戏剧性；**不同**：冯谖重在"才"，郭橐驼兼有"理"（托物言志）。

#### 【模拟 3 · 简答】《香市》如何通过今昔对比表达主旨？

> **标准答案**：本文用**今昔对比**：
> 昔日——"香市"五六十亩挤满，"屋椽像瘦人的肋骨"（按：此喻应指戏台），"篙击篙，舟触舟，肩摩肩"，各种杂耍热闹非凡；

## 考点 3：细节描写作用（高频）

### 一、细节描写作用速查表

| 作用类型 | 答题效果词 |
| --- | --- |
| **刻画人物** | 具体可感、形象立体 |
| **烘托气氛** | 渲染氛围、身临其境 |
| **推动情节** | 伏笔照应、情节转折 |
| **表现主旨** | 深化主题、震撼人心 |
| **侧面烘托** | 借他人反应反衬 |

### 二、标准答题公式（三段式 · 背熟！）

```
本文运用了细节描写，
写了 XX 的 XX 细节（具体引用原文），
（分析）在 XX 场面中用此细节刻画了人物 XX 的 XX 性格 / 烘托了 XX 氛围 / 推动了 XX 情节，
（效果）使人物形象丰满立体 / 氛围感人至深 / 主旨震撼人心，具有强烈的艺术感染力。
```

::: tip 评分要点
- **必须引用原文细节**，不能只说"用细节描写"。
- **作用要多角度分析**：人物 + 氛围 + 主旨，至少两点。
- **侧面烘托**是隐藏考点——"一座大惊""左右皆泣"是借他人反应写人物。
:::

### 三、本站八大经典细节案例（背诵版）

#### 案例 1：断指射塔（《张中丞传后叙》·南霁云）⭐⭐⭐

> **原文**："**因拔所佩刀断一指，血淋漓，以示贺兰**。一座大惊，皆感激为云泣下。"；"**抽矢射佛寺浮图，矢著其上砖半箭**，曰：'吾归破贼，必灭贺兰，此矢所以志也！'"
>
> **作用**：
> ① **刻画人物**：断指——血淋淋的视觉冲击，最震撼地表现刚烈；射塔——把誓言具象化（箭入砖半支），标记不忘。
> ② **侧面烘托**："一座大惊，皆感激为云泣下"——借满座反应反衬其壮烈。
> ③ **表现主旨**：睢阳殉国忠义的极致体现，震撼人心。

#### 案例 2：画像拼凑（《先妣事略》）⭐⭐

> **原文**：母亲死后，子女们**用画像拼凑**记忆中的母亲形象（"或二寸，或三四寸"），"图貌不可得"。
>
> **作用**：以"拼凑画像"的细节写丧母之痛——越是拼凑不出，越是刻骨铭心；**白描传情，越平淡越深沉**。

#### 案例 3：饮螺致哑（《先妣事略》）⭐⭐

> **原文**：母亲因产后体弱误信偏方，"**饮螺致哑**"（按：原意为"乳饮螺"误补），"举之尽，喑不能言"。
>
> **作用**：一句叙述惨剧，不加评论——**克制抒情**的极致，比号啕大哭更催人泪下，凸显频繁生育与愚昧偏方对女性的戕害。

#### 案例 4：灯火至夜分（《先妣事略》）⭐⭐

> **原文**："**灯火荧荧，每至夜分**"——母亲劳作到深夜。
>
> **作用**：不赞一"勤"字，勤自见——**白描手法**写母亲勤劳，比"母亲很勤劳"有力百倍。

#### 案例 5：弹剑作歌（《冯谖客孟尝君》）⭐⭐

> **原文**：冯谖**三次弹剑**"**长铗归来乎！食无鱼 / 出无车 / 无以为家**"。
>
> **作用**：弹剑、揭剑——傲兀自信的"试探"姿态；三次反复，**层层递进**；为后文"市义三窟"的"扬"做铺垫，是欲扬先抑的关键细节。

#### 案例 6：嗔目辟易（《垓下之围》·项羽）⭐⭐⭐

> **原文**：东城快战，项羽"**嗔目而叱**，赤泉侯人马俱惊，**辟易数里**"。
>
> **作用**：夸张式的细节，把项羽神勇写到极致——一声怒目，敌人倒退数里，**侧面烘托**（借敌人反应）神勇绝伦。

#### 案例 7：赠马（《垓下之围》·项羽）⭐⭐

> **原文**：乌江边，项羽**把乌骓马赠给亭长**："吾骑此马五岁，所当无敌，尝一日行千里，不忍杀之，以赐公。"
>
> **作用**：临死赠马——英雄末路仍念故物，写出项羽的**自尊与儿女情长**，烘托苍凉氛围。

#### 案例 8："怒"四现（《纪念傅雷》·傅雷）⭐⭐

> **原文**：傅雷的"**怒**"——生活之怒、学术之怒、艺术之怒、气节之怒（最后以死殉其刚直）。
>
> **作用**：**以一字立骨**，反复出现，层层升华，塑造刚直不阿的知识分子典型；**克制含蓄**，不号啕而气节自现。

## 考点 4：抒情方式（重点）

### 一、三大抒情方式速查表

| 方式 | 别名 | 核心特征 | 答题效果词 | 本站典范 |
| --- | --- | --- | --- | --- |
| **直接抒情** | 直抒胸臆 | 直接说出情感 | **情感强烈、痛快淋漓** | 先妣"天乎痛哉" / 哭小弟"我不哭" |
| **间接抒情** | 寓情于事/景/物/议 | 把情感融入叙事描写 | **含蓄深沉、回味无穷** | 秋夜 / 牡丹的拒绝 / 都江堰 |
| **克制抒情** | 归有光/宗璞式 | 压抑情感，偶一爆发 | **越克制越有力** | 先妣事略 / 纪念傅雷 |

### 二、标准答题公式（三段式 · 背熟！）

```
本文运用了【直接 / 间接 / 克制】抒情方式，
（直接）作者直抒胸臆，"……"（引用），情感强烈；
（间接）把 XX 之情寓于 XX（事/景/物/议）之中，"……"（引用）；
（克制）压抑情感，仅以"……"（偶发直抒）爆发，其余寓情于事，
（效果）使情感【强烈动人 / 含蓄深沉 / 越克制越有力】，增强艺术感染力。
```

### 三、本站抒情案例库

#### ① 直接抒情案例

- 《先妣事略》："**世乃有无母之人，天乎！痛哉！**"——全文情感总爆发，仅此一处，**偶发直抒**反而冲击力极强。
- 《哭小弟》："**小弟，我不哭！**"——反转升华，由悲而壮。

#### ② 间接抒情案例（重点 · 寓情于 X）

| 寓情于 | 案例 | 效果 |
| --- | --- | --- |
| **寓情于事** | 《先妣事略》全篇（平实叙述传深情） | 越平淡越深沉 |
| **寓情于景** | 《秋夜》（秋夜意象寄孤寂抗争）/《前赤壁赋》（"清风明月"寄旷达） | 情景交融 |
| **寓情于物** | 《牡丹的拒绝》（牡丹寄高贵）/《秋夜》（枣树寄韧性） | 托物言志 |
| **寓情于议** | 《都江堰》（文化感悟含赞美）/《我与地坛》（生命哲思含深情） | 情理交融 |

#### ③ 克制抒情案例（最易考 · 重点）

- **《先妣事略》（典范）**：全文仅"伤哉""天乎痛哉"两处直抒，其余全用白描纪事——**"无意于悲而悲不自胜"**。
- **《纪念傅雷》**：以"怒"立骨，不号啕、不煽情——克制含蓄中见气节。
- **《我与地坛》（母亲部分）**：母亲"悄悄地找，偷偷地听"，不直接抒情——**越克制越催泪**。

::: warning 克制抒情答题要点
**克制抒情 ≠ 不抒情**，而是"压抑情感、偶一爆发"——
1. 指出大部分用白描 / 叙事 / 间接抒情（压抑）；
2. 指出偶发的直抒（爆发点）；
3. 强调"越克制越有力""无意于悲而悲不自胜"。
:::

## 七、五考点综合模拟卷（60 分钟）

### 一、选择题（每题 5 分，共 20 分）

**1. 下列对人物形象的分析，最准确的是：**
- A. 项羽是"百战百胜的将军"
- B. 冯谖是"贪得无厌的食客"
- C. 南霁云是"刚烈忠义的勇将" ✔
- D. 母亲是"软弱隐忍的妇女"

**2. 《西湖七月半》的主要叙事手法是：**
- A. 今昔对比
- B. 多场面描写
- C. 分类描摹 ✔
- D. 倒叙对比

**3. 《先妣事略》"饮螺致哑"这一细节的作用，分析不当的是：**
- A. 表现频繁生育与愚昧偏方的戕害
- B. 用一句叙述惨剧，不加评论

- A. 《哭小弟》"我不哭"

### 二、简答题（每题 12 分，共 36 分）

**5. 分析《垓下之围》多场面描写的运用及作用。**

> **参考答案**：本文用三个经典场面结构全篇：
> ① **垓下之围**（悲歌）：四面楚歌、悲歌别姬"力拔山兮气盖世"——奠定悲剧基调；
> ② **东城快战**（勇战）：溃围斩将、嗔目辟易——展现神勇本色；
> ③ **乌江自刎**（壮死）：拒渡赠马、自刎而死——完成英雄落幕。
> **作用**：三个场面**层层递进**，从悲凉到壮烈到苍凉，完整呈现项羽生命最后阶段，结构严密，富有戏剧张力，是"多场面描写"的典范。

**6. 分析《张中丞传后叙》叙议结合的特点。**

> **参考答案**：本文是**叙议结合**的典范：
> ① **前半议论**（段①-④）：韩愈为许远辩诬——驳"畏死降贼"（反驳论点）、驳"城陷自远始"（反驳论证）；
> ② **后半叙事**（段⑤⑥）：补叙南霁云（断指射塔）、张巡（博学就刑）事迹，用细节描写塑造群像；
> ③ **关系**：议论为叙事张本（先辩诬澄清史实），叙事印证议论（英雄事迹证明死守之忠义）——水乳交融。
> **效果**：既有逻辑严密的说服力，又有生动形象的感染力。

**7. 《纪念傅雷》如何以"怒"立骨？属于哪种抒情方式？**

> **参考答案**：本文以"**怒**"字立骨，反复四次：
> ① 生活之怒（不苟且）；② 学术之怒（不容浮夸）；③ 艺术之怒（不妥协庸俗）；④ 气节之怒（以死殉刚直）。
> 层层升华，由生活到学术到艺术到气节，**塑造刚直不阿的知识分子典型**。
> **抒情方式**：**克制抒情**——不号啕、不煽情，以一字立骨，含蓄中见深沉，越克制越有力。

### 三、论述题（22 分）

**8. 以《先妣事略》为例，综合分析其白描、细节、克制抒情的艺术特色。**

> **参考答案要点**：
> **① 白描手法**：通篇用最简练的文字勾画特征，不加渲染——
> - 生育用纪年罗列（"年十六来归，逾年生女"）——干瘪纪事见频繁生育之苦；
> - 劳动用细节（"灯火荧荧，每至夜分"）——不赞一"勤"字，勤自见。
>
> **② 细节描写**：
> - "举之尽，喑不能言"（饮螺致哑）——一句叙述惨剧；
> - "画像拼凑"——以拼凑不出写丧母之痛。
>
> **③ 克制抒情**：全文仅"伤哉""天乎痛哉"两处直抒，其余寓情于事。
>
> **④ 综合效果**：**越平淡越深沉**，"无意于悲而悲不自胜"，比号啕大哭更催人泪下——是悼亡散文的经典之作。

### 四、赏析题（22 分）

**9. 赏析《我与地坛》中"母亲"形象的塑造艺术。**

> **参考答案要点**：
> **① 克制抒情的典范**：母亲"**悄悄地**找，**偷偷地**听"，不直接抒情——隐忍克制的深爱。
> **② 白描传情**："这园中不单是处处都有我的车辙，有过我的车辙的地方也都有过母亲的脚印"——平实叙述，深沉催泪。
> **③ 以小见大**：通过母亲的形象，升华到"**怎样活**"的生命思考——母爱是作者走出绝望的精神支柱。
> **④ 矛盾统一**：母亲既是"理解儿子的母亲"（深爱），又是"独自承受焦虑的母亲"（克制）——立体丰满。
> **⑤ 典型意义**：母亲是**所有隐忍深爱、为子女牺牲的中国母亲**的典型，具有强烈的普遍感染力。
