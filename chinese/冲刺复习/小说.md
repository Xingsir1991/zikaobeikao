# 冲刺复习 · 小说（对接考点）

> 本课件针对五个核心考点精准设计：
> **考点 1：人物形象分析**（最高频）—— 性格 + 例证 + 典型意义
> **考点 2：情节/结构分析**（高频）—— 线式 / 双线 / 梦幻 / 欧·亨利式 / 诗化
> **考点 3：环境描写作用**（高频）—— 渲染 / 烘托 / 推动 / 象征
> **考点 4：艺术手法**（重点）—— 对比 / 象征 / 白描 / 侧面 / 以物衬人 / 以小见大
> **考点 5：结尾艺术**（高频）—— 欧·亨利式 / 反讽 / 点题 / 开放式
> 整合本站 7 篇小说 + 《小说的艺术特点》方法论课件，**直击考场**。

::: warning 冲刺策略
本课件不讲理论铺垫，**只讲考点 + 答法 + 例证 + 模拟**。
复习时按"考点 → 答题公式 → 本站例证 → 模拟练习"四步走。
所有手法题统一用 **三段式**：**指明手法 → 分析文本 → 点明效果**。
:::

## 考点 1：人物形象分析（必考·最高频）

### 一、人物形象题的"标准答题公式"（背熟！）

> **总—分—结 三段式**（适用于所有"分析 XX 人物形象"题）

```
【总】XX 是一个 【性格特征 1】+【性格特征 2】+【身份定位】 的形象。
【分】① 【特征 1】：结合原文……（引具体言行），可见其……
    ② 【特征 2】：结合原文……（引具体言行），可见其……
    ③ 【特征 3（可选）】：……
    （每条都要"特征 + 例证"，光说性格不举例 = 不给分）
【结】这一形象反映了……（社会面貌 / 作者态度 / 典型意义），具有……的艺术价值。
```

::: warning 三大丢分点
1. **只罗列性格词，不举例**（如"她天真、善良、倔强"——空话，必丢分）
2. **不写身份定位**（"贵族少女""底层车夫""旧时代英雄"——这是"典型意义"的入口）
3. **不点典型意义**（人物折射了什么社会现实 / 作者什么态度——这是升华分）
:::

### 二、本站 7 篇核心人物速写（背诵版）

#### ① 沙子龙（断魂枪）—— 旧时代英雄的没落与坚守 ⭐⭐⭐

- **身份**：清末民初由盛转衰的旧时代武林英雄，昔日"神枪沙子龙"，如今客栈掌柜。
- **特征**：
  1. **武艺绝顶**：二十年苦练"五虎断魂枪"，威震西北，"没遇见过敌手"。
  2. **清醒自尊**：深知"今天是火车、快枪"，遂改镖局为客栈，白天不谈武艺。
  3. **孤傲固执**：夜练六十四枪，对徒弟和孙老者一概"不传"，"都跟我入棺材"。
- **典型意义**：凝聚着传统文化在现代工业文明冲击下的**悲剧性坚守**——老舍既同情其珍视尊严，又批判其抱残守缺。

#### ② 婴宁（婴宁）—— 以"笑"立骨的三重统一 ⭐⭐⭐

- **身份**：狐女，由鬼母抚养长大，嫁与王子服。
- **特征**（以"笑"贯穿五重层次）：
  1. **憨笑（天真少女）**："嗤嗤笑不已""狂笑欲堕"，"呆痴裁如婴儿"。
  2. **痴笑（娇憨狐女）**："我不惯与生人睡""大哥欲我共寝"——不解世故。
  3. **黠笑（智黠护身）**：西邻子事件中"指墙底，笑而下"，设枯木蝎子陷阱惩淫——"其黠孰甚焉"。
  4. **含笑不笑（成熟后）**：受官司惊吓后"矢不复笑"。
  5. **反笑为哭（孝义本色）**：夜诉身世，迁葬鬼母——"隐于笑者"。
- **典型意义**：赞美纯真自由的人性，反思礼教束缚；揭示"笑"是险恶世态中的生存智慧。

#### ③ 贾宝玉（宝黛吵架）—— 痴情叛逆的贵族公子 ⭐⭐⭐

- **特征**：
  1. **用情专一**：心里眼里只有黛玉，凡闺英闱秀"皆未有稍及林黛玉者"。
  2. **叛逆性**：嗔张道士提亲，**砸通灵玉**以抗"金玉之说"。
  3. **痴情痴狂**："立刻因你死了也情愿"；"什么捞什骨子，我砸了你完事！"
  4. **软弱性**：被袭人一劝就后悔；无力正面表达，只能以毁物证心。
- **典型意义**：折射封建礼教压抑下青年无法直接表达真情的痛苦，是《红楼梦》"打破传统思想和写法"的载体。

#### ④ 林黛玉（宝黛吵架）—— 敏感多情的贵族少女 ⭐⭐⭐

- **特征**：
  1. **真情专一**：悔剪玉穗，"还得我穿了他才带"。
  2. **敏感多疑**：最忌"金玉"之说；金麒麟事件让她敏感。
  3. **嘴尖舌利**："我也知道白认得了我，那里象人家有什么配的上呢"——一句三刺。
  4. **倔强自尊**："我不希罕"；**剪玉穗**对宝玉砸玉。
- **典型意义**：与宝玉共同揭示"求近反疏"的爱情悖论。

#### ⑤ 约纳（苦恼）—— 孤独的俄罗斯底层小人物 ⭐⭐⭐

- **特征**：
  1. **深沉的丧子之痛**："我的儿子这个星期死了"，四处寻找倾听者。
  2. **逆来顺受**：被骂被揍还"嘻嘻"地笑，不反抗。
  3. **孤独无助**：四次向人倾诉失败，最终向**瘦马**诉说。
  4. **善良木讷**：用"母马失驹"类比讲自己的痛——朴素深刻。
- **典型意义**：是俄罗斯底层小人物的缩影；其孤独是**现代人类共通的存在性孤独**。

#### ⑥ 德拉 / 吉姆（麦琪的礼物）—— 痴情奉献的贫穷夫妻 ⭐⭐⭐

- **德拉**：① 深爱丈夫——卖掉引以为荣的长发买表链；② 细致体贴——专挑"素静而有价值"的表链；③ 自尊敏感——还价被骂"死抠"而"两颊发烧"；④ 柔弱坚韧——剪发后忐忑自嘲。
- **吉姆**：① 深爱妻子——卖**祖传三代金表**买梳子（比头发牺牲更重，金表不可复得）；② 宽容温柔——"不能叫我对妻子的爱减少一丝一毫"；③ 超越物质——"把圣诞节礼物放到一边，暂时保留着吧"。
- **典型意义**：物质贫乏中的精神高贵，"**最不明智**的物质计算"反衬"**最最聪明**"的爱的智慧。

#### ⑦ 香雪（哦，香雪）—— 清纯倔强的山村少女 ⭐⭐⭐

- **特征**：
  1. **清纯纯净**："洁如水晶的眼睛""仿佛一分钟前才诞生的面孔"。
  2. **腼腆胆小却倔强自尊**：看火车缩到最后，却敢跳上火车、走三十里夜路；"台儿沟再穷，她也从没白拿过别人的东西"。
  3. **追求知识、向往尊严**：不像凤娇求发卡纱巾，只执着于"铅笔盒"——"上大学、坐上火车到处跑……再也不会被人盘问"。
  4. **精神觉醒**：夜路归家升起"从未有过的骄傲"。
- **典型意义**：礼赞山村青年对现代文明、知识与尊严的纯真向往，是改革开放初期精神觉醒的缩影。

#### ⑧ 卢生（枕中记）—— 热衷功名的世俗士子 ⭐⭐

- **特征**：
  1. **功名焦虑**："大丈夫生世不谐，困如是也"；理想是"建功树名，出将入相，列鼎而食，选声而听"。
  2. **世俗趋避**：梦中"立碑颂德"见好名，遭诬陷"骇惧哭泣"——并非完人。
  3. **由迷到悟**：梦觉"愕然良久"，"宠辱之道，穷达之运……尽知之矣"。
- **典型意义**：反映唐代士子对功名的普遍热衷与对宦海无常的隐忧。

### 三、考点 1 模拟练习（含答案）

#### 【模拟 1 · 简答】分析沙子龙这一人物形象，并谈谈作者对他的态度。

> **标准答案：**
> 【总】沙子龙是清末民初由盛转衰的**旧时代武林英雄**的代表。
> 【分】① **武艺绝顶**：二十年苦练"五虎断魂枪"未遇敌手，威震西北，"神枪沙子龙"五个字名扬天下。
> ② **清醒自尊**：深知"今天是火车、快枪"，便将镖局改为客栈，白天不谈武艺与往事。
> ③ **孤傲固执**：夜间独自关院门练枪，对求艺的徒弟和孙老者一概拒绝，"那条枪和那套枪都跟我入棺材"，"不传！不传！"
> 【结】这一形象凝聚着**传统文化在现代工业文明冲击下的悲剧性坚守**。作者态度是辩证的：**同情**其珍视尊严、为其唱挽歌；**批判**其抱残守缺、与时代脱节；**感慨**历史无情。

#### 【模拟 2 · 简答】婴宁的"笑"在全篇中有哪些不同层次？这一形象有何典型意义？

> **标准答案：**
> 婴宁的"笑"贯穿全篇，五重层次：
> ① **憨笑**（天真少女）：出场"嗤嗤笑不已""狂笑欲堕"，天性纯真。
> ② **痴笑**（娇憨狐女）："我不惯与生人睡"等痴语，不解世故。
> ③ **黠笑**（智黠护身）：西邻子事件"指墙底，笑而下"，设枯木蝎子陷阱惩淫。
> ④ **含笑不笑**（成熟后）：受官司惊吓后"矢不复笑"。
> ⑤ **反笑为哭**（孝义本色）：夜诉身世，迁葬鬼母。
> **典型意义**：婴宁以"笑"为外壳，集天真、智黠、孝义于一身，是"隐于笑者"——既赞美纯真人性，又暗含对礼教束缚的反思。

#### 【模拟 3 · 选择】下列对人物形象的判断，正确的一项是：

- A. 沙子龙是积极拥抱现代文明的改革者 ✅（是隐退守艺的旧时代英雄）
- B. 约纳是俄罗斯底层小人物，其孤独具有人类共通性 ✔
- C. 香雪追求发卡纱巾，与凤娇本质相同 ✅（香雪求铅笔盒＝精神追求）
- D. 卢生梦觉后更热衷功名 ✅（是由迷到悟，"窒吾欲"）

> **答案：B**

## 考点 2：情节 / 结构分析（高频）

### 一、六大结构类型速查表（背熟！）

| 结构类型 | 核心特征 | 本站案例 | 答题效果词 |
| --- | --- | --- | --- |
| **单线推进** | 一条主线到底 | 《断魂枪》孙老者挑战—求艺 | **脉络清晰、层层推进** |
| **双线结构** | 明暗两线交织，结尾交汇 | 《麦琪的礼物》德拉线 / 吉姆线 | **对称、悬念、反转** |
| **梦幻框架** | 现实—入梦—梦觉三段 | 《枕中记》黄粱一梦 | **虚实相生、时空浓缩** |

| **诗化（散文化）** | 淡化情节，强化意境 | 《哦，香雪》铅笔盒情结 | **意境优美、抒情性强** |
| **心理线索** | 以心理变化为内线 | 《宝黛吵架》"假情试探"恶性循环 | **心理深度、揭示本质** |

### 二、各结构的"标准答题公式"

#### ① 双线结构（必背·高频）

```
本文采用双线结构，
【明线（详写）】是 XX：……（具体情节），
【暗线（略写）】是 XX：……（具体情节，常结尾点破），
两线在结尾交汇，形成……（反转 / 对称 / 主题升华）。
效果：① 对称——凸显……；② 悬念——制造……；③ 主旨——反衬……。
```

**本站案例（麦琪的礼物）：**
- **明线（德拉）**：数钱 → 卖发 → 买白金表链 → 剪发补救 → 送给吉姆。
- **暗线（吉姆）**：卖祖传金表 → 买玳瑁梳子 → 带回家（全程暗写，结尾点破）。
- **交汇**：结尾双礼亮相——**表链无表，梳子无发**，反转爆发。
- **效果**：① **对称**——双方牺牲等价，凸显爱情的对等；② **悬念**——吉姆线暗写制造反转冲击；③ **主旨**——交汇于"暂时无用"的礼物，反衬**爱的无用之用**。

#### ② 梦幻框架（幻化结构）

```
本文采用"现实—入梦—梦觉"的梦幻框架，
【现实层】……（开篇情境），
【入梦枢纽】……（奇幻道具 / 入口），
【梦中层】……（梦中历尽什么），
【梦觉层】……（醒来参照，点破虚幻）。
效果：① 时空浓缩——以短现实时间对长梦中时间，形成反差；
     ② 虚实相生——幻中见真；③ 揭示……哲理主题。
```

**本站案例（枕中记）：**
- 现实层：邯郸邸舍，卢生叹困，"主人蒸黄粱为馔"。
- 入梦枢纽：吕翁授青瓷枕，"窍其两端"，卢生"举身而入"。
- 梦中层：历婚姻、科举、出将、入相、遭诬流贬、封公寿终，凡二十六年（一生）。
- 梦觉层："主人蒸黄粱尚未熟"——梦中八十年，现实一锅饭未熟。
- **效果**：时空强烈反差，凸显"人生如梦"；虚实相生（梦中萧嵩、李林甫皆真实人物）；揭示"宠辱穷达得丧死生"皆幻的哲理，凝练为成语"黄粱一梦"。

#### ③ 欧·亨利式结构（反转式）

```
本文采用反转式结构，前文层层伏笔，结尾陡然反转。
【铺垫】前文埋下……伏笔（具体），
【跟随】读者跟随……（明线人物）以为……，
【反转】结尾揭示……，与读者预期形成强烈反差。
效果：出人意料又在情理之中，形成强烈心灵震撼。
```

**本站案例（麦琪的礼物）：** 见考点 5 详述。

#### ④ 诗化（散文化）结构

```
本文是"诗化小说"（散文化小说），
【淡化情节】全篇无强烈戏剧冲突，只有……的淡化主线，
【内在线索】以……（情感 / 心理变化）为内在线索，
【重意境】大量环境描写、抒情性议论，故事性被有意削弱。
效果：意境优美、抒情性强，以意境代冲突、以情感代情节。
```

**本站案例（哦，香雪）：**
- 淡化主线：火车进山 → 姑娘换物 → 香雪求铅笔盒 → 上车换盒 → 走三十里夜路归家。
- 内在线索：香雪对"铅笔盒"的精神追求与心理变化（害怕→犹豫→骄傲）。
- **四大诗化特征**：① 情节淡化结构松散；② 环境描写突出；③ 语言富有诗意；④ 重意境重情感。

::: tip 诗化小说谱系（易考比较）
废名 → 沈从文《边城》 → 孙犁《荷花淀》（荷花淀派） → 汪曾祺《受戒》 → 铁凝《哦，香雪》
注意：《苦恼》《断魂枪》偏写实，不可归入诗化小说。
:::

#### ⑤ 以细节立骨（一线串珠）

```
本文以"XX"这一核心细节贯穿全篇，
全文反复出现"XX"凡数处（列举层次），
形成一条贯穿全篇的"XX 曲线"。
效果：① 形象鲜明——"XX"成为人物符号；
     ② 暗示变化——由 X 到 Y 暗示人物成长 / 蜕变；
     ③ 以一细节统摄全篇，结构精巧。
```

**本站案例（婴宁·以笑立骨 / 苦恼·四次倾诉）：**
- 《婴宁》：全文写"笑"数十处，有"嗤嗤笑""吃吃笑""狂笑""浓笑""憨笑"等不同形态；笑由多变少、由有到无（憨笑→黠笑→矢不复笑→反笑为哭），形成"笑的曲线"。
- 《苦恼》：以"倾诉"为线索，四次向人倾诉失败（军人→青年→扫院人→车夫），层层递进，最后向马诉说形成高潮。

### 三、考点 2 模拟练习（含答案）

#### 【模拟 1 · 简答】分析《麦琪的礼物》的双线结构及其作用。

> **标准答案：**
> 本文采用**双线结构**：
> ① **明线（详写）——德拉线**：数钱 → 卖发 → 买白金表链 → 剪发补救 → 送给吉姆。

> - **对称**：两条牺牲之线对称展开，凸显爱情的对等（金表祖传三代、长发披到膝下，皆为至宝）。
> - **悬念与反转**：吉姆线暗写，结尾双线交汇才点破，**制造强烈的反转冲击**。
> - **主旨升华**：双线交汇于"暂时无用"的礼物，反衬**爱的无用之用**——物质可空，精神无价。

#### 【模拟 2 · 简答】《枕中记》的梦幻框架有何艺术效果？

> **标准答案：**

> ② **虚实相生**：志怪的青瓷枕（虚）+ 写实的萧嵩、裴光庭、李林甫、罗希奭等真实人物（实）——幻中见真，是唐传奇"用传奇法而以志怪"的典范。
> ③ **伏笔照应**：开篇"主人蒸黄粱为馔"埋伏，结尾"尚未熟"呼应，形成"现实—梦境—现实"的严密闭环。
> ④ **主题升华**："黄粱"成为荣华富贵的隐喻，凝练为成语"黄粱一梦"，强化对功名利禄的消解与反思。


### 三、本站四大经典环境描写案例

#### 案例 1：《苦恼》—— 彼得堡冬夜（自然环境）

> **原文**："暮色昏暗。大片的湿雪绕着刚点亮的街灯懒洋洋地飘飞，落在房顶、马背、肩膀、帽子上，积成又软又薄的一层。车夫约纳·波塔波夫周身雪白，像是一个幽灵。"

- **渲染气氛**：暮色、湿雪、街灯——营造**冷寂、悲凉、麻木**的基调。
- **烘托人物**：约纳"周身雪白，像是一个幽灵"——写其麻木如死人，与雪景融为一体。
- **象征暗示**：现代都市的冰冷正是**人际冷漠**的物化——"那种苦恼是广大无垠的"。

#### 案例 2：《哦，香雪》—— 三十里夜路（自然环境·诗化典范）⭐⭐⭐

> **原文**："一轮满月升起来了，照亮了寂静的山谷……群山被月光笼罩着，像母亲庄严、神圣的胸脯；那秋风吹干的一树树核桃叶，卷起来像一树树金铃铛……她不再害怕了……大山原来是这样的！月亮原来是这样的！"

- **烘托心理（核心）**：从害怕（黑黝黝的大山、窸窸窣窣的小树林）到不再害怕（明净的月亮、歌唱的核桃叶），环境变化**外化**香雪从恐惧到骄傲的心理成长。
- **营造诗意意境**：月光、群山、小溪、核桃叶构成庄严神圣的山月图，是诗化小说意境美的集中体现。
- **象征精神觉醒**：群山"像母亲庄严、神圣的胸脯"、"第一次认出养育她成人的山谷"——象征香雪重新认识故乡、完成精神洗礼。
- **以景结情**：与结尾"古老的群山被感动得颤栗"呼应，将个人成长升华为整个山村的觉醒。

#### 案例 3：《婴宁》—— 山中幽居（自然环境·典型环境塑造典型人物）⭐⭐⭐

> **原文**："乱山合沓，空翠爽肌，寂无人行，止有鸟道"；"门前皆绿柳，墙内桃杏尤繁，间以修竹，野鸟格磔其中"；"白石砌路，夹道红花"；"粉壁光明如镜，窗外海棠枝朵探入室内"。

- **烘托人物性格（核心）**：清新、洁净、幽雅的环境，正是婴宁爱花、清纯脱俗性格的**物化**——是"**典型环境塑造典型人物**"的范例。
- **暗示人物身份**：深山幽谷、鸟道无人、花木繁盛——非人间凡境，**暗伏婴宁"狐仙"身份**，为后文揭谜铺垫。
- **寄托审美理想**：山中幽居是蒲松龄心中的"桃花源"，寄托对远离世俗礼法、回归自然的理想。

#### 案例 4：《断魂枪》—— 开篇时代背景（社会环境）⭐⭐⭐

> **原文**："东方的大梦没法子不醒了。炮声压下去马来与印度野林中的虎啸……龙旗的中国也不再神秘，有了火车呀，穿坟过墓的破坏着风水……今天是火车、快枪，通商与恐怖。"

- **交代时代背景（核心）**：清末民初中国传统社会被西方工业文明全面冲击的历史转折期。
- **渲染悲凉氛围**："东方的大梦没法子不醒了""连祖先与祖先所信的神明全不灵了"——奠定全篇**悲凉、沉重、无可奈何**的基调。
- **暗示人物命运**：直接决定沙子龙"镖局改客栈"的命运——**任何神妙的武艺都已成"昨夜的梦"**，为后文"不传"的固执做必然性的铺垫。
- **揭示主旨**：以"龙旗的中国也不再神秘"点破题旨——写的不是一个人的故事，而是**一个民族、一种文化在现代化冲击下的命运**。

## 考点 4：艺术手法（重点·高频）

### 一、八大艺术手法速查表（背熟！）

| 手法 | 核心特征 | 本站案例 | 答题效果词 |
| --- | --- | --- | --- |
| **① 对比** | 两事物正反对照 | 断魂枪：沙/孙/王三人物对比 | **鲜明突出、立体凸显** |
| **② 象征** | 具体物象寄寓抽象义 | 香雪：铅笔盒=知识/尊严 | **含蓄深沉、引发联想** |
| **③ 白描** | 简练文字勾勒，不加渲染 | 断魂枪：人物肖像数笔传神 | **简练传神、画眼睛** |

| **⑤ 以物衬人** | 物作人的镜像 | 苦恼：瘦马=约纳镜像 | **同类相映、反讽至深** |
| **⑥ 以小见大** | 小题材折射大时代 | 麦琪：一对夫妻→爱的奉献 | **见微知著、以少胜多** |
| **⑦ 心理描写** | 直接揭示内心 | 宝黛：大段心理独白 | **入木三分、揭示本质** |
| **⑧ 细节描写** | 抓典型细微处 | 婴宁：笑；香雪：四十个鸡蛋 | **传神写照、画龙点睛** |

### 二、手法题统一"标准答题公式"（三段式·背熟！）

```
【第一段·指明手法】本篇（本句）运用了 XX 手法（如对比 / 象征 / 白描……），
                  （简释该手法如何运用）。
【第二段·分析文本】结合原文"……"（引具体语句），分析手法如何展开：
                  （具体说明对比项 / 象征本体喻体 / 白描笔法等）。
【第三段·点明效果】这一手法既塑造了……（人物形象），又推动了……（情节），
                  深化了……（主题），是本篇的突出特色。
```

::: warning 三段式必背口诀
**指明手法 → 分析文本 → 点明效果**（明手法 → 析文本 → 说效果）
任何手法题都套这个公式，不会写就按这三步走。
:::

### 三、八大手法详解 + 本站案例

#### ① 对比手法 ⭐⭐⭐（最高频）

**公式**：本文运用对比，将 XX 与 XX 作对比（具体对比项），形成强烈反差，突出了 XX。

**本站案例**：

| 《断魂枪》 | 过去（神枪威风）vs 现在（镖局改客栈） | 时代之变、英雄没落 |
| 《苦恼》 | 马的温情回应 vs 人的冷漠敷衍 | 人对人不如马对人，反讽至深 |
| 《宝黛》 | 宝玉砸玉 vs 黛玉剪穗 | 互为镜像，写求近反疏 |
| 《宝黛》 | 贾母（慈厚）vs 凤姐（跋扈，打小道士） | 凸显贾母仁厚 |

#### ② 象征手法 ⭐⭐⭐

**公式**：本文以 XX 为象征，表层是……，深层象征……（多重意义），含蓄深沉地表达了……。

**本站案例（核心象征物）：**

| 小说 | 象征物 | 象征意义 |
| --- | --- | --- |
| 《断魂枪》 | **五虎断魂枪** | ① 中国传统武术极致；② 整个中国传统文化；③ "断魂"双关——断敌魂亦断己魂，暗示传统技艺必然衰亡；④ 沙子龙的精神与尊严 |
| 《断魂枪》 | **镖局改客栈** | 一个时代的落幕——从"走江湖"到"迎来送往" |
| 《断魂枪》 | **火车、快枪** | 现代工业文明的冲击 |
| 《断魂枪》 | **夜练六十四枪** | 旧时代英雄最后的尊严 |
| 《哦，香雪》 | **铅笔盒**（核心） | ① 知识（上学读书）；② 现代文明（大城市工业品）；③ 人的尊严（不再因贫穷被盘问）——寄托改变命运的全部梦想 |
| 《哦，香雪》 | **火车** | 外部现代工业文明的闯入 |
| 《哦，香雪》 | **大山** | 传统、封闭，又是母亲般的温厚庇护 |
| 《哦，香雪》 | **三十里夜路** | 成长的精神洗礼、"成年礼" |

::: tip 象征 vs 比喻（易混辨析）
- **比喻**：临时性、局部性（"头发像棕色的瀑布"——只此一处）。
- **象征**：贯穿性、整体性，往往是全篇的核心意象（铅笔盒、断魂枪、瘦马——贯穿全文，多重意义）。
答题时：贯穿全文的核心物象写"**象征**"，局部修辞写"**比喻**"。
:::

#### ③ 白描手法 ⭐⭐⭐（老舍特色）

**定义**：用最简练的文字勾勒人物或场景，不施烘托、不加渲染，"画眼睛"传神。

**公式**：本文运用白描，仅以"……"（引简练文字）数笔勾勒，不事渲染，便写出 XX 的神韵，简练传神。

**本站案例（《断魂枪》白描典范）：**
- **人物肖像**："短瘦、利落、硬棒，两眼明得像霜夜的大星"——八个字写尽沙子龙昔日的精悍。
- **孙老者**："脸上窝窝瘪瘪，眼陷进去很深，几根细黄胡"——三笔写出干巴老头。
- **王三胜**："大个子，一脸横肉，弩着对大黑眼珠"——市井气顿现。
- **动作**："给手心一口吐沫，抄起大刀来"——王三胜卖艺栩栩如生。
- **场景**："院子里有几支楼鸽"——一笔写出后院的冷清孤寂。

#### ④ 侧面描写（烘托） ⭐⭐⭐

**定义**：不直接写本体，而写周围人物的反应、环境的反应来衬托本体。

**公式**：本文运用侧面描写（烘托），通过写 XX（他者）的反应/评价，侧面衬托出 XX（本体）的 XX 特征，以宾衬主，含蓄有力。

**本站案例：**
- 《婴宁》：① **未见其人先闻其声**——"嗤嗤笑不已"先于婴宁出场，以声写人；② 通过婢女"贼腔未改"、王子服的痴、吴生的怪、邻女的争承迎，侧面烘托婴宁的魅力。
- 《断魂枪》：① 以徒弟们到处吹腾"神枪沙"烘托沙子龙的威名；② 以孙老者献艺"满院乱飞的燕子忽然归了巢"（已是顶尖高手）烘托沙子龙**更高深**的武境——"神枪"之名不言自立。

#### ⑤ 以物衬人 ⭐⭐⭐（契诃夫特色）

**定义**：以动物 / 无生命之物作为人物的镜像、同类，以物的处境衬人的处境。比侧面描写更进一步——物不只是"衬托"，而是人的"灵魂同类"。

**公式**：本文运用以物衬人，写 XX（物）的 XX 特征，与 XX（人物）的 XX 特征互为镜像（指明）；结尾 XX（具体互动），反衬……（运用）；以物的 XX 对比人的 XX，反讽至深，把……主题推向高潮（效果）。

**本站案例（《苦恼》瘦马）：**
- **镜像**：开篇写瘦马"瘦骨棱棱、棍子般直挺挺的腿、呆呆不动、多半在想心思"——与约纳衰老、孤独、被遗弃的形象**完全对应**。瘦马是约纳的镜像。
- **反衬**：结尾约纳向马倾诉，马"嚼草、听着、**向主人手上呵气**"温情回应 vs 前四次向人倾诉的冷漠敷衍——**人对人不如马对人**。
- **升华**："约纳讲得入了迷，就把他心里的话统统对它讲了"——**只有同类才懂得同类**，把"人类孤独"主题推向高潮。

#### ⑥ 以小见大 ⭐⭐⭐

**定义**：通过小人物、小题材、小场景折射大时代、大主题。

**公式**：本文以 XX（小人物 / 小事）为题材，折射了 XX（大时代 / 大主题），见微知著，以少胜多。

**本站案例：**

| 小说 | "小" | "大" |
| --- | --- | --- |
| 《麦琪的礼物》 | 一对贫穷夫妻的圣诞礼物 | 爱的奉献、物质贫乏中的精神高贵 |
| 《苦恼》 | 一个老车夫丧子之痛 | 现代人类共通的存在性孤独 |
| 《断魂枪》 | 一杆枪、三个人、一夜星光 | 整个传统文化在现代化冲击下的命运 |
| 《哦，香雪》 | 一只铅笔盒、三十里夜路 | 山村青年精神觉醒、改革开放时代浪潮 |
| 《婴宁》 | 一个狐女的笑 | 对自然人性的礼赞、对礼教束缚的反思 |

#### ⑦ 心理描写 ⭐⭐⭐（曹雪芹巅峰）

**公式**：本文运用直接心理描写（心理独白），直接揭示人物 XX 的内心活动，"……"（引原文），把……写得入木三分。

**本站案例（《宝黛吵架》中国古典小说心理描写巅峰）：**
- **大段直接心理独白**：作者直接进入人物内心——"因你也将真心真意瞒了起来，只用假意……如此两假相逢，终有一真"。
- **对称式心理对照**：宝玉想"你竟心里没我"，黛玉想"可知你心里时时有'金玉'"——两人各自的逻辑对称展开，互为镜像。
- **叙述者评论介入**："两个人原本是一个心，但都多生了枝叶，反弄成两个心了"；"都是求近之心，反弄成疏远之意"——以哲理性语言点破情感本质。
- **以生理喻心理**（《苦恼》）："如同那个青年人渴望喝水一样，他渴望说话"——深刻。

#### ⑧ 细节描写 ⭐⭐

**公式**：本文抓住"……"这一典型细节，传神写出 XX 的 XX 特征，画龙点睛。

**本站案例：**
- 《婴宁》：笑的数十种形态（嗤嗤/吃吃/狂笑/浓笑/憨笑）。
- 《哦，香雪》："四十个鸡蛋"反复出现——凸显交换代价与倔强自尊；"哒哒"声——同桌铅笔盒合上的声音，成心头挥之不去的诱惑与屈辱。
- 《断魂枪》：沙子龙"对着墙角立着的大枪点了点头"——无声胜有声；"用手指慢慢摸着凉滑的枪身"——"慢慢"写出珍重与诀别。
- 《宝黛》：宝玉"咬牙恨命往地下一摔"；黛玉"顺手抓起一把剪子"——痴狂与尖刻。

### 四、考点 4 模拟练习（含答案）

#### 【模拟 1 · 论述】分析《苦恼》中"瘦马"的作用。

> **标准答案：**
> 瘦马在文中有三重作用：
> ① **以物衬人（镜像作用）**：开篇写瘦马"瘦骨棱棱、棍子般直挺挺的腿、呆呆不动、多半在想心思"——与约纳衰老、孤独、被遗弃的形象**完全对应**，瘦马是约纳的**镜像**。
> ② **反衬（对比作用）**：马的"嚼草、听着、呵气"温情回应 vs 前四次向人倾诉的冷漠敷衍——**人对人不如马对人**，反讽至深。
> ③ **升华主题**：结尾约纳"讲得入了迷，统统对它讲了"——**只有同类才懂得同类**。人对人的痛苦视而不见，反而是不能言的马给予回应，把"人类孤独"的主题推向高潮。
> 这是契诃夫"以物衬人"手法的最高境界——不是简单写物，而是让物成为人的**灵魂同类**。

#### 【模拟 2 · 论述】"五虎断魂枪"在小说中有何象征意义？

> **标准答案：**
> "五虎断魂枪"是贯穿全文的核心意象，具有多重象征意义：
> ① **象征中国传统武术的极致**：沙子龙二十年苦练的绝技，"神枪沙子龙"五个字"没遇见过敌手"——代表传统武术的最高境界。
> ② **象征整个中国传统文化**：枪与"镖局""龙旗的中国""江湖上的智慧与黑话、义气与声名"连为一体，是整个传统江湖文化体系的浓缩。
> ③ **"断魂"二字的双关**：既"断敌之魂"（神妙绝伦），又"断己之魂"（在火车快枪时代注定随主人入棺材）——暗示传统文化的必然衰亡。
> ④ **象征沙子龙的精神与信仰**："摸着凉滑的枪身"——枪即人，人枪合一；枪的"凉、滑、硬而发颤"就是沙子龙冷峻、坚硬、却在时代中颤栗的心境。
> ⑤ **象征旧时代英雄的尊严**：沙子龙不肯"传"，正是要守护这杆枪不被时代消费——"断魂枪"由此升华为旧时代英雄最后的尊严象征。

#### 【模拟 3 · 简答】《哦，香雪》中"铅笔盒"和"火车"各有什么象征意义？

> **标准答案：**
> **铅笔盒**——① 表层：装有吸铁石的自动铅笔盒，来自大城市的文具；② 深层：象征 **知识（上学读书）、现代文明（大城市工业产品）、人的尊严（不再因贫穷被盘问）**；③ 香雪用 40 个鸡蛋换铅笔盒，寄托了她改变命运、走出大山、获得尊严的全部梦想，是全篇的核心象征。
> **火车**——① 表层：经过台儿沟的绿皮列车；② 深层：象征 **外部现代工业文明的闯入**，是连接封闭山村与外部世界的纽带；③ "挟带着来自山外的陌生、新鲜的清风"，既带来物质（挂面、发卡），更唤醒山村青年的精神向往；④ 香雪"坐上火车到处跑"的幻想，正是山村青年走出大山的渴望。

#### 【模拟 4 · 选择】下列对手法的判断，正确的是：

- A. 《苦恼》以瘦马衬约纳，是侧面描写 ✅（是以物衬人，更深刻）
- B. 《断魂枪》人物肖像"两眼明得像霜夜的大星"是白描 ✔
- C. 《哦，香雪》铅笔盒是比喻 ✅（是贯穿全文的核心象征）
- D. 《麦琪的礼物》双线结构 = 心理线索 ✅（是明暗双线结构）

> **答案：B**

## 考点 5：结尾艺术（高频）

### 一、五种结尾类型速查表

| 结尾类型 | 核心特征 | 本站案例 | 答题效果词 |
| --- | --- | --- | --- |
| **① 欧·亨利式** | 出人意料 + 情理之中 | 麦琪的礼物：表链无表、梳子无发 | **震撼反转、含泪微笑** |
| **② 反讽式** | 结尾事实与预期相反 | 苦恼：人对人不如马对人 | **反讽至深、控诉有力** |
| **③ 点题式** | 结尾点明主旨 | 婴宁："隐于笑者矣"；麦琪："这两个人就是贤人" | **画龙点睛、升华主旨** |
| **④ 开放式（留白）** | 戛然而止，余味无穷 | 断魂枪："不传！不传！" | **余韵悠长、引人深思** |
| **⑤ 抒情式（诗化）** | 结尾直接抒情礼赞 | 香雪："哦，香雪！香雪！" | **情感升华、礼赞高潮** |

### 二、结尾题的"标准答题公式"（背熟！）

```
本文结尾运用了 XX（手法 / 形式），
【内容上】写出了……（具体内容），
【手法上】采用……（反复 / 留白 / 反转 / 点题），
【人物上】塑造 / 揭示了人物……，
【主旨上】深化了……主题，
【结构上】与开头……呼应，首尾圆合，
【效果上】……（震撼 / 含泪微笑 / 余韵悠长）。
```

### 三、本站五大经典结尾案例

#### 案例 1：《麦琪的礼物》—— 欧·亨利式结尾 ⭐⭐⭐（最高典范）

> **结尾**：吉姆送的是一套**玳瑁发梳**（德拉梦寐以求、配她那头美发的梳子），可长发已剪；德拉送的**白金表链**，可金表已被吉姆卖掉。"我把圣诞节礼物放到一边，暂时保留着吧……我将表卖了钱给你买了梳子。"叙述者议论："**这两个人就是《圣经》里提到的贤人。**"

- **出人意料**：读者跟随德拉线以为高潮是送礼，结尾却揭示双方都卖掉珍宝——**表链无表，梳子无发**，双双重叠失效。
- **情理之中**：前文"一块八毛七"的贫穷、"两件珍宝（金表/长发）"的并置、双方深爱彼此——层层伏笔使反转真实可信。
- **含泪的微笑**：物质礼物"暂时无用"，但爱的牺牲才是真正的"麦琪之礼"——把"爱的奉献"主题推向高潮。
- **点题升华**："这两个人就是《圣经》里提到的贤人"——把贫穷夫妻升华为"麦琪"，是对**功利主义**的温柔反叛。

#### 案例 2：《断魂枪》—— 开放式留白 ⭐⭐⭐

> **结尾**："夜静人稀，沙子龙关好了小门，一气把六十四枪刺下来；而后，拄着枪，望着天上的群星，想起当年在野店荒林的威风。叹一口气，用手指慢慢摸着凉滑的枪身，又微微一笑，'**不传！不传！**'"

- **手法**：**反复（叠句）+ 留白**，戛然而止，余味悠长。
- **人物**：这是沙子龙夜练后"拄枪、望群星、叹气、微微一笑"时的自语——简短有力地揭示其明知时代已变、却仍固执守护断魂枪的孤傲与坚守。
- **结构**：与开头"镖局已改成客栈"、中间"都跟我入棺材"形成贯穿全文的"不传"线索，首尾圆合。
- **主旨**：把外在的**动作冲突**（比武）升华为内在的**精神冲突**（坚守与否），深化"传统文化没落与坚守"主题。
- **效果**：以少胜多，留给读者广阔想象——这"不传"究竟是悲壮还是顽固？老舍不评判，把答案留给读者。

#### 案例 3：《苦恼》—— 反讽式 ⭐⭐⭐

> **结尾**："那匹瘦马嚼着草料，听着，向它主人的手上呵气。约纳讲得入了迷，就把他心里的话统统对它讲了……"

- **反讽**：人有言而不听人言，**马无言却以一"呵气"胜过万人**——人对人不如马对人。
- **主题爆发**：全篇的高潮，也是最深沉的悲哀——人类不倾听，只有马倾听。把"人类孤独""人际冷漠"主题推向顶点。
- **冷峻中的温情**：表面客观冷静，但向马诉说的画面既温暖又心酸——**冷的是笔调，热的是人心**。

#### 案例 4：《婴宁》—— "异史氏曰"点题 ⭐⭐⭐

> **结尾**："观其孜孜憨笑，似全无心肝者。而墙下恶作剧，其黠孰甚焉。至凄恋鬼母，反笑为哭，**我婴宁殆隐于笑者矣**。……若解语花，正嫌其作态耳。"

- **史传式评论**：仿《史记》"太史公曰"体例，篇末以"异史氏曰"作评——是对人物的总结，是对世态的评判。
- **画龙点睛**："**隐于笑**"三字点破婴宁形象的核心——外笑内慧、外憨内黠、外痴内孝。
- **对比升华**：将婴宁的天然之笑与世故做作的"解语花"对比——肯定前者，批判后者，升华赞美天真人性、反思礼教的主题。

#### 案例 5：《哦，香雪》—— 抒情式礼赞 ⭐⭐⭐

> **结尾**："古老的群山终于被感动得颤栗了，它发出洪亮低沉的回音，和她们共同欢呼着。**哦，香雪！香雪！**"

- **直接抒情**：以抒情感叹词"哦"凝聚赞美、怜爱、震撼等复杂情感。
- **复沓结构**：连呼"哦，香雪！香雪！"，模仿诗歌的复沓，节奏明快，富有抒情性与音乐美。
- **主旨升华**：不仅是对香雪个人的赞美，更是对所有如香雪般向往文明、追求尊严的山村青年的礼赞。
- **首尾圆合**：与标题"哦，香雪"呼应。
- **以景结情**：群山"被感动得颤栗"——将个人成长升华为整个山村的觉醒。

### 四、考点 5 模拟练习（含答案）

#### 【模拟 1 · 论述】《麦琪的礼物》结尾被称为"欧·亨利式结尾"的典范，请结合文本分析其妙处。

> **标准答案：**
> "欧·亨利式结尾"的特点是**出人意料又在情理之中**，本文妙处有三：
> ① **出人意料的反转**：读者跟随德拉线（卖发买表链）以为高潮是送礼，结尾却揭示吉姆也卖了祖传金表买玳瑁梳子——**表链无表可配，梳子无发可饰**，两件礼物同时"暂时无用"，强烈震撼。
> ② **情理之中的伏笔**：前文"一块八毛七"的贫穷、"两件珍宝（金表/长发）"的并置、双方深爱彼此——层层铺垫，使吉姆同样卖宝的结局**真实可信**，并非生硬意外。
> ③ **含泪的微笑**：反转不是讽刺的嘲笑，而是**会心又心酸**——物质礼物落空，但双方牺牲的爱意才是真正的"麦琪之礼"，把"爱的奉献"主题推向高潮，余味悠长。

#### 【模拟 2 · 论述】小说结尾"不传！不传！"堪称神来之笔，请赏析其妙处。

> **标准答案：**
> ① **手法上**：采用**反复（叠句）+ 留白**，戛然而止，余味悠长；与上文孙老者追问的"不传？"——"不传！"形成呼应与回环。
> ② **人物上**：这是沙子龙夜练六十四枪后"拄着枪、望群星、叹气、微微一笑"时的自语，**简短而有力地揭示其内心**——明知时代已变、绝技再无用处，仍固执守护这杆枪的尊严，孤傲而悲壮。
> ③ **结构上**：与开头"镖局已改成客栈"、与中间"都跟我入棺材"形成**贯穿全文的"不传"线索**，使结构首尾圆合。
> ④ **主旨上**：把外在的**动作冲突**（比武）升华为内在的**精神冲突**（坚守与否），深化了"传统文化在现代化冲击下没落与坚守"的主题，把悲剧意味推向顶点。
> ⑤ **效果上**：以少胜多，给读者留下广阔的想象与思考空间——这"不传"究竟是悲壮还是顽固？老舍不评判，把答案留给读者。

#### 【模拟 3 · 选择】下列对结尾类型的判断，正确的是：

- A. 《苦恼》以约纳向马诉说结尾，是开放式留白 ✅（是反讽式）
- B. 《哦，香雪》"哦，香雪！香雪！"是抒情式礼赞 ✔
- C. 《婴宁》"异史氏曰"是欧·亨利式反转 ✅（是点题式评论）
- D. 《麦琪的礼物》结尾是点题式 ✅（虽含点题，但本质是欧·亨利式反转）

> **答案：B**

## 七、综合模拟卷（60 分钟）

### 一、选择题（每题 5 分，共 20 分）

**1. 下列对各篇人物形象塑造手法的判断，正确的是：**
- A. 《哦，香雪》通过大量心理独白塑造香雪，属于白描 ✅（心理描写 + 细节为主）
- B. 《断魂枪》以"两眼明得像霜夜的大星"刻画沙子龙，是白描手法 ✔
- C. 《苦恼》写瘦马"瘦骨棱棱"是比喻论证 ✅（是以物衬人）
- D. 《婴宁》写婴宁之笑是心理描写 ✅（是细节描写 / 以笑立骨）

**2. 下列对结构类型的判断，正确的是：**
- A. 《苦恼》是双线结构 ✅（是四次倾诉的递进结构）
- B. 《枕中记》是梦幻框架结构 ✔
- C. 《麦琪的礼物》是诗化小说 ✅（是双线结构 + 欧·亨利式）
- D. 《哦，香雪》是欧·亨利式反转 ✅（是诗化小说）

**3. 下列象征意义分析不正确的是：**

### 二、简答题（每题 12 分，共 36 分）

**5. 简要分析约纳这一人物形象及其典型意义。**

> **参考答案：**
> 【总】约纳是一个**孤独的俄罗斯底层小人物**。
> 【分】① **深沉的丧子之痛**：儿子刚死，渴望倾诉，反复说"我的儿子这个星期死了"。
> ② **逆来顺受**：被骂被揍还"嘻嘻"地笑，不反抗。
> ③ **孤独无助**：四次向人（军人、青年、扫院人、车夫）倾诉都失败，最终只能向瘦马诉说。
> ④ **善良木讷**：用"母马失驹"类比讲自己的痛——朴素深刻。
> 【结】约纳是俄罗斯底层小人物的缩影；其孤独是**现代人类共通的存在性孤独**——人人自顾不暇，谁也不听谁。

**6. 分析《宝黛吵架》心理描写的艺术特色及作用。**

> **参考答案：**
> **艺术特色**：
> ① **大段直接心理独白**：作者直接进入人物内心，"因你也将真心真意瞒起来，只用假意……两假相逢，终有一真"——直接揭示内心矛盾，在中国古典小说中极为罕见。
> ② **对称式心理对照**：宝玉想"你竟心里没我"，黛玉想"可知你心里时时有'金玉'"——两人各自逻辑对称展开，互为镜像。
> ③ **叙述者评论介入**："原是一个心，反弄成两个心""求近之心，反弄疏远之意"——以哲理性语言点破情感本质。
>
> **作用**：
> ① **塑造立体人物**：把相爱相猜、相试相伤的微妙心理写得入木三分。
> ② **揭示悲剧本质**："求近反疏"的悖论，揭示封建礼教下真情无法直接表达的痛苦。
> ③ **深化主题**：为后文"诉肺腑""焚稿"奠定心理基础，是《红楼梦》"打破传统思想和写法"（鲁迅）的集中体现。

**7. 简析《哦，香雪》作为"诗化小说"的四大特征。**

> **参考答案：**
> ① **情节淡化，结构松散**：无强烈戏剧冲突，以香雪对铅笔盒的精神追求和夜路归家的心理变化为内在线索。
> ② **环境描写突出，意境优美**：如开篇对台儿沟大山的描写、夜路中"群山像母亲庄严、神圣的胸脯""核桃叶卷起来像一树树金铃铛"等，景物与人物心灵交融。
> ③ **语言富有诗意**：大量比喻、拟人、反复、排比，如"两根纤细、闪亮的铁轨勇敢地盘旋""不停不停，不停不停！"，节奏舒缓抒情。
> ④ **重象征、重情感**：铅笔盒、火车、大山、月亮均具象征义；结尾"哦，香雪！香雪！"直接以抒情颂歌收束。

### 三、论述题（22 分）

**8. 以《断魂枪》中的沙子龙与《哦，香雪》中的香雪为例，比较二人面对"现代化"（火车）的姿态有何不同？作者态度有何差异？**

> **参考答案要点：**
> | 维度 | 沙子龙（断魂枪） | 香雪（哦，香雪） |
> | --- | --- | --- |
> | **面对现代化的姿态** | **拒绝、隐退、守护传统**（关院门练枪，"不传"） | **拥抱、追求、走向现代**（用鸡蛋换铅笔盒） |
> | **对火车的态度** | 视为破坏风水、终结武林的东西 | 视为通往文明与梦想的载体 |
> | **核心动作** | 背对火车：夜练断魂枪 | 走向火车：跳上火车换铅笔盒 |
> | **作者态度** | **同情中有批判**，为其唱挽歌 | **赞美中有忧虑**，为其唱赞歌 |
>
> **深层分析**：
> - 两人都是传统中国人在现代化转型中的缩影，都带有某种悲剧性——沙子龙是"**没落的悲剧**"（旧时代英雄守不住昨日荣光），香雪是"**成长的阵痛**"（山村少女走出大山的代价）。
> - 两部小说代表中国现代文学中**两种典型的"现代化回应姿态"**——**沙子龙是"背对火车"，香雪是"走向火车"**。
> - 老舍写于 1935 年民族危亡之际，更多是反思与挽悼；铁凝写于 1982 年改革开放之初，更多是希望与憧憬。两个时代、两种姿态，共同构成中国人面对现代化的复杂心灵史。
> - 但两人都有可贵的精神：**坚守尊严、守护自我**——沙子龙守护的是传统的尊严，香雪守护的是做人的尊严（"从没白拿过别人的东西"）。

### 四、赏析题（22 分）

**9. 赏析《苦恼》结尾"那匹瘦马嚼着草料，听着，向它主人的手上呵气。约纳讲得入了迷，就把他心里的话统统对它讲了"的艺术效果。**

> **参考答案：**
> 【指明手法】这一结尾运用了**以物衬人 + 反讽 + 客观冷峻**的手法，是全篇的高潮与泪点。
> 【分析文本】
> ① **以物衬人**：瘦马"嚼着草料，听着，向它主人的手上呵气"——简单的三个动作，给予约纳前四次向人倾诉都得不到的温情回应。瘦马是约纳的镜像（同病相怜），此刻成为他**唯一的倾听者**。
> ② **反讽至深**：人有言而不听人言（军人敷衍、青年辱骂、扫院人粗暴、车夫睡着），**马无言却以一"呵气"胜过万人**——人对人不如马对人，这是契诃夫式冷峻的反讽。
> ③ **客观冷峻的笔调**：作者不直接批判"社会太冷漠了"，而是**冷静展示**马的三个动作与约纳"讲得入了迷"的事实，让读者自己体会——倾向藏在客观描写后面，是契诃夫"客观现实主义"的精髓。
> 【点明效果】
> ① **人物上**：完成约纳"孤独小人物"形象的最终定格——他的苦恼"广大无垠"，却只有马愿意倾听。
> ② **主题上**：把"人类孤独""人际冷漠"主题推向高潮——现代都市中，人对人的痛苦视而不见，反而是不能言的马给予回应，这是对人类看不见他人痛苦的深沉控诉。
> ③ **审美上**：冷的是笔调，热的是人心——表面客观冷静，结尾却饱含深沉的悲悯，**含蓄有力，余味无穷**。
