# 议论文阅读与写作 · 三要素

> 论点 · 论据 · 论证 —— 贯穿本站所有议论文篇目的核心方法论
> 本课件结合已收录的 8 篇文章（5 篇文言散文 + 3 篇现代散文）讲解

::: tip 课件定位
这是大学语文**议论文阅读与写作的总纲课件**。
无论文言议论文（诸子 / 奏疏 / 史论）还是现代议论文（学术随笔 / 思想随笔），
万变不离其宗——**论点、论据、论证**三要素。
本站所有议论文刨解，都是这三要素的具体运用。
:::

## 一、议论文三要素总论

### 什么是议论文

议论文是以**议论**为主要表达方式，通过摆事实、讲道理，**阐明作者主张**的文章。

> **议论文 = 观点（论点）+ 材料（论据）+ 逻辑（论证）**

### 三要素关系图

```
                议论文
                  │
    ┌─────────────┼─────────────┐
    ▼             ▼             ▼
  论点          论据          论证
 （主张）     （材料）       （方法）
    │             │             │
  证明什么      用什么证明    怎么证明
    │             │             │
   灵魂          血肉          骨架
```

| 要素 | 作用 | 回答什么 | 比喻 |
| --- | --- | --- | --- |
| **论点** | 作者的主张、见解 | 证明**什么** | 灵魂 |
| **论据** | 支撑论点的材料 | **用什么**证明 | 血肉 |
| **论证** | 用论据证明论点的过程 | **怎么**证明 | 骨架 |

::: warning 三要素口诀
**论点是灵魂，论据是血肉，论证是骨架。**
- 论点错误，全盘皆输
- 论据不足，空洞无物
- 论证不严，难以服人
:::

## 二、论点（作者的主张）

### 1. 什么是论点

论点是作者对论述问题所持的**见解和主张**，是议论文的**核心**。

### 2. 论点的分类

| 类型 | 含义 | 位置 | 例子 |
| --- | --- | --- | --- |
| **中心论点** | 全文的核心主张 | 篇首 / 篇末 / 标题 | 盛衰之理在于人事 |
| **分论点** | 支撑中心论点的下属论点 | 各段开头 | 忧劳兴国、逸豫亡身 |

### 3. 论点位置的三种典型结构（结合本站文章）

#### 结构 A：开门见山（先议后叙）—— 论点在篇首

**代表：《五代史伶官传序》（欧阳修）**

```
段① 论点先行 —— "盛衰之理，虽曰天命，岂非人事哉！"
段②③④ 叙事印证 —— 庄宗受矢 / 盛衰对比 / 升华
```

特点：**论点鲜明，一开始就亮明立场**，后文叙事为论点服务。

#### 结构 B：卒章显志（先叙后议）—— 论点在篇末

**代表：《过秦论》（贾谊）**

```
段①②③④ 叙事铺陈 —— 秦之兴亡史（不置一词评论）
段⑤ 议论点题 —— "仁义不施而攻守之势异也"
```

特点：**前文铺陈越充分，结论越有冲击力**。读者被"为什么"牵引，读到结论时豁然开朗。

#### 结构 C：叙议结合 —— 论点贯穿全文

**代表：《齐桓晋文之事》（孟子）**

```
段① 避开霸道  → 段② 立论"保民而王"
  → 段③④⑤ 正面展开 → 段⑥ 破霸道 → 段⑦ 给方案
```

特点：**论点 + 分论点交替出现**，层层递进。

::: tip 考点提示
**判断中心论点位置**是高频考点。
- 看标题（如《过秦论》暗含"秦有过失"）
- 看首段（开门见山型）
- 看末段（卒章显志型）
- 看反复强调的句子
:::

### 4. 论点的要求

- **正确**：观点符合客观实际
- **鲜明**：赞成什么反对什么，不含糊
- **新颖**：有独到见解（如朱光潜质疑"敲优于推"）
- **深刻**：能揭示本质（如胡适把"容忍"与"自由"挂钩）

## 三、论据（支撑论点的材料）

### 1. 什么是论据

论据是用来证明论点的**事实和道理**。

### 2. 论据的两大类型

| 类型 | 含义 | 包括 | 优点 |
| --- | --- | --- | --- |
| **事实论据** | 具体的事实 | 史实、事例、数据、统计 | 真实可信，具体有力 |
| **道理论据** | 公认的道理 | 名言、格言、科学原理、经典言论 | 权威性强，理论高度 |

### 3. 本站文章的论据运用（案例库）

#### 案例一：《过秦论》—— 史实论据的典范

| 论点 | 论据类型 | 具体材料 |
| --- | --- | --- |
| 仁义不施而攻守之势异 | 事实（史实） | 秦孝公崛起 → 九国败北 → 始皇极盛 → 陈涉速亡 |

特点：**用整个秦朝兴亡史作论据**，铺陈之充分，前无古人。九国合纵（强）败于秦，陈涉（弱）却亡秦——**双重对比**逼出结论。

#### 案例二：《五代史伶官传序》—— 史实 + 引用

| 论点 | 论据类型 | 具体材料 |
| --- | --- | --- |
| 盛衰之理在于人事 | 事实 | 庄宗受矢负矢 → 极盛 → 极衰 |
| 忧劳兴国、逸豫亡身 | 道理（引用） | 引《尚书》"满招损，谦得益" |

特点：**史实论据 + 道理论据结合**。一事（庄宗兴亡）+ 一典（尚书名言）。

#### 案例三：《咬文嚼字》—— 多领域事实论据

| 论点 | 论据类型 | 具体材料 |
| --- | --- | --- |
| 炼字即炼意 | 事实 | 郭沫若改字、史记李广射虎、韩愈贾岛推敲、苏轼"小团月" |
| 惟陈言之务去 | 道理（引用） | 引韩愈"惟陈言之务去" |

特点：**例子跨领域**（文坛掌故 + 史传 + 诗词 + 俗语），**正反结合**（苏轼善用联想义 vs 套版反应误用）。

#### 案例四：《容忍与自由》—— 史实论据的深度运用

| 论点 | 论据类型 | 具体材料 |
| --- | --- | --- |
| 容忍是一切自由的根本 | 事实 | 高尔文烧死塞维图斯（1553） |
|  | 事实 | 自己五十年前"不容忍"《西游记》 |
|  | 道理 | 布尔先生的话"容忍比自由更重要" |

特点：**他者史实 + 自身经历 + 名人名言**，三层论据。

#### 案例五：《如何避免愚蠢的见识》—— 生活化事实论据

| 论点 | 论据类型 | 具体材料 |
| --- | --- | --- |
| 避免愚蠢见识需五原则 | 事实 | 亚里士多德数牙齿、甘地反产业革命、男女/民族优越 |
|  | 道理 | 心理学"套版反应"概念 |

特点：**例子幽默生活化**，权威反例（亚里士多德也犯错）+ 自嘲（自己也错）。

### 4. 论据的要求

- **真实**：材料必须准确无误
- **典型**：有代表性，能反映本质
- **充分**：数量足够支撑论点
- **新颖**：避免陈旧（如 always 举司马迁受刑）
- **与论点一致**：论据必须能推出论点

::: warning 论据常见错误（考试陷阱）
- **以偏概全**：用个别例子推出普遍结论
- **论据与论点脱节**：材料证明不了观点
- **论据不真实**：编造或歪曲事实
- **堆砌而不分析**：光摆例子不讲道理
:::

## 四、论证（用论据证明论点的方法）

### 1. 什么是论证

论证是用论据证明论点的**过程和方法**，是连接论点与论据的**逻辑桥梁**。

> 论证 = **论点 ←（逻辑）← 论据**

### 2. 常见论证方法（核心考点）

#### ① 举例论证（例证法）⭐⭐⭐

**定义**：列举典型事例证明论点。

**本站最佳案例：**
- 《过秦论》：秦四代兴亡史
- 《五代史伶官传序》：庄宗盛衰
- 《咬文嚼字》：郭沫若改字、推敲公案
- 《如何避免愚蠢的见识》：亚里士多德数牙齿

**答题公式：**
> 运用了举例论证，举了 XX 的例子（**指明**），证明了 XX 观点（**分析**），使论证具体有力、真实可信（**效果**）。

---

#### ② 道理论证（引证法）⭐⭐⭐

**定义**：引用名言、格言、经典著作证明论点。

**本站最佳案例：**
- 《五代史伶官传序》：引《尚书》"满招损，谦得益"
- 《咬文嚼字》：引韩愈"惟陈言之务去"
- 《容忍与自由》：引布尔先生"容忍比自由更重要"

**答题公式：**
> 运用了道理论证，引用了 XX 的名言"…"（**指明**），证明了 XX 观点（**分析**），增强了论证的权威性和说服力（**效果**）。

---

#### ③ 对比论证 ⭐⭐⭐

**定义**：把正反两方面的观点、事实作对比，从而证明论点。

**本站最佳案例：**

| 文章 | 对比项 | 论证什么 |
| --- | --- | --- |
| 《过秦论》 | 九国之强 vs 陈涉之弱 | 秦亡之速 |
| 《五代史伶官传序》 | 庄宗盛 vs 庄宗衰 | 人事决定盛衰 |
| 《咬文嚼字》 | "推" vs "敲" | 字词影响意境 |
| 《齐桓晋文之事》 | 不为 vs 不能 | 是意愿非能力 |

**答题公式：**
> 运用了对比论证，将 XX 与 XX 作对比（**指明**），形成了强烈反差（**分析**），突出强调了 XX 观点，使论证更鲜明（**效果**）。

---

#### ④ 比喻论证 ⭐⭐

**定义**：用比喻说明道理，把抽象化为具体。

**本站最佳案例：**

| 文章 | 比喻 | 本体 |
| --- | --- | --- |
| 《齐桓晋文之事》 | 缘木求鱼 | 霸道求王业 |
| 《齐桓晋文之事》 | 折枝 vs 挟太山超北海 | 不为 vs 不能 |
| 《秋水》 | 井蛙 / 夏虫 / 曲士 | 认识三限 |
| 《咬文嚼字》 | 月轮 / 月晕 / 霞光 | 直指义 / 联想义 |
| 《如何避免愚蠢的见识》 | 优越水母如我们优越水母 | 人类自大可笑 |

**答题公式：**
> 运用了比喻论证，把 XX 比作 XX（**指明**），化抽象为具体（**分析**），使道理浅显易懂、生动形象（**效果**）。

---

#### ⑤ 类比论证 ⭐

**定义**：根据两个事物某些属性相同，推断其他属性也相同。

**本站案例：**
- 《秋水》：井蛙（空间）类比夏虫（时间）类比曲士（教化）
- 《齐桓晋文之事》：邹敌楚类比齐敌天下

---

#### ⑥ 归谬法 ⭐

**定义**：假定对方观点成立，推出荒谬结论，从而反驳对方。

**本站案例：**
- 《如何避免愚蠢的见识》：若真确信"2+2=4"，则听到"2+2=5"只会怜悯不会愤怒——反衬"发怒说明不确信"

---

### 3. 论证结构（四种基本结构）

#### A. 总分式（总—分—总）

```
总：提出中心论点
    ↓
分：分几个方面论述（分论点 1/2/3）
    ↓
总：总结归纳
```

**代表**：《咬文嚼字》（引子→立论→例证→深化→反例→总结）

#### B. 递进式（层进式）

```
层 1：是什么
    ↓
层 2：为什么
    ↓
层 3：怎么办
```

**代表**：《如何避免愚蠢的见识》（五条原则层层递进，从观察到异见到辩论到情绪到自大）

#### C. 对照式（正反对比）

```
正面：论点 + 正面论据
    ↓
反面：反面论据 / 反面假设
```

**代表**：《过秦论》（九国强败 vs 陈涉弱成）；《咬文嚼字》（善用联想义 vs 套版反应）

#### D. 并列式

```
分论点 1   ┐
分论点 2   ├ 并列关系
分论点 3   ┘
```

**代表**：《如何避免愚蠢的见识》（五条原则并列）

::: tip 考点提示
**分析论证结构**是高频考点。判断方法：
- 看各段是**并列**还是**递进**（有没有"层进"关系）
- 看有没有**正反对比**
- 看论点是**先提**（总分）还是**后出**（分总）
:::

## 六、议论文写作指南

### 1. 立论（确定论点）的三步

```
第一步：审题 —— 抓住题眼，确定论题
第二步：立意 —— 确定立场，亮明观点
第三步：构思 —— 搭骨架，列分论点
```

**论点写作要求：**
- **一句话能说清**：论点必须能凝练成一句话
- **有立场**：赞成 / 反对，不能骑墙
- **可论证**：必须有材料可证，不能是空泛口号

### 2. 选材（选论据）的原则

| 原则 | 说明 | 反例 |
| --- | --- | --- |
| **真实** | 材料准确 | 编造名人名言 |
| **典型** | 有代表性 | 用个别案例推普遍 |
| **新颖** | 避免陈词滥调 | 又是司马迁受刑、爱迪生发明 |
| **充分** | 数量足够 | 只举一个例子就下结论 |
| **切题** | 能证明论点 | 论据与论点无关 |

::: warning 写作大忌：套版反应
朱光潜批评的"套版反应"——美人必"柳腰桃面"、才子必"学富五车"。
考场作文**最忌陈词滥调**：
- 不要一写坚持就举司马迁、爱迪生
- 不要一写母爱就举孟母三迁
- **要积累新鲜素材**，避免与其他考生撞车
:::

### 3. 论证（展开论述）的骨架模板

#### 模板 A：总分总（最稳妥）

```
【开头·总】亮明中心论点（1 段）
    ↓
【主体·分】
    分论点 1 + 论据 + 分析（1 段）
    分论点 2 + 论据 + 分析（1 段）
    分论点 3 + 论据 + 分析（1 段）
    ↓
【结尾·总】回扣论点，升华主题（1 段）
```

#### 模板 B：对照式（对比鲜明）

```
【开头】亮明论点
    ↓
【正面】论点 + 正面论据（如历史成功案例）
    ↓
【反面】反面论据（如失败案例）/ 反面假设
    ↓
【结尾】正反对比，强化论点
```

#### 模板 C：层进式（有深度）

```
【层 1】是什么 —— 界定概念
    ↓
【层 2】为什么 —— 分析原因
    ↓
【层 3】怎么办 —— 提出对策
    ↓
【结尾】升华
```

### 4. 段内论证的标准结构（PEEL 法）

一个好段落应该有完整的论证链条：

```
P (Point)      观点句 —— 本段分论点
E (Evidence)   证据句 —— 举出论据（事实/道理）
E (Explain)    阐释句 —— 分析论据如何证明论点
L (Link)       收束句 —— 回扣本段观点 / 过渡下段
```

**示例（《五代史伶官传序》段③的 PEEL）：**
- **P**：忧劳可以兴国
- **E**：庄宗负矢出征，系燕父子，函梁君臣之首
- **E'**：正是这种忧劳恭敬的姿态，使他报三矢之仇，成就帝业
- **L**：故"忧劳可以兴国"

::: warning 写作大忌：堆砌论据
最常见的错误：**光摆例子不分析**。
错误写法："庄宗忧劳兴国。勾践卧薪尝胆兴国。项羽破釜沉舟兴国。"——堆砌！
正确写法：每个例子后，都要有"**阐释句**"分析它如何证明论点。
:::

### 5. 语言要求

- **准确**：用词精当，不模糊
- **鲜明**：立场清楚，不含糊
- **概括**：叙述例子要简洁，重在分析
- **有逻辑**：关联词使用（因此 / 然而 / 首先 / 进而）

## 七、考点对接

### 常考题型

1. **找中心论点**（基础）—— 看标题 / 首段 / 末段 / 反复句
2. **辨析论据类型**（基础）—— 事实论据 vs 道理论据
3. **分析论证方法**（重点）—— 例证 / 引证 / 对比 / 比喻 / 类比
4. **分析论证结构**（重点）—— 总分 / 递进 / 对照 / 并列
5. **补写论据**（应用）—— 给论点补一个恰当的例子
6. **论据是否能证明论点**（辨析）—— 判断论据与论点是否一致
7. **议论文写作**（综合）—— 完整写一篇议论文

### 预测题 + 标准答案

#### 【预测题 1 · 选择】下列对议论文三要素的表述，正确的是：

- A. 论点是作者用来证明观点的材料 ✅（错，这是论据）
- B. 论据分为事实论据和道理论据两类 ✔
- C. 论证是指作者的观点和主张 ✅（错，这是论点）
- D. 一篇议论文只能有一个论据 ✅（错，可有多个）

> **答案：B**

#### 【预测题 2 · 简答】简述举例论证的答题公式，并以《咬文嚼字》为例说明。

> **标准答案：**
> **答题公式**：运用了举例论证，举了 XX 的例子，证明了 XX 观点，使论证具体有力、真实可信。
>
> **以《咬文嚼字》为例**：
> 朱光潜运用了举例论证，举了郭沫若把"你是没有骨气的文人"改为"你这没有骨气的文人"的例子（**指明**），证明了"字词更动会引起情感变化"的观点（**分析**），使论证具体有力、真实可信（**效果**）。

#### 【预测题 3 · 简答】比较举例论证与道理论证的异同。

> **标准答案：**
> **相同点**：都是用来证明论点的论证方法。
>
> **不同点**：
> - **举例论证**用**具体事实**（史实、事例、数据），**具体有力**，如《过秦论》用秦兴亡史。
> - **道理论证**用**公认道理**（名言、原理），**权威性强**，如《五代史伶官传序》引《尚书》。
>
> **关系**：二者常**结合使用**，事实 + 道理，论证更全面。如《五代史伶官传序》既有庄宗史实（例证），又有《尚书》引用（引证）。

#### 【预测题 4 · 论述】以《过秦论》为例，分析对比论证的作用。

> **标准答案：**
> 《过秦论》通篇运用**对比论证**，核心有三组：
> ① **九国之强 vs 秦之更强**：九国有百万之师却"逡巡不敢进"，秦"无亡矢遗镞"而胜——以强衬强。
> ② **陈涉之微 vs 秦之强**：陈涉"瓮牖绳枢""斩木为兵"，却亡了强秦——以弱衬强。
> ③ **陈涉 vs 九国**：陈涉地位、武器、军队、谋略皆不如九国，唯独成功——反衬其成功之反常。
>
> **作用**：多重对比把秦之强推到极致，再写其亡之速，**反差越大，越显"不施仁义"的可悲**，结论"仁义不施而攻守之势异也"极具冲击力。这种"欲抑先扬"的对比，是本文最大的艺术特色。

#### 【预测题 5 · 应用】给论点"成功需要坚持"补写一个论据，并分析如何证明论点。

> **标准答案（示例）**：
> **论点**：成功需要坚持
> **论据**：司马迁受宫刑之辱，仍坚持十余年，写成《史记》。
> **分析**：司马迁面对奇耻大辱，若放弃则《史记》不成；正是他十余年如一日的坚持，才成就了"史家之绝唱"。可见，**坚持是通向成功的必经之路**。
>
> **注意**：这是模板化答案。实际写作中，司马迁已是陈词滥调，**建议积累更新鲜的素材**（如当代科学家、运动员的故事）。

#### 【预测题 6 · 写作】以"谈容忍"为题，写一篇不少于 800 字的议论文。

> **写作提示**：
> 1. **立论**：可参考胡适"容忍是一切自由的根本"，也可自己立意
> 2. **结构**：建议总分总
>    - 开头：亮明论点
>    - 主体：① 容忍是美德（正面）② 不容忍的危害（反面）③ 怎样培养容忍（对策）
> 3. **论据**：可用胡适、罗素的例子，但**要加入自己的思考**，避免套版
> 4. **论证**：举例 + 道理 + 对比结合
>
> **避坑**：不要只堆砌胡适、罗素的话，要有自己的分析。

## 十、迁移运用（实战 checklist）

### 阅读议论文时，问自己：

- [ ] 中心论点是什么？在哪一段？（首/末/标题）
- [ ] 有几个分论点？分别是什么？
- [ ] 用了哪些论据？是事实还是道理？
- [ ] 用了什么论证方法？（例证/引证/对比/比喻）
- [ ] 是什么论证结构？（总分/递进/对照/并列）
- [ ] 论据能否证明论点？有无以偏概全？

### 写作议论文时，问自己：

- [ ] 论点是否一句话能说清？是否鲜明？
- [ ] 分论点是否支撑中心论点？
- [ ] 论据是否真实、典型、新颖？是否切题？
- [ ] 每个论据后是否有**分析阐释**（不堆砌）？
- [ ] 是否用了多种论证方法？（不要只会举例）
- [ ] 结构是否清晰？（总分总最稳）
- [ ] 是否避免陈词滥调？（套版反应）
