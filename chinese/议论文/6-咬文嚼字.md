# 《咬文嚼字》

> 朱光潜（现代） · 议论文 / 学术随笔 · 选自《谈文学》

::: tip 本篇定位
本文是美学大师朱光潜谈**文学语言艺术**的随笔，论证"**文学文字的一字不苟，实质是思想情感的精准把握**"。
是现代**学术性议论文**的典范，说理深入浅出、例证丰富，是大学语文现代文阅读的常考篇目。
:::

## 一、知人论世

**作者：** 朱光潜（1897—1986），安徽桐城人，中国现代**美学的奠基人**。留学英法，学贯中西，毕生从事美学研究和教育。代表作《文艺心理学》《谈美》《谈文学》《西方美学史》。世称"**美学宗师**"。

**核心观点：** 美是"心物婚媾"的产物；文学艺术应"**慢慢走，欣赏啊**"（《谈美》结尾）。

**写作背景：**
- 本文选自《谈文学》（1946 年前后），是朱光潜写给青年的**文学入门随笔**
- "咬文嚼字"本是贬义成语，朱光潜**反其意而用之**，赋予其文学上的正面意义

**文体定位：**
- 形式：**学术随笔 / 议论文**
- 特色：说理深入浅出，例证丰富，**辩证思维**突出（不绝对化，常自我质疑）

::: warning 与《容忍与自由》对比（现代散文·议论文双璧）
| | 《咬文嚼字》（朱光潜） | 《容忍与自由》（胡适） |
| --- | --- | --- |
| 领域 | 文学理论（美学） | 思想 / 政治哲学 |
| 论点 | 炼字即炼意 | 容忍是一切自由的根本 |
| 论证 | 例证为主（多文坛掌故） | 例证 + 引证（多史实） |
| 风格 | 学者随笔，温润辩证 | 思想家随笔，恳切坦诚 |
| 共同点 | 都**反绝对化**、重思辨、深入浅出 |
:::

## 二、文本梳理（六段说理结构）

> 全文采用"**引子 → 立论 → 正面例证 → 理论深化 → 反面警示 → 总结**"的层层推进结构。

### 段① 引子：从郭沫若改字说起

**内容要点：**
- 郭沫若剧本《屈原》中"你是没有骨气的文人"→ 演员建议改"你这"→ 更够味
- 郭沫若又把"你有革命家的风度"改"你这革命家的风度"
- 朱光潜评：第一个改得好，第二个改得不好

**朱光潜的分析：**
- 引《水浒》石秀骂梁中书"**你这与奴才做奴才的奴才**"、杨雄醉骂潘巧云连用六个"你这"——证明"你这"式语法带**极端憎恶**的情感，是"坚决的判断 + 强烈情感"
- 但"你有革命家的风度"改"你这"，逻辑上不通（"是"=等号；"有"表领属，性质不同）

**手法：**
- **以趣事入题**：文坛掌故亲切有趣，引出话题
- **例证（正反）**：同一改字法，一处好一处坏——辩证
- **引证**：引《水浒》多个骂人例，佐证"你这"的情感色彩

**逻辑：** 这是**引子**。用一个看似简单的改字故事，引出"字词变化会引起情感变化"的核心话题，为下文立论铺垫。

---

### 段② 立论：炼字即炼意（全文中心论点）

**原文（关键句）：**
> "咬文嚼字，在表面上像只是斟酌文字的分量，**在实际上就是调整思想和情感**。"

**核心论点：** 咬文嚼字 ≠ 表面玩弄文字；**调整文字 = 调整思想情感**。内容和形式相随而变。

**例证：《史记·李广射虎》**
- 原文："李广见草中石，以为虎而射之，中石没镞，视之，石也。更复射，终不能入石矣。"
- 王若虚《史记辨惑》批评"凡多三石字"，要删改成"以为虎而射之，没镞，既知其为石…"
- **朱光潜反驳**：删了就失去"**视之，石也**"那种发现错误而惊讶的意味；"终不能复入石矣"有失望而斩截的意味，改"终不能入"便索然无味

**手法：**
- **驳论**：先引王若虚的"简洁论"，再驳——**简洁不等于好**，失去情感意味的简洁是失败
- **辩证思维**：表面简洁 vs 实质意味——朱光潜看重后者

**逻辑：** 这是**立论段**。提出全文中心论点"**炼字即炼意**"，并用《史记》例证"一字之差，意味全变"。

::: tip 全文中心论点（必背）
**"咬文嚼字，在实际上就是调整思想和情感。"**
（炼字 = 炼意，文字与思想情感一体两面）
:::

---

### 段③ 例证：推敲字即推敲意境

**内容要点：**
- 韩愈月夜听贾岛吟"鸟宿池边树，僧推月下门"，劝改"推"为"敲"——古今传为美谈
- 一般人都赞赏"敲"比"推"好

**朱光潜的独到见解（辩证亮点）：**
- **"推"的意境**：孤僧步月归寺，自掩自推，**冷寂、自在无碍**，有胸襟气度；"推"无声，与"鸟宿"的静夜调和
- **"敲"的意境**：拘礼，寺里有人应门，**有人情温暖**；但"敲"有声，惊起宿鸟、打破岑寂，反增搅扰
- **结论**：究竟哪个好，要看贾岛当时**心里玩索的是哪种境界**——"**在文字上推敲，骨子里实在是在思想情感上推敲**"

**手法：**
- **反弹琵琶（翻案）**：对"敲优于推"的成见提出质疑——体现学者的独立思考
- **意境分析**：把一字之别细化为两种不同的意境画面，具体入微
- **呼应论点**：再次强调"推敲字 = 推敲思想情感"

**逻辑：** 这是**正面例证段**。用最经典的"推敲"公案，再次印证中心论点。可贵在朱光潜**不盲从成见**，体现了学者本色。

---

### 段④ 深化：直指义 vs 联想义（理论升华）

**核心理论（字义二分）：**

| | 直指义（字典义） | 联想义（历史累积义） |
| --- | --- | --- |
| 性质 | 固定、明确 | 游离、多变 |
| 特点 | 偏类型 | 偏个性 |
| 适用 | 科学文字（越精确越好） | 文学文字（尤以诗为甚） |
| 比喻 | 如**月轮**（明显确实） | 如**轮外月晕、晕外霞光**（浓淡随人随时） |
| 难度 | 易用 | 难用（可使意蕴丰富，也可使意义含糊） |

**善用联想义之例：** 苏轼"独携天上小团月，来试人间第二泉"
- "天上小团月"由"小龙团"（茶名）联想而来
- 不知关联则不通；知其关联则品出"明月照泉水 + 清茶泡泉水"的共通意味
- 这是"**点铁成金**"——文学之所以为文学

**手法：**
- **理论二分（分类论证）**：把字义科学地分为两类，分别说明
- **比喻**：月轮 / 月晕 / 霞光——把抽象的字义理论形象化
- **例证**：苏轼诗句——展示联想义的妙用

**逻辑：** 这是**理论深化段**。从具体例子上升到**字义理论**：文学之难，在联想义的驾驭。这是全文理论性最强的段落。

---

### 段⑤ 反例：套版反应之害（反面警示）

**核心概念：套版反应（stock response）**
- **定义**：习惯走熟路，一人走过人人跟着走，越走越平滑俗滥，用套语滥调而不自知
- **表现**：美人必"柳腰桃面""王嫱西施"，才子必"学富五车才高八斗"，离别必"柳岸灞桥"，印书必"付梓""杀青"
- **危害**：
  - 对作者：套版反应与创造动机为仇敌
  - 对读者：引不起新鲜真切的情趣
  - 更深：用字用词套版 → 运思布局也套版 → 人生态度也套版

**药方：** 引韩愈"**惟陈言之务去**"
- 语言跟着思维情感走
- 不肯用俗滥语言 → 自然不肯用俗滥思想 → 文章才是"作"出来的

**手法：**
- **反面论证**：从"善用联想义"（段④正）转向"误用联想义"（段⑤反），正反对照
- **概念阐释**：引入"套版反应"这一文艺心理学术语，增强理论深度
- **列举**：大量套语滥调的具体例子，直观可感
- **引证**：引韩愈名言作药方

**逻辑：** 这是**反面警示段**。承接段④联想义"难用"，指出最易犯的毛病——套版反应。正反结合，论证更完整。

---

### 段⑥ 结论：谨严刻苦（总结升华）

**原文（关键句）：**
> "文学是艰苦的事，只有刻苦自勉，推陈翻新，时时求思想情感和语言的精炼与吻合，你才会逐渐达到艺术的完美。"

**核心精神：**
- **谨严**：一字不肯放松
- **刻苦**：不能懒、不能粗心、不能自满
- **求精**：求思想情感与语言的精炼吻合

**手法：**
- **总结呼应**：呼应开篇"咬文嚼字"的话题，首尾圆合
- **劝勉语气**：用"你"称呼读者，亲切诚恳

**逻辑：** 这是**总结升华段**。从具体的"炼字"上升到文学创作的总体精神——**谨严刻苦**。

## 四、手法鉴赏

### 1. 例证法（本文最突出的论证方法）⭐⭐
全文几乎**全靠例子说理**，例子丰富且来自不同领域：

| 例证类型 | 例子 | 论证什么 |
| --- | --- | --- |
| 文坛掌故 | 郭沫若改字、韩愈贾岛推敲 | 字词与情感密切相关 |
| 史传 | 《史记·李广射虎》 | 一字之差意味全变 |

| 俗语滥调 | 柳腰桃面、学富五车 | 套版反应之害 |

**效果**：化抽象的文学理论为具体可感的实例，**深入浅出**，是学术随笔的典范写法。

### 2. 辩证思维（本文灵魂）⭐⭐
朱光潜**不绝对化**，处处体现辩证：
- "你这"式语法——有时好（憎恶），有时不好（赞美）→ **看语境**
- "推"vs"敲"——不盲从"敲好"的成见，各有意境 → **看境界**
- 联想义——可使意蕴丰富，也可使意义含糊 → **两面看**
- 简洁——王若虚求简洁，但失去意味的简洁是失败 → **辩证反驳**

### 3. 对比论证
- "你是" vs "你这"（情感色彩对比）
- "推" vs "敲"（意境对比）
- 直指义 vs 联想义（性质对比）
- 善用联想义（苏轼）vs 误用联想义（套版反应）

### 4. 比喻
- 直指义如**月轮**，联想义如**月晕、霞光**
- "点铁成金"——文学联想的妙用

### 5. 驳论
段②驳王若虚的"简洁论"——先树立靶子，再反驳，使论证更有力度。

## 五、主旨情感

**中心论点：** **咬文嚼字，在实际上就是调整思想和情感**——文学的语言斟酌，本质是思想情感的精准把握。

**分论点：**
1. 字词更动 = 思想情感更动（炼字即炼意）
2. 文学之难在"联想义"的驾驭（直指义易，联想义难）
3. 最忌"套版反应"（套语滥调），须"惟陈言之务去"

**思想内涵：**
- **形式与内容统一**：文字（形式）与思想情感（内容）不可分，调整形式即调整内容
- **反对马虎草率**：文学须一字不苟，刻苦谨严
- **反对套版俗滥**：创造须破除习惯，推陈出新

**情感态度：**
- 表面：温和的学者说理
- 实质：**对文学严肃性的坚守**，对套版俗滥的批评
- 体现朱光潜"**谨严**"的美学追求

**文学价值：**
- 学术随笔的典范，说理深入浅出
- 体现了朱光潜美学思想的核心：**内容与形式的统一**
- 对今天的写作、阅读仍有指导意义

>

#### 【预测题 3 · 简答】朱光潜对"推"和"敲"的看法与一般人有何不同？体现什么精神？

> **标准答案：**
> - **一般看法**：都赞赏韩愈改"推"为"敲"，认为"敲"优于"推"。
> - **朱光潜的看法**：不盲从成见，认为"推""敲"各有意境——
>   - "推"：孤僧自掩自推，**冷寂、自在**，与"鸟宿"静夜调和；
>   - "敲"：拘礼有人情，但**打破岑寂**，反增搅扰。
>   - 究竟哪个好，要看贾岛**当时心里玩索的是哪种境界**。
>
> **体现的精神**：
> ① **独立思考**：不盲从古今成见；
> ② **辩证分析**：不绝对化，一字之差各有意境；
> ③ **以意逆志**：从作者的创作意图出发判断，而非外在标准。
>
> 这正是学者本色，也是本文"炼字即炼意"论点的生动体现。

#### 【预测题 4 · 选择】下列对文章的理解，不正确的一项是：
- A. "咬文嚼字"在本文中被赋予了正面的文学意义 ✔
- B. 作者认为文字的更动同时也就是思想情感的更动 ✔
- C. 作者完全否定王若虚对《史记》的修改意见 ✅
- D. "套版反应"是文学创作的大敌，须"惟陈言之务去" ✔
>
> **答案：C**
> **讲评：** C 错（"完全否定"太绝对）。作者**辩证地**看待王若虚的修改——承认表面"简洁些"，但指出失去了原文"惊讶""失望"的意味，**并非简单否定**，而是辩证分析。这正是朱光潜说理"不绝对化"的特点。此类题常考"过于绝对"的选项。

#### 【预测题 5 · 简答】解释"套版反应"，并说明其对文学创作的危害。

> **标准答案：**
> - **套版反应**（stock response）：指习惯走熟路、套用现成模式（套语滥调）的心理习惯。如写美人必"柳腰桃面"，写才子必"学富五车"。
> - **危害**：
>   ① 对**作者**：套版反应与创造动机为仇敌，导致思想情感俗滥；
>   ② 对**读者**：引不起新鲜真切的情趣；
>   ③ **更深层的危害**：用字用词套版 → 运思布局也套版 → **甚至人生态度也套版**。
> - **药方**：韩愈"惟陈言之务去"——不肯用俗滥语言，自然也不肯用俗滥思想。

## 八、写作启示（迁移运用）

本文的论证艺术对写作的启示：

1. **例证要多领域**：说理不能只举一类的例子，多领域例证更有说服力
2. **要辩证不绝对**：好的议论文不死板站队，要看到事物的两面
3. **反弹琵琶**：对成见提出质疑（如质疑"敲优于推"），往往更有深度
4. **理论 + 实例结合**：既有"直指义/联想义"的理论，又有苏轼诗的实例——理论不空洞，实例不零散
5. **首尾呼应**：开头"咬文嚼字"引子，结尾回到"谨严"精神，结构圆合

::: warning 写作迁移
这五条是写**议论文、杂文、学术随笔**的通用法则。考场作文若能用到其中 2-3 条（尤其"辩证""多领域例证"），立意和论证会明显提升。
:::
