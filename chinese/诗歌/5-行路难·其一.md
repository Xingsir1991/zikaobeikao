# 《行路难·其一》（李白）

> 李白（盛唐） · 乐府旧题（七言为主，杂言） · 出处：《李太白全集》

::: tip 本篇定位
《行路难·其一》是李白在天宝三载（744）离开长安时所作组诗（三首）中的第一首，集中表现了他怀才不遇、壮志难酬的苦闷，以及最终对未来充满自信、乐观豪迈的精神。其中"长风破浪会有时，直挂云帆济沧海"是千古传诵的励志名句，最能体现李白**豪放飘逸、积极浪漫**的诗风。
:::

---

## 一、知人论世

**作者简介**：李白（701—762），字太白，号青莲居士，唐代最伟大的浪漫主义诗人，祖籍陇西成纪，生于碎叶（今吉尔吉斯斯坦境内），五岁随父迁居四川绵州。少年即显露才华，"十五好剑术""十五观奇书"。25 岁出蜀漫游。天宝元年（742）被玄宗征召入京，供奉翰林，但遭权贵谗毁，天宝三载（744）"赐金放还"，离开长安。

**文学地位**：
- 与杜甫并称"**李杜**"，被誉为"**诗仙**"。
- 中国古典诗歌浪漫主义传统的集大成者（继承《楚辞》—陶渊明—鲍照一脉）。
- 杜甫赞："笔落惊风雨，诗成泣鬼神。"
- 严羽《沧浪诗话》："子美不能为太白之飘逸，太白不能为子美之沉郁。"

**诗风特点**：豪放飘逸、想象瑰丽、夸张大胆、语言清新自然、"清水出芙蓉，天然去雕饰"。

**文体常识——乐府诗**：
- "乐府"本是汉武帝设立的音乐机构，后演变为诗体。
- **乐府旧题**：沿用乐府古题（如《行路难》《将进酒》《蜀道难》）而创作，多写新意。
- 本诗题为《行路难》，本是晋代袁山松、刘宋鲍照所创乐府旧题（鲍照《拟行路难》十八首最有名），多写世路艰难、离别悲伤。李白沿用此题，注入新生命。
- **形式特点**：以七言为主，间杂三言、五言、九言，**句式长短错落，节奏跌宕起伏**，最宜表达奔放跌宕的情感（区别于近体格律诗的整饬）。

**写作背景**：
- 天宝元年（742），李白怀揣"使寰区大定，海县清一"的政治抱负奉诏入京，本以为能"奋其智能，愿为辅弼"。
- 但玄宗只让他做供奉翰林（文学侍从），加上他恃才傲物，"力士脱靴""贵妃捧砚"（虽多传说，但反映其傲岸），遭高力士、杨贵妃、张垍等谗毁。
- 天宝三载（744），李白被"赐金放还"，被迫离开长安。本诗即作于此时。
- 组诗共三首，本篇为第一首，最负盛名。

**李白"赐金放还"的心境**：一方面是怀才不遇、壮志难酬的苦闷愤懑（"行路难，多歧路，今安在"）；另一方面，是李白式的自信豪迈、永不放弃（"长风破浪会有时"）。这种矛盾交织，构成全诗跌宕起伏的情感脉络。

::: warning 李白《行路难》三首与鲍照《拟行路难》对比
| 比较项 | 鲍照《拟行路难》（南朝宋） | 李白《行路难·其一》（唐） |
| --- | --- | --- |
| 基调 | 怀才不遇、悲愤抑郁 | 苦闷→自信，由抑到扬 |
| 结尾 | "对案不能食，拔剑击柱长叹息"——悲愤难解 | "长风破浪会有时，直挂云帆济沧海"——豪情万丈 |
| 风格 | 沉郁苍凉 | 豪放飘逸、积极浪漫 |
| 主题拓展 | 个人悲愤 | 个人抱负 + 时代自信（盛唐气象） |
:::

## 二、原文与翻译

> 全诗 12 句，按情感起伏可分为五层：① "停杯"四句——苦闷停食；② "欲渡"四句——世路艰难；③ "行路难"四句——迷茫失路；④ "闲来"二句——希冀用典；⑤ "长风"二句——豪情作结。

| 原文 | 翻译 |
| --- | --- |
| 金樽清酒斗十千， | 金杯中盛着价值一斗十千钱的清酒， |
| 玉盘珍羞直万钱。 | 玉盘里盛着价值万钱的珍美佳肴。（羞：同"馐"；直：同"值"） |
| 停杯投箸不能食， | （我却）停下酒杯、丢下筷子，无法下咽， |
| 拔剑四顾心茫然。 | 拔出宝剑环顾四周，内心茫然无措。 |
| 欲渡黄河冰塞川， | 想渡过黄河，却被坚冰阻塞了河流； |
| 将登太行雪满山。 | 想登上太行山，却大雪满山。 |
| 闲来垂钓碧溪上， | 闲暇时在碧绿的溪边垂钓（像姜太公那样等待时机）， |
| 忽复乘舟梦日边。 | 忽然又像伊尹那样梦见乘船经过太阳旁边。 |
| 行路难，行路难， | 行路难啊，行路难！ |
| 多歧路，今安在？ | 岔路这么多，如今正确的路（我的归宿）究竟在哪里？ |
| 长风破浪会有时， | 总有一天，我会乘长风破万里浪， |
| 直挂云帆济沧海。 | 高挂云帆，横渡沧海！ |

### 2. 通假字（必考）
| 字 | 通假 | 释义 |
| --- | --- | --- |
| **珍羞** | 羞同"馐" | 美味的菜肴 |
| **直万钱** | 直同"值" | 价值 |

### 3. 古今异义
| 字词 | 古义 | 今义 |
| --- | --- | --- |
| 茫然 | 失意、无所适从 | 完全不知道、不清楚 |
| 会有时 | 终将会有 | 集会、会议 |
| 安在 | 在哪里（疑问代词宾语前置） | "安"今多为"安定、安全" |

### 4. 炼字精妙（⭐高频考点）

**① ⭐⭐"停杯投箸不能食，拔剑四顾心茫然"——"停""投""拔""顾"（动作炼字）**
四个动词连贯而出——"停"杯、"投"箸、"拔"剑、"顾"望——刻画诗人面对美酒佳肴却食不下咽的苦闷形象。
- **"停""投"**：极写内心的烦躁苦闷——美酒佳肴本应畅饮大嚼，却停杯投箸，可见苦闷之深；
- **"拔剑四顾"**：拔剑而无从挥砍、四顾而茫然无措——生动展现英雄无用武之地的悲愤。
- **作用**：化用鲍照《拟行路难》"对案不能食，拔剑击柱长叹息"，但比鲍照更含蓄、更有张力——李白不是"击柱"发泄，而是"四顾茫然"，更见深沉。四个动词连缀，把苦闷失意之情表现得淋漓尽致。

**② "欲渡黄河冰塞川，将登太行雪满山"——"冰塞""雪满"（意象炼字）**
- **"冰塞川""雪满山"**：以"冰""雪"喻仕途的险阻艰难（奸佞当道、抱负无法实现）；
- **"塞""满"**两字，写阻碍之彻底、之无处不在——不是某一段冰、某处雪，而是"塞"满整条河、"满"覆整座山，象征现实的全面阻碍。
- **作用**：以自然险景隐喻仕途艰难，**景中含情、虚实结合**，形象鲜明。

**③ ⭐⭐"长风破浪会有时，直挂云帆济沧海"——"破""挂""济"（豪情炼字）**
- **"破"**（乘风破浪）：一个"破"字，气吞万里，写出势不可挡的冲天气魄；
- **"挂"**（直挂云帆）："直"是径直、毫不犹豫，"挂"是干脆利落地扬帆——豪迈果决；
- **"济"**（济沧海）：渡过沧海，喻实现宏大抱负。
- **作用**：三字连用，把李白式的自信、豪迈、永不放弃的精神推向顶峰，成为千古励志名句。

**④ "直挂云帆"——"直"字**
"直"是径直、毫不犹豫、一往无前之意。一个"直"字，把诗人的豪迈果决写得淋漓尽致——不再是"四顾茫然"，而是"直挂云帆"，前后呼应，写出情感的转折升华。

## 四、意象与意境

### 1. 核心意象分析

**（一）开篇意象——美酒佳肴（极写苦闷的反衬）**
| 意象 | 象征义 |
| --- | --- |
| **金樽清酒（斗十千）** | 极言酒之珍贵精美。 |
| **玉盘珍羞（直万钱）** | 极言菜之珍美华贵。 |
- 这是"以乐景写哀情"——美酒佳肴本是宴饮之乐，诗人却停杯投箸，**以美酒佳肴之盛反衬内心苦闷之深**（一倍增其哀）。

**（二）中段意象——险阻之路（隐喻仕途）**
| 意象 | 象征义 |
| --- | --- |
| **黄河冰塞川** | 想渡河却被坚冰所阻——喻仕途险阻、报国无门。 |
| **太行雪满山** | 想登山却被大雪封山——喻奸佞当道、前途受阻。 |
| **歧路** | 岔路众多、不知所从——喻人生选择的迷茫。 |
- 这些意象以自然险阻隐喻人生仕途的艰难，**虚实结合、景中有象外之象**。

**（三）典故意象——历史先贤（自我期许）**
| 意象 | 象征义 |
| --- | --- |
| **碧溪垂钓（姜太公）** | 吕尚曾垂钓磻溪，遇周文王而兴周——喻等待时机、终遇明主。 |
| **乘舟梦日（伊尹）** | 伊尹将受汤聘前，梦见乘船过日月旁——喻将得遇明君、施展抱负。 |
| **长风破浪（宗悫）** | 南朝宋宗悫少年时说"愿乘长风破万里浪"——喻宏大志向。 |
- 这三个典故**皆含"困而后达、终遇明时"之意**，是诗人在苦闷中的自我激励与信心重塑。

**（四）结尾意象——云帆沧海（理想的升华）**
| 意象 | 象征义 |
| --- | --- |
| **云帆** | 高大如云的船帆——喻远大抱负。 |
| **沧海** | 茫无涯际的大海——喻广阔的天地与终极理想。 |

### 2. 意境描绘
全诗意境由"压抑苦闷"到"迷茫失路"再到"豪情万丈"层层推进：
开篇——金樽玉盘、美酒佳肴，诗人却停杯投箸、拔剑四顾，苦闷压抑；
中段——黄河冰塞、太行雪满，险阻重重，意境苍凉沉重；
转折——碧溪垂钓、乘舟梦日，进入想象与历史时空，意境悠远开阔；
结尾——长风破浪、云帆沧海，意境骤然开阔壮丽，豪气干云。
整首诗意境**跌宕起伏，由抑到扬，由暗到明**，最终达到极致的壮美境界。

### 3. 景情关系
- **以景衬情**：美酒佳肴（乐景）衬苦闷（哀情）；
- **景中喻理**：黄河冰塞、太行雪满（实景）喻仕途险阻（虚意）；
- **借典寓志**：以古人典故寄托自己的志向与信心；
- **景情反转**：从开篇的苦闷压抑到结尾的壮阔豪情，景与情共同推动情感升华。

## 五、艺术手法

### 1. ⭐⭐浪漫主义（李白诗风核心）
- **想象瑰丽**：碧溪垂钓、乘舟梦日、长风破浪、云帆沧海——大胆瑰丽的想象，超越现实。
- **夸张大胆**：金樽斗十千、玉盘直万钱——夸张写美酒佳肴之盛；沧海、长风——夸张写抱负之大。
- **气势磅礴**：黄河、太行、长风、沧海——大意象、大气势，体现盛唐的浪漫主义气魄。

### 2. ⭐⭐⭐用典（核心手法，必考）
全诗用典三处，皆有深意：
| 典故 | 出处 | 诗意 | 用意 |
| --- | --- | --- | --- |
| **碧溪垂钓** | 姜太公（吕尚）钓于磻溪，遇文王 | 闲来垂钓碧溪上 | 自比吕尚，相信终遇明主 |
| **乘舟梦日** | 伊尹梦乘船过日月旁，将受汤聘 | 忽复乘舟梦日边 | 自比伊尹，相信自己终将得志 |
| **长风破浪** | 宗悫"愿乘长风破万里浪" | 长风破浪会有时 | 自比宗悫，表达宏大志向 |
**作用**：①含蓄地表达自己的志向与信心，相信终将得遇明时；②以古人自况，增加诗歌的厚重感与文化底蕴；③从苦闷中寻得精神的支撑与希望。

### 3. ⭐虚实结合
- **实**：停杯投箸、拔剑四顾（实际动作）；黄河冰塞、太行雪满（实景，亦含隐喻）。
- **虚**：碧溪垂钓、乘舟梦日（想象/典故）；长风破浪、云帆沧海（理想境界）。
- **作用**：虚实相生，拓展诗歌时空，使情感有现实基础又有理想升华。

### 4. ⭐⭐对比反衬（结构核心）
- **美酒佳肴 vs 停杯投箸**：以极乐反衬极悲；
- **冰塞雪满（险阻） vs 长风破浪（壮阔）**：由抑到扬；
- **四顾茫然（失意） vs 直挂云帆（果决）**：前后对比，写情感转折。
**作用**：形成强烈的情感跌宕，突出李白式的"由苦闷到豪情"的精神升华。

### 5. 比兴（隐喻象征）
"欲渡黄河冰塞川，将登太行雪满山"——以渡河、登山之险阻，隐喻人生仕途的艰难，是典型的"比"（隐喻）。

### 6. 反复与呼告
"行路难，行路难，多歧路，今安在？"——反复"行路难"，加上呼告式的诘问"今安在"，把迷茫失路的情感推向高潮，节奏短促跌宕。

### 7. 句式参差错落（乐府特色）
全诗句式：七言（金樽清酒斗十千）—七言—七言—七言—七言—七言—七言—七言—三言（行路难）—三言—三言—三言（多歧路，今安在？实际为三、三、三言的杂言）—七言—七言。
**特点**：以七言为主，中段短句（三言）急促，结尾长句（七言）舒展，**句式长短与情感起伏完全契合**——这是乐府诗（杂言）的最大优势。

### 8. 情感脉络（结构艺术）
全诗情感**四起四落、跌宕起伏**：
- 起（苦闷）→ 落（受阻）→ 又起（用典自信）→ 又落（迷茫歧路）→ 大起（豪情作结）。
这种跌宕起伏的结构，最真切地表现了李白复杂的内心世界。

## 六、主旨情感

**中心思想**：本诗通过写诗人面对美酒佳肴却食不下咽的苦闷，以"冰塞川""雪满山"喻仕途险阻，表达**怀才不遇、壮志难酬的苦闷迷茫**；又借吕尚、伊尹、宗悫之典和"长风破浪""云帆沧海"的壮丽想象，表达**对未来坚定自信、积极进取、永不放弃**的豪情壮志。

**情感层次**（四起四落，跌宕起伏）：
1. **极度的苦闷失意**：停杯投箸、拔剑四顾、心茫然（开篇）。
2. **世路艰难的愤懑**：冰塞川、雪满山（隐喻奸佞当道、报国无门）。
3. **怀信自慰的希望**：垂钓碧溪、梦日边——相信终遇明时。
4. **迷茫失路的诘问**：行路难，多歧路，今安在？
5. **⭐豪情万丈的升华**：长风破浪会有时，直挂云帆济沧海！

这种"苦闷→自信"的转折，正是李白精神的精髓——**无论遭遇何种挫折，都不丧失对未来的信心**。

**文学价值**：
- 李白乐府诗的代表作之一，集中体现其**豪放飘逸、积极浪漫**的诗风。
- "长风破浪会有时，直挂云帆济沧海"成为千古励志名句，激励无数后人。
- 是盛唐**时代精神**（自信、进取、豪迈）与个人抱负完美结合的典范。

## 七、考点对接

### 常考题型
1. 赏析"长风破浪会有时，直挂云帆济沧海"（高频名句）。
2. 分析诗歌中用典的作用。
3. 分析本诗情感起伏（跌宕）的结构艺术。
4. "停杯投箸不能食，拔剑四顾心茫然"动作描写的作用。
5. 选择题 + 理解性默写。

### 预测题与标准答案

**【题 1·名句赏析（高频）】** 请赏析"长风破浪会有时，直挂云帆济沧海"。

> **参考答案**：①**用典**：化用南朝宋宗悫"愿乘长风破万里浪"的典故，含蓄而有力。②**炼字传神**："破"字气吞万里，写势不可挡的冲天气魄；"直挂"径直高挂、毫不犹豫，写豪迈果决；"济沧海"喻实现远大抱负。③**修辞综合**：用典 + 夸张（沧海、长风）+ 比喻（云帆喻抱负、沧海喻理想境界）。④**情感升华**：在前面苦闷迷茫的基础上，陡然振起，表达诗人对未来坚定自信、积极进取、永不言弃的豪情壮志，是全诗的诗眼和高潮。⑤**影响**：成为千古励志名句，最能代表李白豪放飘逸的诗风与盛唐气象。

**【题 2·用典分析】** "闲来垂钓碧溪上，忽复乘舟梦日边"用了什么典故？有何作用？

> **参考答案**：①**用典**：上句用姜太公（吕尚）垂钓磻溪、遇周文王而兴周的典故；下句用伊尹将受商汤聘请前、梦见乘船经过日月旁的典故。②**作用一（自我期许）**：吕尚、伊尹都是先困后达、终遇明主而建功立业的人物，诗人以之自比，含蓄地表达相信自己终将得遇明时、施展抱负。③**作用二（情感转折）**：在前面"冰塞川""雪满山"的苦闷压抑之后，借典故给自己注入希望与信心，使情感由抑转扬、跌宕起伏。④**作用三（艺术效果）**：使诗歌含蓄蕴藉、富有文化底蕴，意境悠远开阔。

**【题 3·结构/情感分析】** 本诗的情感是怎样起伏变化的？请结合诗句分析。

> **参考答案**：全诗情感**跌宕起伏、四起四落**：
> ①**苦闷失意（落）**："停杯投箸不能食，拔剑四顾心茫然"——面对美酒佳肴却食不下咽。
> ②**愤懑受阻（落）**："欲渡黄河冰塞川，将登太行雪满山"——以险阻喻仕途艰难。
> ③**自信希冀（起）**："闲来垂钓碧溪上，忽复乘舟梦日边"——借典故重塑信心。
> ④**迷茫诘问（落）**："行路难，行路难，多歧路，今安在？"——失路之悲。
> ⑤**豪情升华（大起）**："长风破浪会有时，直挂云帆济沧海"——豪情万丈作结。
> 这种跌宕起伏的情感脉络，真实而深刻地表现了李白复杂的内心世界，体现他"苦闷中不失自信、挫折中仍怀豪情"的精神特质。

**【题 4·选择题】** 下列对这首诗的赏析，**不正确**的一项是（　）
A. "金樽清酒斗十千，玉盘珍羞直万钱"以夸张手法写美酒佳肴之盛，反衬内心苦闷。
B. "停杯投箸""拔剑四顾"通过连贯的动作描写，生动刻画诗人苦闷失意的形象。
C. "冰塞川""雪满山"是写实，单纯描绘自然景色，没有更深寓意。
D. 全诗情感跌宕起伏，由苦闷迷茫到豪情万丈，体现李白积极浪漫的诗风。

> **参考答案**：C。"冰塞川""雪满山"不仅是写实，更以比兴手法**隐喻仕途险阻、奸佞当道**，"景中含象外之象"，有深刻寓意。

**【题 5·理解性默写】**
（1）《行路难·其一》中通过动作描写表现诗人苦闷彷徨的句子是：______，______。
（2）《行路难·其一》中以自然险阻隐喻仕途艰难的句子是：______，______。
（3）《行路难·其一》中用姜太公、伊尹典故表达希望的句子是：______，______。
（4）《行路难·其一》中表达诗人坚定自信、豪情万丈的千古名句是：______，______。
（5）《行路难·其一》中"______，______"两句化用鲍照《拟行路难》，但反其意而用之，由悲愤转为豪迈。

> **参考答案**：
> （1）停杯投箸不能食，拔剑四顾心茫然。
> （2）欲渡黄河冰塞川，将登太行雪满山。
> （3）闲来垂钓碧溪上，忽复乘舟梦日边。
> （4）长风破浪会有时，直挂云帆济沧海。
> （5）停杯投箸不能食，拔剑四顾心茫然。

**核心记忆三句话**：
1. 《行路难·其一》是李白"赐金放还"离开长安时所作乐府名篇，情感**跌宕起伏、四起四落**——苦闷→受阻→希冀→迷茫→豪情，最终以"长风破浪会有时，直挂云帆济沧海"豪迈作结。
2. 核心手法：① **三处用典**（姜太公、伊尹、宗悫）以古人自况、自励；② **浪漫主义**想象夸张；③ **对比反衬**（美酒佳肴衬苦闷，冰塞雪满衬长风破浪）；④ **虚实结合**。
3. **"长风破浪会有时，直挂云帆济沧海"是诗眼**，"破/挂/济"三字气吞万里，体现李白式自信——无论遭遇何种挫折，都不丧失对未来的信念，是千古励志名句，最能代表盛唐气象与李白豪放飘逸的诗风。
