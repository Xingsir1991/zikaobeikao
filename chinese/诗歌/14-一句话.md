# 《一句话》

> 闻一多（现代） · 现代格律诗（新月派） · 选自《死水》

::: tip 本篇定位
闻一多**"三美"理论**的典范实践之作。全诗以"**咱们的中国**"这一句呐喊为核心，以**火山**与**青天霹雳**的壮烈意象，抒发对**理想中国**的炽热期盼与必能实现的坚定信念。是**现代格律诗（建筑美）+ 音乐美（复沓节奏）+ 绘画美（火山霹雳意象）**的完美统一，也是中国现代爱国诗歌的**经典名篇**。
:::

---

## 一、知人论世

**作者**：闻一多（1899—1946），本名**闻家骅**，字友三，号友山，湖北浠水人。中国现代著名**诗人、学者、民主战士**。**新月诗派**代表人物，与徐志摩、朱湘等并称。早年留学美国，归国后先后任教于多所大学。1946年7月在昆明遭国民党特务**暗杀**，是中国现代文学史与民主革命史上**壮烈殉道的诗人**。毛泽东赞其"**宁可倒下去，不愿屈服**"。

**文学地位**：
- 闻一多是**中国现代格律诗**的奠基者，提出著名的"**三美主张**"——**音乐美、建筑美、绘画美**。
- 代表诗集《**红烛**》《**死水**》。《一句话》即收入《死水》（1928）。
- 闻一多的诗**深沉炽热、格律严整**，融中国古典诗词意境与西方格律诗技法于一炉，是中国新诗走向成熟的标志。

**写作背景**：本诗写于1925年闻一多**留美归国前后**。当时中国处于**军阀混战、列强欺凌、民不聊生**的黑暗时期。闻一多身在海外，深感祖国的贫弱与屈辱，又目睹人民沉默中蕴蓄的革命力量。他以诗人的敏感与斗士的热血，预言**沉默的中国终将爆发**——如同火山冲破缄默、青天霹雳炸响——诞生一个**崭新、强大、"咱们的"中国**。"咱们的"三字**饱含人民当家作主的理想**。

**文体常识**：
- **体裁**：现代新诗（自由诗走向格律化的产物）。
- **流派**：**新月派**（现代格律诗派）。
- **主张**：实践"**三美**"——音乐美、建筑美、绘画美。
- **体式**：全诗两节，每节8行，**章节对称、句式整饬**，节末复沓同一句呐喊。

::: warning 新月派·三美主张对比
| 项目 | 内容 |
|---|---|
| 流派 | **新月派**（现代格律诗派），代表：闻一多、徐志摩、朱湘等 |
| 三美·音乐美 | 节奏匀称、韵律和谐、复沓回环（如两节末尾"爆一声：咱们的中国！"） |
| 三美·建筑美 | 节式对称、句式整饬、排列美观（两节各8行，结构镜像对应） |
| 三美·绘画美 | 意象鲜明、色彩浓烈、画面感强（火山、青天霹雳、铁树开花） |
| 题材 | 爱国呐喊 / 政治抒情诗 |
:::

## 二、原文与大意

### 第一节：火山的缄默与预言

| 原文 | 大意 |
|---|---|
| **有一句话说出就是祸，** | 有一句话（指"咱们的中国"），一旦说出来就会招来灾祸， |
| **有一句话能点得着火。** | 但这句话本身就有点燃革命烈火的力量。 |
| **别看五千年没有说破，** | 别看五千年来中国人民一直没有把它说出口， |
| **你猜得透火山的缄默？** | 你能猜透火山（爆发前）那种深沉的缄默吗？ |
| **说不定是突然着了魔，** | 说不定哪一天，人民会突然像着了魔一样， |
| **突然青天里一个霹雳，** | 在晴朗的天空里突然响起一个霹雳， |
| **爆一声：** | 爆发出一声呐喊—— |
| **"咱们的中国！"** | "**咱们的中国！**"（属于我们人民的中国！） |

### 第二节：坚定的预言与警告

| 原文 | 大意 |
|---|---|
| **这话叫我今天怎么说？** | 这句话叫我今天怎么说呢？（难以直言，含蓄） |
| **你不信铁树开花也可，** | 你就是不信铁树能开花也随你， |
| **那么有一句话你听着：** | 那么有一句话你听好—— |
| **等火山忍不住了缄默，** | 等到火山再也忍不住它的缄默（即人民忍无可忍）， |
| **不要发抖，伸舌头，顿脚，** | （到那时）你可不要吓得发抖、伸舌头、顿脚， |
| **等到青天里一个霹雳，** | 等到青天里炸响一个霹雳—— |
| **爆一声：** | 爆发出一声呐喊—— |
| **"咱们的中国！"** | "**咱们的中国！**" |

## 三、字词与炼字（关键意象词分析）

### 关键词释义
- **咱们的中国**：全诗的"诗眼"。"咱们"是**劳苦大众、全体人民**的口吻——这个中国是"**我们人民的**"中国，而非军阀官僚的中国。短短五个字，寄托**人民当家作主**的理想，是全诗核心意象与呐喊。
- **祸 / 火**：说出这句话会招"祸"（反动派的迫害），但它本身能点"火"（革命烈火）。**"祸"与"火"同韵相对**，写出这句话的危险性与爆发力。
- **五千年**：指中华民族悠久历史。五千年的沉默，蓄积五千年的力量。
- **火山的缄默**：火山爆发前表面平静、内里岩浆奔突。**喻人民沉默中蕴蓄的巨大革命力量**。
- **着了魔**：形容情不自禁、不可遏制的爆发状态。
- **青天里一个霹雳**：晴空突起的炸雷。**喻突如其来的革命爆发**，出人意料又势不可挡。
- **铁树开花**：铁树（苏铁）极少开花，喻**极难发生却终将发生之事**。你不信铁树开花也罢——总有一天它会发生。
- **发抖、伸舌头、顿脚**：受惊吓的三种动作。**讽刺那些不相信革命、届时又惊慌失措的人**。

### ⭐ 关键意象词分析（炼字）

1. **⭐⭐⭐ "咱们的"**——全诗诗眼。三字定调：这个中国是**人民的中国**。包含**民主、平等、人民当家作主**的理想——不仅是民族觉醒，更是民主革命。短促有力、口语化，**极具鼓动性与感召力**。

2. **⭐⭐⭐ "火山的缄默"**——核心隐喻。"火山"有**蕴蓄、危险、必将爆发**的特性；"缄默"写**表面沉寂**。"火山+缄默"组合，精准写出**中国人民沉默中蕴蓄的巨大力量**——表面无声，内里岩浆奔突，**终将惊天一爆**。这是全诗最精彩的意象。

3. **⭐⭐ "点得着火"与"霹雳"**——两个意象都写**爆发的力量**。"火"是点燃（革命的火星），"霹雳"是天崩（革命的惊天爆发）。由"**点得着火**"到"**青天霹雳**"，**力量层层升级**。

4. **⭐⭐ "忍不住"**——"等火山**忍不住**了缄默"。"忍不住"三字写爆发的**不可避免、势所必然**——不是要不要爆发的问题，而是**何时爆发**的问题。语气坚定。

5. **⭐ "祸/火"双关与押韵**——"说出就是**祸**""点得着**火**"。**"祸"是对反动派而言（镇压），"火"是对人民而言（革命）**；同韵相对，一字之差，立场分明。

### ⭐ 名句背诵
- "有一句话说出就是祸，有一句话能点得着火。"
- "你猜得透火山的缄默？"
- "爆一声：'咱们的中国！'"
- "等火山忍不住了缄默……爆一声：'咱们的中国！'"

## 四、意象与意境

### 核心意象表

| 意象 | 特征 | 象征义 |
|---|---|---|
| **一句话** | 祸／火／铁树开花 | **"咱们的中国"——革命理想、民主新中国** |
| **火山（缄默）** | 沉默而蕴能量 | **沉默中蓄势待发的人民力量** |
| **青天霹雳** | 突如其来、惊天动地 | **突如其来的革命爆发** |
| **铁树开花** | 罕见、必现 | **看似不可能、终将实现** |
| **发抖/伸舌头/顿脚** | 受惊之态 | **讽刺不相信革命者届时惊慌** |

### 意境分析
全诗营造**炽热、压抑、蓄势待发、终将爆发**的意境。前数句以"**火山的缄默**""**五千年没有说破**"渲染**压抑沉闷、岩浆奔突于地下**的氛围——沉重、紧张、蓄势；中间以"**青天里一个霹雳**"形成**惊天爆破**；最后以"**爆一声：咱们的中国！**"喷薄而出。全诗**由压抑到爆发、由沉默到呐喊**，意境由**深沉凝重**骤转为**壮烈奔放**，极富张力。

### 景情关系（以意象寄情）
- 全诗不直写"我热爱中国"，而是通过**火山、霹雳、铁树**等壮烈意象**寄托**炽热的爱国激情与坚定的革命信念——**以象寓意、以景蓄情**。
- 意象的"**爆发性**"（火山、霹雳）与情感的"**爆发性**"（呐喊）高度统一，**情与象合、声与势应**。

## 五、艺术手法

### 1. ⭐⭐⭐ "三美"主张的完美实践（最核心）

这是本诗最重要的艺术特色——闻一多**"音乐美、建筑美、绘画美"**主张的典范实践：

#### （1）⭐⭐⭐ 建筑美（结构最显著）
- 全诗**两节，每节8行**，**节式完全对称**——这是最显眼的"建筑美"。
- 两节相对应的诗行**句式相同或相近**：如第一节"**有一句话说出就是祸/有一句话能点得着火**"对应第二节"**这话叫我今天怎么说/你不信铁树开花也可**"；"**别看五千年没有说破**"对应"**那么有一句话你听着**"；"**说不定是突然着了魔**"对应"**不要发抖，伸舌头，顿脚**"；"**突然青天里一个霹雳/爆一声/咱们的中国**"对应"**等到青天里一个霹雳/爆一声/咱们的中国**"。
- **整饬的方块结构**，如建筑物般**匀称、稳定、美观**——"建筑美"的直观体现。

#### （2）⭐⭐⭐ 音乐美
- **节奏匀称**：每行大致四顿，节拍匀齐，读来铿锵有节奏。
- **复沓回环**：两节末尾"**爆一声：咱们的中国！**"完全重复，形成**回环复沓**，如歌谣副歌，**一唱三叹、余音绕梁**。
- **韵律和谐**：全诗押"o/uo"韵（祸、火、破、默、魔、说、可、着、脚等），**一韵到底**，音韵和谐统一。
- **口语化而有节奏**：用日常口语（"咱们""着了魔""伸舌头"）却符合格律，**以俗为雅、声情并茂**。

#### （3）⭐⭐ 绘画美
- 意象鲜明、画面感强：**火山**（炽红、奔突）、**青天霹雳**（白光乍破、电闪雷鸣）、**铁树开花**（罕见奇景）。
- 色彩浓烈、视觉冲击力强——每一意象都是一幅**强烈的画面**。
- 这些意象营造出**壮烈、火爆、奔放**的视觉意境，是绘画美的体现。

### 2. ⭐⭐⭐ 隐喻/象征（核心表现手法）

全诗通篇以**隐喻、象征**抒情，不直白说理：

| 隐喻/象征 | 本体（所指） |
|---|---|
| **一句话** | "咱们的中国"——民主新中国理想 |
| **火山的缄默** | 沉默中蕴蓄力量的人民 |
| **点得着火** | 这句话点燃革命的力量 |
| **青天霹雳** | 突如其来的革命爆发 |
| **铁树开花** | 看似不可能、终将实现 |
| **发抖伸舌头顿脚** | 不信革命者届时惊慌 |

**象征效果**：将抽象的政治信念、革命激情，转化为**具象、可感、壮烈**的意象，**含蓄蕴藉又极具感染力**，比直白呐喊更深刻、更有张力。

### 2.5 ⭐⭐ 反复（复沓）
- **两节末尾完全重复**"爆一声：咱们的中国！"——**强调、回环、一唱三叹**。
- **两节句式镜像对应**——整篇反复回环。
- **反复**增强节奏感与抒情强度，是音乐美的重要来源。

### 3. ⭐⭐ 对比（矛盾张力）
- "**祸**"与"**火**"对比——立场分明（反动派 vs 人民）；
- "**五千年没有说破**"与"**突然青天霹雳**"对比——千年沉默与一朝爆发；
- "**你不信**"与"**等火山忍不住**"对比——怀疑者与坚定者。
对比使全诗**充满张力**。

### 4. ⭐ 设问与反诘
- "**你猜得透火山的缄默？**"——反诘，引发思考；
- "**这话叫我今天怎么说？**"——设问，承上启下。
增强**抒情性与互动感**。

### 5. ⭐ 口语入诗
- "**咱们**""**着了魔**""**伸舌头、顿脚**"等口语入诗，**通俗易懂、亲切有力**，体现新诗的"新"。

### 6. ⭐ 卒章显志（点题）
两节末尾"**爆一声：咱们的中国！**"——卒章显志，点明全诗主旨。**反复呐喊、掷地有声**。

## 六、主旨情感

本诗以"**咱们的中国**"这一句呐喊为中心，抒发闻一多对**民主、强大、属于人民的新中国**的炽热期盼，以及对其**必将实现**的坚定信念。

**情感/主旨层次**：
1. **炽热的爱国激情**——对"咱们的中国"（人民的中国）的深情呼唤；
2. **对人民力量的坚定信念**——以"火山的缄默""忍不住"表达**沉默中蓄势、终将爆发**的坚定信念；
3. **对反动势力的蔑视与警告**——以"祸""发抖、伸舌头、顿脚"讽刺镇压者、怀疑者届时必将惊慌；
4. **对革命爆发的预言**——"青天霹雳""铁树开花"预言**新中国终将诞生**，势不可挡。

全诗**炽热奔放、铿锵有力**，是现代爱国诗歌的**经典呐喊**，也是闻一多作为**诗人兼斗士**的精神写照——他以诗预言，更以生命实践（1946年被暗杀殉道）。

## 七、考点对接

### 常考题型
- "三美"主张在本诗中的体现（**最高频**，几乎必考）
- "咱们的中国"的含义
- 核心意象（火山、霹雳、铁树开花）的象征意义
- 复沓 / 反复的作用
- 现代格律诗 / 新月派常识
- 名句默写（"爆一声：咱们的中国！"）
- 情感主旨概括

### 预测题

**1. 本诗如何体现闻一多"三美"主张？请结合诗句分析。**

> **参考答案**：本诗是闻一多"三美"主张的典范实践：
> ① **建筑美**：全诗**两节、每节8行**，节式完全对称；两节对应诗行句式相同相近（如"有一句话说出就是祸"对"这话叫我今天怎么说"，"说不定是突然着了魔"对"不要发抖，伸舌头，顿脚"），排列整饬美观，如建筑物般匀称稳定；
> ② **音乐美**：全诗**一韵到底**（押"祸/火/破/默/魔/说/可/着/脚"等o/uo韵），节奏匀称铿锵；两节末尾"**爆一声：咱们的中国！**"完全复沓，回环往复、一唱三叹；口语入诗而声情并茂；
> ③ **绘画美**：意象鲜明浓烈、画面感强——**火山缄默**（炽红奔突）、**青天霹雳**（电闪雷鸣）、**铁树开花**（罕见奇景），视觉冲击力强，营造壮烈奔放的意境。
> 三美合一，形式与内容高度统一，是现代格律诗的典范。

**2. "咱们的中国"这句话在全诗中有何深刻含义？**

> **参考答案**："咱们的中国"是全诗的**诗眼与核心呐喊**，含义深刻：
> ① **"咱们"的口吻**："咱们"是**劳苦大众、全体人民**的口吻——这个中国是"**人民的中国**"，而非军阀官僚的中国，寄托**人民当家作主、民主平等**的理想；
> ② **革命的预言**：这句话"**能点得着火**"，是点燃革命的火星；其爆发如"**青天霹雳**"，预言**民主新中国终将诞生**；
> ③ **时代的禁忌**：这句话"**说出就是祸**"，在当时是**反动派不容的革命言论**，故"五千年没有说破"，是黑暗时代的禁忌呐喊。
> 短短五字，融**爱国激情、民主理想、革命预言**于一炉，是全诗精神所在。

**3. 诗中"火山的缄默"和"铁树开花"两个意象有何象征意义？**

> **参考答案**：两意象皆为**隐喻/象征**，表达对人民力量与新中国的坚定信念：
> ① **"火山的缄默"**——火山表面平静、内里岩浆奔突，象征**中国人民在沉默中蕴蓄的巨大革命力量**。"你猜得透火山的缄默？"反诘人民力量之深不可测；"等火山忍不住了缄默"则预言爆发**不可避免、势所必然**。
> ② **"铁树开花"**——铁树极少开花，喻**看似不可能发生、终将实现之事**。"你不信铁树开花也可"——你不信也罢，**它终会发生**，表达对革命胜利的**坚定信念**。
> 两意象**以壮烈的自然之象寄寓革命之理**，含蓄而有力，是本诗绘画美与象征手法的完美结合。

**4. 两节末尾"爆一声：咱们的中国！"完全重复，有何作用？**

> **参考答案**：此句运用**反复（复沓）**手法，作用有三：
> ① **强化主旨**：反复呐喊"咱们的中国"，**突出全诗核心**，使爱国激情与革命预言深入人心；
> ② **音乐美**：反复形成**回环往复、一唱三叹**的效果，如歌谣副歌，增强节奏感与抒情强度，是"三美·音乐美"的体现；
> ③ **建筑美**：两节末尾完全对称，构成**整饬对称的结构美**，是"三美·建筑美"的体现；
> ④ **情感爆发**：反复呐喊如**两次重锤敲击**，将全诗情感推向高潮，**掷地有声、振聋发聩**。

**5. 简述本诗的艺术特色及主旨。**

> **参考答案**：
> **艺术特色**：① 实践闻一多"**三美**"主张——建筑美（两节8行对称）、音乐美（一韵到底+末句复沓）、绘画美（火山霹雳意象）；② 通篇**象征隐喻**（火山缄默喻人民蓄势、青天霹雳喻革命爆发、铁树开花喻理想必现）；③ **反复、对比、反诘**等手法交织；④ **口语入诗**而格律严整。
> **主旨**：以"**咱们的中国**"这一句呐喊为中心，抒发对**民主、强大、属于人民的新中国**的炽热期盼，预言沉默的人民终将如火山霹雳般爆发，新中国必将诞生——是现代爱国诗歌的经典呐喊，体现闻一多诗人兼斗士的精神。
