# 《蒹葭》（《诗经·秦风》）

> 佚名（西周至春秋） · 四言古体 · 出处：《诗经·秦风》

::: tip 本篇定位
《蒹葭》是《诗经·秦风》中的名篇，被誉为"千古怀人诗之祖"。全诗以"在水一方"的"伊人"为追寻对象，营造朦胧、空灵、可望而不可即的意境，开了中国古典诗歌"企慕情境"的先河。
:::

---

## 一、知人论世

**作品出处**：选自《诗经》。《诗经》是中国第一部诗歌总集，收录西周初年至春秋中叶诗歌 305 篇，又称"诗三百"，分"风、雅、颂"三部分，"风"即十五国风，为各地民歌。《蒹葭》属"秦风"，是秦国（今陕西一带）的民歌。

**"秦风"特色**：秦国地处西北，民风彪悍刚健，秦风多慷慨悲歌（如《无衣》"岂曰无衣，与子同袍"）。但《蒹葭》却是一篇婉约缠绵的怀人诗，是秦风中罕见的清丽之作，故前人有"诗之为言，性情而已；秦风有此，弥见其真"之叹。

**文体常识——四言诗**：
- 句式：每句四字，二二节拍（蒹葭/苍苍，白露/为霜）。
- 章法：**重章叠句**——各章句法相同或相近，只更换少数字词，反复咏叹。
- 手法：**赋、比、兴**——"兴"即先言他物以引起所咏之物，本诗为典型的"兴而含比"。

**创作背景**：本诗意旨历来众说纷纭，主要有三：
1. **爱情说**（主流）：写对意中人可望而难即的思念追求。
2. **招贤说**：以"伊人"喻贤才隐士，表达求之不得的怅惘。
3. **理想说**："伊人"为某种美好理想或境界的象征。

::: warning 与《关雎》对比（同为《诗经》爱情名篇）
| 比较项 | 《关雎》（周南） | 《蒹葭》（秦风） |
| --- | --- | --- |
| 情感基调 | 喜悦和谐，"琴瑟友之""钟鼓乐之" | 朦胧怅惘，可望不可即 |
| 起兴之物 | 关雎和鸣（水鸟） | 蒹葭白露（秋景） |
| 结局 | 求而得之，皆大欢喜 | 求而不得，一往情深 |
| 风格 | 温柔敦厚，明快 | 苍凉凄清，空灵 |
:::

## 二、原文与翻译

> 全诗三章，章八句，重章叠句。下面分章翻译。

### 第一章
| 原文 | 翻译 |
| --- | --- |
| 蒹葭苍苍，白露为霜。 | 河畔芦苇青苍苍，夜来白露凝成霜。 |
| 所谓伊人，在水一方。 | 我所思念的那个人，在水的那一方。 |
| 溯洄从之，道阻且长。 | 逆流而上去追寻她，道路险阻又漫长。 |
| 溯游从之，宛在水中央。 | 顺流而下去找寻她，她仿佛就在水的中央。 |

| 原文 | 翻译 |
| --- | --- |

| 所谓伊人，在水之湄。 | 我所思念的那个人，就在水边高崖地。 |
| 溯洄从之，道阻且跻。 | 逆流而上去追寻她，道路险阻又陡峭。 |

| 原文 | 翻译 |

| 所谓伊人，在水之涘。 | 我所思念的那个人，就在水边那岸沿。 |

## 三、字词与炼字

### 1. 重点字词注音释义
| 字词 | 读音 | 释义 |
| --- | --- | --- |
| 蒹葭 | jiān jiā | 芦苇。蒹：未吐穗的芦苇；葭：初生的芦苇。 |
| 苍苍 / 萋萋 / 采采 | cāng cāng / qī qī / cǎi cǎi | 均形容茂盛繁密的样子，程度递进。 |
| 伊人 | yī rén | 那个人（指意中所指之人）。"伊"，指示代词"那"。 |
| 湄 / 涘 | méi / sì | 均指水边、岸边。湄：水草交接处；涘：水边。 |
| 溯洄 | sù huí | 逆流而上。溯：逆水而行。 |
| 溯游 | sù yóu | 顺流而下。 |
| 跻 / 右 | jī / yòu | 跻：升高、陡峭；右：迂回、弯曲。 |
| 坻 / 沚 | chí / zhǐ | 均指水中的小洲、高地。坻：水中高地；沚：水中沙滩。 |
| 宛 | wǎn | 宛然、仿佛、好像。 |
| 晞 / 已 | xī / yǐ | 晞：干；已：停止。 |

### 2. 炼字精妙（⭐高频考点）

**① "苍苍"——"萋萋"——"采采"（叠词炼字）**
三章芦苇分别用"苍苍""萋萋""采采"，皆为叠词形容茂盛，但有层次递进：苍苍突出**秋意肃杀、苍凉凝重**；萋萋突出**茂密葱茏**；采采突出**繁多鲜明**。叠词连用使音节舒缓悠长，构成回环往复的音乐美，强化了抒情氛围。

**② "为霜"——"未晞"——"未已"（时间炼字）**
写露水由"结霜"到"未干"再到"未止"，是**时间的推移**：从天未亮（露凝为霜）到日出（露尚未干）再到清晨（露水仍滴落）。暗示主人公从黎明到清晨长时间伫立追寻、苦苦等候，**以时间之长写相思之深**，含蓄蕴藉。

**③ ⭐⭐"宛"字（核心炼字）**
"宛在水中央"的"宛"（仿佛、好像），是全诗诗眼。
- **妙处一**：写出"伊人"若隐若现、似近实远、似有若无的朦胧美感；
- **妙处二**：暗示主人公追求的徒劳与可望不可即的怅惘——看似就在眼前，伸手却抓不到；
- **妙处三**：化实为虚，使整首诗笼罩在一层亦真亦幻的氛围中，境界空灵。一个"宛"字，画龙点睛，造就了千古传诵的"在水一方"意境。

## 四、意象与意境

### 1. 核心意象分析
| 意象 | 象征义 / 情感色彩 |
| --- | --- |
| **蒹葭（芦苇）** | 秋意萧瑟、凄清苍凉的背景；亦象征主人公绵长不绝、绵密柔韧的情思。 |
| **白露（为霜→未晞→未已）** | 暗示时令（深秋）与时间推移；"霜"营造凄清氛围，"露"是清冷、易逝之感的寄托。 |
| **秋水（河流）** | 阻隔与思念的载体——水隔开两人，水又连着思念；"秋水"已凝定为成语，喻清澈明亮（"秋水伊人"）。 |
| **伊人** | 思慕对象。可实指心上人，亦可虚指贤才、理想、美好境界——具有**多义性与朦胧美**。 |
| **道路（阻且长/跻/右）** | 追寻途中的艰难险阻，象征人生求索的坎坷、理想实现的不易。 |

### 2. 意境描绘
全诗营造的是一种**凄清、朦胧、空灵、可望不可即**的意境。
设想其画面：深秋清晨，芦苇苍苍、白露凝霜，烟波浩渺的秋水之上，一个身影在水的一方若隐若现。主人公一会儿逆流而上，路险且长；一会儿顺流而下，那人仿佛就在水中央，可总也走不到、抓不住。芦苇在秋风中摇曳，水汽氤氲，意境似真似幻，情思绵长不绝。

### 3. 景情关系
- **以景起兴，景中含情**：开篇秋景苍凉凄清，已定下全诗怅惘低回的基调。
- **情景交融，物我合一**：蒹葭的苍凉即主人公心境的苍凉；秋水的阻隔即相思的阻隔；"宛在"的虚幻即追求的徒劳。
- **景为情设，情因景生**：所有的景物描写（芦苇、白露、秋水、道路）都服务于"求而不得"的情感表达。

## 五、艺术手法

### 1. ⭐重章叠句（最突出手法）
三章结构基本相同，仅更换"苍苍/萋萋/采采""为霜/未晞/未已""一方/之湄/之涘""且长/且跻/且右""中央/坻/沚"等少量字词。

**作用**：
- **抒情上**：一唱三叹，回环往复，使感情层层加深、层层递进（不是简单的重复，而是深化）。
- **音乐上**：节奏舒缓、旋律优美，便于歌唱和记忆，体现民歌特色。
- **叙事上**：通过时间（霜→晞→已）和空间（一方→湄→涘）的细微变化，写出追寻过程的漫长和执著。
- **意境上**：造成一种回环往复、绵邈无尽的韵致，强化"求之不得"的怅惘。

### 2. ⭐兴而含比（赋比兴中的"兴"）
以"蒹葭苍苍，白露为霜"起兴——先言芦苇白露之秋景，再引出"所谓伊人"。既是起兴，又含比喻：芦苇的茂密苍凉、白露的凄清易逝，正暗合主人公幽思怅惘、思念绵长的心境。这种"兴中有比、比兴交融"是《诗经》最典型的艺术手法。

### 3. 动静结合与虚实相生
- **实写**（动作）："溯洄从之""溯游从之"——主人公奔波追寻的动态。
- **虚写**（幻象）："宛在水中央"——伊人若隐若现的虚幻景象。
- 实中有虚，虚中有实，亦真亦幻，营造朦胧空灵之美。

### 4. 重叠词（叠词）的运用
"苍苍""萋萋""采采"等大量叠词，音韵和谐、节奏鲜明，增强形象性和音乐美。

### 5. 以少总多、含蓄蕴藉
全诗不直接抒情、不直陈其事，借景抒怀，"伊人"所指不明，给读者留下广阔的想象空间，所谓"言有尽而意无穷"。

::: warning ⭐三段式答题模板（"分析艺术手法"类）
**第一段——明手法**：本诗运用了……手法（如重章叠句、起兴、叠词、虚实相生）。
**第二段——析运用**：结合诗句具体分析，如"以蒹葭白露起兴……""三章仅换数字……"。
**第三段——点作用**：这种手法……地表现了……情感/意境，达到了……的艺术效果（如回环往复、一唱三叹、朦胧空灵等）。
:::

## 六、主旨情感

**中心思想**：本诗写主人公于深秋清晨在河边反复追寻"伊人"而终不可得的历程，表达了**对美好事物（爱情/理想）执着追求却可望不可即的怅惘与哀婉**。

**情感层次**：
1. **深深的思慕与追寻**："溯洄从之""溯游从之"，反复追寻，一往情深。
2. **求之不得的怅惘**："宛在水中央"——可望而不可即的失落怅惘。
3. **执着不悔的坚韧**：尽管道路"阻且长""且跻""且右"，仍不舍追寻，体现主人公九死不悔的执着。
4. **含蓄朦胧的多义美**：因"伊人"指向不明，使本诗超越了具体爱情，上升为对一切美好理想的追求与企盼，具有哲理意味。

**文学价值**：
- 千古怀人诗、企慕情境的鼻祖，影响后世无数"在水一方""可望不可即"的母题。
- "秋水伊人""在水一方"已成为汉语常用成语。
- 王国维《人间词话》评："《诗·蒹葭》一篇，最得风人深致。"

## 七、考点对接

### 常考题型
1. 鉴赏"宛"字的炼字妙处。
2. 分析"重章叠句"的艺术效果。
3. "蒹葭苍苍，白露为霜"在诗中的作用（起兴）。
4. 分析诗歌营造的意境及景情关系。
5. 理解性默写。

### 预测题与标准答案

**【题 1·炼字鉴赏】** "宛在水中央"中的"宛"字历来为人称道，请简要赏析。

> **参考答案**："宛"是"仿佛、好像"之意（释义）。一个"宛"字，写出了"伊人"若隐若现、似近实远的朦胧美感（析运用）；同时暗示主人公苦苦追寻却始终可望不可即的怅惘与徒劳，化实为虚，使全诗笼罩在一层亦真亦幻的氛围中（析作用）。它和"水中央"配合，把企慕而难即的微妙心境表现得含蓄蕴藉、韵味无穷，是全诗的诗眼（点效果）。

**【题 2·手法分析】** 本诗主要运用了什么表现手法？请结合诗句分析其效果。

> **参考答案**：主要运用了**重章叠句**和**起兴（兴而含比）**两种手法。
> ①**重章叠句**：全诗三章句法基本相同，只换少量字词（如"苍苍/萋萋/采采""为霜/未晞/未已"）。一唱三叹、回环往复，强化了思慕追寻之情；同时通过时间推移和空间转换写出追寻过程的漫长执著。
> ②**起兴**："蒹葭苍苍，白露为霜"先描写芦苇白露的凄清秋景，再引出"所谓伊人"。既是起兴又含比喻——以景之凄凉衬情之怅惘，景情交融，奠定全诗凄清朦胧的基调。

**【题 3·意境分析】** 这首诗营造了怎样的意境？请简要分析。

> **参考答案**：营造了**凄清、朦胧、空灵、可望不可即**的意境。诗人以深秋清晨的芦苇、白露、秋水为背景，营造凄清苍凉的氛围；又以"宛在水中央"写出"伊人"若隐若现、似有若无的朦胧；加之主人公逆流、顺流反复追寻而不得的动态，构筑出一个执着追寻却终归虚幻的空灵境界，将思慕与怅惘之情表现得含蓄隽永。

**【题 4·选择题】** 下列对这首诗的赏析，**不正确**的一项是（　）
A. "蒹葭苍苍，白露为霜"以秋景起兴，渲染了萧瑟凄清的氛围。
B. 三章运用重章叠句，反复咏叹，使情感层层加深、回环往复。
C. "所谓伊人"中的"伊人"指意中人，也可以泛指一切美好理想。
D. 全诗直抒胸臆，明确表达了主人公对伊人的爱情与思念。

> **参考答案**：D。本诗不直抒胸臆，而是含蓄蕴藉、借景抒情；"伊人"所指不明，具有多义性。

**【题 5·理解性默写】**
（1）《蒹葭》中点明时令、渲染凄清气氛的句子是：______，______。
（2）《蒹葭》中表现主人公逆流而上追寻伊人、道路艰难的句子是：______，______。
（3）成语"秋水伊人""在水一方"出自本诗的句子是：______，______。

> **参考答案**：
> （1）蒹葭苍苍，白露为霜。
> （2）溯洄从之，道阻且长。
> （3）所谓伊人，在水一方。

**核心记忆三句话**：
1. 《蒹葭》是《诗经·秦风》怀人名篇，以**重章叠句**和**蒹葭白露起兴**，营造**凄清朦胧、可望不可即**的意境。
