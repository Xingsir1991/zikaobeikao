# 《八声甘州·对潇潇暮雨洒江天》

> 柳永（北宋） · 慢词·八声甘州 · 选自《乐章集》

::: tip 本篇定位
这是柳永**慢词**的代表作，也是宋词中**思乡怀人**的千古名篇。上阕以宏阔笔力描绘**江天暮雨、霜风残照**的萧瑟秋景，下阕转入**思乡怀人、羁旅愁苦**的深情抒发。苏轼盛赞其"**人皆言柳耆卿词俗，然如'霜风凄紧，关河冷落，残照当楼'，唐人高处，不过如此**"——其写景之笔不减唐人高处。
:::

## 一、知人论世

### 1. 作者其人
- **柳永（约 984—约 1053）**，原名**三变**，字**耆卿**，排行第七，又称"**柳七**"；官至屯田员外郎，世称"**柳屯田**"。
- 福建崇安人，北宋**第一个专力写词**的词人，**婉约派**代表人物。
- 屡试不第，流落江湖，混迹市井勾栏，与歌妓乐工交往甚密，其词多写**都市繁华、男女离别、羁旅愁思**。
- **柳永两大贡献**：①**大力创作慢词**（长调），扩充了词的体制；②**大量吸收市井俚语**入词，使词通俗化、"凡有井水饮处，皆能歌柳词"。
- 代表作：《雨霖铃》《八声甘州》《望海潮》等。有《**乐章集**》传世。

### 2. 创作背景
- 柳永一生**漂泊江湖、羁旅天涯**，多次漫游江南、两湖一带，求仕不第、萍踪不定。
- 此词当作于其**漫游江南、登高思乡**之时。秋雨黄昏，江天暮色，登楼远望，思乡怀人，写下这首**羁旅思乡的代表作**。
- 上阕写景，下阕抒情，是典型的**"上景下情"**结构。

::: warning 柳永词的艺术特色
| 维度 | 内容 |
|------|------|
| 词派 | 婉约派代表（与李清照前后呼应） |
| 体制 | 大力创作**慢词**（长调），扩展词的篇幅容量 |
| 语言 | 大量使用**市井俗语**，通俗流丽 |
| 题材 | 都市繁华、男女离别、**羁旅愁思** |
| 风格 | 细密铺叙、情景交融、白描见长 |
| 影响 | "凡有井水饮处，皆能歌柳词" |
:::

### 3. 文体常识——慢词与《八声甘州》词牌
- **慢词**：指**字数较多的长调词**（91 字以上），区别于字数较少的**小令**。
  - 慢词节奏舒缓、篇幅宏大，便于**铺叙**景物、展开情节。
  - 柳永是**第一个大量创作慢词**的词人，使慢词与小令并驾齐驱，扩大了词的表现力。
- **《八声甘州》词牌**：
  - 又名"甘州""潇潇雨"，**双调**，**97 字**（一说 98 字）。
  - 上下阕，节奏舒缓跌宕，**宜于抒写羁旅愁思、苍凉感慨**。
  - "八声"指全词押八次韵（八个韵脚），故名。
  - 本词押平水韵"**尤**"韵（秋、楼、休、流、收、留、舟、愁），一韵到底。

## 二、原文与翻译

> 对潇潇暮雨洒江天，一番洗清秋。渐霜风凄紧，关河冷落，残照当楼。是处红衰翠减，苒苒物华休。唯有长江水，无语东流。
>
> 不忍登高临远，望故乡渺邈，归思难收。叹年来踪迹，何事苦淹留。想佳人妆楼颙望，误几回、天际识归舟。争知我，倚阑杆处，正恁凝愁！

**【上阕（写景）】**
面对着傍晚时分潇潇秋雨洒落江天，一阵雨洗出一个清冷的秋。渐渐地，寒霜似的风凄冷地紧刮起来，关山河川变得冷落萧条，一抹残阳正照在楼头。到处是红花凋零、绿叶枯减，一切美好的景物都在慢慢凋残。只有那长江水，默默无语地向东奔流。

**【下阕（抒情）】**
实在不忍心登高远望，望向那遥远渺茫的故乡，归乡的思绪难以收敛。可叹这一年来的漂泊踪迹，究竟为了什么事苦苦淹留在外？遥想家中的佳人，此刻正在妆楼上凝神远望，不知多少回错把天边的船当成我归来的船。她怎会知道，我此刻倚靠栏杆的地方，正这样凝望着、愁绪万千！

## 三、字词与炼字

### 1. 词语释义
| 字词 | 释义 |
|------|------|
| **潇潇** | 雨势急骤的样子（拟声 + 状貌）。 |
| **暮雨** | 傍晚的雨。 |
| **洒江天** | 雨洒落在江面和天空（"江天"——江面与天空，开阔远景）。 |
| **一番洗清秋** | 一阵（雨）洗出一个清冷的秋天。一番——一阵。 |
| **渐** | 时间副词，**渐渐**——写景由远及近、由大到小的层次。 |
| **霜风** | 带寒霜的冷风，秋风之寒者。 |
| **凄紧** | 凄冷而紧刮（风势急、寒意重）。"紧"——急、烈。 |
| **关河** | 关山河川，泛指山河、地理（暗含行旅之远）。 |
| **冷落** | 萧条冷清。 |
| **残照当楼** | 夕阳的余晖照在楼头。残照——夕阳余晖；当——对着、照在。 |
| **是处** | 到处、处处。 |
| **红衰翠减** | 红花衰败、绿叶凋谢（**借代**：红代花，翠代叶）。 |
| **苒苒（rǎn rǎn）** | 渐渐地、慢慢地（一说同"荏苒"，时间流逝貌）。 |
| **物华休** | 万物的光华、美好都已凋残。物华——美好的景物；休——凋残、停止。 |
| **无语东流** | 默默无语地向东奔流。 |
| **不忍登高临远** | 不忍心登高望远（因望则思乡，思则更苦）。 |
| **渺邈（miǎo miǎo）** | 遥远渺茫。 |
| **归思（sì）难收** | 归乡的思绪难以收敛、平息。思——思绪、思念（此处读 sì，名词）。 |
| **年来踪迹** | 一年来的漂泊足迹。 |
| **何事** | 为什么、为了什么事。 |
| **苦淹留** | 苦苦地久留在外。淹留——长期滞留。 |
| **颙（yóng）望** | 抬头凝望、呆望。颙——仰望、专注地望。 |
| **误几回** | 错认了多少回。 |
| **天际识归舟** | 远远地从天边认出（以为是）归来的船。化用谢朓"天际识归舟，云中辨江树"。 |
| **争知** = "怎知" | 怎么会知道。**争 = 怎**（疑问代词，方言词，柳永常用俗语）。 |
| **倚阑杆** | 倚靠栏杆。阑 = 栏（通假）。 |
| **正恁（nèn）** | 正这样、正在这般。恁——如此、这样（俗语）。 |
| **凝愁** | 凝结不解的愁苦。 |

### 2. 炼字与名句赏析（⭐重点）

- ⭐⭐⭐**"洗"字**（"一番洗清秋"）：
  "洗"字是**炼字的典范**——本为雨洗万物，这里却说雨"洗"出一个清秋，**化实为虚、以动写静**。一"洗"字，把暮雨后秋天的**清冷、洁净、澄澈**境界和盘托出，新颖奇绝，胜过千言万语的描摹。

- ⭐⭐⭐**"渐"字**（"渐霜风凄紧"）：
  "渐"字写出**景物的层次与动态**——风是渐渐凄紧的，景是渐渐萧条的，由暮雨到霜风到残照，**层次分明、由远及近、由大到小**。一字统领"霜风凄紧、关河冷落、残照当楼"三句，是写景的总枢。

- ⭐⭐⭐**"紧"字**（"霜风凄紧"）：
  王国维、苏轼都极赏此"紧"字。前人写秋风多用"急""厉"，柳永用"紧"——**既写风的急烈、又写寒气的逼人**，含**触觉与听觉的双重感受**，胜过寻常"急"字。

- ⭐⭐⭐**"红衰翠减"**（借代炼意）：
  以"红"代花、以"翠"代叶，**色彩借代**——既写出花叶凋零，又以浓艳之色反衬衰败之惨，**绚烂与衰飒对照**，富有视觉冲击。

- ⭐⭐⭐**"无语东流"**（以景写情的极致）：
  "唯有长江水，无语东流"——长江无语东流，是**景**；但"无语"二字却含**情**：①江水无情，任凭景物凋零而默然流去；②江水似有恨而**无言**，映衬词人之愁深难言；③江水东流，**喻时光流逝、人生漂泊**。**以景结情、含蓄不尽**。

- ⭐⭐**"不忍"**（"不忍登高临远"）：
  "不忍"二字含**痛苦与矛盾**——本欲登高望乡，又怕望乡更增愁苦。一字之中，写出**思乡之切与愁苦之深**的矛盾心理。

- ⭐⭐**"误几回"**（侧面描写，对写）：
  "误几回、天际识归舟"——不直写自己思家，而**设想佳人错认归舟**，以**对方的失望**反衬自己的思归之切。这是**"对写"（从对方写起）**的经典手法（与杜甫"今夜鄜州月，闺中只独看"异曲同工）。

## 四、意象与意境

### 1. 核心意象及象征义
| 意象 | 字面义 | 象征义 / 情感内涵 |
|------|--------|------------------|
| **潇潇暮雨** | 傍晚急雨洒落江天 | 萧瑟凄清，烘托愁绪 |
| **清秋** | 雨后清冷的秋天 | 季节之衰、心境之冷 |
| **霜风凄紧** | 寒风凛冽 | 寒意逼人、孤旅凄苦 |
| **关河冷落** | 山河萧条 | 行旅之远、漂泊之孤 |
| **残照当楼** | 夕阳余晖照楼 | 迟暮之感、孤寂之悲 |
| **红衰翠减** | 花叶凋零 | 万物衰败、青春易逝 |
| **长江无语东流** | 江水默然东去 | 时光流逝、人生漂泊、愁深难言 |
| **故乡渺邈** | 故乡遥远渺茫 | 思乡之切、归乡之难 |
| **归舟**（误识） | 错认归舟 | 佳人盼望之切、自己思归之深 |
| **阑干、凝愁** | 倚栏凝愁 | 羁旅愁思、无可奈何 |

### 2. 意境画面（⭐苏轼所赏"唐人高处"）
- **上阕（写景）——阔大萧瑟、苍凉壮美**：
  一幅**秋雨黄昏的江天暮景图**——潇潇暮雨洒满江天，雨洗出一个清秋；霜风渐起、凄冷急烈，关山河川冷落萧条，残阳夕照斜倚楼头；到处花凋叶落，万物的光华渐渐凋残；唯有长江水默默东流。意境**开阔高远而又萧瑟苍凉**，故苏轼赞其"唐人高处，不过如此"。
- **下阕（抒情）——缠绵哀婉、深情绵邈**：
  由登高望乡转入思归，再设想佳人妆楼错认归舟，最后回到自己倚栏凝愁。意境由**宏阔转为深细**，由**写景转入抒情**，由**自己推及对方**，情感层层深入，缠绵不绝。

### 3. 景情关系（⭐重点）
- **上景下情、情景交融**：上阕纯写景（暮雨秋景），下阕纯抒情（思乡怀人），但**景中含情、情中带景**——"唯有长江水，无语东流"既是景又是情，景情合一。
- **以阔景衬深愁**：上阕秋景愈是**阔大萧瑟**，愈衬托出下阕**思乡之愁的深沉无尽**——**壮景反衬孤愁**，是柳永的妙笔。
- **乐景哀情 / 衰景哀情**：此处为**衰景写哀情**——以萧瑟秋景渲染愁绪，景与情相一致（不同于《虞美人》的乐景写哀反衬）。

## 五、艺术手法

### ⭐⭐⭐ 核心手法：上景下情、铺叙展衍（慢词特色）
- 这是柳永**慢词铺叙手法**的典型：上阕以**层层铺叙**写秋景（暮雨→霜风→残照→花叶→长江），层次分明；下阕以**层层转折**抒情（不忍登高→思乡难收→叹淹留→想佳人→回到自己），情思缠绵。
- "铺叙展衍、备足无余"是柳永慢词的标志性特征。

### 其他重要手法
- ⭐⭐⭐**借景抒情 / 情景交融**：上阕暮雨秋景全是抒情之资，"无语东流"以景结情。
- ⭐⭐⭐**对写（从对方写起，虚实相生）**：
  - "想佳人妆楼颙望，误几回、天际识归舟"——不直写自己思乡，而**设想佳人在家盼望、错认归舟**；
  - 以**对方之苦衬自己之愁**，加倍写出相思之深。与杜甫"今夜鄜州月，闺中只独看"、白居易"想闺中只独看"同妙。
- ⭐⭐**白描**：写景不堆砌典故，以朴素笔触勾勒（潇潇暮雨、霜风残照、红衰翠减），**平易而传神**。
- ⭐⭐**借代**："红衰翠减"以颜色代花叶。
- ⭐⭐**化用前人诗句**：
  - "天际识归舟"——化用**谢朓《之宣城郡出新林浦向板桥》"天际识归舟，云中辨江树"**；
  - "误几回"——化用**温庭筠《望江南》"过尽千帆皆不是"**之意。
- ⭐⭐**对比 / 反衬**：
  - **阔景与孤愁对比**——壮阔江天反衬游子孤独；
  - **物与人对比**——长江东流不息 vs 游子淹留难归；
  - **自己与佳人对比**——双方相互思念却天各一方。
- ⭐**以景结情**："唯有长江水，无语东流"——以景作结，含蓄不尽。
- ⭐**比喻（暗喻）**：长江东流暗喻**时光流逝、愁绪不绝**（与李煜"一江春水"异曲同工）。
- **设问 / 反诘**："何事苦淹留？""争知我……"——以反诘加重愁情。
- **时空交织**：时间（暮雨→残照→年来踪迹）与空间（江天→关河→故乡→妆楼）交织，使愁思无时无处不在。
- **俗语入词**："争知"（怎知）、"正恁"（正这样）、"颙望"等——体现柳永**以俗语入词**的特点。

::: warning 三段式答题模板（慢词鉴赏分析）
**第一步（明内容 / 手法）**：本词为柳永**慢词**代表作，采用**"上景下情"**结构，运用**铺叙、借景抒情、对写**等手法。
**第二步（释运用）**：① 上阕层层铺叙暮雨、霜风、残照、长江等秋景（如"霜风凄紧，关河冷落，残照当楼"）；② 下阕"想佳人妆楼颙望，误几回、天际识归舟"为**对写**——从对方写起，虚实相生。
**第三步（说效果）**：阔大萧瑟之景反衬深沉思乡之愁，对写加倍写出相思之深，表达**羁旅思乡、怀人伤别**之情，体现柳永慢词**铺叙展衍**的艺术特色。
:::

## 六、主旨情感

全词通过**秋雨黄昏江天暮景**的阔大萧瑟描写，转入**羁旅思乡、怀人伤别**的深情抒发，表达了柳永**漂泊天涯的孤寂愁苦、对故乡的深切思念、对远方佳人的眷恋与愧疚**。

**核心情感四重**：①羁旅漂泊之苦；②思乡归家之切；③怀人念远之深；④自责淹留之愧（"何事苦淹留"）。

苏东坡之所以盛赞其写景"唐人高处，不过如此"，正因柳永以慢词之宏阔体制、铺叙之细密笔法，把**游子秋思**写得既壮阔又深细，达到了情景交融的极高境界。

## 七、考点对接

### 常考题型
1. 炼字题（"洗""渐""紧""不忍""误"等，**高频**）。
2. 手法题（上景下情、铺叙、对写、借景抒情、借代、化用）。
3. 名句赏析（"霜风凄紧，关河冷落，残照当楼""想佳人妆楼颙望""唯有长江水，无语东流"）。
4. 情感题（羁旅思乡、怀人）。
5. 结构题（上下阕关系、层次铺叙）。

### 预测题

**【题 1·炼字】** 请赏析"对潇潇暮雨洒江天，一番洗清秋"中"洗"字的妙处。

> **参考答案**：
> ① "洗"字是**炼字的典范**——本为暮雨洗万物，词人却写雨"洗"出一个清秋，**化实为虚、以动写静**；
> ② 一"洗"字，把雨后秋天的**清冷、洁净、澄澈**境界和盘托出，胜过千言万语的描摹，新颖奇绝；
> ③ "洗"字还与"洒"字呼应，一"洒"一"洗"，写出暮雨的气势与秋天的清冷，**奠定全词萧瑟清冷的基调**，为下文抒情蓄势。

**【题 2·名句赏析】** 苏轼盛赞"霜风凄紧，关河冷落，残照当楼"为"唐人高处，不过如此"。请分析其妙处。

> **参考答案**：
> ① 此三句以**白描**手法，勾勒出一幅**阔大萧瑟的秋日江天暮景图**——霜风凄烈、关河萧条、残阳当楼，意境苍凉壮美；
> ② **层次分明**——"渐"字统领，由风（霜风）到地（关河）再到日（残照），由远及近、由大到小，**铺叙展衍**，是慢词写景的典型笔法；
> ③ "紧""冷落""残"等字炼意精到，**景物中饱含凄凉萧瑟之情**，景情交融；
> ④ 苏轼所赏，正是其**写景的阔大境界与沉郁气象**，直追唐人高处，突破了花间词的狭小格局。

**【题 3·手法】** 下阕"想佳人妆楼颙望，误几回、天际识归舟"运用了什么独特手法？有何效果？

> **参考答案**：
> ① 运用了**"对写"（从对方写起，虚实相生）**的手法——不直写自己思念佳人，而**设想佳人在家登楼盼望、多少回错把远处的船认作自己归舟**；
> ② **以对方之苦衬自己之愁**——加倍写出相思之深（与杜甫"今夜鄜州月，闺中只独看"同妙）；同时化用谢朓"天际识归舟"、温庭筠"过尽千帆皆不是"之意，**化用前人诗意**；
> ③ 虚实结合——此为**虚写**（想象），与上文实写（自己登高思乡）相呼应，**使情感更为深细缠绵**，是柳永慢词抒情的妙笔。

**【题 4·手法】** "唯有长江水，无语东流"历来为人称道，请赏析其妙处。

> **参考答案**：
> ① **以景结情**——上阕纯写景，以此句作结，含蓄不尽；
> ② **景中有情**——"无语"二字含三重意味：江水无情而默然流去；江水似有恨而**无言**，映衬词人愁深难言；江水东流**喻时光流逝、人生漂泊**；
> ③ **对比反衬**——长江东流不息，反衬游子淹留难归；万物凋零而江水永恒，反衬人生短促与漂泊无依；
> ④ 此句**以景写情、景情合一**，为下阕抒情作铺垫，承上启下，是全词写景的**诗眼**。

**【题 5·默写与常识】**
① 柳永《八声甘州》中被苏轼盛赞"唐人高处不过如此"的三句是："________，________，________。"
② 以"对写"手法设想佳人盼望、错认归舟的句子是："________，________。"
③ 以景结情、含蓄不尽的句子是："________，________。"
④ 简答：柳永是北宋____派代表，本词属____（小令 / 慢词），词牌是《____》。

> **参考答案**：
> ① 渐霜风凄紧，关河冷落，残照当楼；
> ② 想佳人妆楼颙望，误几回、天际识归舟；
> ③ 唯有长江水，无语东流；
> ④ 婉约；慢词；八声甘州。

**核心记忆三句话**：
1. **柳永《八声甘州》——慢词思乡怀人的代表作**，"上景下情"，铺叙展衍，苏轼盛赞其"霜风凄紧，关河冷落，残照当楼"——"唐人高处，不过如此"。
2. **核心手法**：①**对写**（"想佳人妆楼颙望，误几回、天际识归舟"）——从对方写起，加倍写相思；②**以景结情**（"唯有长江水，无语东流"）；③**炼字精到**（"洗""渐""紧"）。
3. **核心情感**：羁旅漂泊之苦 + 思乡怀人之深 + 淹留自责之愧，是柳永漂泊一生、天涯游子的典型心声。
