# 《早雁》

> 杜牧（晚唐） · 七言律诗 · 选自《樊川文集》

::: tip 本篇定位
这是一首**托物寄兴（咏物抒怀）**的七言律诗。表面咏雁，实则以"早雁"喻指**遭受回鹘侵扰、流离失所的北方边关百姓**，含蓄地表达对国事的忧虑与对苦难人民的深切同情。咏物而不滞于物，寄托深远，是晚唐咏物诗的代表作。
:::

---

## 一、知人论世

### 1. 作者其人
- **杜牧（803—852）**，字**牧之**，京兆万年（今陕西西安）人。宰相杜佑之孙。
- 晚唐杰出诗人，与李商隐并称"**小李杜**"（区别于李白、杜甫的"大李杜"）。
- 诗风**清丽俊爽、雄姿英发**，尤擅**七言绝句**和**咏史怀古**，代表作有《泊秦淮》《山行》《清明》《阿房宫赋》等。
- 有经邦济世之志，关注边事国政，诗中常含**忧时伤世**之情。世称"杜紫微""杜樊川"，有《樊川文集》。

### 2. 时代背景（⭐理解寄托的关键）
- 此诗作于**唐武宗会昌二年（842 年）秋**（一说作于会昌年间）。
- 时**回鹘（hé）**（即回纥，维吾尔族先祖）南侵，**大举寇边**，攻掠天德、振武等北方边镇，"虏杀掠甚众"，大批边民流离失所、南逃避难。
- 八月秋半，北方边塞因战乱而提早惊飞的大雁（即"**早雁**"——比正常南归时节提前惊飞之雁），触动了杜牧对边关百姓的同情。
- "虏弦开"即指**回鹘弯弓搭箭**、发动侵略战争，故大雁惊飞四散。

### 3. 文体常识——咏物寄兴诗
- **咏物诗**：以自然界之物（动植物、器物）为描写对象，要求**"不即不离"**——既贴合物的特征，又不滞于物。
- **托物寄兴（比兴）**：借咏物来寄托情怀、寄寓感慨。最高境界是"**物我合一、言在此而意在彼**"。
- 本诗是典型的**咏物寄兴**——字字写雁，又字字写人，"早雁"实为**流离边民**的化身。

## 二、原文与翻译

> 金河秋半虏弦开，云外惊飞四散哀。
> 仙掌月明孤影过，长门灯暗数声来。
> 须知胡骑纷纷在，岂逐春风一一回？
> 莫厌潇湘少人处，水多菰米岸莓苔。

**【首联】** 八月秋半之时，金河一带回鹘拉开了侵略的弓弦；大雁在云天之外惊惶地四散飞逃，发出阵阵哀鸣。

**【颔联】** 月明之夜，大雁孤单的身影从汉建章宫的仙人承露盘前飞过；长门宫灯火暗淡处，传来它的一声声哀叫。

**【颈联】** 须知那胡人的骑兵还纷纷驻扎在北方边塞，怎能指望大雁像往年那样逐春风一一北归呢？

**【尾联】** 大雁啊，请不要嫌弃潇湘一带人烟稀少，那里水中多菰米、岸上多莓苔，正可暂时安身避难。

## 三、字词与炼字

### 1. 词语释义
| 字词 | 释义 |
|------|------|
| **金河** | 河名，在今内蒙古呼和浩特一带（唐代边地），代指北方边塞。 |
| **秋半** | 八月（仲秋），指秋季过半。此时雁尚不该南飞，因战乱而**提前惊飞**，故称"早雁"。 |
| **虏弦开** | 胡人（回鹘）拉开了弓弦，发动战争。"虏"是对回鹘的蔑称。**双关**：既指射猎雁之弓弦，又指回鹘寇边的战弓。 |
| **惊飞四散** | 惊惶飞起、四处逃散（喻边民流离）。 |
| **仙掌** | 西汉建章宫前神明台上的**铜仙人承露盘**（捧铜盘承接仙露），在长安。借指京城、皇宫。 |
| **长门** | 汉代**长门宫**（冷宫，陈阿娇被废居之处），在长安。借指孤寂冷落的宫苑。 |
| **孤影 / 数声** | 孤单的身影、几声哀鸣——状大雁（边民）孤苦哀怨。 |
| **胡骑（jì）** | 胡人的骑兵，指回鹘军队。 |
| **逐春风** | 追逐春风北归（雁为候鸟，春暖北归）。 |
| **一一回** | 一只接一只飞回北方。 |
| **潇湘** | 湘江及其支流潇水，在今湖南一带（南方），代指南方偏远水乡。 |
| **菰（gū）米** | 菰（一种水生植物）的籽实，可食，雁喜食之。 |
| **莓苔** | 苔藓，雁的食物之一。 |

### 2. 炼字赏析（⭐重点）

- ⭐⭐⭐**"虏弦开"**（一语双关，托物之眼）：
  字面是猎人拉弓射雁的弓弦，实指**回鹘寇边的战弓**。一"开"字，把边塞战事与大雁惊飞的关系揭示无遗——雁之惊，非因秋寒，而是因"虏弦"。这是全诗**托物寄兴的"诗眼"**，由物及人、由景及事，分量极重。

- ⭐⭐**"惊"字**：
  "惊飞四散"——一"惊"字写出大雁（边民）的**惊惶、恐惧、猝不及防**之态，传神地写出战乱突至、百姓仓皇逃难的情状，物（雁）与人（边民）合而为一。

- ⭐⭐**"哀"字**：
  一"哀"字定下全诗感情基调，大雁四散之哀鸣，即**边民流离失所之哀号**，言简意深。

- ⭐⭐**"孤""暗"**（颔联炼字）：
  "**孤**影过"——以"孤"写雁之形单影只，亦喻边民**离群失所、孤苦无依**；"长门灯**暗**"——以"暗"写宫苑冷落萧条，烘托**凄清哀怨**氛围。月明、灯暗对照，孤影、数声相映，意境凄美。

- ⭐**"莫厌"**（尾联炼字）：
  "莫厌潇湘少人处"——一"莫厌"如对失所之人温言相劝，殷殷叮嘱，**饱含诗人对苦难边民的深切同情与关切**，语浅情深。

## 四、意象与意境

### 1. 核心意象及象征义（⭐托物寄兴的关键）
| 意象（写雁） | 字面义 | 象征义（喻人） |
|--------------|--------|----------------|
| **早雁**（秋半惊飞之雁） | 因战乱提前南飞的大雁 | **流离失所的北方边关百姓** |
| **虏弦开** | 猎弓 / 战弓拉开 | **回鹘侵边的战争** |
| **惊飞四散** | 大雁惊散逃飞 | **边民仓皇南逃、家破人亡** |
| **孤影 / 数声哀** | 雁过孤影、哀鸣 | **边民孤独无依、哀苦无告** |
| **仙掌、长门** | 汉宫建筑（京城） | 喻指朝廷；孤影过仙门，哀声入冷宫——**朝廷虽闻哀声却无力相救** |
| **胡骑纷纷在** | 胡人骑兵驻扎 | **北方边患未除、战火未息** |
| **逐春风一一回** | 雁春归北方 | **百姓盼望北归故土（却不可能）** |
| **潇湘、菰米、莓苔** | 南方水乡、雁之食物 | **南方暂可避难安身之所** |

### 2. 意境画面
全诗营造了**凄清哀怨、忧深思远**的意境。秋半的金河、拉开的虏弦、云外四散的哀鸣；月明之夜孤雁的影子掠过仙掌、长门暗淡的灯火下传来数声哀叫；北方胡骑未去、归乡无望；唯有潇湘水乡，菰米莓苔，可暂作栖息之地。一幅**边塞战乱、雁（民）离失所的凄凉图卷**徐徐展开，意境**凄美而沉痛**。

### 3. 景情关系
- **以物喻人、物我交融**：全诗无一字直说"边民"，但字字不离边民之苦。"早雁"既是雁，更是人——**言在此而意在彼**。
- **寓情于景、情景交融**：仙掌月明、长门灯暗——清冷景物烘托哀怨情感。
- **托物言志（忧国忧民）**：通过对早雁的同情，寄托诗人**忧时伤世、关怀民生**的情怀。

## 五、艺术手法

### ⭐⭐⭐ 核心手法：托物寄兴（咏物抒怀）
- 全诗表面**咏雁**，实则以雁**喻人**（流离边民），这是中国古典诗歌**"比兴"**传统的体现。
- 要求咏物诗**"不即不离"**——既贴合物（雁）的特征（候鸟、秋半惊飞、月下孤影、食菰米莓苔、逐春风北归），又有所寄托（边民流离、归乡无望）。本诗达到了**物我交融、不粘不脱**的高妙境界。
- **"早雁"是核心意象**——"早"字点明战乱致使雁（民）提前南逃，是全诗寄托的关键。

### 其他重要手法
- ⭐⭐⭐**一语双关**："虏弦开"——既是射雁之弓，又是寇边之战弓。一语双关，由物及事，是全诗**点睛之笔**。
- ⭐⭐**借代**："虏"代回鹘，"胡骑"代回鹘军队，"仙掌""长门"代朝廷宫苑（汉代唐）。
- ⭐⭐**用典（借古讽今）**：用汉代"仙掌（建章宫铜仙人承露盘）""长门（冷宫）"代指唐代宫苑，**以汉喻唐**——既避免直刺当朝，又增加历史厚重感。
- ⭐⭐**对比（反衬）**：
  - "**月明**"对"**灯暗**"——明暗对比，烘托凄清；
  - "**仙掌**（华贵）"对"**长门**（冷落）"——盛衰对比；
  - "**孤影**（视觉）"对"**数声**（听觉）"——视听结合，渲染孤苦。
- ⭐**设问 / 反问**："岂逐春风一一回？"——以反问句强调归乡之不可能，**含蓄而沉痛**。
- ⭐**拟人（语带劝慰）**："莫厌潇湘少人处"——以对雁温言相劝的口吻，表达对边民的同情，情真意切。
- **视听结合**：孤影（视）、数声（听）；惊飞（动）、哀鸣（声）。
- **虚实结合**：前四句实景（雁惊飞过仙掌长门），后四句虚写（设想胡骑未去、归乡无望、劝留潇湘），虚实相生。

::: warning 三段式答题模板（咏物寄兴诗分析）
**第一步（明手法）**：本诗运用了**托物寄兴（咏物抒怀 / 比兴）**的手法，以"早雁"为咏写对象。
**第二步（析物与寄）**：① 扣物——抓住雁"秋半惊飞""孤影哀鸣""逐春风而归""食菰米莓苔"等特征；② 寄托——以早雁喻指遭受回鹘侵扰、流离失所的北方边民，"虏弦开"双关寇边战事。
**第三步（说效果）**：达到物我交融、不即不离之境，含蓄地表达了对国事的忧虑与对边民的深切同情，言在此而意在彼，意味深长。
:::

## 六、主旨情感

本诗是一首**托物寄兴**的政治抒情诗。诗人借咏"早雁"，曲折而深沉地表达了对会昌年间**回鹘侵边、百姓流离**的关切与忧虑，以及对苦难边民的深切同情。字里行间流露着诗人**忧国忧民、伤时念乱**的情怀，体现了晚唐有识之士对国势衰颓的深沉忧患。

**核心情感四重**：①对边患（虏弦）的忧愤；②对边民（早雁）流离的同情；③对归乡无望的悲慨；④对苦难者"莫厌潇湘"的温言劝慰。

## 七、考点对接

### 常考题型
1. 手法题（托物寄兴 / 咏物抒怀，几乎是必考点）。
2. 炼字题（"虏弦开""惊""孤""暗""莫厌"等）。
3. 意象分析题（早雁的象征义）。
4. 情感题（忧国忧民 / 同情边民）。
5. 双关、用典、对比等具体手法。

### 预测题

**【题 1·核心手法】** 这首诗在表现手法上最突出的特点是什么？请结合全诗分析。

> **参考答案**：
> 最突出的特点是**托物寄兴（咏物抒怀）**。
> ① 全诗字字咏"雁"，抓住其"秋半惊飞""孤影哀鸣""逐春风北归""食菰米莓苔"等待征；
> ② 实则以"早雁"喻指**遭受回鹘侵边、流离失所的北方边民**——"虏弦开"双关回鹘寇边之战事，"惊飞四散"喻边民仓皇南逃，"孤影数声"喻其孤苦无告，"岂逐春风一一回"写其归乡无望，"莫厌潇湘"乃劝其暂避南方；
> ③ 全诗**不即不离、物我交融**，言在此而意在彼，含蓄地表达了诗人忧时伤世、关怀民瘼的深沉情怀，是晚唐咏物寄兴诗的代表作。

**【题 2·炼字】** 请赏析首联"金河秋半虏弦开"中"虏弦开"三字的妙处。

> **参考答案**：
> ① "虏弦开"运用了**一语双关**手法：字面指猎人拉弓射雁之弓弦，实指**回鹘寇边的战弓**；
> ② 由"虏弦开"引出"惊飞四散哀"——揭示雁（边民）惊飞的原因，不是秋寒，而是战争，**点明寄托的关键**；
> ③ 一"开"字将咏物（射雁）与寄兴（寇边）巧妙勾连，是全诗**托物寄兴的"诗眼"**，言简意丰，分量极重。

**【题 3·意象】** 诗中的"早雁"是一个怎样的意象？它寄托了诗人怎样的情感？

> **参考答案**：
> "早雁"是全诗的**核心意象**，具有**物我双重性**：
> ① **物的一面**——指因北方战乱（虏弦）而于秋半（早于常规南归时节）被迫惊飞南逃的大雁，特征是孤苦、惊惶、流离；
> ② **寄托的一面**——喻指会昌年间因回鹘侵边而**流离失所、仓皇南逃的北方边民**；
> ③ 这一意象寄托了诗人对边患的忧愤、对苦难百姓的深切同情、对归乡无望的悲慨，以及对苦难者的温言劝慰，体现了诗人**忧国忧民**的情怀。

**【题 4·手法】** 颔联"仙掌月明孤影过，长门灯暗数声来"运用了哪些艺术手法？有何效果？

> **参考答案**：
> ① **用典（借古喻今）**："仙掌"（汉建章宫铜仙人承露盘）、"长门"（汉冷宫）——以汉代唐，借宫苑之典，既避直刺，又增厚重；
> ② **对比（反衬）**："月明"对"灯暗"——明暗对比；"仙掌"（华贵）对"长门"（冷落）——盛衰对比，烘托凄清哀怨；
> ③ **视听结合**："孤影过"（视觉）、"数声来"（听觉），渲染孤苦凄凉；
> ④ 全联借宫廷景象，烘托早雁（边民）的孤苦哀怨，暗含**朝廷虽闻哀声却无力相救**的深意。

**【题 5·情感】** 尾联"莫厌潇湘少人处，水多菰米岸莓苔"表达了诗人怎样的情感？

> **参考答案**：
> ① 表层是劝慰大雁：不要嫌弃潇湘一带人烟稀少，那里水中多菰米、岸上多莓苔，可作栖息觅食之所；
> ② 深层是诗人以**温言劝慰**的口吻，对**流离失所的边民**表达深切的同情与关怀——既然北方胡骑未除、归乡无望，不如暂且在南方安身；
> ③ "莫厌"二字情真意切，含**无奈中的劝慰、同情中的关切**，体现了诗人忧国忧民、关怀民生疾苦的深沉情怀。

**核心记忆三句话**：
1. **杜牧《早雁》——托物寄兴的代表作**，字字写雁、字字写人，"早雁"喻指遭受回鹘侵边的北方流民。
2. **"虏弦开"一语双关**——既是射雁之弓，又是寇边战弓，是全诗托物寄兴的诗眼。
3. **主旨：忧国忧民**——通过对早雁的描写，含蓄表达对边患的忧虑与对流离边民的深切同情，体现晚唐有识之士的忧患意识。
