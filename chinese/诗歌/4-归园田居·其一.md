# 《归园田居·其一》（陶渊明）

> 陶渊明（东晋） · 五言古诗（五古） · 出处：《陶渊明集》

::: tip 本篇定位
《归园田居·其一》是陶渊明辞彭泽令归隐后所作组诗的第一首，集中表达了他"久在樊笼里，复得返自然"的喜悦和对官场"尘网"的厌恶。本诗是**中国田园诗的开山之作**，奠定了陶诗"自然平淡、意境深远"的美学风格。
:::

---

## 一、知人论世

**作者简介**：陶渊明（约 365—427），一名潜，字元亮，私谥"靖节先生"，浔阳柴桑（今江西九江）人，东晋末南朝宋初伟大诗人。曾祖陶侃曾为晋大司马，至渊明时家道中落。他先后任江州祭酒、镇军参军、建威参军、彭泽令等职，仅 80 余日即辞官归隐（"不为五斗米折腰"）。是中国**第一位田园诗人**，开创了田园诗派，与谢灵运的山水诗并称"陶谢"。

**文学地位**：
- **田园诗派的开创者**，被誉为"隐逸诗人之宗"。
- 钟嵘《诗品》推为"古今隐逸诗人之宗"。
- 苏轼评："其诗质而实绮，癯而实腴。"
- 对后世王维、孟浩然、李白、苏轼等影响深远。

**文体常识——五言古诗（五古）**：
- 起源于汉代（如《古诗十九首》），成熟于建安，至陶渊明达到新高峰。
- **特点**：每句五字，句数不限（本诗 20 句），不求对仗、不限平仄、不讲究严格押韵，**形式自由、语言质朴、抒情性强**。
- 区别于近体诗（律绝）的格律严谨。

**写作背景**：
- 陶渊明 41 岁（405 年）任彭泽令，恰逢郡里派督邮来县，属吏劝他束带迎见，他叹曰："我岂能为五斗米折腰向乡里小儿！"即日辞官归隐，作《归去来兮辞》。
- 归隐后作《归园田居》五首（组诗），本篇为第一首，写归隐后的感受。
- "一去三十年"句，"三十年"是夸饰之辞（实为十三年，29 岁初仕至 41 岁归隐），一说为"十三年"之误，言官场生活之漫长。

**陶诗总体风格**：自然平淡、质朴真率、意境深远、富理趣。"一语天然万古新，豪华落尽见真淳"（元好问评）。

::: warning 与谢灵运山水诗对比（陶谢并称）
| 比较项 | 陶渊明（田园诗） | 谢灵运（山水诗） |
| --- | --- | --- |
| 题材 | 田园风光、农村生活 | 名山大川、自然山水 |
| 风格 | 平淡自然、意境深远 | 雕琢工丽、富丽精工 |
| 情感 | 真率淡泊、寄寓哲理 | 玄理 + 山水赏玩 |
| 句式 | 五言为主 | 多山水佳句、对仗 |
| 地位 | 田园诗派之祖 | 山水诗派之祖 |
:::

## 二、原文与翻译

> 全诗 20 句，可分三层：① 首四句 + "误落"六句，写归隐缘由；② 中间十句，写田园风光与生活；③ 末四句，写归隐的喜悦。

| 原文 | 翻译 |
| --- | --- |
| 少无适俗韵， | 从小就没有迎合世俗的性格与情趣， |
| 性本爱丘山。 | 我的本性原本就喜爱山川丘壑。 |
| 误落尘网中， | 误入官场的尘网（束缚）之中， |
| 一去三十年。 | 一去就是许多年（三十年，夸张言久）。 |
| 羁鸟恋旧林， | 笼中的鸟眷恋它原来的树林， |
| 池鱼思故渊。 | 池中的鱼思念它原来的深潭。 |
| 开荒南野际， | 在南边田野的边际开垦荒地， |
| 守拙归园田。 | 安守本分（不随俗）归隐田园。 |
| 方宅十余亩， | 住宅四周有十多亩田地， |
| 草屋八九间。 | 茅草搭的房屋有八九间。 |
| 榆柳荫后檐， | 榆树柳树遮蔽着屋后的房檐， |
| 桃李罗堂前。 | 桃树李树排列在堂前。 |
| 暧暧远人村， | 远处的村落隐隐约约， |
| 依依墟里烟。 | 村里的炊烟依稀可辨、袅袅升起。 |
| 狗吠深巷中， | 狗在深巷中吠叫， |
| 鸡鸣桑树颠。 | 鸡在桑树顶上鸣啼。 |
| 户庭无尘杂， | 门庭里没有世俗尘杂之事， |
| 虚室有余闲。 | 闲居的居室有充裕的清闲。 |
| 久在樊笼里， | （就像）长久关在樊笼里一样， |
| 复得返自然。 | 如今又能回到大自然（自由自在）。 |

### 2. 古今异义与字义辨析
| 字词 | 古义 | 今义 |
| --- | --- | --- |
| 适俗韵（韵） | 气质、风度、情趣 | 韵母、韵律 |
| 守拙（拙） | 不巧饰、守本性（褒义/谦辞） | 笨拙（贬义） |
| 虚室（虚） | 空闲、清静（道家境界） | 虚假、空虚 |
| 自然 | ①大自然 ②顺应本性 | 仅指自然界 |
| 颠 | 顶部 | 颠簸、跌倒 |

### 3. 炼字精妙（⭐高频考点）

**① ⭐⭐"暧暧远人村，依依墟里烟"——"暧暧""依依"（叠词炼字）**
- **"暧暧"**：写远村隐隐约约、若隐若现的朦胧感，远而不清；
- **"依依"**：写炊烟袅袅轻柔上升、悠然飘散的情态，柔美含情。
- 两组叠词**绘形传神**，把远景的朦胧与近景的柔和对照写出，营造恬静安谧的田园意境；同时**音韵和谐**，节奏舒缓，与诗人心境的悠然自得相契合。

**② "榆柳荫后檐，桃李罗堂前"——"荫""罗"（动词炼字）**
- **"荫"**（遮蔽）：以动写静，写出榆柳高大繁茂、绿荫遮檐的画面；
- **"罗"**（排列）：写出桃李成行、罗列堂前的整齐美景。
- 两字皆用作动词，**化静为动**，使画面生机勃勃，富有立体感和层次感。

**③ ⭐"羁鸟恋旧林，池鱼思故渊"——"羁""恋""思"（情感炼字）**
- **"羁"**（束缚）：一"羁"字写尽官场对人性的束缚压抑；
- **"恋""思"**：以鸟、鱼的本性喻人之本性——对自由的向往和对田园的眷恋是天性使然，不可磨灭。
- 全联以物自比，**对仗工整、形象鲜明**，是千古传诵的归隐名句，集中表现诗人对自由的渴望。

**④ "守拙归园田"——"守拙"**
- "拙"是巧之反，看似自谦，实为坚守本真、不迎合世俗的骄傲。"守拙"二字，体现陶渊明傲岸不屈的人格——宁可"拙"，也要守住本心，不肯随波逐流。

**⑤ ⭐⭐"复得返自然"——"自然"（诗眼）**
- "自然"一词**双关**：表层指回到大自然（田园山林）；深层指**回归本性、顺应天性**（与开篇"性本爱丘山"呼应）。
- 一个"返"字，写尽久困樊笼后重获自由的欣喜——"复得"二字尤见珍贵。这是全诗的诗眼和主旨所在。

## 四、意象与意境

### 1. 核心意象分析

**（一）比喻性意象（抒怀）**
| 意象 | 象征义 |
| --- | --- |
| **尘网 / 樊笼** | 喻指官场——充满尘俗、束缚人性的地方（贯穿首尾）。 |
| **羁鸟 / 池鱼** | 自喻——官场中失去自由、怀念本性的自己。 |
| **旧林 / 故渊** | 喻指田园——本性所归、自由自在的所在。 |
| **适俗韵** | 世俗的迎合与虚伪——诗人所不愿学。 |

**（二）田园意象（写景）**
| 意象 | 象征义 / 情感色彩 |
| --- | --- |
| **方宅草屋**（十余亩、八九间） | 朴素简陋的田园生活，体现诗人知足安贫。 |
| **榆柳桃李**（荫后檐、罗堂前） | 田园的生机勃勃、绿荫满目。 |
| **远村墟烟**（暧暧、依依） | 远景朦胧、田园的恬静安宁。 |
| **狗吠鸡鸣**（深巷、桑颠） | 以动衬静，写乡村的幽静与生机（化用汉乐府《鸡鸣》"鸡鸣高树颠，狗吠深宫中"）。 |
| **户庭虚室** | 远离尘俗的清静闲适。 |

### 2. 意境描绘
全诗营造的是一幅**恬静、安谧、自然、悠然**的田园生活画卷：
近处——方宅草屋、榆柳桃李围绕；远处——朦胧村落、袅袅炊烟；耳畔——狗吠深巷、鸡鸣树颠。一切都是那样自然、宁静、充满生机。这是诗人历经官场尘网之后回归本真、重获自由的精神家园——既是写实（田园实景），更是写意（心灵归宿）。

### 3. 景情关系（⭐情景交融的典范）
- **景中含情**：所有田园景物（草屋、榆柳、远村、墟烟）都浸透诗人归隐的喜悦——"一切景语皆情语"。
- **以景衬情**：恬静安谧的田园景物，反衬出对官场"尘网""樊笼"的厌恶；美好的田园，正是诗人心灵的归宿。
- **由景入理**：景物描写背后寄托"返自然"（回归本性）的人生哲理，景情理三者交融，这是陶诗高于一般田园诗之处。

## 五、艺术手法

### 1. ⭐白描手法（陶诗标志性手法）
中间十句写田园，纯用**白描**——不施雕琢、不用典故、不事夸张，只用朴素自然的语言勾勒画面："方宅十余亩，草屋八九间""榆柳荫后檐，桃李罗堂前""狗吠深巷中，鸡鸣桑树颠"。
**作用**：质朴自然，如话家常，正契合田园生活的本色与诗人淡泊的心境，体现陶诗"豪华落尽见真淳"的美学追求。

### 2. ⭐⭐情景交融
景物描写（田园风光）与情感抒发（归隐之喜）水乳交融。恬静的田园景物既是写实，更是诗人内心喜悦与宁静的投射——景因情设、情因景生。

### 3. ⭐比喻（自喻）——羁鸟、池鱼
"羁鸟恋旧林，池鱼思故渊"——以"羁鸟""池鱼"自比，以"旧林""故渊"喻田园。
**作用**：形象生动地表达对自由的渴望、对官场的厌恶；对仗工整，含蓄隽永。

### 4. ⭐⭐对比（贯穿全诗的核心结构）
- **尘网/樊笼 vs 田园/自然**：官场的束缚与田园的自由对比；
- **羁鸟/池鱼 vs 旧林/故渊**：被囚禁与归本性的对比；
- **"误落""一去"（过去）vs "复得""返"（现在）**：今昔对比。
**作用**：通过对比，鲜明地表现归隐后的喜悦和对官场的厌恶，结构上首尾呼应（"尘网"开篇、"樊笼"作结）。

### 5. 动静结合（以动衬静）
"狗吠深巷中，鸡鸣桑树颠"——以狗吠鸡鸣的动（声音），反衬乡村的宁静。
**作用**：以动衬静，愈见其静，营造幽静安宁的田园意境（化用《诗经》"鸡栖于埘，日之夕矣，羊牛下来"与汉乐府的写法）。

### 6. 远近结合、视听结合
- **远近**：近景（方宅、草屋、榆柳、桃李）+ 远景（远村、墟烟），层次分明。
- **视听**：视觉（草屋、绿荫、远村、炊烟）+ 听觉（狗吠、鸡鸣），画面立体丰富。

### 7. 叠词的运用
"暧暧""依依"等叠词，既绘形传神，又增加音韵美。

### 8. 首尾呼应、结构圆融
开篇"少无适俗韵，性本爱丘山"——结句"久在樊笼里，复得返自然"；"尘网"开篇——"樊笼"作结。首尾呼应，结构完整，主题鲜明。

::: warning ⭐三段式答题模板（"赏析某联"或"艺术手法"类）
**第一段——明手法、释句意**：本联运用了……手法（如白描/情景交融/比喻/动静结合），写出了……（如"恬静的田园风光"）。
**第二段——析运用**：结合具体字词分析，如"方宅草屋的朴素""榆柳桃李的生机""狗吠鸡鸣以动衬静"等。
**第三段——点作用**：这样写营造了……意境，表达了诗人……情感（如归隐的喜悦、对官场的厌恶、对自然的热爱），体现了陶诗……的风格。
:::

## 六、主旨情感

**中心思想**：本诗通过写归隐田园后的所见所感，表达了诗人**对自然、自由的热爱和对官场虚伪束缚的厌恶**，体现他**傲岸不屈、坚守本真、不慕荣利**的高洁人格和**返璞归真**的人生追求。

**情感层次**：
1. **对官场的厌恶**："误落尘网""一去三十年""久在樊笼里"——官场是束缚、是尘俗、是煎熬。
2. **对本性的坚守**："少无适俗韵，性本爱丘山""守拙"——天性如此，不可改变。
3. **对自由的渴望**："羁鸟恋旧林，池鱼思故渊"——对自由的向往是天性。
4. **归隐的喜悦**："复得返自然""户庭无尘杂，虚室有余闲"——重获自由的欣喜与宁静。
5. **人生哲理的领悟**："返自然"既是回到田园，更是回归本性——寄寓道家顺应自然、返璞归真的生命哲理。

**文学价值**：
- 中国田园诗的开山之作，奠定田园诗派的美学基础。
- 开创了"归隐"主题的经典范式，影响后世王维、孟浩然、李白、苏轼等无数文人。
- "尘网""樊笼""羁鸟""池鱼""返自然"等意象与表达，已成为汉语常用典故和成语。

## 七、考点对接

### 常考题型
1. 赏析"羁鸟恋旧林，池鱼思故渊"。
2. 分析中间写景部分的艺术手法（白描、动静、远近、叠词）。
3. "守拙""自然"等字词的含义。
4. 选择题（手法、情感、字词）。
5. 理解性默写。

### 预测题与标准答案

**【题 1·名句赏析（高频）】** 请赏析"羁鸟恋旧林，池鱼思故渊"。

> **参考答案**：①**运用比喻（自喻）**：以"羁鸟""池鱼"自比，喻在官场失去自由的自己；以"旧林""故渊"喻自己向往的田园。②**对仗工整**：羁鸟对池鱼、旧林对故渊，句式整饬，音韵和谐。③**情感深刻**："恋""思"二字写尽对自由的渴望、对田园的眷恋——这是天性使然，不可磨灭。④**承上启下**：紧承"性本爱丘山"，引出下文"开荒南野际，守拙归园田"的归隐行动。两句形象鲜明、含蓄隽永，是千古传诵的归隐名句。

**【题 2·手法分析】** 这首诗写景部分（"方宅十余亩……鸡鸣桑树颠"）运用了哪些艺术手法？请简要分析。

> **参考答案**：①**白描**：用朴素自然的语言勾勒田园景物（方宅、草屋、榆柳、桃李），不施雕琢，正契合田园本色和诗人淡泊心境。②**动静结合（以动衬静）**："狗吠深巷中，鸡鸣桑树颠"以动写静，反衬乡村的幽静安宁。③**远近结合**：近景（方宅草屋、榆柳桃李）与远景（远村、墟烟）层次分明。④**视听结合**：视觉（绿荫、炊烟）与听觉（狗吠、鸡鸣）交融，画面立体。⑤**叠词传神**："暧暧""依依"绘形传神、音韵和谐。这些手法共同营造恬静安谧的田园意境，情景交融地表达归隐的喜悦。

**【题 3·炼字/字词】** "守拙归园田"中"守拙"和"复得返自然"中"自然"各有何含意？

> **参考答案**：①**"守拙"**："拙"是巧之反，看似自谦，实为**坚守本真、不迎合世俗**的骄傲——宁可"拙"也要守住本心，不肯随波逐流，体现诗人傲岸不屈的人格。②**"自然"**：**双关**。表层指大自然（田园山林）；深层指**顺应本性、回归天性**（与开篇"性本爱丘山"呼应）。"返自然"既是回到田园，更是回归本性——这是全诗的诗眼和主旨。

**【题 4·选择题】** 下列对这首诗的赏析，**不正确**的一项是（　）
A. "少无适俗韵，性本爱丘山"开篇直抒胸臆，奠定全诗感情基调。
B. "一去三十年"是夸张之辞，极言为官时间之久、内心煎熬之深。
C. 诗中"尘网""樊笼""羁鸟""池鱼"等意象都含有比喻义，寄托对官场的厌恶。
D. 全诗以浓墨重彩、雕琢华丽的语言描绘田园，表达对自然的赞美。

> **参考答案**：D。陶诗风格是**自然平淡、质朴真率**，"豪华落尽见真淳"，并非"浓墨重彩、雕琢华丽"。

**【题 5·理解性默写】**
（1）《归园田居·其一》中表达诗人本性喜爱自然、不愿迎合世俗的句子是：______，______。
（2）《归园田居·其一》中以鸟鱼自喻、表达对自由渴望的千古名句是：______，______。
（3）《归园田居·其一》中以动衬静、写乡村宁静的句子是：______，______。
（4）《归园田居·其一》中点明主旨、表达归隐喜悦的诗眼句是：______，______。

> **参考答案**：
> （1）少无适俗韵，性本爱丘山。
> （2）羁鸟恋旧林，池鱼思故渊。
> （3）狗吠深巷中，鸡鸣桑树颠。
> （4）久在樊笼里，复得返自然。

```
                                    坚守本真·返璞归真
                                    （田园诗开山之作）
```

**核心记忆三句话**：
1. 《归园田居·其一》是**田园诗开山之作**，五古名篇，结构上以**"尘网—樊笼"对比"田园—自然"**首尾呼应，表达归隐之喜。
2. 核心手法：① **白描**写田园（朴素自然）；② **比喻自喻**——"羁鸟恋旧林，池鱼思故渊"写对自由的渴望；③ **动静结合**——"狗吠深巷中，鸡鸣桑树颠"以动衬静；④ **情景交融**。
3. **"复得返自然"是诗眼**，"自然"双关——既指大自然，更指回归本性；体现陶渊明傲岸不屈、返璞归真的人格，"豪华落尽见真淳"的美学。
