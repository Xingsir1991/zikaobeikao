# 《登高》

> 杜甫（盛唐） · 七言律诗 · 选自《杜工部集》

::: tip 本篇定位
此诗作于杜甫晚年（767 年）夔州重阳登高之时，为"古今七律第一"（胡应麟《诗薮》评）。八句皆对仗，章法、句法、字法无一不精，集沉郁顿挫之大成，是律诗格律与抒情完美融合的巅峰之作。
:::

---

## 一、知人论世

### 1. 作者其人
- **杜甫（712—770）**，字**子美**，自号**少陵野老**，世称"杜工部""杜少陵"。
- 祖籍襄阳，生于巩县（今河南巩义）。出身"奉儒守官"之家，自称"杜陵布衣"。
- 我国文学史上最伟大的**现实主义诗人**，与李白并称"**李杜**"。其诗被称为"**诗史**"，本人被尊为"**诗圣**"。
- 诗风**沉郁顿挫**：内容深厚凝重（沉郁），音节气势波澜起伏（顿挫）。

### 2. 时代背景
- 此诗作于**唐代宗大历二年（767 年）秋**，杜甫时年 56 岁，流寓夔州（今重庆奉节）。
- 安史之乱虽已平定（763 年），但藩镇割据、吐蕃入侵，国势仍危。
- 杜甫此时**疾病缠身**（肺病、风痹、糖尿病），好友李白、高适、严武相继辞世，孤独无依，登高远眺，触景伤怀。

::: warning 杜甫与李白对比
| 维度 | 李白 | 杜甫 |
|------|------|------|
| 时代 | 盛唐气象 | 由盛转衰 |
| 流派 | 浪漫主义 | 现实主义 |
| 称号 | 诗仙 | 诗圣 |
| 风格 | 飘逸豪放 | 沉郁顿挫 |
| 体式 | 擅古体、乐府 | 擅律诗 |
:::

### 3. 文体常识——七言律诗
- **四联八句**：首联、颔联、颈联、尾联。
- **字数句式**：每句七字，共 56 字。
- **押韵**：偶句押韵（首句可入韵），一韵到底，本诗押平水韵**"十灰"韵**（哀、回、来、台、杯）。
- **平仄**：严格讲究粘对。
- **对仗**：颔联、颈联必须对仗。**本诗八句皆对仗**，为全唐所仅见，故被胡应麟推为"古今七律第一"。

## 二、原文与翻译

> 风急天高猿啸哀，渚清沙白鸟飞回。
> 无边落木萧萧下，不尽长江滚滚来。
> 万里悲秋常作客，百年多病独登台。
> 艰难苦恨繁霜鬓，潦倒新停浊酒杯。

**【首联】** 秋风劲急、天宇高远，猿猴的叫声凄凉哀怨；水清沙白的小洲上，鸟儿在盘旋飞舞（无处栖身）。

**【颔联】** 茫茫无际的落叶，在秋风中"萧萧"地飘零而下；奔流不息的长江，掀起"滚滚"波浪汹涌而来。

**【颈联】** 离家万里、长年漂泊，于悲凉秋景中作客他乡；一生多病、孤独无依，独自登上高台远眺。

**【尾联】** 历尽艰难、深怀遗恨，白发如霜染满了双鬓；困顿潦倒、愁苦难遣，近来连这浊酒也被迫停饮了。

## 三、字词与炼字

### 1. 词语释义
| 字词 | 释义 |
|------|------|
| **猿啸哀** | 猿猴哀鸣。三峡多猿，啼声凄厉，《水经注》载"巴东三峡巫峡长，猿鸣三声泪沾裳"。 |
| **渚（zhǔ）** | 水中的小洲。 |
| **回** | 盘旋、回旋。鸟因风急难飞，盘旋无依。 |
| **落木** | 落叶。"木"指树，"落木"出自屈原《九歌》"袅袅兮秋风，洞庭波兮木叶下"。 |
| **萧萧** | 拟声词，草木摇落之声。 |
| **滚滚** | 形容江水奔流翻涌之势。 |
| **万里** | 极言离家之远。 |
| **作客** | 客居他乡（区别于"做客"，指寄居流寓）。 |
| **百年** | 借指一生、暮年。 |
| **繁霜鬓** | 两鬓白发如霜增多。繁，增多。 |
| **潦倒** | 衰颓、失意。 |
| **新停** | 最近停止（因病戒酒）。 |

### 2. 炼字赏析（⭐重点）

- ⭐⭐**"萧萧""滚滚"**（叠词炼字）：
  "萧萧"摹落叶之声，"滚滚"状江水之势。叠词既**绘声绘色**，又构成**时空无限**之感——落木"无边"显其广，长江"不尽"显其长，将短暂渺小的个体置于永恒浩渺的宇宙之中，反衬人之渺小、生命之短促。声调上平仄相对（萧萧平平，滚滚仄仄），音韵铿锵。

- ⭐⭐**"急""高""哀""清""白""回"**（首联六字炼字）：
  首联十四字写六景，**字字写景、字字含情**。"急"——风之猛烈（秋寒逼人）；"高"——天宇辽阔（衬托人之渺小）；"哀"——猿声凄厉（心境悲凉）；"清""白"——色彩清冷（孤寂）；"回"——鸟盘旋难栖（喻诗人漂泊无依）。**以鸟之盘旋喻人之流寓**，物我交融。

- ⭐**"常""独"**（颈联炼字）：
  "常"——**时间之长**（年年作客、岁岁漂泊）；"独"——**孤独之深**（无亲无友、形单影只）。二字将空间（万里）、时间（百年）、情感（悲秋、多病、孤独）四重愁苦层层叠加。

- ⭐**"繁""新"**（尾联炼字）：
  "繁"——白发增多之状，写岁月催人；"新"——最近才停酒，写病情加重。二字写出**衰老与疾病的双重打击**，更显潦倒之悲。

## 四、意象与意境

### 1. 核心意象及象征义
| 意象 | 字面义 | 象征义 / 情感内涵 |
|------|--------|------------------|
| **急风、高天、哀猿** | 秋日萧瑟 | 环境凄厉，烘托悲凉心境 |
| **清渚、白沙、回鸟** | 江边清冷景象 | 孤寂清冷；**回鸟=诗人漂泊无依** |
| **落木萧萧** | 万木凋零 | 生命衰败、韶华易逝、家国凋敝 |
| **长江滚滚** | 大江东去 | 时光流逝、宇宙永恒、人生短暂 |
| **万里、百年** | 空间与时间 | 时空双重苦难，漂泊与衰老 |
| **霜鬓、浊酒** | 白发、戒酒 | 老病潦倒、壮志难酬 |

### 2. 意境画面
全诗营造了**雄阔高远而又悲凉沉郁**的意境。前四句写**秋江登高所见之景**：天空高远、秋风劲急、猿啼哀怨、鸟雀盘旋，落木萧萧、长江滚滚，构成一幅**苍茫壮阔、肃杀萧条**的三峡秋日图。后四句转入**抒情**，景与情交融，由自然之秋转入人生之秋，意境由壮阔到悲怆，层层深入。

### 3. 景情关系（⭐重点）
- **前四句写景、后四句抒情**，但景中含情、情中有景。
- **以阔景衬悲情**（以乐景反衬哀情、以壮景反衬孤苦）：长江落木之壮阔，反衬诗人之渺小孤独；宇宙之永恒，反衬人生之短促。**"一切景语皆情语"**之典范。

## 五、艺术手法

### ⭐⭐⭐ 核心手法：八句皆对仗（古今七律第一）
- 律诗本只要求颔联、颈联对仗，**本诗八句皆对仗**，前无古人。
- **首联**："风急天高"对"渚清沙白"，"猿啸哀"对"鸟飞回"——**句中自对**兼**上下相对**。
- **颔联**："无边落木"对"不尽长江"，"萧萧下"对"滚滚来"——**流水对**，气势磅礴。
- **颈联**："万里"对"百年"，"悲秋"对"多病"，"常作客"对"独登台"——**数字对**（万里/百年）+ **叠意对**，十四字含**八层悲意**。
- **尾联**："艰难苦恨"对"潦倒新停"，"繁霜鬓"对"浊酒杯"——收束有力。

### 其他重要手法
- ⭐⭐**情景交融**：前四句借景抒情，后四句直抒胸臆，景为情设，情因景生。
- ⭐⭐**对偶 / 叠词**：萧萧、滚滚；落木、长江——音韵铿锵，回环往复。
- ⭐**反衬**：以长江落木之**壮阔永恒**反衬人之**渺小短暂**。
- ⭐**借景抒情（寓情于景）**：以秋日萧瑟之景写漂泊悲秋之情。
- **层层铺陈（八层悲意）**：颈联"万里悲秋常作客，百年多病独登台"，**十四字含八重愁苦**：①作客（漂泊）②常作客（久漂泊）③秋作客（悲秋）④万里作客（远漂泊）⑤登台（重九登高）⑥独登台（无亲无友）⑦百年多病（一生多病）⑧多病独登台（病而孤）。
- **视听结合**：风急天高（视）、猿啸哀（听）、萧萧（听）、滚滚（视/听）。

::: warning 三段式答题模板（艺术手法分析）
**第一步（明手法）**：本诗运用了……手法（如八句皆对仗、情景交融、叠词等）。
**第二步（释运用）**：结合诗句具体分析，如"萧萧""滚滚"为叠词，写落叶之声、江水之势。
**第三步（说效果）**：表达……情感，有……效果（音韵铿锵、时空无限、衬托悲凉等）。
:::

## 六、主旨情感

全诗通过登高所见的秋江萧瑟壮阔之景，抒发了诗人**漂泊思乡之苦、老病孤独之悲、忧国伤时之恨、壮志难酬之叹**。前四句写**自然之秋**，后四句写**人生之秋**，将个人身世之悲与家国沦丧之痛融为一体，体现了杜诗"沉郁顿挫"的典型风格，也凝结着"诗圣"忧国忧民的博大情怀。

**核心情感四重**：①悲秋伤时；②漂泊思乡；③老病孤独；④壮志难酬。

## 七、考点对接

### 常考题型
1. 炼字题（如赏析"萧萧""滚滚""急""独"等）。
2. 手法题（八句皆对仗、情景交融、反衬、叠词）。
3. 情感题（颈联"八层悲意"分析）。
4. 比较题（与杜甫其他登高 / 秋兴诗比较）。
5. 名句默写（颔联、颈联为高频默写句）。

### 预测题

**【题 1·炼字】** 请赏析颔联"无边落木萧萧下，不尽长江滚滚来"中"萧萧""滚滚"两词的妙处。

> **参考答案**：
> ①"萧萧""滚滚"为叠词，"萧萧"摹写落叶纷纷飘落之声，"滚滚"描绘长江波浪汹涌之势（释意）；
> ②叠词运用使音节铿锵、富有节奏感和音乐美，且增强了气势（音韵效果）；
> ③与"无边""不尽"配合，营造出**时空无限、宇宙永恒**的壮阔意境，反衬出**诗人生命的短暂与个体的渺小**，含蓄地抒发了韶华易逝、壮志难酬的悲凉之情（情感效果）。手法 + 效果 + 情感，三层兼备。

**【题 2·手法】** 本诗被誉为"古今七律第一"，请结合全诗分析其在对仗上的突出成就。

> **参考答案**：
> 律诗本只要求颔联、颈联对仗，而本诗**八句皆对仗**，是杜诗格律精严的极致体现。首联"风急天高"与"渚清沙白"句中自对且上下相对；颔联"无边落木"对"不尽长江"为流水对，气势磅礴；颈联"万里悲秋常作客"对"百年多病独登台"是数字对，十四字含八层悲意；尾联"艰难苦恨"对"潦倒新停"收束有力。全诗对仗工整而不板滞，**气韵流动、沉郁顿挫**，故被胡应麟推为"古今七律第一"。

**【题 3·情感】** 前人评颈联"万里悲秋常作客，百年多病独登台"含"八层悲意"，请具体分析。

> **参考答案**：
> 十四字包含八重悲苦：①"作客"——漂泊他乡之悲；②"常作客"——久漂不归之悲；③"秋作客"——秋日悲秋之悲；④"万里作客"——离家遥远之悲；⑤"登台"——重九登高易感之悲；⑥"独登台"——无亲无友之悲；⑦"百年多病"——一生多病之悲；⑧"多病独登台"——病而孤苦之悲。层层叠加，将个人之悲与秋景之衰融为一体，沉郁至极。

**【题 4·意境】** 本诗前四句与后四句在意境与情感上如何转换？请简要分析。

> **参考答案**：
> 前四句**写景**，描绘风急、天高、猿哀、渚清、沙白、鸟回、落木萧萧、长江滚滚等秋江萧瑟壮阔之景，营造雄阔悲凉的意境；后四句**抒情**，由自然之秋转入人生之秋，抒发漂泊、老病、孤独、忧国之悲。**由景及情，情景交融**，前之壮景反衬后之孤悲，层次分明而又浑然一体。

**【题 5·默写】**
① 杜甫《登高》中，写景宏大、被后人誉为"古今七言律第一联"的是："________，________。"
② 集中表现诗人多重悲苦、含八层悲意的一联是："________，________。"
③ 表现诗人因病戒酒、潦倒失意的句子是："________，________。"

> **参考答案**：①无边落木萧萧下，不尽长江滚滚来；②万里悲秋常作客，百年多病独登台；③艰难苦恨繁霜鬓，潦倒新停浊酒杯。
