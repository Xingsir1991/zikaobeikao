# 《九歌·湘夫人》（屈原）

> 屈原（战国·楚） · 楚辞体（骚体） · 出处：《楚辞·九歌》

::: tip 本篇定位
《湘夫人》是屈原《九歌》中祭祀湘水之神的乐歌，与《湘君》为姊妹篇。全诗以湘君（男主角）等待、思念、幻想、失落的心理历程，表达对湘夫人的深切眷恋，是楚辞中典型的"巫神相恋"祭神抒情诗，开后世浪漫主义先河。
:::

---

## 一、知人论世

**作者简介**：屈原（约前340—前278），名平，字原，战国后期楚国贵族、政治家、伟大爱国诗人。他"博闻强志，明于治乱，娴于辞令"，曾任左徒、三闾大夫，主张联齐抗秦、举贤授能。后遭谗被疏，流放沅湘，于楚国都城郢被秦军攻破后，自沉汨罗江而死。作品有《离骚》《九歌》《九章》《天问》等，是中国文学史上第一位伟大的个人诗人。

**文体常识——楚辞体（骚体）**：
- **句式**：以"兮"字为标志，参差错落，多为"○○○兮○○○"的六七言句式，突破《诗经》四言束缚。
- **内容**：多写巫风神灵、楚地山川，富有神话色彩与浪漫情调。
- **开创者**：屈原。"楚辞"之名始见于汉代，刘向辑录成集。
- **名物**：多香草美人（兰、芷、薜荔、杜衡等），形成"香草美人"比兴传统。
- **押韵**：通篇押韵，多换韵。

**《九歌》常识**："九歌"本古乐曲名，屈原在楚地民间祭神乐歌基础上加工润色，共 11 篇：《东皇太一》《东君》《云中君》《湘君》《湘夫人》《大司命》《少司命》《河伯》《山鬼》《国殇》《礼魂》。是祭祀天地神灵的乐歌。

**湘水二神**：湘君（男神）与湘夫人（女神）是楚地所奉湘水配偶神。《湘君》写湘夫人候湘君不至，《湘夫人》写湘君候湘夫人不至，二篇互为表里。

**写作背景**：一般认为作于屈原被放逐沅湘时期，借祭神之曲寄寓自己的政治理想与现实哀怨——以神神相恋喻君臣遇合，以思神不至喻忠君而见弃。

::: warning 楚辞与《诗经》对比
| 比较项 | 《诗经》（北方） | 《楚辞》（南方） |
| --- | --- | --- |
| 句式 | 四言为主，二二节拍 | 句式参差，以"兮"字为标志 |
| 体裁 | 现实主义源头 | 浪漫主义源头 |
| 风格 | 温柔敦厚、质朴 | 浪漫瑰丽、奇幻 |
| 内容 | 民歌、政治、农事 | 神话、巫风、个人抒情 |
| 作者 | 多佚名 | 以屈原为代表（个人创作） |
| 影响 | "风"传统 | "骚"传统（"风骚"并称） |
:::

## 二、原文与翻译

> 全诗可分为四节：① 开篇写候神不至的愁思；② 反复思念与失意；③ 幻想筑室水中迎神；④ 终不可见，捐袂遗褋、采芳远赠，表达怅惘与旷达。

### 2. 炼字精妙（⭐高频考点）

**① ⭐⭐"袅袅兮秋风，洞庭波兮木叶下"——"袅袅""下"（千古名句）**
- **"袅袅"**：既写秋风微弱轻柔、不绝如缕，又暗写情思之绵长不断。一字双关，景中含情。
- **"波""下"**（动词）：水波翻动、落叶飘飞，写秋的动态萧瑟。"下"字干脆利落，把落叶飘零的画面定格，富有动态美与画面感。
- **整句妙处**：以三句极简之笔，写出秋风、水波、落叶——视、听、感交融，渲染出洞庭秋色凄清寥落的氛围，烘托主人公愁绪万千，被王国维誉为"千古言秋之祖"（与宋玉"悲哉秋之为气也"并称），对后世"落叶""悲秋"意象影响深远。

**② "目眇眇兮愁予"——"眇眇""愁予"**
- "眇眇"叠词，写极目远望、望眼欲穿，"愁予"直抒胸臆，把看不清伊人的焦急与愁苦和盘托出——"愁"字奠定全诗基调。

**③ "鸟何萃兮蘋中，罾何为兮木上"——"何"字（反问炼字）**
- 两个"何"字构成反问，鸟本应在树上却聚到水草中，鱼网本应在水中却架在树上——"所求非地、所托非所"的错位，暗示约期已到、佳人未来、一切期待落空，物反其常则人反其愿，抒情含蓄而强烈。

**④ "思公子兮未敢言"——"未敢言"**
- 三字写尽暗恋的苦楚——情深到不敢说、不能说、无人可说，情感压抑而炽烈。

## 四、意象与意境

### 1. 核心意象分析

**（一）自然意象——秋水洞庭**
| 意象 | 象征义 / 情感色彩 |
| --- | --- |
| **秋风 / 木叶 / 洞庭波** | 秋之萧瑟凄清，烘托主人公孤寂怅惘的心境；"木叶"成为后世"落叶悲秋"经典意象。 |
| **沅芷 / 澧兰 / 流水潺湲** | 香草喻佳人，流水喻绵长思念，"望穿秋水"而不得。 |
| **白薠 / 江皋 / 西澨** | 主人公四处眺望、奔波追寻的空间背景。 |

**（二）香草意象——"香草美人"传统（⭐）**
全诗香草密集出现：**芷、兰、荪、椒、桂、辛夷、薜荔、蕙、石兰、杜衡、百草、杜若**等十余种。
- **表层**：装点湘夫人居室，营造芬芳华美、圣洁脱俗的氛围。
- **深层**：以香草喻高洁品格、美好理想（楚辞"香草美人"传统），寄托诗人对美好政治理想的向往。
- 这些香草是屈原人格与理想的物化。

**（三）反常意象——"鸟何萃""罾何为"（物反其常）**
鸟栖水草、网挂树梢、麋食庭中、蛟困水裔——一系列反常之景，写"所托非所、所求不得"，是主人公约期已至、神人不至、希望落空的象征。

### 2. 意境描绘
全诗营造的是**凄美、空灵、缠绵、哀婉**而又**瑰丽奇幻**的意境。开篇即以"秋风袅袅、洞庭波涌、落叶纷飞"构筑深秋洞庭的寥廓凄清背景，主人公登白薠远望、期佳人不至、思而不敢言，怅惘低回。中段笔锋一转，进入瑰丽的想象世界：在水中用香草构筑华美的居室，九嶷众神如云而至迎接——浪漫奇幻、色彩斑斓。结尾却又回到现实，捐袂遗褋、采杜若赠远者，"时不可兮骤得"的怅惘与"聊逍遥兮容与"的旷达交织。

### 3. 景情关系
- **以秋景起兴，渲染氛围**：秋风落叶定下全诗悲秋基调。
- **借香草美人，托物言志**：香草既是装饰，又是品格理想的象征。
- **虚实相生，幻实对照**：幻想中筑室迎神的瑰丽（虚）与现实神终不至的怅惘（实）形成对照，使情感更见深沉。

## 五、艺术手法

### 1. ⭐楚辞体（骚体）句式
以"兮"字为节奏标志的六七言参差句式，如"帝子降兮北渚""袅袅兮秋风"。节奏舒缓绵长、错落有致，便于抒发缠绵委婉、深沉回环的情感，突破《诗经》四言的整饬，开了七言诗的先河。

### 2. ⭐⭐情景交融、借景抒情（核心手法）
"袅袅兮秋风，洞庭波兮木叶下"是情景交融的千古名句——秋景萧瑟凄清，正映衬主人公思而不遇的孤寂怅惘。景为情设，情因景生，**一切景语皆情语**。

### 3. ⭐"香草美人"比兴手法
以香草（芷兰桂椒等）喻高洁人格与美好理想，以"佳人/公子/帝子"喻贤君或理想对象。这是屈原开创的、影响深远的比兴传统，使诗歌在祭神爱情之外别有寄托。

### 4. ⭐丰富的想象与浪漫主义
幻想在水中以香草建造华美居室迎接神灵，众神如云而至——想象瑰丽、色彩斑斓，是屈原浪漫主义的集中体现。这种"水中筑室"的幻想，是对美好理想的极致构建。

### 5. 反衬、反问与对偶
- **反问**："鸟何萃兮蘋中，罾何为兮木上？""麋何食兮庭中？蛟何为兮水裔？"——以物之反常反衬人之失意，含蓄有力。
- **对仗**（虽为骚体但偶句工整）："桂栋兮兰橑，辛夷楣兮药房""罔薜荔兮为帷，擗蕙櫋兮既张"——句式对称、铺陈华美。

### 6. 铺陈（赋的手法）
中段铺写"筑室水中"用荷盖、荪壁、紫坛、芳椒、桂栋、兰橑、辛夷、药房、薜荔、蕙、白玉、石兰、杜衡、百草——铺陈不厌其烦，是典型的"赋"的手法（铺采摛文），渲染居室的华美圣洁。

### 7. 重章叠句的变体与一唱三叹
"思公子兮未敢言""时不可兮骤得"等情感的反复吟咏，"沅有芷兮澧有兰"等句式的对称呼应，造成一唱三叹的韵致。

### 8. 虚实相生
幻想的"筑室迎神"是虚，现实的"神终不至"是实；理想之美与现世之憾相互映衬，深化主题。

::: warning ⭐三段式答题模板（"赏析某句"或"艺术手法"类）
**第一段——明手法、释句意**：本句运用了……手法，写出了……（如"借景抒情，写秋风落叶洞庭波"）。
**第二段——析运用**：结合具体字词分析，如"袅袅""波""下"等字如何写景传情。
**第三段——点作用、明情感**：这样写营造了……意境，烘托了……情感，达到了……艺术效果（如情景交融、托物言志、含蓄蕴藉）。
:::

## 六、主旨情感

**中心思想**：本诗借湘君候湘夫人不至的故事，抒发**对美好理想的深切追求与现实失意的怅惘哀婉**；同时寄寓屈原自己忠君爱国却见放逐、壮志难酬的政治哀怨——以神神不遇喻君臣不合。

**情感层次**：
1. **期盼与等候**："与佳期兮夕张"——精心布置、满怀期待。
2. **焦急与愁苦**："目眇眇兮愁予""思公子兮未敢言"——望眼欲穿、情思压抑。
3. **失望与反常的愤懑**："鸟何萃兮蘋中，罾何为兮木上"——以物反常写希望落空。
4. **幻想与极致的构筑**：筑室水中、香草满庭——以瑰丽想象寄托美好理想。
5. **最终的旷达与执著并存**："时不可兮骤得，聊逍遥兮容与"——既有不能得到的释然旷达，又不舍对美好的追求（"将以遗兮远者"）。

**文学价值**：
- 楚辞浪漫主义的代表作，对后世"悲秋""香草美人""巫山云雨"等文学传统影响深远。
- "袅袅兮秋风，洞庭波兮木叶下"被誉为"千古言秋之祖"，杜甫"无边落木萧萧下"等均受其沾溉。
- 开创了"神人/君臣相恋相弃"的母题，成为后世寄寓政治失意的经典范式。

## 七、考点对接

### 常考题型
1. 赏析"袅袅兮秋风，洞庭波兮木叶下"千古名句。
2. 分析"香草美人"手法及其作用。
3. 理解"鸟何萃兮蘋中，罾何为兮木上"的妙处。
4. 分析本诗浪漫主义的表现。
5. 理解性默写。

### 预测题与标准答案

**【题 1·名句赏析】** 请赏析"袅袅兮秋风，洞庭波兮木叶下"。

> **参考答案**：这两句是千古传诵的写秋名句。
> ①**炼字传神**："袅袅"写秋风的轻柔绵长，又暗喻情思不绝；"波""下"两个动词，写洞庭水波翻涌、落叶纷纷飘落的动态画面，富有动感与画面感。
> ②**情景交融**：以洞庭秋色的萧瑟凄清，烘托湘君候神不至的孤寂怅惘，景为情设、情因景生。
> ③**艺术效果**：三句十八字，融视觉（落叶）、听觉（风声）、触觉（凉意）为一体，营造凄美空灵的意境，被后人誉为"千古言秋之祖"，对后世"落叶悲秋"文学影响深远。

**【题 2·手法分析】** 屈原作品常以"香草美人"为喻，请结合本诗简要分析。

> **参考答案**："香草美人"是屈原开创的比兴传统。
> ①**香草**：本诗密集出现芷、兰、荪、椒、桂、辛夷、杜衡、杜若等十余种香草，既装点湘夫人居室、渲染圣洁华美，又象征高洁的品格与美好的政治理想。
> ②**美人**：诗中"帝子""公子""佳人"指湘夫人，又可喻贤君或美好理想。
> ③**作用**：使全诗在祭神爱情的表层之下别有政治寄托，含蓄地表达诗人忠君爱国、追求美好理想而终不可得的哀怨，形成"言在此而意在彼"的含蓄美与象征意义。

**【题 3·句意理解】** "鸟何萃兮蘋中，罾何为兮木上"两句有何妙处？

> **参考答案**：①**运用反问**：两个"何"字构成反问，语气强烈。②**写反常之景**：鸟本应在树上却聚到水草中，渔网本应在水中却架在树上——"物反其常"。③**以物衬人、托物寓理**：以物之反常反衬人之失意，暗示主人公精心布置佳期却盼望落空，所求非所、所托皆空，含蓄而强烈地表达了希望破灭的怅惘愤懑。

**【题 4·综合选择】** 下列对这首诗的赏析，**不正确**的一项是（　）
A. "帝子"指湘夫人，传说为尧之女娥皇、女英，"眇眇""愁予"奠定全诗愁苦基调。
B. "筑室兮水中"一段运用铺陈，写湘君幻想以香草构筑华美居室迎接湘夫人，富浪漫色彩。
C. "时不可兮骤得，聊逍遥兮容与"表现了主人公彻底绝望、消极避世的情绪。
D. 全诗通过香草美人、想象幻想，寄寓了诗人对美好理想的追求和政治失意的哀怨。

> **参考答案**：C。结尾既有怅惘也有旷达与执著，并非"彻底绝望、消极避世"——"将以遗兮远者"仍含追求之意。

**【题 5·理解性默写】**
（1）《湘夫人》中被誉为"千古言秋之祖"、渲染凄清氛围的句子是：______，______。
（2）《湘夫人》中以反常之景反衬主人公失望之情的句子是：______，______？
（3）《湘夫人》中表现主人公思念却不敢明言、压抑深情的句子是：______，______。
（4）《湘夫人》结尾表达时机难再、姑且自遣旷达的句子是：______，______。

> **参考答案**：
> （1）袅袅兮秋风，洞庭波兮木叶下。
> （2）鸟何萃兮蘋中，罾何为兮木上。
> （3）思公子兮未敢言。（另：沅有芷兮澧有兰）
> （4）时不可兮骤得，聊逍遥兮容与。

## 八、知识图谱

```
                《九歌·湘夫人》（屈原·楚辞体）
                            │
       ┌────────────────────┼────────────────────┐
       │                    │                    │

   突破四言               期→望→幻→失→旷         ②香草美人（比兴）
       │                    │                  ③浪漫想象
       │              ┌─────┴─────┐            ④铺陈+反问
       │              │           │            ⑤虚实相生
       │          香草美人       悲秋意象
       │          （理想）       （秋水洞庭）        │
       │                                    【炼字·千古名句】
       │                              "袅袅兮秋风，洞庭波兮木叶下"
       │                              "袅袅"——景中含情、绵长
       │                              "波""下"——动感画面
       │                                            │
       └──────────────────────────────►         【主旨】

                                       而终不可得的怅惘哀怨

```

**核心记忆三句话**：
1. 《湘夫人》是屈原《九歌》中的祭神抒情名篇，以**楚辞骚体"兮"字句式**写湘君候湘夫人不至的思慕与失意，寄寓诗人**忠君见弃、壮志难酬**的政治哀怨。
2. **"袅袅兮秋风，洞庭波兮木叶下"是千古名句**，情景交融渲染凄美秋境；中段以**香草美人比兴**和**瑰丽想象**（水中筑室）构建理想境界，是屈原浪漫主义的集中体现。
3. 结尾"时不可兮骤得，聊逍遥兮容与"既有求之不得的怅惘，又有不舍追求的旷达——全诗在凄美哀婉中见执著，于瑰丽想象中寓深悲，是楚辞艺术的典范。
