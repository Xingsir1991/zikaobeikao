# 《虞美人·春花秋月何时了》

> 李煜（五代·南唐） · 词·虞美人 · 选自《全唐五代词》

::: tip 本篇定位
这是南唐后主**李煜的绝命词**，也是其词作的最高成就。以"问君能有几多愁？恰似一江春水向东流"的千古名喻，将抽象的亡国之愁化为具体奔涌的江水，被誉为**"以水喻愁"的登峰造极之作**。王国维《人间词话》评："词至李后主而眼界始大，感慨遂深。"
:::

---

## 一、知人论世

### 1. 作者其人
- **李煜（937—978）**，字**重光**，初名从嘉，南唐第三位国君，世称"**南唐后主**""李后主"。
- **961—975 年**在位，南唐被北宋所灭后，**投降宋朝**，被封"违命侯"，囚禁于汴京（今开封）。
- 工书画、精音律、尤擅**词**。其词以**降宋为界**，分为前后两期：
  - **前期**：写宫廷豪华生活、男女之情，绮丽柔靡（如《玉楼春》）。
  - **后期**：写**亡国之痛、故国之思、囚徒之悲**，凄凉悲壮、感慨深沉（如《虞美人》《浪淘沙》《破阵子》）。
- 王国维称其"**变伶工之词为士大夫之词**"——把词从花间派的"艳科"提升为抒写人生重大感慨的文学样式，是**词史上的关键转折人物**。

### 2. 创作背景（绝命词）
- 作于李煜**降宋后被囚禁的第三年（978 年）**的七夕（他的生日）。
- 当时他已被囚三年，过着"此中日夕，只以眼泪洗面"的生活。
- 据说此词写成后，因其中"小楼昨夜又东风，故国不堪回首月明中"等句，触怒宋太宗赵光义（嫌其"故国不堪回首"仍念旧国），**被赐牵机药毒死**。故此词被称为李煜的"**绝命词**"。

::: warning 李煜前后期词对比
| 维度 | 前期（南唐国君时） | 后期（降宋囚禁时） |
|------|--------------------|--------------------|
| 内容 | 宫廷享乐、男女恋情 | 亡国之痛、故国之思 |
| 风格 | 绮丽柔靡、华丽 | 凄凉悲壮、深沉 |
| 代表 | 《玉楼春》《浣溪沙》 | 《虞美人》《浪淘沙》《破阵子》 |
| 情感 | 浅近享乐 | 沉痛深邃 |
:::

### 3. 文体常识——词与《虞美人》词牌
- **词**：又称"**诗余**""**长短句**""**曲子词**"，是配合燕乐演唱的歌辞。
  - 词有**词牌**（曲调名称，规定字数、句式、平仄、押韵）和**题目**（内容，常省略）。
  - 按字数分为：**小令**（58 字以内）、**中调**（59—90 字）、**长调**（91 字以上，又称**慢词**）。
  - 一段称"**单调**"，两段称"**双调**"（上阕/下阕，或上片/下片），三段、四段称"三叠""四叠"。
- **《虞美人》词牌**：
  - 双调，**56 字**，上下阕各四句，两仄韵两平韵（平仄换韵）。
  - 句式：七、五、七、九（上下阕相同）。本词中"了、少、东风、中"等换韵严格。
  - 节奏短促跌宕，最宜抒写哀怨激荡之情。

## 二、原文与翻译

> 春花秋月何时了？往事知多少。小楼昨夜又东风，故国不堪回首月明中。
>
> 雕栏玉砌应犹在，只是朱颜改。问君能有几多愁？恰似一江春水向东流。

**【上阕】**
春花开了、秋月圆了，这样的美景何时才能了结（我宁愿它们不再来）？回首往事，又有多少悲欢离合尽在其中！昨夜，囚居的小楼上又刮起了东风（春天又来了），在明月之中，我不忍回首那已经亡掉的故国。

**【下阕】**
南唐故宫那些雕花的栏杆、白玉的台阶，想必应该还在吧？只是当年的人，朱颜已经改变（青春已逝、容颜憔悴）。问你能有多少愁苦？恰似一江浩荡的春水，滚滚向东流去，无穷无尽！

## 三、字词与炼字

### 1. 词语释义
| 字词 | 释义 |
|------|------|
| **春花秋月** | 春天的花、秋天的月，指代**一年四季的美好景物**。此处为**反用**——乐景反衬哀情，故盼其"何时了"。 |
| **何时了（liǎo）** | 何时了结、何时完结。了，完结。 |
| **往事** | 指过去做国君时的繁华岁月，与亡国后的屈辱形成对比。 |
| **小楼** | 指词人被囚禁于汴京的居所（与昔日"雕栏玉砌"的宫殿形成对比）。 |
| **又东风** | "又"字点明**春又来、年又过**，囚居岁月流逝；东风即春风。 |
| **故国** | 指已灭亡的**南唐故国**（都城金陵，今南京）。 |
| **不堪回首** | 不忍心回忆。"不堪"——不能承受、不忍。 |
| **雕栏玉砌** | 雕花的栏杆、玉石的台阶，借指南唐故宫的**华美宫殿**。 |
| **应犹在** | 应当还在（揣度之词）。应——大概、应该。 |
| **朱颜改** | 红润的容颜已经改变（衰老憔悴）；亦暗指江山易主、物是人非。 |
| **几多** | 多少。 |
| **一江春水** | 指长江（南唐都城金陵临长江）。 |

### 2. 炼字与名句赏析（⭐重点）

- ⭐⭐⭐**"又"字**（"小楼昨夜又东风"）：
  一个"又"字，**分量极重**——说明春已不止一次来，囚居已不止一年。岁月无情流逝，而亡国之痛不仅未减，反而**年复一年加深**。"又"字含**无奈、痛苦、绝望**多重意味，是炼字的典范。

- ⭐⭐⭐**"改"字**（"只是朱颜改"）：
  "改"字与上句"应犹在"相对——**宫殿犹在，而人事已非**：①词人自己朱颜改（衰老憔悴）；②"朱颜"也可指旧时宫人，皆已星散；③更暗指江山易主、**物是人非**。一"改"字写尽亡国之变、人生之痛。

- ⭐⭐**"何时了"**（反诘炼意）：
  良辰美景"春花秋月"本是赏心乐事，词人却问"何时了"——**以乐景写哀、反衬倍增其哀**。亡国之囚，美景反成刺激、反添痛苦，故盼其早日了结。一问之中，含无尽悲愤。

- ⭐⭐⭐**"问君能有几多愁？恰似一江春水向东流"**（千古名喻）：
  这是"**以水喻愁**"的登峰造极之作，妙处有三：
  ① **化抽象为具象**——把看不见、摸不着的"愁"，化为可见的"一江春水"，使愁有了形体；
  ② **化有限为无限**——春水滔滔、连绵不绝、滚滚东流，喻愁之**深、广、长、不断**；
  ③ **设问 + 比喻 + 夸张**——"问君能有几多愁"是设问，"恰似一江春水"是比喻兼夸张，三者合一，气势磅礴，成为写愁的千古绝唱。

## 四、意象与意境

### 1. 核心意象及象征义
| 意象 | 字面义 | 象征义 / 情感内涵 |
|------|--------|------------------|
| **春花秋月** | 四季良辰美景 | **乐景反衬哀情**——美景更勾起亡国之痛 |
| **小楼** | 囚居之所 | 囚徒的孤寂屈辱 |
| **东风** | 春风 | 岁月流逝、春又重来（"又"字含沉痛） |
| **明月** | 中天皓月 | 月圆人不圆、故国不堪回首 |
| **故国** | 南唐故国 | 亡国之痛、思念之深 |
| **雕栏玉砌** | 故宫华美建筑 | 昔日繁华、今日成空（**物是**） |
| **朱颜** | 红润容颜 | 词人衰老、人事变迁（**人非**） |
| **一江春水** | 滚滚东流的长江 | **亡国之愁**——深广、绵长、无穷无尽 |

### 2. 意境画面
全词营造了**凄美哀婉、沉痛深邃**的意境。上阕：春花秋月本是美景，词人却盼其"何时了"；囚居小楼，东风又起，明月当空，却"不堪回首"故国——**乐景与哀情对照**，画面是孤寂的囚楼与清冷的明月。下阕：遥想故宫雕栏玉砌仍在，只是朱颜已改；无尽哀愁，恰似一江春水滚滚东流——**由景入情，由实入虚**，画面从静态的宫殿转入动态的江水，意境由凄美转入壮阔悲沉。

### 3. 景情关系（⭐重点）
- **乐景写哀（反衬）**：春花、秋月、东风、明月，本是美好景物，对亡国之囚却成了痛苦的刺激——**以乐景反衬哀情，倍增其哀**。
- **物是人非**："雕栏玉砌应犹在（物是），只是朱颜改（人非）"——典型的**物是人非**感慨。
- **化景物为情思**：一江春水本是景物，却成为愁的化身——**景即情、情即景**，物我合一。

## 五、艺术手法

### ⭐⭐⭐ 核心手法：以水喻愁（化虚为实）
- "问君能有几多愁？恰似一江春水向东流"——将抽象的愁化为具体的江水，是**化虚为实**的经典。
- 此喻**前无古人**，影响深远——此后写愁多有效仿：秦观"落红万点愁如海"、李清照"只恐双溪舴艋舟，载不动许多愁"等，皆滥觞于此。

### 其他重要手法
- ⭐⭐⭐**设问（自问自答）**：
  - "春花秋月何时了？往事知多少"——设问，引出无限感慨；
  - "问君能有几多愁？恰似一江春水向东流"——设问 + 比喻，气势磅礴，**千古绝唱**。
- ⭐⭐⭐**比喻（明喻）**："恰似一江春水向东流"——以春水喻愁。
- ⭐⭐⭐**夸张**："一江春水"形容愁之多之深，夸张而不失真。
- ⭐⭐**反衬（乐景写哀）**：以春花、秋月、东风、明月等美好景物反衬亡国之愁。
- ⭐⭐**对比（物是人非）**：雕栏玉砌（犹在）与朱颜（已改）对比，凸显**物是人非、江山易主**之痛。
- ⭐⭐**虚实结合**：
  - 实写——小楼、东风、明月（眼前实景）；
  - 虚写——故国回首、雕栏玉砌（心中忆念）、一江春水（想象之喻）。
- ⭐**时空交织**：
  - 时间——"春花秋月""何时了""又东风"（岁月无尽）；
  - 空间——"小楼""故国""一江春水向东流"（空间浩渺）；
  - 时空交织，使愁苦**无时不在、无处不在**。
- **今昔对比**：今（囚楼朱颜改）与昔（故宫雕栏玉砌）对比。
- **隔句呼应**：上下阕结构对称（七、五、七、九），隔句换韵，跌宕有致。

::: warning 三段式答题模板（词的鉴赏分析）
**第一步（明手法）**：本词运用了……手法（如以水喻愁、设问、反衬等）。
**第二步（释运用）**：结合具体词句分析，如"问君能有几多愁？恰似一江春水向东流"为设问 + 比喻。
**第三步（说效果）**：把抽象的愁化为具体的江水，**深广绵长、无穷无尽**，表达了亡国之痛与故国之思，成为千古绝唱。
:::

## 六、主旨情感

全词通过春花秋月、小楼东风、雕栏玉砌、朱颜已改等意象的层层铺写，抒发了李煜**深沉的亡国之痛、无尽的故国之思、人生无常的悲剧感慨**。最后以"一江春水"的千古名喻，将这种愁苦升华到**深广无穷、奔涌不绝**的境界。

**核心情感三重**：①亡国之痛（故国不堪回首）；②囚居之悲（小楼朱颜改）；③人生无常之叹（往事、春花秋月何时了）。

王国维谓："后主之词，真所谓以血书者也。"——字字血泪，是**亡国之君的绝命悲歌**。

## 七、考点对接

### 常考题型
1. 名句赏析（"一江春水""又东风""朱颜改"等，**必考**）。
2. 手法题（以水喻愁、设问、反衬、虚实、对比）。
3. 情感题（亡国之痛、故国之思）。
4. 炼字题（"又""改""何时了"）。
5. 词的格律常识（词牌、上下阕、小令）。

### 预测题

**【题 1·名句赏析】** 请赏析"问君能有几多愁？恰似一江春水向东流"。

> **参考答案**：
> ① 此句运用了**设问 + 比喻（明喻）+ 夸张**的手法：先以"问君能有几多愁"设问，再以"恰似一江春水向东流"作答；
> ② **化虚为实**——把抽象无形、难以言说的"愁"，化为具体可见的"一江春水"，使愁有了**形体**；
> ③ **以水喻愁**——春水滔滔、连绵不绝、滚滚东流，喻愁之**深（水量大）、广（一江）、长（向东流不断）**，将亡国之痛、故国之思推向**无穷无尽**的境界；
> ④ 此喻新颖奇绝、气势磅礴，被誉为"以水喻愁"的**登峰造极之作**，成为千古绝唱。

**【题 2·炼字】** "小楼昨夜又东风"中的"又"字有何妙处？

> **参考答案**：
> ① "又"字表示**不止一次、再度发生**——春风不止一次吹来，囚居岁月已经流逝多年；
> ② "又"字含**无奈、痛苦、绝望**多重意味——岁月无情流逝，而亡国之痛不仅未减，反而**年复一年加深**；
> ③ 同时"又"字与下句"不堪回首"呼应，自然引出对故国的沉痛追忆，是炼字的典范，**一字含无限悲慨**。

**【题 3·手法】** 词中"雕栏玉砌应犹在，只是朱颜改"运用了什么手法？有何效果？

> **参考答案**：
> ① 运用**对比（物是人非）**手法：以"雕栏玉砌应犹在"（宫殿犹在）与"只是朱颜改"（容颜已改）相对照；
> ② **物是人非**——宫殿依旧，而江山易主、人事全非，词人由国君沦为囚徒，朱颜也因囚禁憔悴；
> ③ 这一对比将**亡国之痛、人生无常之悲**凝聚于一句之中，含蓄深沉，与下文"一江春水"的奔涌之情形成由抑到扬的转换，结构上承上启下。

**【题 4·手法】** "春花秋月何时了"一句，词人为何盼良辰美景"何时了"？这运用了什么手法？

> **参考答案**：
> ① 春花秋月本是赏心乐事，但对于**亡国之囚**而言，美景反而成了痛苦的刺激——美景依旧而人事全非，美景愈盛愈勾起对故国繁华的追忆，故盼其"何时了"；
> ② 这运用了**以乐景写哀情（反衬）**的手法，**倍增其哀**——美景反衬哀情，比直接写哀更为沉痛；
> ③ 同时"何时了"也是**反诘（设问）**，含词人对岁月无尽、痛苦无休的悲愤绝望，定下全词沉痛的基调。

**【题 5·默写与常识】**
① 李煜《虞美人》中，以水喻愁、成为千古绝唱的名句是："________？________。"
② 词中表现物是人非、亡国之痛的一句是："________，________。"
③ 表现岁月流逝、囚居之悲的句子是："________，________。"
④ 简答：《虞美人》这一体裁属于____（诗/词），其词牌为《____》，全词共____字，分为____阕。

> **参考答案**：
> ① 问君能有几多愁？恰似一江春水向东流；
> ② 雕栏玉砌应犹在，只是朱颜改；
> ③ 小楼昨夜又东风，故国不堪回首月明中；
> ④ 词；虞美人；56 字（双调小令）；两（上下阕）。

**核心记忆三句话**：
1. **李煜《虞美人》——南唐后主的绝命词**，"词至李后主而眼界始大，感慨遂深"，是其后期词的最高成就。
2. **"问君能有几多愁？恰似一江春水向东流"——以水喻愁的千古绝唱**，化虚为实、化有限为无限，设问 + 比喻 + 夸张三者合一。
3. **核心手法与情感**：以乐景写哀（反衬）+ 物是人非（对比），抒发**亡国之痛、故国之思、人生无常之悲**，字字血泪。
