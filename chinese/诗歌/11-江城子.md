# 《江城子·乙卯正月二十日夜记梦》

> 苏轼（北宋） · 悼亡词（婉约中见沉痛） · 选自《东坡乐府》

::: tip 本篇定位
中国文学史上**第一首真正意义上的悼亡词**，开创以词写悼亡的先河。苏轼将豪放词人的笔触转入儿女私情与生死之痛，于婉约沉挚中见至情，与潘岳《悼亡诗》、元稹《遣悲怀》、纳兰性德悼亡词并称悼亡名篇。乙卯年为熙宁八年（1075），作者时知密州，距妻子王弗去世已整整十年。
:::

---

## 一、知人论世

**作者**：苏轼（1037—1101），字**子瞻**，号**东坡居士**，眉州眉山人。北宋文坛领袖，诗、词、文、书、画皆臻一流。诗与黄庭坚并称"**苏黄**"，词与辛弃疾并称"**苏辛**"，文列"**唐宋八大家**"，书列"宋四家"（苏黄米蔡）之首。开创**豪放词派**，"一洗绮罗香泽之态"。

**文学地位**：苏轼打破了"诗庄词媚"的界限，"以诗为词"，拓展了词的题材与境界。然此篇《江城子》却反其道而行——以豪放大家写柔情悼亡，沉痛处比婉约派更婉约，足见大家手段。

**写作背景**：王弗是苏轼结发之妻，知书达理，"敏而静"，常劝谏苏轼提防小人。结婚时王弗16岁，苏轼19岁；王弗27岁早逝于京师，对苏轼打击巨大。词题"**乙卯正月二十日夜记梦**"——乙卯年（1075）苏轼任密州知州，年已38岁，丧妻十年。这一夜梦见亡妻，醒而作此词。

**文体常识**：
- **词牌**：《江城子》，唐词单调35字，蜀人欧阳炯始作双调70字。全词上下片各七句五平韵。
- **题材**：悼亡词（中国词史上悼亡开山之作）。
- **流派**：此篇虽出豪放大家之手，本身属**婉约沉挚**一类。

## 二、原文与翻译

> **小序**：乙卯正月二十日夜记梦

### 上片：现实——死别之痛

| 原文 | 逐句翻译 |
|---|---|
| **十年生死两茫茫，** | 生死相隔整整十年，彼此音讯渺茫。 |
| **不思量，自难忘。** | 即使不去刻意思量，也自然难以忘怀。 |
| **千里孤坟，无处话凄凉。** | 你远在千里之外的孤坟，我满腹凄凉无处诉说。 |
| **纵使相逢应不识，** | 纵然现在相逢，你也该认不出我了—— |
| **尘满面，鬓如霜。** | 我风尘满面，两鬓已白如寒霜。 |

### 下片：梦境与现实——重逢与醒来

| 原文 | 逐句翻译 |
|---|---|
| **夜来幽梦忽还乡，** | 昨夜忽得一梦，悄然回到了家乡， |
| **小轩窗，正梳妆。** | 你正坐在小窗前对镜梳妆。 |
| **相顾无言，惟有泪千行。** | 我们相对却一句话也说不出，只有眼泪止不住地流下千行。 |
| **料得年年肠断处，** | 想来每年最令我肝肠寸断的地方， |
| **明月夜，短松冈。** | 就是那明月高悬、长满短松的孤坟所在的山冈。 |

## 三、字词与炼字

### 生僻字与词义
- **乙卯**：干支纪年，即熙宁八年（1075）。
- **思量（liáng）**：想念、思念。此处读阳平，押平声韵。
- **孤坟**：王弗葬于四川眉山（苏轼故乡），苏轼时在密州（今山东诸城），相隔数千里，故曰"千里孤坟"。
- **轩窗**：窗户。"轩"为有窗的小室。
- **幽梦**：隐约、迷茫的梦。
- **松冈**：植有松树的山冈，指王弗墓地。古人坟地多植松柏。

### ⭐ 炼字分析

1. **"两茫茫"之"两"**——炼字在"两"。"两茫茫"非"我"一方思念茫然，而是**生死双方**都音讯渺茫。一个"两"字将生死并提，既写自己思而不见的痛苦，也设想亡妻九泉之下的孤寂，**一笔写两面**，加倍写痛苦。
2. **"自难忘"之"自"**——"自"是"自然、本能地"。与上文"不思量"相对：不是天天刻意思念，但**自然而然、刻骨铭心**地忘不掉。这才是真情之深。一个"自"字胜过千言"我思我念"。
3. **"纵使"与"应"**——"纵使"是退一步的设想（假设相逢），"应"是揣测之词（应该不认识了吧）。词人把十年来仕途奔波、容颜衰老的沧桑，浓缩在亡妻"不识"的假设里，**以反常写常情**，令人鼻酸。
4. **"忽"字**——"夜来幽梦**忽**还乡"。"忽"字写出梦境来得突然、不期而然，与下文梦醒后的怅然形成对照。
5. **"惟有泪千行"之"惟有"**——千言万语堵在喉头化作泪水，"惟有"二字写尽**相对无言的心酸**，比千言万语更具感染力。

### ⭐ 名句默写考点
- "十年生死两茫茫，不思量，自难忘。"
- "纵使相逢应不识，尘满面，鬓如霜。"
- "相顾无言，惟有泪千行。"
- "料得年年肠断处，明月夜，短松冈。"

## 四、意象与意境

### 核心意象表

| 意象 | 象征义 / 情感指向 |
|---|---|
| **孤坟** | 亡妻独葬远方，阴阳孤寂；亦喻自己形单影只 |
| **千里** | 空间之远——仕途奔波与故乡坟茔的遥隔 |
| **尘满面、鬓如霜** | 十年宦海沉浮、容颜衰老；自伤迟暮 |
| **小轩窗、正梳妆** | 梦中青春永驻的妻子形象，定格十年前的生活场景 |
| **泪千行** | 千言万语化作泪水，"此时无声胜有声" |
| **明月夜、短松冈** | 凄清冷寂的坟茔，亦真亦幻，余味无穷 |

### 意境分析
全词营造**凄清、沉痛、哀婉而苍凉**的意境。上片以"千里孤坟""鬓如霜"铺陈**现实的死别之痛**：苍老、孤独、无处倾诉；下片转入梦境——"小轩窗，正梳妆"是十年前温馨场景的复活，然而"相顾无言，惟有泪千行"将温情瞬间击碎为更深切的悲恸。结尾"明月夜，短松冈"由梦境跌回现实，将永恒的思念寄寓于凄冷的墓地月夜，**言尽而意不尽**。

### 景情关系
- **以景结情**：结尾"明月夜，短松冈"不直抒哀痛，以冷月荒坟之景作结，余味悠长。
- **以少总多、白描传神**：全篇不用一个典故、不施浓墨重彩，纯以家常场景（梳妆、流泪）和朴素意象（孤坟、松冈、明月）写出**至情**，是**白描**手法的典范。

## 五、艺术手法

### 1. ⭐⭐⭐ 虚实结合（现实—梦境—现实，全篇结构核心）

这是本词最重要的艺术特色。全词形成**"现实（实）—梦境（虚）—现实（实）"**的回环结构：

| 层次 | 内容 | 虚/实 | 作用 |
|---|---|---|---|
| 上片 | 十年死别、千里孤坟、自己衰老 | **实写**（现实） | 铺陈死别之痛、相思之深 |
| 下片前四句 | 梦中还乡、妻子梳妆、相对流泪 | **虚写**（梦境） | 以幻写真，以温馨衬悲恸 |
| 下片末二句 | 梦醒后想象孤坟月夜 | 虚实相生 | 以景结情，余味不尽 |

**虚实相生的效果**：梦境愈温馨（梳妆），醒来愈凄凉；相逢愈短暂（梦中），思念愈绵长（年年肠断）。**以虚衬实、以乐景写哀，倍增其哀**。

### 2. ⭐⭐⭐ 白描手法
全词**不用一典**，纯用白描——以最朴素的语言、最日常的画面（妻子窗前梳妆、两人对视流泪）写出最深沉的生死之痛。如"小轩窗，正梳妆"六字即勾出亡妻栩栩如生的形象，胜过千言万语。这是苏轼"**绚烂之极归于平淡**"的体现。

### 3. ⭐⭐ 对写（从对方落笔）
"千里孤坟，无处话凄凉"——既写自己无处诉说，也设想亡妻孤坟寂寞、无人相伴。"两茫茫"之"两"字更是典型对写，**一笔两写、双向抒情**。

### 4. ⭐⭐ 细节描写（动作、神态）
"小轩窗，正梳妆"——一个"正"字定格妻子生前的日常瞬间，仿佛时间倒流；"相顾无言，惟有泪千行"——以"无言"写"有情"，**此时无声胜有声**。

### 5. ⭐ 以乐景写哀情
梦中重逢、妻子梳妆本是温馨画面，但因生死阻隔，反而衬托出更深的悲恸——**以乐衬哀，其哀倍增**。

### 6. ⭐ 想象与联想（推己及人）
"料得年年肠断处"——由眼前推及未来"年年"，由自己推及亡妻所在，时空双线延展，使情感**绵延无绝**。

::: warning 三段式答题模板（艺术手法题）
**手法 + 分析 + 效果（情感）**

> 例题：请赏析"夜来幽梦忽还乡，小轩窗，正梳妆"的艺术手法。
>
> ① **手法**：词人运用**虚实结合（梦境）与白描**手法。
> ② **分析**：上片实写十年死别的现实之痛，此处忽转入梦境，以白描笔法再现妻子生前窗前梳妆的温馨场景；梦中相对无言、惟有泪千行。
> ③ **效果**：**以乐景写哀情、以虚衬实**，梦境愈温馨，醒来愈凄凉，将夫妻深情与生死之痛表现得淋漓尽致，催人泪下。
:::

## 六、主旨情感

本词是苏轼为悼念亡妻王弗而作，借**记梦**的形式，抒发对亡妻**生死不渝的深情**与**阴阳永隔的哀痛**。

情感层次：
1. **死别之痛**——十年生死两茫茫，无处话凄凉；
2. **相思之深**——不思量，自难忘；
3. **自伤迟暮**——尘满面，鬓如霜（亦含十年宦海沉浮的沧桑）；
4. **重逢之喜与骤失之悲**——梦中相顾无言，惟有泪千行；
5. **绵长无尽的思念**——料得年年肠断处，明月夜，短松冈。

全词**情真意切、朴素深沉**，是悼亡文学中**至情至性**的典范。诚如前人所评："**无声之悲，至情之语**。"

## 七、考点对接

### 常考题型
- 赏析"虚实结合"的结构艺术（**最高频**）
- 赏析白描手法及其表达效果
- 炼字赏析（"两""自""应""忽""惟有"等）
- "以乐景写哀情"的分析
- 名句默写（"十年生死两茫茫""相顾无言，惟有泪千行""明月夜，短松冈"）
- 情感主旨概括

### 预测题

**1. 词的上下片在写法上有何不同？请结合"虚实结合"加以分析。**

> **参考答案**：上片**实写**现实——追述十年死别、千里孤坟与自己容颜衰老之苦，铺陈相思与凄凉；下片**虚写**梦境——写梦中还乡、妻子梳妆、相对无言流泪，再由梦境跌回现实，想象孤坟月夜的凄清。**虚实相生、以虚衬实**：梦境愈温馨，现实愈凄凉；梦中相逢愈短暂，醒来思念愈绵长。这种结构使全词情感跌宕起伏，深化了悼亡之痛。

**2. 请赏析"十年生死两茫茫，不思量，自难忘"中"两"与"自"的妙处。**

> **参考答案**："两茫茫"之"两"字一笔两写，既写自己思而不见的茫然，又设想亡妻九泉之下的孤寂，**双向抒情、加倍写痛**；"自难忘"之"自"意为"自然、本能地"，与"不思量"相对，写并非刻意思量却**自然而然、刻骨铭心**地难以忘怀，**以反常写常情，见出至深真情**。二字配合，朴素中见功力。

**3. "小轩窗，正梳妆。相顾无言，惟有泪千行"运用了什么描写手法？有何效果？**

> **参考答案**：运用**细节描写（神态、动作）与白描**手法。以白描笔法定格妻子生前窗前梳妆的日常瞬间，"正"字使时间仿佛倒流、亡妻栩栩如生；"相顾无言，惟有泪千行"以"无言"写"有情"，**此时无声胜有声**。全无雕饰却催人泪下，将久别重逢、悲喜交集的复杂情感表现得淋漓尽致。

**4. 结尾"明月夜，短松冈"有何艺术效果？**

> **参考答案**：结尾运用**以景结情**手法。不直抒哀痛，而以明月高悬、短松荒坟的凄清冷寂之景作结，将梦醒后无尽的思念与哀痛**寄寓于景物**，言有尽而意无穷，给读者留下悠长的回味与想象空间，深化了全词悼亡的主题。

**5. 简述本词作为"悼亡词"在文学史上的地位，并概括其主旨情感。**

> **参考答案**：本词是中国词史上**第一首真正意义上的悼亡词**，开创了以词写悼亡的先河。词人借记梦形式，抒发对亡妻王弗生死不渝的深情与阴阳永隔的哀痛，兼有自伤迟暮之慨。全篇**情真意切、朴素深沉**，以白描、虚实结合等手法，成为悼亡文学中至情至性的典范。
