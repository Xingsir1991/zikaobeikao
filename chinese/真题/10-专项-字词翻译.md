# 专项·字词解释与文言翻译

> 全国自学考试《大学语文》（04729）—— 议论文 / 记叙文 / 小说文言部分高频考点

::: tip 本专项定位
大学语文试卷中，**字词解释（10 分左右）+ 文言翻译（6—10 分）** 是基础拿分大项，难度中等、套路固定。
本专项围绕 16 篇核心文言课文（先秦至清代），覆盖**词类活用、古今异义、通假字、特殊句式**四大难点，
**会一道即通一类**，是冲刺阶段最高性价比的训练材料。
:::

---

## 一、考点说明

### 考什么

| 考查类型 | 出题方式 | 分值 |
| --- | --- | --- |
| **加点字解释** | 给一句文言，在 4—6 个字下加点，写出释义 | 10 分左右 |
| **文言翻译** | 给一段 2—4 句的文言原文，要求译为现代汉语 | 6—10 分 |

### 出题方向（高频考点）

1. **词类活用**（每年必考 2—3 处）
   - 名词作状语：如「席卷」「蚕食」「日削月割」
   - 使动用法：如「却客」「斗诸侯」「亡我」
   - 意动用法：如「客我」「渔人甚异之」「先国家之急而后私仇」
2. **古今异义**（每年必考 1—2 处）
   - 如「其实」「慷慨」「地方」「至于」「于是」
3. **通假字**（每年必考 1—2 处）
   - 如「内」通「纳」、「蚤」通「早」、「属」通「嘱」、「冯」通「凭」、「辟」通「避」
4. **特殊句式**（翻译题重点）
   - **判断句**（……者……也）
   - **被动句**（为……所 / 见……于）
   - **倒装句**（宾语前置「未之有也」、状语后置「战于长勺」、定语后置「居庙堂之高」）
   - **省略句**（省主语、省介词「于」、省宾语）

## 二、答题公式

### 字词解释「三步法」

::: tip 公式
**①词义 → ②词性/活用 → ③语境义**
:::

```
1. 先判定加点字的本义（本义是什么）
2. 再看是否活用（是名→状、使动、意动，还是通假？）
3. 最终代入句中，写出语境义（在"这一句"里的意思）
```

**举例：**「**蚕**食诸侯」
- 本义：蚕，名词
- 活用：名词作状语（像蚕吃桑叶一样）
- 语境义：像蚕吃桑叶那样（逐渐吞并）

### 文言翻译「六字诀」

::: tip 公式：留 · 删 · 换 · 调 · 补 · 译
:::

| 字诀 | 含义 | 举例 |
| --- | --- | --- |
| **留** | 专有名词、地名、年号保留不译 | 「壬戌之秋」「孟尝君」「齐宣王」 |
| **删** | 无实义的虚词可删 | 「之」「而」「夫」「盖」 |
| **换** | 古字换今字 | 「窃」→ 私下；「过」→ 错误 |
| **调** | 倒装句调回现代语序 | 「未之有也」→「未闻之有也」（宾语前置） |
| **补** | 省略成分补出 | 「（秦人）拱手而取西河之外」 |
| **译** | 关键实词、活用、句式必须落实 | 「却客」→ 拒绝客卿 |

**翻译得分三要件：**
1. **关键词落实**（得分点，每句 1—2 个，错 1 个扣 1 分）
2. **句式体现**（判断、被动、倒装必须译出）
3. **语句通顺**（直译为主、意译为辅，不能脱离原文编造）

## 四、答案与解析

### 第一部分：字词解释答案

**第 1 题答案：**
- ① **之**：往、到……去（动词）
- ② **衅钟**：古代祭钟仪式，杀牲取血涂钟的缝隙（衅 = 杀牲血祭）
- ③ **舍**：释放、放掉（通「释」，动词）

::: details 解析
「牛何之」是**宾语前置**，「之」是动词不是代词，这是高频考点；「衅」是古代祭祀专名；「舍」通「释」，古今字。
:::

---

**第 2 题答案：**
- ① **老**（第一个）：赡养、敬养（形容词作动词，意动用法）
- ② **幼**（第一个）：爱护、抚养（形容词作动词，意动用法）
- ③ **运**于掌：在手掌上转动，比喻容易（动词）

::: details 解析
这是**词类活用**经典例子：形容词活用为动词，且带**意动意味**（「以吾老为老」「以吾幼为幼」）。「老吾老」「幼吾幼」结构相同，前者为动词，后者为名词（老人 / 幼儿）。译为现代汉语要补充「赡养」「抚养」这类动词。
:::

---

**第 3 题答案：**
- ① **旋**：转、掉转（动词）
- ② **闻道百**：听闻了很多道理（百 = 很多，数词虚指）
- ③ **莫己若**：没有谁比得上自己（宾语前置：莫若己）

::: details 解析
「闻道百」中的「百」是**数词虚指**，意为「很多」，不能译成具体数字「一百」。「莫己若」是**宾语前置**（否定句中代词宾语前置），正常语序是「莫若己」（没有谁能比得上我）。这两个都是每年必考的活用与句式点。
:::

---

**第 4 题答案：**
- ① **窃**：私下、私自（谦辞）
- ② **过**：错误（形容词作名词）
- ③ **向使**：假使、当初如果（连词）
- ④ **却**：拒绝、排斥（动词）
- ⑤ **内**：通「纳」，接纳（通假字）

::: details 解析
「内」是**通假字**经典考点，通「纳」；「过」是**形容词作名词**；「却客」的「却」是使动，「使客卿退却」即「拒绝」。翻译全句：私下认为是个错误……假使四君拒绝客卿而不接纳。
:::

---

**第 5 题答案：**
- ① **据**：凭借、依靠（动词）
- ② **固**：险固的地势（形容词作名词）
- ③ **席卷 / 包举 / 囊括**：像卷席子一样 / 像打包袱一样 / 像装口袋一样（名词作状语）
- ④ **内** / **外**：对内 / 对外（名词作状语）
- ⑤ **斗**：使……相斗（使动用法）

::: details 解析
本段是**词类活用密集区**：
- 「席卷、包举、囊括」**三个名词作状语连用**，极写孝公野心；
- 「内」「外」**方位名词作状语**，表方向；
- 「斗诸侯」是**使动**，「使诸侯相斗」。
这是《过秦论》最高频的考点段落，务必烂熟。
:::

---

**第 6 题答案：**
- ① **岂非……哉**：难道不是……吗！（表反问的固定句式）
- ② **原**：推究、推本溯源（形容词作动词）
- ③ **忧劳**：忧愁劳苦（这里作名词，意为「忧愁劳苦的事」）；**逸豫**：安逸享乐
- ④ **兴**国：使……兴盛（使动）；**亡**身：使……灭亡（使动）

::: details 解析
「忧劳可以兴国，逸豫可以亡身」是**千古名句**，其中「兴」「亡」均为**使动用法**——「使国家兴盛」「使自身覆亡」。整句是中心论点之一，必背。
:::

---

**第 7 题答案：**
- ① **属**：通「嘱」，嘱托、请托（通假字）
- ② **铗**（jiá）：剑、剑柄（名词，本句指剑）
- ③ **恶**（wù）：厌恶、讨厌（动词）

::: details 解析
「属」通「嘱」是**通假字**高频考点；「长铗」即长剑，是冯谖弹剑歌的核心意象；「左右皆恶之」的「恶」读 wù，是动词「厌恶」，不是「恶（è）劣」的恶。
:::

---

**第 8 题答案：**
- ① **壁**：驻扎、安营（名词作动词）
- ② **慷慨**：悲愤激昂、情绪激动（**古今异义**，今义为「大方不吝啬」）
- ③ **逝**：向前奔驰、跑（动词）

::: details 解析
「壁」名词作动词，「军壁垓下」=「军队驻扎在垓下」；「慷慨」是**典型古今异义**——古义是「悲愤激昂」（如项羽此时穷途末路、悲歌慷慨），今义是「不吝啬」。翻译必须译古义。
:::

---

**第 9 题答案：**
- ① **食**（sì）：给……吃、喂食（动词，这里指「吃掉」）（读 sì，动词）
- ② **诟**：诋毁、辱骂、诽谤（动词）
- ③ **见**：见识、见解（名词）

::: details 解析
「食其所爱之肉」是睢阳城粮尽时人吃人的惨烈史实，「食」是动词（sì，给人/物吃）；「诟」是**高中常考实词**，意为辱骂诋毁；「儿童之见」的「见」是名词「见识」，与今义「看见」不同。
:::

---

**第 10 题答案：**
- ① **业**：以……为业（名词作动词）
- ② **蚤**：通「早」；**实**：结果（名词作动词）；**蕃**：繁多（形容词，指结果多）
- ③ **顺木之天**：顺应树木的天性；**致其性**：使它的本性得以实现
- ④ **莳**（shì）：栽种、移栽（动词）

::: details 解析
「蚤」通「早」是**通假字**；「实」是**名词作动词**「结果实」；「顺木之天，以致其性」是全文**核心句**，必须完整译出。答题时这种「八字格言式」的句子，要把每个字的意思都落实。
:::

---

**第 11 题答案：**
- ① **管**：钥匙（名词，古今异义，今义为「管子」）
- ② **潜师**：秘密出兵、偷袭之师（潜 = 偷偷地；师 = 军队）
- ③ **访**：询问、咨询（动词，古今异义）
- ④ **劳师**：使军队疲劳（使动用法）
- ⑤ **无乃**：恐怕、大概（表推测的委婉语气，固定句式「无乃……乎」）

::: details 解析
「管」是**古今异义**——古义「钥匙」（与「管钥」通），今义「管子」；「访」古义是「咨询」，与今义「访问」不同；「无乃……乎」是**表委婉推测**的固定句式，译为「恐怕……吧」，常考。
:::

---

**第 12 题答案：**
- ① **既望**：农历每月十六日（望 = 十五；既 = 已经过了）
- ② **属**：通「嘱」，劝请、倾注（通假字，这里指「举杯劝客饮酒」）
- ③ **一苇**：比喻小船（借代 / 比喻，像一片芦苇那样轻小）
- ④ **冯**：通「凭」，乘、凌空（通假字，「冯虚御风」= 凌空驾风）

::: details 解析
「既望」是**文化常识**考点——「望」是农历十五，「既望」即十六；「属」通「嘱」与第 7 题《冯谖》同一通假字；「冯虚」的「冯」通「凭」，译为「凌空」，是《前赤壁赋》高频考点。
:::

### 第二部分：文言翻译答案

**第 13 题参考译文：**

> 敬爱自家的老人，从而推广到敬别家的老人；爱护自家的儿女，从而推广到爱护别家的儿女，（那么）治理天下就像在手掌心里转动东西一样容易了。《诗经》上说：「给妻子做榜样，再推广到兄弟，进而治理好一家一国。」说的就是把这种（爱自家人的）用心推广到他人身上罢了。

::: details 评分要点
- **关键词**：「老」「幼」（意动用法）、「运于掌」（比喻）、「刑」（通「型」，做榜样）、「斯心」（这种用心）、「加诸彼」（推广到他人）——每错一个扣 1 分。
- **句式**：「老吾老，以及人之老」是平行结构，须译出「……从而推广到……」的递进关系。
:::

---

**第 14 题参考译文：**

> 泰山不拒绝泥土，所以能成就它的高大；河海不挑剔细小的水流，所以能造就它的深邃；王者不拒绝民众，所以能彰明他的德行。因此，泰山不舍弃土壤，才能成就其大；这就是五帝三王之所以无敌于天下的原因。

::: details 评分要点
- **关键词**：「让」（辞让、拒绝）、「择」（挑剔、舍弃）、「就」（成就、造就）、「却」（拒绝）、「明」（使动，使……彰明）。
- **手法**：三个「……不……，故能……」构成**排比 + 比喻论证**——以泰山、河海喻王者，说理形象，译文中应保留这种排比气势。
- **逻辑**：「此五帝三王之所以无敌也」是**判断句**（「……之所以……也」），须译出判断语气。
:::

**第 15 题参考译文：**

> 再说，天下并没有缩小变弱；雍州的土地、崤山函谷关的险固，都和从前一样。陈涉的地位，并不比齐、楚、燕、赵、韩、魏、宋、卫、中山的国君更尊贵；锄头木棍并不比钩戟长矛更锋利；被贬谪戍守的民众，并不比九国的军队更强；深谋远虑、行军用兵的策略，也比不上从前（六国）的谋士。然而成功与失败的变化（却如此不同），功业恰好相反。假使让山东诸国与陈涉比量长短大小、衡量权势力量，那就不能同日而语了。

::: details 评分要点
- **关键词**：「小弱」（形作动，变小变弱）、「自若」（和原来一样）、「尊于」「铦于」「抗于」（都是状语后置 / 介宾后置）、「乡时」（通「向」，从前）、「度长絜大」（衡量长短大小）、「同年而语」（同日而语）。
- **句式**：四个「非……于……」是**状语后置**的否定比较句，必须调回现代语序「不比……更……」。
- **手法**：整段为**多重对比**，强调陈涉各方面都不如六国，却胜过秦国，进而逼出「仁义不施」的主旨。
:::

**第 16 题参考译文：**

> 当他用绳子绑着燕王父子，用木匣装着梁朝君臣的首级，进入太庙，把箭还给先王，并禀告大功告成的时候，他意气的旺盛，真可以说雄壮极了！等到仇敌已经消灭，天下已经平定，一个士兵夜里一声呼喊，叛乱的人从四面响应，（庄宗）慌张地向东逃出，还没见到叛贼士卒就溃散了，君臣互相看着，不知道逃往哪里，以至于对天发誓、剪断头发，眼泪流下来沾湿了衣襟，（这情景）又是多么衰颓啊！

::: details 评分要点
- **关键词**：「组」（丝带、绳子）、「函」（名作动，用木匣装）、「还矢」（把箭还给）、「一夫」（一个人，指士兵）、「四应」（数词作状语，从四面）、「仓皇」（慌张）、「誓天断发」（对天发誓剪断头发）。
- **手法**：本段为**盛衰对比**——「可谓壮哉」与「何其衰也」对照鲜明，是欧阳修「**先叙后议、叙议结合**」的典范。
- **省略**：「一夫夜呼」省主语，「仓皇东出」也省主语，需补出「庄宗」。
:::

**第 17 题参考译文：**

> （冯谖）驱车直奔齐国都城，早晨就去求见孟尝君。孟尝君对他回来得这么快感到奇怪，穿好衣服戴好帽子接见他，说：「债都收完了吗？回来怎么这么快！」（冯谖）回答说：「收完了。」「买了什么东西回来？」冯谖说：「您说『看我家缺少什么就买什么』。我私下考虑，您宫中堆积珍宝，狗马充满外面的马厩，美女列于堂下。您家所缺少的，只是『义』罢了！我私下为您买了『义』。」

::: details 评分要点
- **关键词**：「长驱」（驱车直奔）、「疾」（快）、「衣冠」（名作动，穿好衣戴好帽）、「责」（通「债」，古今字）、「反」（通「返」，通假字）、「实外厩」「充下陈」（充实、充满）、「以义耳」（只是义罢了，「以」=「只有」）。
- **手法**：这是冯谖「**焚券市义**」的核心情节，体现其**远见卓识**——孟尝君问「市何物」，冯谖答「市义」，一问一答，戏剧性强。
- **省略**：多处省主语「冯谖」，翻译需补出。
:::

**第 18 题参考译文：**

> 其他种树的人却不是这样：树根蜷曲、泥土更换；他们培土的时候，如果不是过多就是不够。如果有能够与此相反的人，却又爱得太深、忧得太过，早上看看、晚上摸摸，已经离开了还要再回来看。更有甚者，用指甲抓破树皮来检验它是活是死，摇动树根来观察土培得是松是紧，于是树木的本性就一天天丧失了。虽然说爱它，其实是害了它；虽然说担忧它，其实是与它为仇：所以都不如我啊。我又有什么特别的本事呢！

::: details 评分要点
- **关键词**：「拳」（蜷曲）、「易」（更换）、「恩」（深情，这里指爱得太深）、「爪」（名作动，用指甲抓）、「验」（检验）、「疏密」（松紧）、「离」（丧失、背离）、「不我若」（宾语前置，不如我）。
- **句式**：「虽曰爱之，其实害之」中「其实」是**古今异义**——古义「它的实际」，今义「实际上」。「故不我若也」是**否定句宾语前置**，需调为「故不若我」。
- **手法**：与他植者对比，反衬郭橐驼「**顺木之天**」的智慧。
:::

**第 19 题参考译文：**

> 蹇叔说：「使军队疲劳地去偷袭远方，我没听说过（这种打法能成功）。军队疲惫、力量耗尽，远方的主人（郑国）必有防备，恐怕不行吧？军队所做的事，郑国一定会知道。劳苦而无所得，必定会产生怨愤不满之心。况且行军千里，谁会不知道呢？」秦穆公没有采纳他的意见。秦穆公召见孟明、西乞、白乙三位将领，派他们从东门之外出兵。蹇叔哭着为军队送行，说：「孟明啊！我看见军队出发，却看不见它回来了！」

::: details 评分要点
- **关键词**：「劳师」（使动）、「远主」（远方的主人，指郑国）、「无乃……乎」（固定句式，恐怕……吧）、「勤而无所」（劳苦而无所得）、「悖心」（怨愤之心）、「辞」（拒绝、不采纳）。
- **句式**：「无乃不可乎」是**委婉推测句**，译为「恐怕不行吧」；「其谁不知」中「其」表反问语气。
- **手法**：蹇叔的清醒预见与秦穆公的刚愎形成**对比**，为后文崤之战惨败埋下伏笔。
:::

**第 20 题参考译文：**

> 客人也知道那水和月亮吗？流逝的水像这样（不断流去），却实际上未曾真正消失；时圆时缺的月亮像那样，却终究没有增减。如果从它变化的一面来观察，那么天地间的万事万物连一眨眼的时间都不能保持原状；如果从它不变的一面来观察，那么万物与我们都是无穷无尽的，又有什么可羡慕的呢！

::: details 评分要点
- **关键词**：「逝者」（流逝的水 / 事物）、「未尝往」（未曾消失）、「盈虚」（时圆时缺）、「卒」（终究）、「消长」（增减）、「盖」（发语词，表假设，可译为「如果」）、「曾」（竟然，加强语气）、「一瞬」（一眨眼）。
- **哲学内涵**：这是苏轼「**变与不变**」宇宙观的集中表达——变中见不变，故物我皆无尽，无须羡慕长江之无穷。译文须传达这种哲思。
- **句式**：「而又何羡乎」是**反问句**，需译出反问语气（「又有什么可羡慕的呢」）。
:::

1. **优先记高频活用**：席卷 / 包举 / 囊括 / 蚕食 / 日削月割 / 斗诸侯 / 却客 / 兴国亡身；
2. **通假字集中突破**：内=纳、蚤=早、属=嘱、冯=凭、辟=避、责=债、反=返、见=现；
3. **特殊句式清单化**：判断、被动、四种倒装（宾语前置 / 状语后置 / 定语后置 / 主谓倒装），每种记 2—3 个代表例句；
4. **翻译口诀反复念**：留删换调补译，关键实词不放过，句式必须落实，语句通顺即可拿满分。
:::
