# 📚 大学语文

> 涵盖古代文学、现代文阅读、写作三大板块。

## 使用说明

- 左侧目录展示本科目所有章节，点击切换
- 每篇笔记底部有 **「标记为已掌握」** 按钮，用于在 [进度看板](/progress) 累计统计
- 顶部搜索框支持中文全文检索

## 添加新资料

把整理好的 md 文件直接放进 `chinese/` 目录，例如：

```
chinese/
├── index.md              # 本页
├── 先秦文学.md
├── 唐诗宋词.md
└── 现代文阅读技巧.md
```

并在 `.vitepress/config.mts` 的 `sidebarGroup` 中补充对应条目（后续资料多了可考虑自动生成）。

## 重点提示（示例）

::: tip 备考要点
- 古代文学按朝代梳理脉络，重点作家作品要记牢
- 现代文阅读掌握答题模板（概括 / 分析 / 鉴赏）
- 作文积累通用素材与结构
:::
