# 《宝黛吵架》（清虚观打醮后的口角）

> 曹雪芹（清） · 长篇章回小说节选 · 《红楼梦》第29回"享福人福深还祷福　痴情女情重愈斟情"

::: tip 本篇定位
本节选自《红楼梦》第29回，是宝黛爱情线的关键篇目——围绕"金玉良缘"的心理试探与情感冲突，展现两人"原是一个心，反弄成两个心"的痴情纠葛，是全书最细腻的心理描写段落之一。贾母"不是冤家不聚头"一语，与宝玉砸玉、黛玉剪穗并称三大名场面，奠定了宝黛爱情"求近反疏"的悲剧基调。
:::

## 一、知人论世

### 作者·曹雪芹（约1715—约1763）

- **名号**：名霑，字梦阮，号雪芹、芹圃、芹溪。
- **家世**：清代内务府正白旗包衣。曾祖曹玺之妻孙氏为康熙乳母；祖父曹寅曾任康熙伴读、江宁织造，深得康熙信任——曹家鼎盛时"赫赫扬扬，已将近百载"（四代任江宁织造65年）。
- **巨变**：雍正六年（1728），曹頫（雪芹叔父/嗣父）因经济亏空、政治牵连被革职抄家，家族由"锦衣纨绔"跌入"举家食粥酒常赊"的困境。雪芹由南京迁回北京，晚年居于西郊，"披阅十载，增删五次"，在贫困中创作《红楼梦》。
- **思想**：亲历家族由盛转衰，对封建社会的腐朽有切肤之痛；多才多艺（诗、画、医），思想深刻，融儒、道、佛于一炉。

### 作品·《红楼梦》

- **成书**：原名《石头记》，前80回为曹雪芹所作；后40回一般认为高鹗（一说无名氏）续写，程伟元、高鹗整理出版（程甲本120回，1791）。
- **地位**：中国古典小说的最高峰——鲁迅评"自有《红楼梦》出来以后，传统的思想和写法都打破了"。被誉为"封建社会的百科全书"。
- **主题**：以贾宝玉、林黛玉、薛宝钗的爱情婚姻悲剧为主线，写贾府由盛而衰的过程，揭示封建社会必然崩溃的命运。
- **版本**：脂评本（80回抄本，如甲戌本、庚辰本，附脂砚斋批语）；程高本（120回刻本）。
- **三大手法**：
  ① 草蛇灰线、伏脉千里的结构；
  ② 千红一哭、万艳同悲的悲剧观；
  ③ 假语村言、真事隐去的双关笔法（甄士隐=真事隐；贾雨村=假语存）。

### 第29回背景（节选情节定位）

- 时间：端阳节前，元春出资在清虚观打醮（道教祈福仪式）三天。
- 起因：贾母率众人去清虚观看戏，张道士（荣国公替身、当今"终了真人"）在众人面前为宝玉提亲——这成为本节冲突的导火索。
- "金玉良缘"暗线：宝玉有"通灵玉"（刻"莫失莫忘，仙寿恒昌"），宝钗有"金锁"（刻"不离不弃，芳龄永继"）——二者成一对，是"金玉之说"。黛玉无物配玉，最忌人提"金玉"，怀疑宝玉心意。
- 第28回元春赐节礼，独宝玉宝钗一样（黛玉不同），已伏"金玉"暗示。本节"金麒麟"再起波澜（湘云有金麒麟，宝玉想为黛玉留），使黛玉更敏感。

## 二、情节梳理

节选自第29回后半，按情节发展分为六段。

### 第一段：清虚观打醮，张道士提亲（背景）

**段1—段7（前半）**：
- 端阳初一日，贾母率女眷去清虚观打醮，车辆人马"乌压压的占了一街"——渲染贾府"赫赫扬扬"的排场。
- 凤姐打小道士：凤姐扬手照脸打了想躲出去的小道士一个筋斗，骂"野牛肐的，胡朝那里跑"——与贾母的"快带了那孩子来，别唬著他""可怜见的"形成鲜明对比，凸显贾母的慈厚、凤姐的跋扈。
- 张道士（当日荣国公替身、当今"终了真人"、王公藩镇称之"神仙"）见宝玉，叹其"形容身段，言谈举动，怎么就同当日国公爷一个稿子"——贾母由不得满脸泪痕（这里"国公爷"指贾代善，贾母之夫，宝玉之祖父）。
- ⭐**关键情节**：张道士向贾母提亲——"前日在一个人家看见一位小姐，今年十五岁，生的倒也好个模样儿……我想著哥儿也该寻亲事了"。贾母婉拒："上回有和尚说了，这孩子命里不该早娶……不管他根基富贵，只要模样配的上就好……只是模样性格儿难得好的。"
  - **作用**：这是宝黛冲突的导火索——宝玉因此"心中大不受用"；黛玉也中暑生病。
- 张道士又把众道士的法器（金璜玉玦等三五十件）送作贺物，其中有一个赤金点翠的麒麟——
- ⭐**金麒麟细节**：贾母说"这件东西好象我看见谁家的孩子也带著这么一个"，宝钗说"史大妹妹有一个，比这个小些"。宝玉听说便将麒麟揣在怀里，又怕人看见，拿眼瞟人——只见黛玉"瞅著他点头儿，似有赞叹之意"。宝玉不好意思，又掏出来假意要给黛玉，黛玉说"我不希罕"。
  - **作用**：金麒麟事件让黛玉敏感——宝玉心里"有别人"（既怕湘云，又被黛玉看穿），成为次日口角的伏笔。

### 第二段（段27末—段29）：张道士提亲引发风波，回家生气

**段27末**："那贾母因昨日张道士提起宝玉说亲的事来，谁知宝玉一日心中不自在，回家来生气，嗔著张道士与他说了亲，口口声声说从今以后不再见张道士了……二则林黛玉昨日回家又中了暑：因此二事，贾母便执意不去了。"

- ⭐**双重伏因**：
  1. 宝玉因张道士提亲而不自在（因为提的是别人，不是黛玉）；
  2. 黛玉中暑——身体不适，情绪敏感。
- 二人都因张道士提亲而不快：宝玉怪人乱提亲（怕黛玉误会）；黛玉疑宝玉接受"金玉"之说。这是矛盾的种子。

### 第三段（段29）：宝黛第一次口角——砸玉

**段29前半**：宝玉因黛玉中暑来探望，黛玉说"你只管看你的戏去"。宝玉想："别人不知道我的心还可恕，连他也奚落起我来。"沉下脸说"我白认得了你。罢了，罢了！"黛玉冷笑："我也知道白认得了我，那里象人家有什么配的上呢。"
- ⭐**核心冲突点**："配的上"暗指"金玉之说"——黛玉以话刺探宝玉，宝玉觉得被冤枉。
- 宝玉直问脸上："你这么说，是安心咒我天诛地灭？"——又提昨儿赌咒之事。黛玉方想起前日的话，羞愧又急，颤颤兢兢说"我要安心咒你，我也天诛地灭"，反怪宝玉"怕阻了你的好姻缘，你心里生气，来拿我煞性子"。

### 第四段（段29中段）：⭐⭐⭐经典心理描写——"原是一个心反弄成两个心"

**段29中段（叙述者插入的大段心理分析）**：

作者先追溯宝玉、黛玉"痴病"的根源：
- 宝玉自幼与黛玉"耳鬓厮磨，心情相对"，看了"邪书僻传"（指《西厢记》《牡丹亭》），凡所见闺英闱秀，皆不及黛玉——"早存了一段心事，只不好说出来，故每每或喜或怒，变尽法子暗中试探"。
- 黛玉"偏生也是个有些痴病的，也每用假情试探"。
- ⭐"因你也将真心真意瞒了起来，只用假意，我也将真心真意瞒了起来，只用假意，如此两假相逢，终有一真。其间琐琐碎碎，难保不有口角之争。"

**宝玉此刻心理**：
"别人不知我的心，还有可恕，难道你就不想我的心里眼里只有你！你不能为我烦恼，反来以这话奚落堵我。可见我心里一时一刻白有你，你竟心里没我。"
——宝玉的逻辑：你若懂我，就应知道我心里只有你，不该提"配得上"刺我。

**黛玉此刻心理**：
"你心里自然有我，虽有'金玉相对'之说，你岂是重这邪说不重我的。我便时常提这'金玉'，你只管了然自若无闻的，方见得是待我重，而毫无此心了。如何我只一提'金玉'的事，你就著急，可知你心里时时有'金玉'，见我一提，你又怕我多心，故意著急，安心哄我。"
——黛玉的逻辑：你若心里真没"金玉"，我提时你该无所谓；你越着急，说明你心里越有"金玉"。

⭐⭐⭐**点睛之笔**：
"看来两个人原本是一个心，但都多生了枝叶，反弄成两个心了。"
——叙述者跳出情节的评论：两人本一心相爱，却因各自的猜疑、试探，把一颗心分成两颗，越求近越疏远。

**宝玉又想**："我不管怎么样都好，只要你随意，我便立刻因你死了也情愿。你知也罢，不知也罢，只由我的心，可见你方和我近，不和我远。"
**黛玉又想**："你只管你，你好我自好，你何必为我而自失。殊不知你失我自失。可见是你不叫我近你，有意叫我远你了。"
⭐"如此看来，却都是求近之心，反弄成疏远之意。"
——这是中国古典小说中罕见的大段细腻心理独白，把少男少女相爱而相猜、相试而相伤的微妙心理写得入木三分。

### 第五段（段33—39）：砸玉+剪穗——冲突高潮

**段33**：宝玉听见"好姻缘"三字，赌气向颈上抓下通灵宝玉，咬牙恨命往地下一摔："什么捞什骨子，我砸了你完事！"那玉坚硬非常，竟文风没动。宝玉回身找东西来砸。
- ⭐⭐⭐**砸玉的象征**：
  - 砸玉是砸"金玉之说"——宝玉以毁掉玉来表达对"金玉良缘"的抗拒，是对黛玉的真心证明。
  - 也是情感无法表达、转为自毁的痴狂。
- 黛玉大哭："何苦来，你摔砸那哑吧物件。有砸他的，不如来砸我。"
- 袭人赶来劝阻，宝玉冷笑："我砸我的东西，与你们什么相干！"

**段35**：袭人劝宝玉"倘或砸坏了，叫他心里脸上怎么过的去"——这话说到黛玉心坎上，黛玉越发伤心大哭，吐出刚吃的香薷饮解暑汤。
- 紫鹃劝黛玉"倘或犯了病，宝二爷怎么过的去呢"——这话说到宝玉心坎上，宝玉"又见林黛玉脸红头胀，一行啼哭，一行气凑，一行是泪，一行是汗，不胜怯弱"，自己后悔，也滴下泪来。
- ⭐"四个人都无言对泣"——袭人、紫鹃也跟着流泪，场面写尽痴情男女的相互折磨。

**段37**：袭人劝宝玉"你不看别的，你看看这玉上穿的穗子，也不该同林姑娘拌嘴"——穗子是黛玉亲手穿的。黛玉听了"也不顾病，赶来夺过去，顺手抓起一把剪子来要剪"。袭人紫鹃刚要夺，已经剪了几段。
- ⭐**剪穗的象征**：
  - 穗子是黛玉亲手为宝玉所穿，是两人感情的物化；
  - 黛玉剪穗，与宝玉砸玉呼应——宝玉毁玉以证心，黛玉毁穗以证气。
  - 但剪穗也是另一种"相残"：你砸玉，我就剪穗，谁都不肯先低头。
- 黛玉哭道："我也是白效力。他也不希罕，自有别人替他再穿好的去。"——又暗刺"金玉"（宝钗或别人会替他穿）。
- 宝玉说"你只管剪，我横竖不带他，也没什么"。

**段39**：老婆子们见闹得大，回报贾母、王夫人。二人进来见"宝玉也无言，林黛玉也无话"，便将祸移到袭人紫鹃身上，教训了一顿。贾母带宝玉出去，方才平服。

### 第六段（段41—段43）：⭐⭐⭐"不是冤家不聚头"——点题升华

**段41**：过了两日（初三薛蟠生日），宝玉黛玉都不去看戏，各自后悔。
- 宝玉"无精打采""推病不去"；
- 黛玉心里后悔："昨儿千不该万不该剪了那玉上的穗子。管定他再不带了，还得我穿了他才带。"
- 二人虽不见面，心却互相惦记——这是吵架后的"和好"前奏。

**段43**：⭐⭐⭐**贾母的抱怨——全篇点睛**：
贾母见他两个都生气，急得抱怨说：
"我这老冤家是那世里的孽障，偏生遇见了这么两个不省事的小冤家，没有一天不叫我操心。真是俗语说的，'不是冤家不聚头'。几时我闭了这眼，断了这口气，凭著这两个冤家闹上天去，我眼不见心不烦，也就罢了。偏又不咽这口气。"自己抱怨著也哭了。

- ⭐"不是冤家不聚头"——俗语，冤家：仇人，引申为"相爱的人"（爱极反成怨）。这里指宝玉黛玉越是相爱，越要相吵，是命中注定的缘分。
- **关键效果**："这话传入宝林二人耳内。原来他二人竟是从未听见过'不是冤家不聚头'的这句俗语，如今忽然得了这句话，好似参禅的一般，都低头细嚼此话的滋味，都不觉潸然泣下。"
  - 两人因从未听过此话，得之如参禅——一语道破他们"求近反疏"的本质。
- ⭐⭐⭐**结尾名句**：
"虽不曾会面，然一个在潇湘馆临风洒泪，一个在怡红院对月长吁，却不是人居两地，情发一心！"
- 这是全篇的诗意升华——身体分居两地（潇湘馆、怡红院），感情却发自同一颗心。呼应前文"原是一个心"，构成首尾圆合。

**段45**：袭人劝宝玉下气认错，留悬念"且听下回分解"。

## 三、人物形象分析（核心）

### ⭐（一）贾宝玉——痴情叛逆的贵族公子

| 形象层面 | 表现 |
| --- | --- |
| 用情专一 | 心里眼里只有黛玉，凡所见闺英闱秀皆不及黛玉；金麒麟也假意要留给黛玉 |
| 叛逆性 | 嗔张道士提亲，砸通灵玉以抗"金玉之说"——叛逆家长安排的"金玉良缘" |
| 痴情痴狂 | "立刻因你死了也情愿"；砸玉、找东西来砸，"我砸我的东西，与你们什么相干" |
| 内心矛盾 | 想表达"心里只有你"，却说不出来；只会用赌咒、砸玉来证明 |
| 软弱性 | 被袭人一劝就后悔；推病不去看戏；最后还得认错 |

**塑造手法**：
- ⭐⭐⭐**心理独白**：作者直接揭示宝玉的内心活动（"宝玉此刻心理"），是中国古典小说中罕见的大段心理分析。
- **言行细节**：揣麒麟、掏麒麟、给穗穗、砸玉——一连串动作揭示其矛盾心理。
- ⭐**典型细节——砸玉**：砸玉既是痴情（抗金玉之说），也是软弱（无力正面表达，转而自毁）。

### ⭐（二）林黛玉——敏感多情的贵族少女

| 形象层面 | 表现 |
| --- | --- |
| 真情专一 | 心里只有宝玉，悔剪玉穗，"还得我穿了他才带" |
| 敏感多疑 | 最忌"金玉"之说；金麒麟事件让她敏感"宝玉心里有别人" |
| 嘴尖舌利 | "我也知道白认得了我，那里象人家有什么配的上呢"——一句三刺 |
| 痴情痴病 | 一行啼哭、一行气凑、一行是泪、一行是汗；吐出解暑汤；剪玉穗 |
| 倔强自尊 | "我不希罕"；剪穗以对砸玉——不肯先低头 |
| 心细如发 | 见宝玉揣金麒麟便"瞅著他点头儿，似有赞叹之意"——一眼看穿宝玉心思 |

**塑造手法**：
- ⭐⭐⭐**心理独白**：与宝玉对称的心理分析，揭示"假情试探"的恶性循环。
- ⭐**言行细节**：甩手帕"碰在眼睛上"；冷笑"配的上"；抓剪子剪玉穗——动作细致传神。
- **对比衬托**：宝玉砸玉，黛玉剪穗——互相对仗，谁都不让谁，但都是因爱生嗔。

### ⭐（三）贾母——慈爱无奈的最高家长

- **形象**：贾府的最高权威，对宝玉黛玉疼爱有加，但也无奈于两人的口角。
- **关键作用**：
  1. 慈厚（与小道士）：与凤姐打小道士对比，贾母"快带了那孩子来，别唬著他"——慈厚之至。
  2. 婉拒提亲（暗示倾向）：张道士提亲，贾母以"命里不该早娶""只要模样配的上就好"婉拒——含蓄地否定了"金玉"，是宝黛爱情的暗中支持者。
  3. ⭐"不是冤家不聚头"：以俗语点破宝黛情感本质——是全篇的点题者。
- **塑造手法**：语言、神态（抱怨、流泪）。

### （四）袭人、紫鹃——贴身丫鬟的陪衬

- **袭人**（宝玉大丫鬟）：贤惠、世故、劝和。一句"倘或砸坏了，叫他心里脸上怎么过的去"——点到黛玉心坎上。
- **紫鹃**（黛玉大丫鬟）：忠诚、体贴。"倘或犯了病，宝二爷怎么过的去呢"——点到宝玉心坎上。
- ⭐两人都"看到对方主人"，反而比主人们更懂得彼此的心——是宝黛"假情试探"的反衬。
- 最后"四个人都无言对泣"——主仆同悲，写尽痴情局中人的无奈。

### （五）王熙凤——精明跋扈（侧面）

- 打小道士：一扬手照脸一掌，骂"野牛肐的"——跋扈、不恤下人；
- 与贾母的慈厚形成对比；
- 在本节中是贾府排场（清虚观打醮）的执行者，凸显"赫赫扬扬"。

### （六）张道士——世故圆滑的宗教人物

- "当日荣国公的替身""当今封为终了真人""王公藩镇都称他为神仙"——地位显赫；
- 提亲、送法器——既是好意，也是越界；
- 贾珍对他的态度（"再多说，我把你这胡子还挦了呢"）——表面亲密，实则轻慢——折射宗教人物在权贵面前的依附地位。

## 四、艺术手法鉴赏

### ⭐⭐⭐（一）细腻入微的心理描写——本节最大特色

**手法**：作者直接进入人物内心，以大段心理独白揭示宝黛相爱相猜的微妙心理——是中国古典小说中极为罕见的心理描写深度。

**三段式答题**：
- **指出手法**：作者运用直接心理描写（心理独白），把宝黛二人"假情试探"的内心活动层层揭示。
- **分析文本**：
  - 先追溯根源：两人"心情相对"，"早存了一段心事，只不好说出来"，故"变尽法子暗中试探"；
  - 写此刻心理：宝玉想"你竟心里没我"；黛玉想"可知你心里时时有'金玉'"——两人都在自己的逻辑里越走越远；
  - 作者评论："两个人原本是一个心，但都多生了枝叶，反弄成两个心了"；"都是求近之心，反弄成疏远之意"。
- **概括效果**：把少男少女相爱而相猜、相试而相伤的微妙心理写得入木三分，揭示"求近反疏"的悖论，是宝黛爱情悲剧本质的深刻揭示——这种心理深度在中国古典小说中堪称绝唱。

### ⭐⭐（二）"砸玉+剪穗"——言行细节的戏剧化

**手法**：通过两个高度戏剧化的动作（砸玉、剪穗），将抽象的心理冲突外化为可感的场面。

**分析**：
- 砸玉：宝玉赌气抓下通灵玉"咬牙恨命往地下一摔"，"什么捞什骨子，我砸了你完事"——玉是"金玉之说"的物化，砸玉是抗"金玉"、证真心的痴狂；
- 剪穗：黛玉"也不顾病，赶来夺过去，顺手抓起一把剪子来要剪"——穗是她亲手所穿，是两人感情的物化；剪穗与砸玉形成对仗，谁都不让谁；
- 两个动作互为镜像，把"求近反疏"的悲剧具象化。

**效果**：使心理冲突有了外在的戏剧张力，是情节高潮，也是形象刻画的高潮。

### ⭐⭐⭐（三）"不是冤家不聚头"——画龙点睛的点题

**手法**：借贾母之口说出俗语"不是冤家不聚头"，作为全篇的点题之笔。

**三段式答题**：
- **指出手法**：作者借人物（贾母）之口说出俗语"不是冤家不聚头"，作为全篇的点题之笔。
- **分析文本**：贾母抱怨两个"不省事的小冤家"，引用俗语"不是冤家不聚头"——"冤家"本指仇人，引申为"相爱的人"；这里指宝黛越相爱越相吵的命中注定。两人"从未听见过"此话，得之"好似参禅的一般，都低头细嚼此话的滋味，都不觉潸然泣下"。
- **概括效果**：一语道破宝黛"求近反疏"的情感本质，使其后的"人居两地，情发一心"有了哲理依据；同时深化了悲剧基调——既相爱，又必相伤，是命中注定的孽缘。

### ⭐⭐（四）对比映衬

| 对比项 | 内容 | 作用 |
| --- | --- | --- |
| 贾母 vs 凤姐（对小道士） | 慈厚 vs 跋扈 | 凸显贾母仁厚 |
| 宝玉砸玉 vs 黛玉剪穗 | 都毁彼此信物 | 互为镜像，写求近反疏 |
| 袭人、紫鹃 vs 宝黛 | 丫鬟懂主人的心，主人反不懂彼此 | 反衬宝黛的"假情试探" |
| "一个心" vs "两个心" | 原本一心 vs 多生枝叶成两心 | 揭示矛盾本质 |
| 潇湘馆 vs 怡红院 | 临风洒泪 vs 对月长吁 | 分居两地，情发一心 |
| 金玉之说 vs 木石前盟 | 世俗匹配 vs 精神共鸣 | 暗示两种爱情观的对立 |

### ⭐（五）伏笔与照应（草蛇灰线）

- **伏笔**：
  - 张道士提亲（金玉之外的"第三者"）——引发宝黛冲突；
  - 金麒麟事件（湘云有金麒麟）——让黛玉敏感，宝玉揣麒麟的细节埋下冲突；
  - "命里不该早娶"（贾母婉拒提亲）——伏后文宝黛爱情的拖延与悲剧；
- **照应**：
  - 段首"原是一个心"与段尾"情发一心"首尾圆合；
  - 砸玉与剪穗前后呼应；
  - 贾母"小冤家"与宝玉"老冤家"互文。

### ⭐（六）语言特色——《红楼梦》语言的典范

- **白话京腔**：流利、生动、口语化，是成熟的白话文学语言。
- **人物语言个性化**：
  - 贾母："我这老冤家是那世里的孽障""两个不省事的小冤家"——口语化、含深情；
  - 凤姐："野牛肐的，胡朝那里跑""猴儿猴儿，你不怕下割舌头地狱"——泼辣、俏皮；
  - 宝玉："什么捞什骨子，我砸了你完事"——痴狂任性；
  - 黛玉："我不希罕""我也是白效力"——尖刻自尊；
  - 张道士：典雅周全（"无量寿福""托老太太万福万寿"）；
- **叙述语言的雅俗交融**：叙事部分既有白话的流畅，又有诗意的升华（"人居两地，情发一心""一个在潇湘馆临风洒泪，一个在怡红院对月长吁"）。

### （七）环境与场面描写

- 清虚观打醮：车马"乌压压的占了一街"，钟鸣鼓响，张法官执香披衣——渲染贾府排场，是"赫赫扬扬"的写照；
- 怡红院内的砸玉剪穗：相对封闭的内闱空间，使冲突集中、心理外化；
- 潇湘馆、怡红院的"两地一心"：空间分隔反衬情感合一。

## 六、主旨情感

### （一）表层主题：宝黛因"金玉之说"引发的情感冲突

本节通过清虚观打醮后宝黛的口角、砸玉剪穗、贾母抱怨、二人"情发一心"等情节，写两人因"金玉之说"互相试探、互相伤害、又互相互动的痴情纠葛。

### （二）深层意蕴

1. ⭐⭐⭐**揭示"求近反疏"的情感悖论**：
   - 两人"原本是一个心"，却因各自的猜疑、试探，"多生枝叶，反弄成两个心"；
   - 越是深爱，越是敏感；越是试探，越是伤害——这是宝黛爱情悲剧的本质。
   - 作者以罕见的心理深度，揭示了一代青年在封建礼教下无法直接表达真情的痛苦。

2. ⭐⭐**批判封建婚姻制度与"金玉之说"**：
   - "金玉良缘"是家长（王夫人、薛姨妈、元春）撮合的世俗匹配；
   - "木石前盟"是宝玉黛玉的精神共鸣；
   - 宝玉砸玉，是对"金玉之说"的反抗——以毁玉证真心；
   - 但宝玉无力正面抗争，只能以痴狂、自毁表达——折射封建大家长制下个体的无力。
   - 贾母婉拒提亲（"命里不该早娶""模样性格儿难得好的"）——含蓄否定"金玉"，是宝黛爱情的暗中支持，但最终也未能成全。

3. **歌颂真情，悲悯青春**：
   - 宝黛"情发一心"，虽分居两地而心相通——这是作者对真情的歌颂；
   - 但两人越求近越疏远，越相爱越相伤——"千红一哭，万艳同悲"的悲剧基调已定；
   - 贾母"偏又不咽这口气"——老一辈的牵挂也无济于事，预示悲剧的不可避免。

4. **展现贾府的赫赫排场与潜在危机**：
   - 清虚观打醮的"乌压压占了一街"是贾府鼎盛的写照；
   - 但凤姐打小道士、张道士的依附地位、家族内闱的口角纠纷——已暗示家族内部的精神危机；
   - 这是"享福人福深还祷福"（回目）的反讽——越享福越要祷福，越显其内心的不安。

5. **文学价值**：
   - 中国古典小说心理描写的巅峰——大段心理独白、层层揭示，前所未有；
   - "原是一个心反弄成两个心""不是冤家不聚头""人居两地，情发一心"等成为千古名句；
   - 是宝黛爱情线的关键节点，为后文的"诉肺腑"（32回）、"黛玉焚稿"（97回）等情节奠定心理与情感基础。

### （二）预测题（含答案）

#### 预测题 1（心理描写——核心）

**问**：本节对宝黛二人的心理描写被称为中国古典小说的巅峰，请结合文本分析其艺术特色与作用。

**参考答案**：
**艺术特色**：
1. **大段直接心理独白**：作者直接进入人物内心，把宝黛"假情试探"的心理层层揭示——"因你也将真心真意瞒了起来，只用假意……如此两假相逢，终有一真"——这种直接揭示内心矛盾的手法，在以叙事为主的中国古典小说中极为罕见。
2. **对称式心理对照**：宝玉想"你竟心里没我"，黛玉想"可知你心里时时有'金玉'"——两人各自的逻辑对称展开，互为镜像，使读者看到两个"痴人"在各自的执念里越走越远。
3. **叙述者的评论介入**：作者跳出情节作评论——"两个人原本是一个心，但都多生了枝叶，反弄成两个心了""都是求近之心，反弄成疏远之意"——以哲理性的语言点破情感本质。

**作用**：
1. **塑造立体的人物形象**：把宝黛相爱而相猜、相试而相伤的微妙心理写得入木三分，使人物有了前所未有的心理深度。
2. **揭示悲剧本质**：通过"求近反疏"的悖论，揭示宝黛爱情在封建礼教下无法直接表达、只能相互试探伤害的悲剧本质。
3. **深化主题**：为后文"诉肺腑""焚稿"等情节奠定心理基础，是宝黛爱情线的关键节点。
这种心理描写深度，是《红楼梦》"传统的思想和写法都打破了"（鲁迅）的集中体现。

#### 预测题 2（点题俗语——核心）

**问**：贾母口中的"不是冤家不聚头"在全文中有何作用？为什么二人听后"好似参禅的一般"？

**参考答案**：
**作用**：
1. **点题升华**：贾母借这句俗语，一语道破宝黛"越相爱越相吵"的情感本质——"冤家"既指仇人，也指爱人（爱极反怨），是反语。这句话总结了前文砸玉剪穗的冲突，使情节有了哲理高度。
2. **化矛盾为领悟**：两人"从未听见过"此话，得之"好似参禅的一般，都低头细嚼此话的滋味，都不觉潸然泣下"——一语道破他们的相爱相伤，使他们从对立转为共鸣。
3. **结构上的转折**：此语一出，二人虽不见面，却"一个在潇湘馆临风洒泪，一个在怡红院对月长吁""人居两地，情发一心"——由对立转为同心，是情节的转折点，为后文和好做铺垫。

**为什么"好似参禅"**：
1. **从未听过**：两人"竟是从未听见过'不是冤家不聚头'的这句俗语"——得之如新悟；
2. **一语道破**：此语精准点出他们的情感本质——求近反疏、相爱相伤，正中他们内心深处的困惑；
3. **如获顿悟**：禅宗讲究"顿悟"，二人得此语如参透玄机，由痴嗔转为清醒，故"潸然泣下"——是悲欣交集的领悟。

#### 预测题 3（人物形象——核心）

**问**：宝玉"砸玉"与黛玉"剪穗"两个细节有何深意？请结合人物形象分析。

**参考答案**：
**两个细节的深意**：
1. **宝玉砸玉——抗"金玉"的痴狂**：
   - 通灵玉是"金玉之说"的物化（与宝钗金锁成对），是家长安排的"金玉良缘"的象征；
   - 宝玉"咬牙恨命往地下一摔"，"什么捞什骨子，我砸了你完事"——砸玉是抗"金玉"、证真心的痴狂表达；
   - 但只能以毁物（而非正面言说）来证明，折射其在封建家长制下无力正面表达真情的软弱。
2. **黛玉剪穗——对砸玉的镜像回应**：
   - 玉上的穗子是黛玉亲手所穿，是两人感情的物化；
   - 黛玉"也不顾病，赶来夺过去，顺手抓起一把剪子"剪了几段——与宝玉砸玉形成对仗，谁都不肯先低头；
   - "我也是白效力。他也不希罕，自有别人替他再穿好的去"——又暗刺"金玉"，是敏感自尊的体现。

**人物形象**：
- 两个细节互为镜像，写出了宝黛"求近反疏"的悲剧：越想证明真心，越互相伤害；
- 宝玉的痴狂与黛玉的尖刻，都是"痴情"的不同表现——情到深处反成怨；
- 二人都不会正面表达爱，只能通过毁物来宣泄，是封建礼教压抑下青春心理的扭曲外化。

#### 预测题 4（主题探究——核心）

**问**：作者说宝黛"原是一个心，反弄成两个心""都是求近之心，反弄成疏远之意"，结合全文谈谈你对这一"求近反疏"悖论的理解。

**参考答案**：
"求近反疏"是宝黛爱情的本质悖论，本节作了深刻揭示：
1. **根源：假情试探的恶性循环**：
   - 两人"心情相对"，本"是一个心"，但因封建礼教不能直接表达，"将真心真意瞒了起来，只用假意"——以假情试探对方真心；
   - "如此两假相逢，终有一真"——但试探中必有误解，"其间琐琐碎碎，难保不有口角之争"。
2. **表现：各自的执念**：
   - 宝玉的逻辑：你若懂我，就不该提"配得上"刺我；
   - 黛玉的逻辑：你若心里没"金玉"，我提时你该无所谓；
   - 两人各自的逻辑都对，但都从自己的角度出发，互不理解，越求近越疏远。
3. **高潮：砸玉剪穗**：抽象的心理矛盾外化为毁物的冲突，把"求近反疏"具象化。
4. **化解：贾母"不是冤家不聚头"**：一语道破情感本质，使二人由对立转为"情发一心"——但这是暂时的领悟，悲剧基调已定。

**意义**：
- 揭示了封建礼教压抑下，真情无法直接表达、只能通过试探伤害来曲折传递的痛苦；
- 这种心理深度在中国古典小说中前所未有，是《红楼梦》打破"传统思想和写法"（鲁迅）的集中体现；
- 也具有超越时代的普适性——凡深爱者，必有相猜相伤，是爱情本身的悖论。

#### 预测题 5（环境与场面）

**问**：本节开头花大量笔墨写清虚观打醮的排场（车马、道士、张法官等），与后文宝黛吵架有何关系？

**参考答案**：
1. **情节上的因果关系**：
   - 正是在清虚观，张道士当众为宝玉提亲——这是宝黛冲突的直接导火索；
   - 也是在清虚观，金麒麟事件（宝玉揣麒麟给黛玉被看穿）让黛玉敏感；
   - 没有清虚观的提亲与金麒麟，就没有次日的口角。
2. **主题上的反讽对照**：
   - 清虚观打醮的"赫赫扬扬"（车马乌压压、钟鸣鼓响、张法官执香）是贾府鼎盛的写照；
   - 但繁华背后，凤姐打小道士（跋扈）、张道士依附权贵（世故）、家族内闱的口角——已暗示家族的精神危机；
   - 回目"享福人福深还祷福"——越享福越要祷福，反讽其内心的不安。
3. **结构上的铺垫作用**：
   - 大段排场描写为张道士提亲做铺垫；
   - 提亲引发宝玉不自在、黛玉中暑，二人都不去看戏——为口角创造条件；
   - 场面的"外"（清虚观）与"内"（怡红院砸玉）形成内外对照，使情节张弛有度。
4. **人物形象的多元展现**：
   - 贾母（慈厚）、凤姐（跋扈）、张道士（世故）、宝玉（叛逆）——在清虚观场面中各有表现，为后文冲突中的人物形象做铺垫。

#### 预测题 6（综合比较）

**问**：结合本节，谈谈你对《红楼梦》"打破传统的思想和写法"（鲁迅语）的理解。

**参考答案**：
鲁迅所评"打破传统"，在本节有突出体现：
1. **打破"才子佳人"模式**：
   - 传统才子佳人小说写爱情，多是"一见钟情—私订终身—金榜题名—奉旨成婚"的套路；
   - 本节写宝黛的相爱相猜、求近反疏——是真实的、有矛盾、有心理深度的爱情，没有套路的圆满，只有真实的痛苦。
2. **打破"好人完全好、坏人完全坏"的人物模式**：
   - 宝玉痴情但软弱，黛玉真情但尖刻，贾母慈厚但无奈，凤姐精明但跋扈——人物都是多面的、立体的；
   - 没有脸谱化的善恶对立。
3. **打破"重情节轻心理"的写法**：
   - 中国古典小说多以情节、动作叙事为主；
   - 本节用大段心理独白揭示人物内心——"原是一个心反弄成两个心"——前所未有的心理深度。
4. **打破"大团圆"结局**：
   - 传统小说多以"有情人终成眷属"收场；
   - 本节虽以"情发一心"的领悟收尾，但"求近反疏"的悖论已预示悲剧——宝玉黛玉最终未能结合。
5. **思想上的突破**：
   - 通过宝黛爱情，质疑封建婚姻制度与"金玉之说"；
   - 通过贾府排场与内闱危机，预示封建家族的必然崩溃——"千红一哭，万艳同悲"的悲剧观，是对封建社会的深刻批判。
本节是《红楼梦》打破传统的集中体现，标志着中国古典小说走向成熟与深刻。

## 附录：原文

> 　　话说宝玉正自发怔，不想黛玉将手帕子甩了来，正碰在眼睛上，倒唬了一跳，问是谁。林黛玉摇著头儿笑道："不敢，是我失了手。因为宝姐姐要看呆雁，我比给他看，不想失了手。"宝玉揉著眼睛，待要说什么，又不好说的。
>
> 　　一时，凤姐儿来了，因说起初一日在清虚观打醮的事来，遂约著宝钗，宝玉，黛玉等看戏去。宝钗笑道："罢，罢，怪热的。什么没看过的戏，我就不去了。"凤姐儿道："他们那里凉快，两边又有楼。咱们要去，我头几天打发人去，把那些道士都赶出去，把楼打扫干净，挂起帘子来，一个闲人不许放进庙去，才是好呢。我已经回了太太了，你们不去我去。这些日子也闷的很了。家里唱动戏，我又不得舒舒服服的看。"
>
> 　　贾母听说，笑道："既这么著，我同你去。"凤姐听说，笑道："老祖宗也去，敢情好了！就只是我又不得受用了。"贾母道："到明儿，我在正面楼上，你在旁边楼上，你也不用到我这边来立规矩，可好不好？"凤姐儿笑道："这就是老祖宗疼我了。"贾母因又向宝钗道："你也去，连你母亲也去。长天老日的，在家里也是睡觉。"宝钗只得答应著。
>
> 　　贾母又打发人去请了薛姨妈，顺路告诉王夫人，要带了他们姊妹去。王夫人因一则身上不好，二则预备著元春有人出来，早已回了不去的，听贾母如今这样说，笑道："还是这么高兴。"因打发人去到园里告诉："有要逛的，只管初一跟了老太太逛去。"这个话一传开了，别人都还可已，只是那些丫头们天天不得出门槛子，听了这话，谁不要去。便是各人的主子懒怠去，他也百般撺掇了去，因此李宫裁等都说去。贾母越发心中喜欢，早已吩咐人去打扫安置，都不必细说。单表到了初一这一日，荣国府门前车辆纷纷，人马簇簇。那底下凡执事人等，闻得是贵妃作好事，贾母亲去拈香，正是初一日乃月之首日，况是端阳节间，因此凡动用的什物，一色都是齐全的，不同往日。少时，贾母等出来。贾母坐一乘八人大轿，李氏，凤姐儿，薛姨妈每人一乘四人轿，宝钗，黛玉二人共坐一辆翠盖珠缨八宝车，迎春，探春，惜春三人共坐一辆朱轮华盖车。然后贾母的丫头鸳鸯，鹦鹉，琥珀，珍珠，林黛玉的丫头紫鹃，雪雁，春纤，宝钗的丫头莺儿，文杏，迎春的丫头司棋，绣桔，探春的丫头侍书，翠墨，惜春的丫头入画，彩屏，薛姨妈的丫头同喜，同贵，外带著香菱，香菱的丫头臻儿，李氏的丫头素云、碧月，凤姐儿的丫头平儿、丰儿、小红，并王夫人两个丫头也要跟了凤姐儿去的金钏，彩云，奶子抱著大姐儿带著巧姐儿另在一车，还有两个丫头，一共又连上各房的老嬷嬷奶娘并跟出门的家人媳妇子，乌压压的占了一街的车。贾母等已经坐轿去了多远，这门前尚未坐完。这个说："我不同你在一处"，那个说"你压了我们奶奶的包袱"，那边车上又说"蹭了我的花儿"，这边又说"碰折了我的扇子"，咭咭呱呱，说笑不绝。周瑞家的走来过去的说道："姑娘们，这是街上，看人笑话。"说了两遍，方觉好了。前头的全副执事摆开，早已到了清虚观了。宝玉骑著马，在贾母轿前。街上人都站在两边。将至观前，只听钟鸣鼓响，早有张法官执香披衣，带领众道士在路旁迎接。贾母的轿刚至山门以内，贾母在轿内因看见有守门大帅并千里眼，顺风耳，当方土地，本境城隍各位泥胎圣像，便命住轿。贾珍带领各子弟上来迎接。凤姐儿知道鸳鸯等在后面，赶不上来搀贾母，自己下了轿，忙要上来搀。可巧有个十二三岁的小道士儿，拿著剪筒，照管剪各处蜡花，正欲得便且藏出去，不想一头撞在凤姐儿怀里。凤姐便一扬手，照脸一下，把那小孩子打了一个筋斗，骂道："野牛肏的，胡朝那里跑！"那小道士也不顾拾烛剪，爬起来往外还要跑。正值宝钗等下车，众婆娘媳妇正围随的风雨不透，但见一个小道士滚了出来，都喝声叫"拿，拿，拿！打，打，打！"
>
> 　　贾母听了忙问："是怎么了？"贾珍忙出来问。凤姐上去搀住贾母，就回说："一个小道士儿，剪灯花的，没躲出去，这会子混钻呢。"贾母听说，忙道："快带了那孩子来，别唬著他。小门小户的孩子，都是娇生惯养的，那里见的这个势派。倘或唬著他，倒怪可怜见的，他老子娘岂不疼的慌？"说著，便叫贾珍去好生带了来。贾珍只得去拉了那孩子来。那孩子还一手拿著蜡剪，跪在地下乱战。贾母命贾珍拉起来，"叫他别怕。问他几岁了。"那孩子通说不出话来。贾母还说"可怜见的"，又向贾珍道："珍哥儿，带他去罢。给他些钱买果子吃，别叫人难为了他。"贾珍答应，领他去了。这里贾母带著众人，一层一层的瞻拜观玩。外面小厮们见贾母等进入二层山门，忽见贾珍领了一个小道士出来，叫人来带去，给他几百钱，不要难为了他。家人听说，忙上来领了下去。
>
> 　　贾珍站在阶矶上，因问："管家在那里？"底下站的小厮们见问，都一齐喝声说："叫管家！"登时林之孝一手整理著帽子跑了来，到贾珍跟前。贾珍道："虽说这里地方大，今儿不承望来这么些人。你使的人，你就带了往你的那院里去，使不著的，打发到那院里去。把小幺儿们多挑几个在这二层门上同两边的角门上，伺候著要东西传话。你可知道不知道，今儿小姐奶奶们都出来，一个闲人也到不了这里。"林之孝忙答应"晓得"，又说了几个"是"。贾珍道："去罢。"又问："怎么不见蓉儿？"一声未了，只见贾蓉从钟楼里跑了出来。贾珍道："你瞧瞧他，我这里也还没敢说热，他倒乘凉去了！"喝命家人啐他。那小厮们都知道贾珍素日的性子，违拗不得，有个小厮便上来向贾蓉脸上啐了一口。贾珍又道："问著他！"那小厮便问贾蓉道："爷还不怕热，哥儿怎么先乘凉去了？"贾蓉垂著手，一声不敢说。那贾芸，贾萍，贾芹等听见了，不但他们慌了，亦且连贾璜，贾㻞，贾琼等也都忙了，一个一个从墙根下慢慢的溜上来。贾珍又向贾蓉道："你站著作什么？还不骑了马跑到家里，告诉你娘母子去！老太太同姑娘们都来了，叫他们快来伺候。"贾蓉听说，忙跑了出来，一叠声要马，一面抱怨道："早都不知作什么的，这会子寻趁我。"一面又骂小子："捆著手呢？马也拉不来。"待要打发小子去，又恐后来对出来，说不得亲自走一趟，骑马去了，不在话下。
>
> 　　且说贾珍方要抽身进去，只见张道士站在旁边陪笑说道："论理我不比别人，应该里头伺候。只因天气炎热，众位千金都出来了，法官不敢擅入，请爷的示下。恐老太太问，或要随喜那里，我只在这里伺候罢了。"贾珍知道这张道士虽然是当日荣国府国公的替身，曾经先皇御口亲呼为"大幻仙人"，如今现掌"道录司"印，又是当今封为"终了真人"，现今王公藩镇都称他为"神仙"，所以不敢轻慢。二则他又常往两个府里去，凡夫人小姐都是见的。今见他如此说，便笑道："咱们自己，你又说起这话来。再多说，我把你这胡子还挦了呢！还不跟我进来。"那张道士呵呵大笑，跟了贾珍进来。
>
> 　　贾珍到贾母跟前，控身陪笑说："这张爷爷进来请安。"贾母听了，忙道："搀他来。"贾珍忙去搀了过来。那张道士先哈哈笑道："无量寿佛！老祖宗一向福寿安康？众位奶奶小姐纳福？一向没到府里请安，老太太气色越发好了。"贾母笑道："老神仙，你好？"张道士笑道："托老太太万福万寿，小道也还康健。别的倒罢，只记挂著哥儿，一向身上好？前日四月二十六日，我这里做遮天大王的圣诞，人也来的少，东西也很干净，我说请哥儿来逛逛，怎么说不在家？"贾母说道："果真不在家。"一面回头叫宝玉。谁知宝玉解手去了才来，忙上前问："张爷爷好？"张道士忙抱住问了好，又向贾母笑道："哥儿越发发福了。"贾母道："他外头好，里头弱。又搭著他老子逼著他念书，生生的把个孩子逼出病来了。"张道士道："前日我在好几处看见哥儿写的字，作的诗，都好的了不得，怎么老爷还抱怨说哥儿不大喜欢念书呢？依小道看来，也就罢了。"又叹道："我看见哥儿的这个形容身段，言谈举动，怎么就同当日国公爷一个稿子！"说著两眼流下泪来。贾母听说，也由不得满脸泪痕，说道："正是呢，我养这些儿子孙子，也没一个像他爷爷的，就只这玉儿像他爷爷。"
>
> 　　那张道士又向贾珍道："当日国公爷的模样儿，爷们一辈的不用说，自然没赶上，大约连大老爷，二老爷也记不清楚了。"说毕呵呵又一大笑，道："前日在一个人家看见一位小姐，今年十五岁了，生的倒也好个模样儿。我想著哥儿也该寻亲事了。若论这个小姐模样儿，聪明智慧，根基家当，倒也配的过。但不知老太太怎么样，小道也不敢造次。等请了老太太的示下，才敢向人去说。"贾母道："上回有和尚说了，这孩子命里不该早娶，等再大一大儿再定罢。你可如今打听著，不管他根基富贵，只要模样配的上就好，来告诉我。便是那家子穷，不过给他几两银子罢了。只是模样性格儿难得好的。"
>
> 　　说毕，只见凤姐儿笑道："张爷爷，我们丫头的寄名符儿你也不换去。前儿亏你还有那么大脸，打发人和我要鹅黄缎子去！要不给你，又恐怕你那老脸上过不去。"张道士呵呵大笑道："你瞧，我眼花了，也没看见奶奶在这里，也没道多谢。符早已有了，前日原要送去的，不指望娘娘来作好事，就混忘了，还在佛前镇著。待我取来。"说著跑到大殿上去，一时拿了一个茶盘，搭著大红蟒缎经袱子，托出符来。大姐儿的奶子接了符。张道士方欲抱过大姐儿来，只见凤姐笑道："你就手里拿出来罢了，又用个盘子托著。"张道士道："手里不干不净的，怎么拿，用盘子洁净些。"凤姐儿笑道："你只顾拿出盘子来，倒唬我一跳。我不说你是为送符，倒象是和我们化布施来了。"众人听说，哄然一笑，连贾珍也掌不住笑了。贾母回头道："猴儿猴儿，你不怕下割舌头地狱？"凤姐儿笑道："我们爷儿们不相干。他怎么常常的说我该积阴骘，迟了就短命呢！"
>
> 　　张道士也笑道："我拿出盘子来一举两用，却不为化布施，倒要将哥儿的这玉请了下来，托出去给那些远来的道友并徒子徒孙们见识见识。"贾母道："既这们著，你老人家老天拔地的跑什么，就带他去瞧了，叫他进来，岂不省事？"张道士道："老太太不知道，看著小道是八十多岁的人，托老太太的福倒也健壮，二则外面的人多，气味难闻，况是个暑热的天，哥儿受不惯，倘或哥儿受了腌臜气味，倒值多了。"贾母听说，便命宝玉摘下通灵玉来，放在盘内。那张道士兢兢业业的用蟒袱子垫著，捧了出去。
>
> 　　这里贾母与众人各处游玩了一回，方去上楼。只见贾珍回说："张爷爷送了玉来了。"刚说著，只见张道士捧了盘子，走到跟前笑道："众人托小道的福，见了哥儿的玉，实在可罕。都没什么敬贺之物，这是他们各人传道的法器，都愿意为敬贺之礼。哥儿便不希罕，只留著在房里顽耍赏人罢。"贾母听说，向盘内看时，只见也有金璜，也有玉玦，或有事事如意，或有岁岁平安，皆是珠穿宝贯，玉琢金镂，共有三五十件。因说道："你也胡闹。他们出家人是那里来的，何必这样，这不能收。"张道士笑道："这是他们一点敬心，小道也不能阻挡。老太太若不留下，岂不叫他们看著小道微薄，不象是门下出身了。"贾母听如此说，方命人接了。宝玉笑道："老太太，张爷爷既这么说，又推辞不得，我要这个也无用，不如叫小子们捧了这个，跟著我出去散给穷人罢。"贾母笑道："这倒说的是。"张道士又忙拦道："哥儿虽要行好，但这些东西虽说不甚希奇，到底也是几件器皿。若给了乞丐，一则与他们无益，二则反倒遭塌了这些东西。要舍给穷人，何不就散钱与他们。"宝玉听说，便命收下，等晚间拿钱施舍罢了。说毕，张道士方退出去。
>
> 　　这里贾母与众人上了楼，在正面楼上归坐。凤姐等占了东楼。众丫头等在西楼，轮流伺候。贾珍一时来回："神前拈了戏，头一本《白蛇记》。"贾母问"《白蛇记》是什么故事？"贾珍道："是汉高祖斩蛇方起首的故事。第二本是《满床笏》。"贾母笑道："这倒是第二本上？也罢了。神佛要这样，也只得罢了。"又问第三本，贾珍道："第三本是《南柯梦》。"贾母听了便不言语。贾珍退了下来，至外边预备著申表，焚钱粮，开戏，不在话下。
>
> 　　且说宝玉在楼上，坐在贾母旁边，因叫个小丫头子捧著方才那一盘子贺物，将自己的玉带上，用手翻弄寻拨，一件一件的挑与贾母看。贾母因看见有个赤金点翠的麒麟，便伸手拿了起来，笑道："这件东西好象我看见谁家的孩子也带著这么一个的。"宝钗笑道："史大妹妹有一个，比这个小些。"贾母道："是云儿有这个。"宝玉道："他这么往我们家去住著，我也没看见。"探春笑道："宝姐姐有心，不管什么他都记得。"林黛玉冷笑道："他在别的上还有限，惟有这些人带的东西上越发留心。"宝钗听说，便回头装没听见。宝玉听见史湘云有这件东西，自己便将那麒麟忙拿起来揣在怀里。一面心里又想到怕人看见他听见史湘云有了，他就留这件，因此手里揣著，却拿眼睛瞟人。只见众人都倒不大理论，惟有林黛玉瞅著他点头儿，似有赞叹之意。宝玉不觉心里没好意思起来，又掏了出来，向黛玉笑道："这个东西倒好顽，我替你留著，到了家穿上你带。"林黛玉将头一扭，说道："我不希罕。"宝玉笑道："你果然不希罕，我少不得就拿著。"说著又揣了起来。刚要说话，只见贾珍贾蓉的妻子婆媳两个来了，彼此见过，贾母方说："你们又来做什么，我不过没事来逛逛。"一句话没说了，只见人报："冯将军家有人来了。"原来冯紫英家听见贾府在庙里打醮，连忙预备了猪羊香烛茶银之类的东西送礼。凤姐儿听了，忙赶过正楼来，拍手笑道："嗳呀！我就不防这个。只说咱们娘儿们来闲逛逛，人家只当咱们大摆斋坛的来送礼。都是老太太闹的。这又不得不预备赏封儿。"刚说了，只见冯家的两个管家娘子上楼来了。冯家两个未去，接著赵侍郎也有礼来了。于是接二连三，都听见贾府打醮，女眷都在庙里，凡一应远亲近友，世家相与都来送礼。贾母才后悔起来，说："又不是什么正经斋事，我们不过闲逛逛，就想不到这礼上，没的惊动了人。"因此虽看了一天戏，至下午便回来了，次日便懒怠去。凤姐又说："打墙也是动土，已经惊动了人，今儿乐得还去逛逛。那贾母因昨日张道士提起宝玉说亲的事来，谁知宝玉一日心中不自在，回家来生气，嗔著张道士与他说了亲，口口声声说从今以后不再见张道士了，别人也并不知为什么原故，二则林黛玉昨日回家又中了暑：因此二事，贾母便执意不去了。凤姐见不去，自己带了人去，也不在话下。
>
> 　　且说宝玉因见林黛玉又病了，心里放不下，饭也懒去吃，不时来问。林黛玉又怕他有个好歹，因说道："你只管看你的戏去，在家里作什么？"宝玉因昨日张道士提亲，心中大不受用，今听见林黛玉如此说，心里因想道："别人不知道我的心还可恕，连他也奚落起我来。"因此心中更比往日的烦恼加了百倍。若是别人跟前，断不能动这肝火，只是林黛玉说了这话，倒比往日别人说这话不同，由不得立刻沉下脸来，说道："我白认得了你。罢了，罢了！"林黛玉听说，便冷笑了两声，"我也知道白认得了我，那里象人家有什么配的上呢。"宝玉听了，便向前来直问到脸上："你这么说，是安心咒我天诛地灭？"林黛玉一时解不过这个话来。宝玉又道："昨儿还为这个赌了几回咒，今儿你到底又准我一句。我便天诛地灭，你又有什么益处？"林黛玉一闻此言，方想起上日的话来。今日原是自己说错了，又是著急，又是羞愧，便颤颤兢兢的说道："我要安心咒你，我也天诛地灭。何苦来！我知道，昨日张道士说亲，你怕阻了你的好姻缘，你心里生气，来拿我煞性子。"原来那宝玉自幼生成有一种下流痴病，况从幼时和黛玉耳鬓厮磨，心情相对，及如今稍明时事，又看了那些邪书僻传，凡远亲近友之家所见的那些闺英闱秀，皆未有稍及林黛玉者，所以早存了一段心事，只不好说出来，故每每或喜或怒，变尽法子暗中试探。那林黛玉偏生也是个有些痴病的，也每用假情试探。因你也将真心真意瞒了起来，只用假意，我也将真心真意瞒了起来，只用假意，如此两假相逢，终有一真。其间琐琐碎碎，难保不有口角之争。即如此刻，宝玉的心内想的是："别人不知我的心，还有可恕，难道你就不想我的心里眼里只有你！你不能为我烦恼，反来以这话奚落堵我。可见我心里一时一刻白有你，你竟心里没我。"心里这意思，只是口里说不出来。那林黛玉心里想著："你心里自然有我，虽有'金玉相对'之说，你岂是重这邪说不重我的。我便时常提这'金玉'，你只管了然自若无闻的，方见得是待我重，而毫无此心了。如何我只一提'金玉'的事，你就著急，可知你心里时时有'金玉'，见我一提，你又怕我多心，故意著急，安心哄我。"
>
> 　　看来两个人原本是一个心，但都多生了枝叶，反弄成两个心了。那宝玉心中又想著："我不管怎么样都好，只要你随意，我便立刻因你死了也情愿。你知也罢，不知也罢，只由我的心，可见你方和我近，不和我远。"那林黛玉心里又想著："你只管你，你好我自好，你何必为我而自失。殊不知你失我自失。可见是你不叫我近你，有意叫我远你了。"如此看来，却都是求近之心，反弄成疏远之意。如此之话，皆他二人素习所存私心，也难备述。
>
> 　　如今只述他们外面的形容。那宝玉又听见他说"好姻缘"三个字，越发逆了己意，心里干噎，口里说不出话来，便赌气向颈上抓下通灵宝玉，咬牙恨命往地下一摔，道："什么捞什骨子，我砸了你完事！"偏生那玉坚硬非常，摔了一下，竟文风没动。宝玉见没摔碎，便回身找东西来砸。林黛玉见他如此，早已哭起来，说道："何苦来，你摔砸那哑吧物件。有砸他的，不如来砸我。"二人闹著，紫鹃雪雁等忙来解劝。后来见宝玉下死力砸玉，忙上来夺，又夺不下来，见比往日闹的大了，少不得去叫袭人。袭人忙赶了来，才夺了下来。宝玉冷笑道："我砸我的东西，与你们什么相干！"
>
> 　　袭人见他脸都气黄了，眼眉都变了，从来没气的这样，便拉著他的手，笑道："你同妹妹拌嘴，不犯著砸他，倘或砸坏了，叫他心里脸上怎么过的去？"林黛玉一行哭著，一行听了这话说到自己心坎儿上来，可见宝玉连袭人不如，越发伤心大哭起来。心里一烦恼，方才吃的香薷饮解暑汤便承受不住，"哇"的一声都吐了出来。紫鹃忙上来用手帕子接住，登时一口一口的把一块手帕子吐湿。雪雁忙上来捶。紫鹃道："虽然生气，姑娘到底也该保重著些。才吃了药好些，这会子因和宝二爷拌嘴，又吐出来。倘或犯了病，宝二爷怎么过的去呢？"宝玉听了这话说到自己心坎儿上来，可见黛玉不如一紫鹃。又见林黛玉脸红头胀，一行啼哭，一行气凑，一行是泪，一行是汗，不胜怯弱。宝玉见了这般，又自己后悔方才不该同他较证，这会子他这样光景，我又替不了他。心里想著，也由不的滴下泪来了。袭人见他两个哭，由不得守著宝玉也心酸起来，又摸著宝玉的手冰凉，待要劝宝玉不哭罢，一则又恐宝玉有什么委曲闷在心里，二则又恐薄了林黛玉。不如大家一哭，就丢开手了，因此也流下泪来。紫鹃一面收拾了吐的药，一面拿扇子替林黛玉轻轻的扇著，见三个人都鸦雀无声，各人哭各人的，也由不得伤心起来，也拿手帕子擦泪。四个人都无言对泣。
>
> 　　一时，袭人勉强笑向宝玉道："你不看别的，你看看这玉上穿的穗子，也不该同林姑娘拌嘴。"林黛玉听了，也不顾病，赶来夺过去，顺手抓起一把剪子来要剪。袭人紫鹃刚要夺，已经剪了几段。林黛玉哭道："我也是白效力。他也不希罕，自有别人替他再穿好的去。"袭人忙接了玉道："何苦来，这是我才多嘴的不是了。"宝玉向林黛玉道："你只管剪，我横竖不带他，也没什么。"
>
> 　　只顾里头闹，谁知那些老婆子们见林黛玉大哭大吐，宝玉又砸玉，不知道要闹到什么田地，倘或连累了他们，便一齐往前头回贾母王夫人知道，好不干连了他们。那贾母王夫人见他们忙忙的作一件正经事来告诉，也都不知有了什么大祸，便一齐进园来瞧他兄妹。急的袭人抱怨紫鹃为什么惊动了老太太，太太，紫鹃又只当是袭人去告诉的，也抱怨袭人。那贾母，王夫人进来，见宝玉也无言，林黛玉也无话，问起来又没为什么事，便将这祸移到袭人紫鹃两个人身上，说"为什么你们不小心伏侍，这会子闹起来都不管了！"因此将他二人连骂带说教训了一顿。二人都没话，只得听著。还是贾母带出宝玉去了，方才平服。
>
> 　　过了一日，至初三日，乃是薛蟠生日，家里摆酒唱戏，来请贾府诸人。宝玉因得罪了林黛玉，二人总未见面，心中正自后悔，无精打采的，那里还有心肠去看戏，因而推病不去。林黛玉不过前日中了些暑溽之气，本无甚大病，听见他不去，心里想："他是好吃酒看戏的，今日反不去，自然是因为昨儿气著了。再不然，他见我不去，他也没心肠去。只是昨儿千不该万不该剪了那玉上的穗子。管定他再不带了，还得我穿了他才带。"因而心中十分后悔。
>
> 　　那贾母见他两个都生了气，只说趁今儿那边看戏，他两个见了也就完了，不想又都不去。老人家急的抱怨说："我这老冤家是那世里的孽障，偏生遇见了这么两个不省事的小冤家，没有一天不叫我操心。真是俗语说的，'不是冤家不聚头'。几时我闭了这眼，断了这口气，凭著这两个冤家闹上天去，我眼不见心不烦，也就罢了。偏又不咽这口气。"自己抱怨著也哭了。这话传入宝林二人耳内。原来他二人竟是从未听见过"不是冤家不聚头"的这句俗语，如今忽然得了这句话，好似参禅的一般，都低头细嚼此话的滋味，都不觉潸然泣下。虽不曾会面，然一个在潇湘馆临风洒泪，一个在怡红院对月长吁，却不是人居两地，情发一心！
>
> 　　袭人因劝宝玉道："千万不是，都是你的不是，往日家里小厮们和他们的姊妹拌嘴，或是两口子分争，你听见了，你还骂小厮们蠢，不能体贴女孩儿们的心。今儿你也这么著了。明儿初五，大节下，你们两个再这们仇人似的，老太太越发要生气，一定弄的大家不安生。依我劝，你正经下个气，陪个不是，大家还是照常一样，这么也好，那么也好。"那宝玉听见了不知依与不依，要知端详，且听下回分解。
