# 《麦琪的礼物》

> 欧·亨利（美国·19 世纪末 20 世纪初） · 短篇小说 · 1906 年（出自《四百万》） · 译文

::: tip 本篇定位
本文是美国短篇小说大师欧·亨利的代表作，"**出人意料的结尾**"（欧·亨利式结尾）的典范。
写贫穷夫妻**德拉（黛拉）**与**吉姆**，在圣诞前夜各自卖掉引以为荣之物（长发 / 金表）为对方买礼物，结果两件礼物都"暂时无用"，却让爱的光芒照彻全篇。
是"**含泪的微笑**""**双线结构**""**小人物的温情**"的世界名篇。
:::

## 一、知人论世

**作者：** 欧·亨利（O. Henry，1862—1910），原名威廉·西德尼·波特，**美国**著名短篇小说家。与法·莫泊桑、俄·契诃夫并称**"世界三大短篇小说之王"**。

**生平：** 当过药店学徒、牧牛人、出纳员、银行职员。曾因账目问题含冤入狱三年，在狱中开始写作。底层经历使他对小人物充满同情，笔下多写市民社会里挣扎的普通人。

**文学地位：**
- 美国现代短篇小说的奠基者之一
- 以"**欧·亨利式结尾**"（O. Henry ending）闻名世界——出人意料又在情理之中
- 被誉为"**曼哈顿的桂冠散文作家**"

**文学特色：**
- **精巧构思**：情节峰回路转，结尾陡然反转
- **含泪的微笑**：写小人物的辛酸，却总带温情与幽默
- **语言幽默**：善用比喻、夸张、谐谑的叙述语调
- **关注底层**：笔下多是贫苦小职员、店员、工人

**写作背景：**
- 19 世纪末 20 世纪初美国工业化加速，贫富分化加剧
- 出自小说集《四百万》（The Four Million，1906），书名寓意"纽约有四百万人，不是四百人"——驳斥权贵"纽约上流社会只有四百人"的说法，宣告**普通小人物也值得书写**
- 圣诞节在西方是家人互赠礼物的最重要节日，本篇以"圣诞礼物"切入

**文体定位：**
- 形式：**短篇小说**（译文）
- 特色：**双线结构 + 欧·亨利式结尾 + 含泪的微笑**

::: warning 与《苦恼》对比（外国短篇小说双璧）

## 二、情节梳理（双线交织 → 结尾反转）

> 全文以**圣诞节前夜"为爱人买礼物"**为线索，**德拉线**为明线（详写），**吉姆线**为暗线（略写，结尾点破），两线在结尾交汇，形成震惊。

### 开端：贫困的窘境（德拉数钱）

**情节：** 德拉反复数钱，只有"**一块八毛七**"，而明天就是圣诞节。她扑在沙发上默默流泪。

**作用：**
- **悬念与基调**：一开篇就用精确到分的"一块八毛七"奠定**贫穷 + 愁苦**的基调
- "六毛还是用小硬币凑成的"——细节写尽**精打细算的辛酸**
- "明天就是圣诞节"的反复——**时间压力**推动情节

### 发展一：引出两件珍宝（伏笔）⭐⭐⭐

**情节：** 介绍这个八元租金的寒酸之家，并点出夫妻俩引以为荣的两件东西：**吉姆的金表**（祖传三代）与**德拉的长发**（"棕色的瀑布"，拖到膝盖下）。德拉决意卖发。

**作用（伏笔——全篇枢纽）：**
- "两件东西"的并置，看似闲笔，实为全篇**最大的伏笔**
- 示巴女王、所罗门国王的典故——**极言两件宝物的珍贵**，为后文"牺牲"做铺垫
- "头发一直拖到膝盖下面，简直变成一件衫子罩着她"——浓墨重彩，为卖发蓄势

### 发展二：德拉卖发、买表链（德拉线）

**情节：** 德拉以二十元卖掉头发，用二十一元买下一条**白金表链**（"素静而有价值"，正配吉姆和他的表）。回家用卷发钳补救，剪成短发像"逃学的中学生"。

**作用：**
- 表链的描写 = **以物写人**：表链"素静而有价值"，正是吉姆的写照
- "出于爱情和慷慨而造成的灾害"——幽默中藏辛酸
- 剪发后"像逃学的中学生"——**含泪的微笑**：自嘲里有忐忑，怕丈夫嫌弃

### 高潮：吉姆归来，奇特的表情

**情节：** 吉姆进门，"像猎狗闻到鹌鹑"般一动不动，盯着德拉，表情"既不是发怒，也不是惊讶……不是她预料中的任何表情"。德拉哀求解释，吉姆才掏出一个小包。

**作用：**
- 吉姆的"奇特表情"是**最大的悬念**——读者此时只知德拉卖发，猜不透吉姆为何震惊
- "像猎狗闻到鹌鹑"——**动作比喻**写瞬间的呆滞
- 为结尾的反转**持续蓄势**

### 结局（反转）：两件礼物都"暂时无用"⭐⭐⭐

**情节：** 吉姆送的是一套**玳瑁发梳**（德拉梦寐以求、配她那头美发的梳子），可长发已剪；德拉送的**白金表链**，可金表已被吉姆卖掉。两人相视，含泪而笑。吉姆说："我们把圣诞节礼物放到一边，暂时保留着吧……我将表卖了钱给你买了梳子。"

**手法（欧·亨利式结尾）：**
- **出人意料**：谁也没料到双方都卖掉了珍宝——表链无表可配，梳子无发可饰
- **情理之中**：前文"两件珍宝"的伏笔、"一块八毛七"的贫穷、双方深爱彼此——**牺牲是必然，只是恰好交错**
- **含泪的微笑**：礼物"暂时无用"，但爱"永远有用"——悲喜交织

::: warning 双线结构（必背）
- **明线（详写）——德拉线**：数钱 → 卖发 → 买表链 → 剪发补救 → 送给吉姆
- **暗线（略写）——吉姆线**：卖金表 → 买玳瑁梳子 → 带回家（全程暗写，直到结尾点破）
- **交汇点**：结尾两件礼物同时亮相——**表链无表，梳子无发**，反转爆发
:::

## 三、人物形象分析

### 德拉：痴情奉献的妻子

| 性格面 | 表现 | 手法 |
| --- | --- | --- |
| **深爱丈夫** | 宁可卖掉引以为荣的长发，也要为吉姆买一件"配得上他"的礼物 | 行动 + 心理 |
| **细致体贴** | 走遍全城，专挑"素静而有价值"的表链（与吉姆相配） | 细节 |
| **自尊敏感** | 买东西还价被骂"死抠"而"两颊发烧" | 心理描写 |
| **柔弱又坚韧** | 剪发后忐忑自嘲，但仍热切地献上礼物 | 语言 + 动作 |

**形象意义：**
- 德拉是**欧·亨利笔下的小人物**——贫穷、善良、为爱无私
- 她的牺牲把"麦琪"的真义——**爱人胜过爱己之宝**——具象化

### 吉姆：沉默深情的丈夫⭐

| 性格面 | 表现 | 手法 |
| --- | --- | --- |
| **深爱妻子** | 同样卖掉祖传金表，买下妻子梦寐以求的梳子 | 暗写 + 结尾点破 |
| **瘦而庄重** | "才二十二岁，却要负担一个家庭"，外套旧、无手套 | 外貌白描 |
| **宽容温柔** | 见德拉短发，不动怒不责备，只说"不能叫我对妻子的爱减少一丝一毫" | 语言 |
| **超越物质** | "我们把圣诞节礼物放到边，暂时保留着吧"——**不被物役，以爱为重** | 语言 |

**形象意义：**
- 吉姆虽是暗线，却是**与德拉对称的另一半**
- 他卖掉的是**祖传三代金表**——比德拉卖头发牺牲更重（金表无法再生，头发可再生），欧·亨利借此让**爱的分量对等**

::: tip 两个人的礼物"暂时无用"，爱情却"永远有用"
欧·亨利的高明：他写的不是"礼物的失败"，而是**"爱的胜利"**——
两件物质礼物都落了空，但**双方为彼此牺牲的心意**，才是真正的、永不失效的"麦琪的礼物"。
:::

## 四、艺术手法鉴赏

### 1. 欧·亨利式结尾（出人意料又在情理之中）⭐⭐⭐

这是本文最重要的、也是欧·亨利招牌的手法：

- **出人意料**：读者跟着德拉线走，以为高潮是"德拉送表链"，结尾却揭示**吉姆也卖了表买梳子**——表链无表、梳子无发，双双重叠失效，震惊读者。
- **情理之中**：处处伏笔使反转**真实可信**：
  - "一块八毛七"——贫穷是双方共同处境，吉姆同样没钱
  - "两件东西他们都引以为荣"——金表与长发并置，**埋下双向牺牲的种子**
  - 双方深爱彼此——**牺牲是必然**，只是恰好交错
- **含泪的微笑**：反转不是讽刺的嘲笑，而是**会心又心酸的微笑**——礼物落空，爱却圆满。

::: warning 手法三段式答题（欧·亨利式结尾）
**指明手法 → 分析如何运用 → 点明表达效果**
欧·亨利式结尾 = 文末吉姆送出玳瑁梳子，与德拉送出的白金表链同时亮相，揭示双方都卖掉珍宝（指明）+ 前文"一块八毛七"的贫穷、"两件珍宝"的并置、双方深爱彼此等伏笔层层铺垫，使反转真实可信（运用）+ 出人意料又在情理之中，形成强烈的心灵震撼，把"爱的奉献"主题推向高潮，留下含泪的微笑（效果）。
:::

### 2. 双线结构（明暗交织）⭐⭐⭐

- **明线（详写）德拉**：卖发 → 买表链 → 送礼
- **暗线（略写）吉姆**：卖表 → 买梳子 → 送礼
- **作用**：
  - **对称**：两条牺牲之线对称展开，凸显**爱情的对等**
  - **悬念**：吉姆线全程暗写，结尾才点亮，**制造反转的冲击力**
  - **主题**：双线交汇于"暂时无用"的礼物，反衬**爱的无用之用**

### 3. 含泪的微笑（悲喜交融）⭐⭐⭐

- 欧·亨利风格的灵魂：**写贫穷的辛酸，却用幽默谐谑的笔调**，让读者笑着流泪
- 例：
  - "卡片上的'迪林厄姆'……似乎在认真考虑最好紧缩成一个'迪'字"——**幽默拟人**写家道中落
  - "出于爱情和慷慨而造成的灾害"——把剪坏的头发称"灾害"，**谐谑中有心酸**
  - 像科尼岛上"歌剧合唱队的歌手"、"逃学的中学生"——**自嘲里藏着忐忑**
- **效果**：悲剧的内容 + 喜剧的笔调 = **含泪的微笑**，比纯悲剧更动人

### 4. 伏笔与照应 ⭐⭐⭐

- **"两件珍宝"并置**（金表 + 长发）——为双方各自牺牲埋下种子
- **"一块八毛七"的贫穷**——暗示吉姆同样没钱，必走牺牲之路
- **德拉剪发的忐忑**（怕吉姆嫌弃）——与吉姆进门"奇特的表情"形成**反差照应**
- **白金表链"正配得上那块怀表"**——结尾"表已卖"的反转正是对这句的**反讽照应**

### 5. 以物写人 / 物我相照 ⭐⭐

- **白金表链 = 吉姆的写照**："素静而有价值，纯粹以质地取胜……就像他本人"——**以物喻人**
- **玳瑁梳子 = 德拉的美**：配她那头"棕色瀑布"——礼物是**对美的礼赞**
- **信箱无信、电钮无人摁**：物写人际的孤寂与清贫

### 6. 对比（多重对比） ⭐⭐

| 对比项 | 效果 |
| --- | --- |
| **示巴女王 / 所罗门国王** vs **贫穷夫妻** | 以夸张典故反衬贫穷，凸显珍宝之贵 |
| **两件珍宝** vs **八元租金的寒舍** | 贫穷中的精神富足 |
| **礼物的"暂时无用"** vs **爱的"永远有用"** | 升华主旨：物质可空，精神无价 |
| **德拉线（明）** vs **吉姆线（暗）** | 结构对称，凸显对等的爱 |

### 7. 典故运用 ⭐⭐

- **示巴女王**（《圣经》）：以财富闻名的女王——反衬德拉头发的光泽
- **所罗门国王**（《圣经》）：以财富智慧闻名的王——反衬金表的贵重
- **麦琪（东方三博士）**（《圣经》）：首倡圣诞赠礼的贤人——**点题**，结尾把夫妻二人比作"贤人"
- **科尼岛**：纽约游乐区，"歌剧合唱队歌手"暗示浓妆艳抹——德拉自嘲短发

### 8. 幽默谐谑的叙述语调 ⭐⭐

- 叙述者跳出来说"别理会这糟糕的比喻吧"——**自我调侃的旁白**
- "一名数学家或一名才子会给你一个错误的答案"——**反讽物质计算**
- 这种**亲切谐谑的"我"**让苦涩变温柔，是欧·亨利标志性语调

### 文化常识（重点）

::: tip "麦琪"点题——全篇的钥匙
- **"麦琪"** = Magi = 《圣经》中**朝拜圣婴的东方三博士**
- 他们是**首倡圣诞赠礼的人**，所送黄金、乳香、没药皆为"聪明的礼物"
- 欧·亨利在结尾**反转点题**：两个"笨孩子"卖掉珍宝为对方买礼物，看似"最不明智"，实则"**最最聪明**"——**他们就是当代的麦琪**
- **点题的双重性**：① 物质礼物虽"暂时无用"，但爱的奉献是真正的"麦琪之礼"；② **真正的智慧 = 爱的牺牲**，而非物质的精明计算

## 六、主旨情感

**中心思想：** 本文通过贫穷夫妻德拉与吉姆在圣诞前夜各自卖掉最珍爱的东西（长发 / 金表）为对方买礼物的故事，赞美了**纯真无私的爱情与奉献精神**，揭示了**物质匮乏中精神的富有**。

**思想内涵：**
1. **爱的奉献**：双方都以自己最珍贵的宝贝换取对方的快乐——**爱人胜过爱己之宝**
2. **小人物的温情**：贫穷、挣扎，却保有爱与尊严——**物质贫乏，精神高贵**
3. **真正的"聪明"是爱**：物质计算"错误"（礼物都落空），但爱的智慧"正确"——**人不是数学题**
4. **含泪的微笑**：生活的辛酸里仍有人性的光辉——**欧·亨利的人道主义温情**

**情感基调：温馨含泪**
- 表面：幽默谐谑的叙述
- 深层：对小人物辛酸的深切同情
- 结尾：礼物的"无用"与爱的"有用"交叠——**含泪的微笑**

**文学价值：**
- "欧·亨利式结尾"的**最高典范**，世界短篇小说经典
- "双线结构"的教科书级运用
- 开"含泪的微笑"风格之先河
- 对后世短篇小说（尤其反转式结尾）影响深远

::: warning 主旨升华（结尾"麦琪"点题）
结尾叙述者跳出议论："这两个人就是《圣经》里提到的贤人。"
——欧·亨利把一对贫穷夫妻**升华为"麦琪"**：
物质上，他们送的礼物"暂时无用"，是"最不明智"的；
精神上，他们的牺牲是"**最最聪明**"的——**爱的奉献本身就是最珍贵的礼物**。
这是对**功利主义价值观**的温柔反叛，也是对**人间真情**的最高礼赞。

## 七、考点对接

### 常考题型
1. **欧·亨利式结尾**（重点·高频）—— 出人意料又在情理之中
2. **双线结构**（重点·高频）—— 德拉线 / 吉姆线
3. **"含泪的微笑"**（论述题）—— 欧·亨利风格
4. **伏笔与照应**（高频）—— 两件珍宝的并置
5. **"麦琪"点题**（重点）—— 标题含义
6. **人物形象**（高频）—— 德拉 / 吉姆
7. **与《苦恼》比较**（论述题）—— 外国短篇双璧

### 预测题 + 标准答案

#### 【预测题 1 · 论述】本文结尾被称为"欧·亨利式结尾"的典范，请结合文本分析其妙处。

> **标准答案：**
> "欧·亨利式结尾"的特点是**出人意料又在情理之中**，本文妙处有三：
> ① **出人意料的反转**：读者跟随德拉线（卖发买表链）以为高潮是送礼，结尾却揭示吉姆也卖了祖传金表买玳瑁梳子——**表链无表可配，梳子无发可饰**，两件礼物同时"暂时无用"，强烈震撼。
> ② **情理之中的伏笔**：前文"一块八毛七"的贫穷、"两件珍宝（金表/长发）"的并置、双方深爱彼此——层层铺垫，使吉姆同样卖宝的结局**真实可信**，并非生硬意外。
> ③ **含泪的微笑**：反转不是讽刺的嘲笑，而是**会心又心酸**——物质礼物落空，但双方牺牲的爱意才是真正的"麦琪之礼"，把"爱的奉献"主题推向高潮，余味悠长。

#### 【预测题 2 · 简答】分析本文"双线结构"及其作用。

> **标准答案：**
> 本文采用**双线结构**：
> ① **明线（详写）——德拉线**：数钱 → 卖发 → 买白金表链 → 剪发补救 → 送给吉姆。
> ② **暗线（略写）——吉姆线**：卖祖传金表 → 买玳瑁梳子 → 带回家（全程暗写，结尾点破）。
> ③ **作用**：
> - **对称**：两条牺牲之线对称展开，凸显**爱情的对等**——双方牺牲等价（金表祖传三代、长发披到膝下，皆为至宝）。
> - **悬念与反转**：吉姆线暗写，使读者只知德拉卖发，结尾双线交汇才点破吉姆卖表，**制造强烈的反转冲击**。
> - **主旨升华**：双线交汇于"暂时无用"的礼物，反衬**爱的无用之用**——物质可空，精神无价。

#### 【预测题 3 · 论述】谈谈你对标题"麦琪的礼物"的理解。

> **标准答案：**
> 标题"麦琪的礼物"是全篇的**钥匙**，有双重含义：
> ① **字面（典故）**："麦琪"（Magi）即《圣经》中朝拜初生耶稣的**东方三博士**，他们**首倡圣诞赠礼**，所送黄金、乳香、没药被视为"聪明的礼物"。
> ② **深层（点题）**：欧·亨利在结尾**反转点题**——两个贫穷"笨孩子"卖掉至宝为对方买礼物，物质上"最不明智"（礼物都暂时无用），精神上却"**最最聪明**"。叙述者明言"**这两个人就是《圣经》里提到的贤人**"。
> ③ **主旨升华**：标题暗示**真正的"麦琪之礼"不是物质的礼物，而是为爱牺牲的心意**。爱比黄金、乳香、没药更珍贵——这是对**功利主义价值观**的温柔反叛，也是对**人间真情**的最高礼赞。

#### 【预测题 4 · 简答】分析德拉的形象。

> **标准答案：**
> 德拉是一个**痴情奉献的贫穷妻子**：
> ① **深爱丈夫**：宁可卖掉引以为荣的长发，也要为吉姆买一件"配得上他"的礼物。
> ② **细致体贴**：走遍全城，专挑"素静而有价值"的表链（与吉姆相配），以物见人。
> ③ **自尊敏感**：还价被骂"死抠"而"两颊发烧"——写贫穷者的体面。
> ④ **柔弱而坚韧**：剪发后忐忑自嘲，仍热切献上礼物，"上帝慈悲，让他觉得我这模样依然漂亮"——以忐忑写深情。
>
> **意义**：德拉是欧·亨利笔下的小人物——**贫穷、善良、为爱无私**，她的牺牲把"麦琪"的真义（爱人胜过爱己之宝）具象化。

#### 【预测题 5 · 选择】下列对小说的理解，不正确的一项是：
- A. "一块八毛七"的精确数字奠定了贫穷愁苦的基调 ✔
- B. 金表与长发的并置是全篇最大的伏笔 ✔
- C. 结尾的反转纯粹是巧合，缺乏前文铺垫 ✅
- D. "含泪的微笑"指用幽默笔调写辛酸，悲喜交融 ✔
>
> **答案：C**
> **讲评：** C 错。结尾反转**并非生硬巧合**，而是"**情理之中**"——前文"一块八毛七"的贫穷（吉姆同样没钱）、"两件珍宝"的并置（金表/长发）、双方深爱彼此等伏笔层层铺垫，使吉姆卖表的结局真实可信。这正是"欧·亨利式结尾"的精髓：**出人意料又在情理之中**。

> 要晓得，詹姆士·迪林厄姆·扬一家有两件东西是他们两人都引以为荣的。一件是吉姆的金表，那是他祖父传给他父亲，父亲又传给他的；另一件就是黛拉的头发。

> 你看此刻黛拉的长发披散开来，摇曳生姿，闪闪发亮，像一道棕色的瀑布。头发一直拖到她膝盖下面，简直变成一件衫子罩着她。

> 这是一条带饰物的白金表链，式样朴素高雅，纯粹以质地取胜而不是靠耀眼的装璜……它正配得上那块怀表……这表链就像他本人，素静而有价值。

> 吉姆停在门内，像一只猎狗闻到鹌鹑的气味那样一动不动。他的眼睛盯住了黛拉，里面有一种她无法理解的表情……既不是发怒，也不是惊讶，也不是赞成，也不是恐惧，不是她预料中的任何表情。

> 因为摆在桌上的是一套梳子——整套的梳子……正是黛拉好久以来在百老汇大街橱窗里看得不胜羡慕的那一种……而现在，她有了梳子，可是这些痴心妄想的装饰品用以装饰的长发却不存在了。

> "黛拉，"他说，"我们把圣诞节礼物放到一边，暂时保留着吧。这两件礼物太好了，只是目前还不能用。我将表卖了钱给你买了梳子。"

> 而我在这里跟你笨嘴拙舌地讲述了这间屋子里两个笨孩子的不足为奇的掌故，他们最不明智地为对方牺牲了他们家的最宝贵的财富。但是……他们俩是最最聪明的。他俩到哪里都是聪明的。**这两个人就是《圣经》里提到的贤人。**
