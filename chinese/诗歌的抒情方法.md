# 诗歌的抒情方法

> 诗歌是情感的产物。"诗者，志之所之也，在心为志，发言为诗。"
> 抒情方法是诗歌鉴赏的**核心考点**，贯穿本站所有诗歌篇目。
> 本课件结合已收录的 18 首诗歌讲解。

::: tip 课件定位
这是大学语文**诗歌鉴赏的方法论课件**之一（与未来的《诗歌意象与格律》并列）。
诗歌鉴赏千头万绪，但归根结底是两个问题：
**① 抒发了什么情感？② 怎样抒发的？（抒情方法）**
本课件专攻第二个问题。
:::

---

## 一、抒情方法总论

### 什么是抒情方法

抒情方法，即诗人**表达情感的方式和手段**。同样是写"愁"，有人直说"我很愁"，有人借落叶残菊来表达，有人借古人古事来寄托——这就是抒情方法的不同。

> **抒情方法 = 直接抒情 + 间接抒情**

### 抒情方法分类总图

```
                    抒情方法
                       │
          ┌────────────┴────────────┐
          ▼                         ▼
      直接抒情                   间接抒情
   （直抒胸臆）              （诗的主要方式）
          │                         │
     直接说出情感        ┌──────┬──────┬──────┬──────┐
                       ▼      ▼      ▼      ▼      ▼
                    借景    托物    借古    因事    借理
                    抒情    言志    咏怀    抒情    抒情
                     │      │      │      │      │
                  (最重要) (咏物)  (用典) (叙事) (理趣)
```

### 直接 vs 间接的比例

- **古典诗词**：以**间接抒情**为主（含蓄蕴藉是中国诗的美学追求）
- **现代新诗**：直接抒情比例上升（但仍多用意象象征）
- **越好的诗越含蓄**："不着一字，尽得风流"（司空图）

::: warning 抒情方法口诀
**直接抒情：直说。**
**间接抒情：借景、借物、借古、借事、借理。**
诗的好坏，往往看"借"得高明不高明。
:::

## 二、直接抒情（直抒胸臆）

### 1. 定义

诗人**不假外物，直接把情感说出来**。又称"直抒胸臆"。

### 2. 常见标志词

> "哀""愁""悲""怒""喜""叹""可怜""伤心""肠断"……

### 3. 本站案例

| 诗歌 | 直抒句 | 情感 |
| --- | --- | --- |
| 《虞美人》 | "问君能有几多愁？**恰似一江春水向东流**" | 亡国之愁（兼用比喻） |
| 《登高》 | "**艰难苦恨**繁霜鬓" | 老病孤愁 |
| 《江城子》 | "**相顾无言，惟有泪千行**" | 悼亡之痛 |
| 《摸鱼儿》 | "**闲愁最苦**" | 壮志难酬 |
| 《一句话》 | "**咱们的中国！**" | 炽热爱国激情 |
| 《祖国啊》 | "**我是你十亿分之一**" | 与祖国血脉相连 |
| 《行路难》 | "**长风破浪会有时**" | 自信豪情 |
| 《声声慢》 | "**这次第，怎一个愁字了得**" | 愁到极点（反话正说） |

### 4. 直抒胸臆的效果与注意

- **优点**：情感强烈、鲜明、有冲击力
- **风险**：直白易流于浅露，需配合意象/比喻才有力
- **高手用法**：很少纯直抒，往往是**间接抒情铺垫后的爆发点**

::: tip 直抒的经典位置
直接抒情常出现在诗歌的**结尾**（卒章显志）或**高潮处**——
经过前面的借景/借物铺垫，情感积聚到顶点，一句直抒，力透纸背。
如《登高》末联"艰难苦恨繁霜鬓"——前三联写景铺垫，末联直抒。
:::

## 三、间接抒情（一）：借景抒情（最重要）⭐⭐⭐

### 1. 定义

诗人**把情感寄托在自然景物中**，通过写景来抒情。这是中国古典诗词**最核心**的抒情方式。

### 2. 景与情的关系（高频考点）

#### ① 情景交融（景即情）

景与情**融为一体**，分不清是写景还是抒情。

**代表：** 《登高》
- "无边落木萧萧下，不尽长江滚滚来"——既是秋景，又是身世飘零、韶光易逝之悲
- 景即情，情即景

#### ② 以乐景写乐情 / 以哀景写哀情（正衬）

景的色调与情的色调**一致**。

| 关系 | 案例 |
| --- | --- |
| **乐景衬乐情** | 《再别康桥》"那河畔的金柳，是夕阳中的新娘"——美景写依依之情 |
| **哀景衬哀情** | 《声声慢》"满地黄花堆积，憔悴损"——残秋写愁苦；《登高》"猿啸哀"写悲 |

#### ③ 以乐景写哀情 / 以哀景写乐情（反衬）⭐⭐⭐

景的色调与情的色调**相反**——**反衬**，效果更强烈。这是**最高级的借景抒情**。

**代表案例：**

| 诗歌 | 乐/哀景 | 哀/乐情 | 效果 |
| --- | --- | --- | --- |
| 《蒹葭》 | 秋水苍苍（清冷景） | 追慕怅惘 | 以清冷衬求之不得 |
| 《江城子》 | "小轩窗，正梳妆"（温馨往昔） | 生死之悲 | **以乐景写哀情**，更增其悲 |
| 《归园田居》 | "羁鸟恋旧林，池鱼思故渊"（困境） | 归隐之乐 | 以束缚反衬自由 |

::: warning 核心考点：以乐写哀
**"以乐景写哀，以哀景写乐，一倍增其哀乐。"**（王夫之《姜斋诗话》）
- 《江城子》写亡妻生前"小轩窗，正梳妆"的温馨——越温馨，越显今日的凄凉
- 这是反衬的最高境界，务必掌握
:::

### 3. 借景抒情的常见手法

| 手法 | 说明 | 案例 |
| --- | --- | --- |
| **触景生情** | 先写景，引发情感 | 《登高》登高见秋景，生漂泊之悲 |
| **寓情于景** | 情含在景中，不直接说 | 《声声慢》满眼黄花、梧桐细雨，皆含愁 |
| **以景结情** | 结尾用景，情在景中 | 《再别康桥》"悄悄是别离的笙箫" |

## 四、间接抒情（二）：托物言志（咏物抒怀）⭐⭐⭐

### 1. 定义

诗人**借自然界的事物（物）来寄托自己的志向、品格或情感**。"物"是载体，"志/情"是本质。

### 2. 托物言志 vs 借景抒情（易混辨析）⭐

| | 借景抒情 | 托物言志 |
| --- | --- | --- |
| 借什么 | 景（场面、环境） | 物（具体事物） |
| 抒什么 | 情（一时的情感） | 志（长久的志向/品格） |
| 特点 | 多个景物构成画面 | 集中写一物，抓其特征喻人 |
| 案例 | 《登高》借秋景 | 《早雁》借大雁 |

### 3. 本站案例

| 诗歌 | 所托之物 | 所言之志 |
| --- | --- | --- |
| 《早雁》 | 大雁（遭弓射、失群） | 边关百姓遭战乱、流离失所（**托物寄兴**） |
| 《早雁》 | "莫厌潇湘少人处，水多菰米岸莓苔" | 劝慰流民暂避，含关怀 |
| 《我愿意是急流》 | 急流/废墟/云朵/旗帜 | 我愿为爱人奉献一切（**多重比喻**） |
| 《摸鱼儿》 | 春光消逝、落花 | 青春流逝、国事不可为（**伤春喻志**） |

::: tip 咏物诗的判定
咏物诗三要素：
1. **以物为题**（早雁、梅花、石灰）
2. **描物之态**（抓住物的特征）
3. **喻人之志**（物 = 人的品格/遭遇）
《早雁》是典型：咏雁的遭射失群 → 喻百姓的战乱流离。
:::

## 五、间接抒情（三）：借古咏怀（用典 / 怀古）⭐⭐

### 1. 定义

诗人**借历史人物、历史事件或前人诗文**，来寄托自己的情感或志向。包括**用典**和**怀古**。

### 2. 两种形式

#### ① 用典（引经据典）

借用经典中的故事或语句，含蓄表达。

**代表：** 《行路难》
- "闲来垂钓碧溪上，忽复乘舟梦日边"——用**姜太公钓鱼**（文王得遇）和**伊尹梦日**（商汤得贤）的典故
- 表达：**我虽遇挫，终有被重用的一天**（自信）

#### ② 咏史怀古

借历史遗迹或事件抒发感慨。

**代表：** 《长恨歌》
- 借唐玄宗与杨贵妃的爱情悲剧，抒"**天长地久有时尽，此恨绵绵无绝期**"的长恨

**代表：** 《摸鱼儿》
- 用汉武帝陈皇后"长门"典故（"长门事，准拟佳期又误"）——以**宫怨喻己之被疏远**

### 3. 借古咏怀的效果

- **含蓄深沉**：不说自己的事，借古人说，更耐人寻味
- **以古鉴今**：借历史反思现实
- **扩大容量**：一句话连接古今

::: warning 用典题答题套路
① 指出典故内容（典故是什么）→ ② 分析诗人用意（借古说什么）→ ③ 点明情感（抒发了什么）
:::

## 六、间接抒情（四）：因事抒情（叙事中抒情）⭐⭐

### 1. 定义

诗人**通过叙述事件、描写行为**来抒情，情感蕴含在叙事中。

### 2. 本站案例

| 诗歌 | 所叙之事 | 抒之情 |
| --- | --- | --- |
| 《陌上桑》 | 罗敷拒使君、盛夸夫婿 | 赞美罗敷的机智坚贞 |
| 《长恨歌》 | 玄宗贵妃的爱情悲剧 | 长恨与同情 |
| 《江城子》 | "夜来幽梦忽还乡"（梦境） | 悼亡——梦中相见更增悲 |
| 《归园田居》 | "开荒南野际，守拙归园田" | 归隐之乐 |

### 3. 叙事诗的抒情特点

- **叙事诗**（如《陌上桑》《长恨歌》）以叙事为骨架，但情感贯穿始终
- **抒情诗中的叙事**（如《江城子》的梦境）往往是**情感的载体**

::: tip 《江城子》的"因事抒情"
"夜来幽梦忽还乡。小轩窗，正梳妆。相顾无言，惟有泪千行。"——
这是**梦中叙事**：梦见亡妻正在窗前梳妆，两人相对无言，泪流满面。
梦境越温馨，醒来越凄凉——**以梦境之虚，写思念之实**。
:::

## 七、间接抒情（五）：借理抒情（理趣）⭐⭐

### 1. 定义

诗人**通过对人生、宇宙的哲理思考**来抒情，情在理中，理由情出。这是**宋诗/宋词**的重要特色。

### 2. 本站案例

| 诗歌 | 所借之理 | 抒之情 |
| --- | --- | --- |
| 《前赤壁赋》（记叙文组，可参照） | "变与不变" | 旷达超脱 |
| 《行路难》 | "长风破浪会有时"（信念） | 由悲转豪 |
| 《摸鱼儿》 | "千金纵买相如赋，脉脉此情谁诉" | 才不见用的郁愤 |

### 3. 理趣的特点

- **不是枯燥说理**，而是**情感驱动下的感悟**
- **情理交融**：有情的温度，有理的深度
- **宋人最长于此**（苏轼、辛弃疾）

::: warning 借理抒情 vs 议论文说理
- **议论文说理**：目的是"说服你"（逻辑论证）
- **诗歌借理抒情**：目的是"表达我的情感"（理为情服务）
同样是讲"变与不变"，《前赤壁赋》是**借景说理以抒情**，不是写议论文。
:::

## 九、抒情方法答题套路（核心考点）

### 1. 判断抒情方法

**提问方式**："这首诗是怎样抒情的？""请分析本诗的抒情方式。"

**答题步骤：**
```
① 判断方法：直接 / 借景 / 托物 / 用典 / 因事
② 结合诗句：举出具体句子（如"……句"）
③ 分析情感：抒发了什么情感
④ 点明效果：这样写有什么好处（含蓄 / 强烈 / 深沉）
```

### 2. 借景抒情题（最高频）

**答题模板：**
> 本诗运用了**借景抒情**（/寓情于景/情景交融）的手法，
> 诗人写了 XX 景（**举例句**，描绘画面），
> 营造了 XX 的氛围，
> 抒发了 XX 的情感，
> 达到了"**一切景语皆情语**"的效果，含蓄深沉。

### 3. 直抒胸臆题

**答题模板：**
> 本诗运用了**直抒胸臆**的手法，
> 诗人直接说出"XX"（**举原句**），
> 强烈抒发了 XX 的情感，
> 情感鲜明，感染力强。

### 4. 托物言志题

**答题模板：**
> 本诗运用了**托物言志**的手法，
> 诗人描写了 XX 之物（**物的特征**），
> 借此寄托了 XX 的志向/品格，
> 含蓄蕴藉，物我合一。

### 5. 以乐写哀（反衬）题（高频难点）

**答题模板：**
> 本诗运用了**以乐景写哀情**（反衬）的手法，
> 诗人写了 XX 的乐景（**温馨/美好的画面**），
> 反衬 XX 的哀情，
> "**以乐景写哀，一倍增其哀**"，使悲情更加深沉动人。

## 十、考点对接

### 常考题型

1. **判断抒情方法**（基础）—— 直接/间接
2. **借景抒情分析**（重点·高频）—— 景情关系
3. **以乐写哀**（难点·高频）—— 反衬
4. **托物言志**（重点）—— 咏物诗
5. **用典分析**（重点）—— 典故内容 + 用意
6. **比较两诗抒情异同**（论述题）—— 同题材不同方法

### 预测题 + 标准答案

#### 【预测题 1 · 简答】《江城子》是怎样抒发悼亡之情的？

> **标准答案：**
> 本词综合运用多种抒情方法：
> ① **直抒胸臆**："十年生死两茫茫，不思量，自难忘"——直接说出生死阻隔的悲痛。
> ② **因事抒情（梦境）**："夜来幽梦忽还乡，小轩窗，正梳妆"——借梦中相见叙事，**以梦境之虚写思念之实**。
> ③ **以乐景写哀情（反衬）**："小轩窗，正梳妆"是亡妻生前的温馨画面，**越温馨，越显今日的凄凉**——"以乐景写哀，一倍增其哀"。
> ④ **借景抒情**："明月夜，短松冈"——以孤坟冷景收束，余味深长。
>
> **效果**：多种方法交织，悼亡之情深沉动人，被誉为"悼亡词之冠"。

#### 【预测题 2 · 论述】分析"以乐景写哀情"的艺术效果。

> **标准答案：**
> "以乐景写哀情"是**反衬**手法的最高境界，王夫之《姜斋诗话》："以乐景写哀，以哀景写乐，一倍增其哀乐。"
> ① **反差原理**：美好的景物与悲伤的情感形成强烈反差，反差越大，悲情越突出。
> ② **以《江城子》为例**："小轩窗，正梳妆"——写亡妻生前梳妆的温馨画面。这画面越温馨，越衬托出"生死两茫茫"的凄凉——**因为温馨已永远不再**。
> ③ **比直接写哀更有力**：如果直接写"我很悲伤"，反而不感人；借温馨反衬，读者自己体会到落差，更加心痛。
> ④ **美学意义**：体现了中国诗"含蓄蕴藉"的美学——不直说，让读者自己感受。

#### 【预测题 3 · 简答】《登高》是如何做到"情景交融"的？

> **标准答案：**
> 《登高》被誉为"情景交融"的典范：
> ① **前两联写景**："风急天高猿啸哀，渚清沙白鸟飞回。无边落木萧萧下，不尽长江滚滚来。"——描绘萧瑟壮阔的秋景。
> ② **景中含情**：
>    - "猿啸哀"——听觉之悲
>    - "鸟飞回"——孤鸟盘旋，喻己漂泊
>    - "无边落木"——韶光易逝之悲
>    - "不尽长江"——滚滚东流，喻时光与愁绪无穷
> ③ **一切景语皆情语**：看似写景，实则字字含悲——秋之萧瑟 = 人之孤愁。
> ④ **后两联由景入情**："万里悲秋常作客……艰难苦恨繁霜鬓"——由景生情，直抒漂泊老病之悲。
>
> **效果**：景与情水乳交融，分不清是写景还是抒情，达到了"情景交融"的最高境界。

#### 【预测题 4 · 简答】《早雁》的"托物言志"体现在哪里？

> **标准答案：**
> 《早雁》运用**托物言志（托物寄兴）**：
> ① **所托之物**：大雁——"金河秋半虏弦开"，大雁遭弓射、被迫早飞、失群流浪。
> ② **所言之志**：以早雁比喻**边关遭战乱、流离失所的百姓**（唐武宗时回鹘侵扰）。
> ③ **关键句**：
>    - "莫厌潇湘少人处，水多菰米岸莓苔"——劝慰大雁（百姓）暂避南方，含深切关怀。
> ④ **效果**：不直接写战乱百姓，而借咏雁含蓄表达——**含蓄深沉，耐人寻味**，是咏物诗的典范。

#### 【预测题 5 · 比较】比较《声声慢》与《虞美人》写"愁"的异同。

> **标准答案：**
> **相同点**：
> ① 都写**愁**，都以**自然景物**衬托。
> ② 都有**直抒胸臆**的句子。
>
> **不同点**：
> | | 《声声慢》（李清照） | 《虞美人》（李煜） |
> | --- | --- | --- |
> | 愁的内容 | 国破家亡、晚年孤苦 | 亡国之痛、故国之思 |
> | 主要手法 | **借景抒情**（残秋黄花、梧桐细雨）+ 叠词 | **直抒 + 比喻**（"一江春水"喻愁） |
> | 愁的形象 | 细碎、绵密、无处不在（"寻寻觅觅"） | 滔滔不绝、汹涌澎湃（春水东流） |
> | 情调 | 凄婉细腻（婉约） | 沉痛奔放（亡国之君的悲壮） |
>
> **总结**：同是写愁，李清照的愁是**细密的哀婉**，李煜的愁是**汹涌的悲恸**——抒情方法不同，呈现的愁态迥异。

#### 【预测题 6 · 选择】下列对抒情方法的分析，不正确的一项是：
- A. 《登高》"无边落木萧萧下"是借景抒情 ✔
- B. 《江城子》"小轩窗，正梳妆"是以乐景写哀情 ✔
- C. 《早雁》全诗是直抒胸臆，直接表达对战乱的忧虑 ✅
- D. 《行路难》用姜太公钓鱼的典故是借古咏怀 ✔
>
> **答案：C**
> **讲评：** C 错。《早雁》是**托物言志**（借大雁喻百姓），不是直抒胸臆。全诗没有直接说"我忧虑战乱"，而是通过咏雁的遭射失群，含蓄表达——这正是托物言志"含蓄"的特点。

## 十一、知识图谱（总览）

```
                  诗歌的抒情方法
                       │
          ┌────────────┴────────────┐
          ▼                         ▼
      直接抒情                   间接抒情
   （直抒胸臆）                    │
          │              ┌────┬────┴───┬────┬────┐
     说出情感              ▼    ▼        ▼    ▼    ▼
          │            借景  托物     借古  因事  借理
     案例：              抒情  言志     咏怀  抒情  抒情
     虞美人(春水愁)       │    │        │    │    │
     一句话(咱们的中国)  正衬  咏物    用典  叙事  理趣
                       反衬  寄兴    怀古
                     (以乐写哀)        │
                       │               │
                    案例：           案例：
                    登高/声声慢      行路难/摸鱼儿
                    蒹葭/八声甘州
                       │
                   最普遍、最重要
```

::: tip 核心记忆五句话
1. **总纲**：直接抒情（直说）+ 间接抒情（借）
2. **借景抒情**：情景交融 / 正衬 / 反衬（以乐写哀）
3. **托物言志**：借物喻人（早雁/我愿意是急流）
4. **借古咏怀**：用典 / 怀古（行路难/摸鱼儿）
5. **答题**：判断方法 → 举诗句 → 析情感 → 说效果
:::

## 十二、迁移运用（鉴赏实战 checklist）

### 鉴赏一首诗的抒情时，问自己：

- [ ] 诗人抒发的是什么情感？（愁/喜/愤/慕/志）
- [ ] 是直接说出的，还是借外物表达的？
- [ ] 如果借景：景是乐景还是哀景？与情一致还是相反？
- [ ] 如果托物：物有什么特征？喻人的什么品格？
- [ ] 如果用典：典故讲什么？诗人借它说什么？
- [ ] 直抒的句子出现在哪里？（开头/结尾/高潮）
- [ ] 这种抒情方法有什么效果？（含蓄/强烈/深沉）

### 写作诗歌鉴赏时，套用：

```
【判断】本诗运用了 XX 抒情方法。
【举证】诗人写了"……"（原句），……（描绘画面/解释典故）。
【析情】借此抒发了 XX 的情感。
【效果】这样写 XX（含蓄深沉 / 强烈鲜明 / 反差动人）。
```
