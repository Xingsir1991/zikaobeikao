# 《马伶传》

> 侯方域（清初） · 文言人物传记 · 选自《壮悔堂文集》

::: tip 本篇定位
本文是清初散文家侯方域创作的文言人物传记名篇，记叙明代金陵演员马锦为在演技上超越对手，甘当三年门卒、以当朝权臣为"师"苦学严嵩神态，最终反败为胜的故事。文章表面写伶人，实则**借技艺之争暗刺权奸、讽刺官场**，是"以伶喻政"的寓言式传记，与司马迁《史记·优孟传》一脉相承，属大学语文重点文言篇目。
:::

---

## 一、知人论世

### 作者生平
**侯方域**（1618—1655），字**朝宗**，明末清初著名文学家，河南商丘人。出身世家，祖父侯执蒲、父亲侯恂均官至尚书。侯方域少年即以文章才华闻名，与**冒襄、陈贞慧、方以智**并称"**明末四公子**"。他是复社领袖人物，与秦淮名妓**李香君**的爱情故事被孔尚任写入传奇《**桃花扇**》，广为人知。

入清后，侯方域于顺治八年（1651）被迫应乡试，中副榜，心中悔恨，将其书斋命名为"**壮悔堂**"，文集亦名《壮悔堂文集》，表达痛悔降清之意。其散文**才气纵横、雄奇奔放**，上继司马迁、唐宋八大家，下开清初古文风气，与魏禧、汪琬并称"**清初散文三大家**"。

### 写作背景
本文所记虽托为"明之留都"金陵（南京）梨园事，实为**寓言式传记**。明末魏忠贤阉党与东林党的斗争、严嵩（分宜）等权奸的擅权，是士人心中挥之不去的政治阴影。侯方域借演员学"严嵩"演技的故事，**影射当朝权臣顾秉谦之流与严嵩本是一丘之貉**，言在此而意在彼，是典型的"借古讽今""指桑骂槐"。

> **关键背景知识**：
> - **严嵩**：明代权奸，江西分宜人，故文中称"分宜"，长期把持朝政、陷害忠良。
> - **顾秉谦**：明末昆山人，阉党首领之一，魏忠贤的死党。文中马伶"走京师为其门卒三年"，学的是顾秉谦的举止，却演出了严嵩的神态——作者借此暗示**顾秉谦与严嵩本质上无别**，皆为奸臣。
> - 《鸣凤记》：明代传奇，演杨继盛（椒山先生）等弹劾严嵩事，是抨击权奸的名剧。

### 文体定位
**文言人物传记**（记叙文的一种重要类型）。本文继承《史记》开创的**纪传体**传统，又有创新：

1. **以伶人（艺人）为传主**：打破了"为帝王将相立传"的传统，与《史记·滑稽列传》《优孟传》一脉相承，体现了"凡有一技之长、可传之事者皆可立传"的进步史观。
2. **寓议论于叙事**：文末"太史公曰"式的"侯方域曰"，仿《史记》论赞体，点明主旨。
3. **寓言性质**：以小故事寄寓大讽刺，是传记中的特殊品种。

## 二、文本梳理（逐段翻译+分析）

### 第一段：背景介绍——金陵梨园之盛

> **原文**：马伶者，金陵梨园部也。金陵为明之留都，社稷百官皆在，而又当太平盛时，人易为乐。其士女之问桃叶渡、游雨花台者，趾相错也。梨园以技鸣者，无虑数十辈，而其最著者二：曰兴化部，曰华林部。

**【译文】**马伶（马锦），是金陵（南京）梨园戏班的人。金陵是明朝的留都，宗庙社稷和百官都设在这里，又正值太平盛世，人们容易寻欢作乐。那些寻访桃叶渡、游览雨花台的男男女女，脚趾互相交错（极言人多）。梨园戏班中以技艺闻名的大约有几十个，而其中最著名的有两个：一个叫兴化部，一个叫华林部。

**【字词】**
| 字词 | 释义 | 类别 |
|------|------|------|
| 梨园部 | 戏班（唐玄宗设梨园教习歌舞，后为戏班代称） | 文化常识 |
| 留都 | 旧朝迁都后，在旧都仍保留的都城设置（明成祖迁都北京后，南京为留都） | 文化常识 |
| 社稷 | 本指土神和谷神，代指国家政权 | 文化常识 |
| 问桃叶渡 | 探访桃叶渡（金陵名胜，王献之送妾桃叶渡江处） | 典故 |
| 趾相错 | 脚趾相交错，形容人多拥挤 | 成语 |
| 无虑 | 大约、大概（"虑"通"略"，或谓"无庸虑及"） | 古今异义（今无此词） |
| 辈 | 同类的人（此处为"批、伙"义） | 量词活用 |

**【叙事手法】**：本段为**环境铺垫（背景介绍）**。先点金陵繁华、梨园兴盛，为后文两大戏班的对决提供舞台。**"最著者二"**一句为下文伏笔，引出核心冲突。

**【逻辑作用】**：交代故事发生的时代、地点、环境，奠定"繁华—角逐"的基调。

### 第二段：第一回合——马伶败走（故事开端）

> **原文**：一日，新安贾合两部为大会，遍征金陵之贵客文人，与夫妖姬静女，莫不毕集。列兴化于东肆，华林于西肆，两肆皆奏《鸣凤》，所谓椒山先生者。迨半奏，引商刻羽，抗坠疾徐，并称善也。当两相国论河套，而西肆之为严嵩相国者曰李伶，东肆则马伶。坐客乃西顾而叹，或大呼命酒，或移座更近之，首不复东。未几更进，则东肆不复能终曲。询其故，盖马伶耻出李伶下，已易衣遁矣。马伶者，金陵之善歌者也。既去，而兴化部又不肯辄以易之，乃竟辍其技不奏，而华林部独著。

**【译文】**一天，一位新安（徽州）商人把两个戏班合在一起举办盛大的演出，遍请金陵的贵客文人，还有妖艳的姬妾和娴静的女子，全都到齐了。安排兴化部在东边的戏场，华林部在西边的戏场，两个戏场都上演《鸣凤记》——就是演椒山先生（杨继盛）那部戏。演到一半时，他们引商刻羽（讲究音律），高低快慢都十分和谐，两边都说好。当演到两个相国争论河套事的时候，西边戏场扮演严嵩相国的是李伶，东边戏场扮演严嵩的是马伶。观众们于是都回头向西看并赞叹，有人大喊着叫上酒，有人挪动座位离得更近，头再也不转向东边。没多久继续演下去，东边的戏场就不能把曲子演完了。询问原因，原来是马伶以技艺不如李伶为耻，已经换了衣服逃走了。马伶，是金陵擅长歌唱的人。他离开后，兴化部又不肯随便找人替代他，竟然停止了他们的演出，于是华林部一家独秀。

**【字词】**
| 字词 | 释义 | 类别 |
|------|------|------|
| 新安贾（gǔ） | 徽州商人（新安为徽州古称，贾：商人） | 文化常识/通假 |
| 妖姬静女 | 妖艳的姬妾和娴静的女子（"静女"用《诗经》典故） | 典故 |
| 肆 | 集市、店铺，此指戏场（戏班子搭台演出处） | 古今异义 |
| 《鸣凤》 | 《鸣凤记》，传奇剧本，演杨继盛等弹劾严嵩事 | 文化常识 |
| 椒山先生 | 杨继盛（号椒山），明嘉靖名臣，因弹劾严嵩被害 | 文化常识 |
| 迨（dài） | 等到、到了 | 文言虚词 |
| 引商刻羽 | 指讲究音律、歌唱精妙（商、羽为五音之二） | 成语 |
| 抗坠疾徐 | 高低快慢（抗：高亢；坠：低沉；疾：快；徐：慢） | 词组 |
| 论河套 | 严嵩与夏言争论是否收复河套地区（明代政事） | 历史典故 |
| 易衣遁 | 换了衣服逃走（遁：逃） | 词组 |
| 辄（zhé） | 就、总是、随便 | 文言虚词 |
| 辍（chuò） | 停止 | 文言实词 |

**【叙事手法】**：
- ⭐⭐⭐ **同台竞演、对比描写**：两部同演一剧、同扮一角色（严嵩），观众反应（"西顾而叹""首不复东"）形成鲜明对比，胜负立判。
- ⭐⭐⭐ **侧面烘托**：用观众反应（"大呼命酒""移座更近"）烘托李伶技艺之高，比直接描写更有说服力。
- ⭐⭐ **细节描写**：马伶"耻出李伶下，已易衣遁矣"——一个"耻"字、一个"遁"字，写出马伶的自尊心与不甘，为下文苦学伏笔。

**【逻辑作用】**：设置核心冲突（马伶败于李伶），埋下"耻—学—胜"的因果线索，是故事的起因。

### 第三段：第二回合——马伶大胜（故事高潮）

> **原文**：去后且三年而马伶归，遍告其故侣，请于新安贾曰："今日幸为开宴，招前日宾客，愿与华林部更奏《鸣凤》，奉一日欢。"既奏，已而论河套，马伶复为严嵩相国以出，李伶忽失声，匍匐前称弟子。兴化部是日遂凌出华林部远甚。其夜，华林部过马伶："子，天下之善技也，然无以易李伶。李伶之为严相国至矣，子又安从授之而掩其上哉？"马伶曰："固然，天下无以易李伶；李伶即又不肯授我。我闻今相国昆山顾秉谦者，严相国俦也。我走京师，求为其门卒三年，日侍昆山相国于朝房，察其举止，聆其语言，久乃得之。此吾之所为师也。"华林部相与罗拜而去。

**【译文】**马伶离开后将近三年才回来，他遍告旧日的同伴，向新安商人请求说："今天希望您开设宴席，邀请上次那些宾客，我愿意和华林部再演一次《鸣凤记》，奉上一日的欢乐。"演出开始，演到争论河套的情节时，马伶再次扮演严嵩相国出场，李伶忽然不能出声（惊叹佩服），伏在地上爬向前自称弟子。兴化部这一天就远远超过了华林部。当晚，华林部的人拜访马伶（询问）："您是天下擅长技艺的人，但是没有人能取代李伶。李伶扮演严嵩相国已经达到极致了，您又从哪里学到的而能超过他呢？"马伶说："确实，天下没有人能取代李伶；李伶又不肯教我。我听说当今相国昆山人顾秉谦，和严嵩是同类人。我跑到京城，请求做他门下的差役三年，每天在朝房里侍奉昆山相国，观察他的举止，聆听他的言语，很久以后才掌握了。这就是我的老师。"华林部的人于是纷纷向他罗列行礼（叩拜）后离去。

**【字词】**
| 字词 | 释义 | 类别 |
|------|------|------|
| 且 | 将近、将要（"且三年"=将近三年） | 文言虚词 |
| 更（gēng）奏 | 再次演奏/演出 | 多音多义 |
| 失声 | （因极度惊叹而）失去常态地叫不出声 | 词义 |
| 匍匐前 | 伏在地上爬上前（表示极端敬服） | 动作 |
| 凌出 | 超过、胜过（凌：超越；出：超出） | 实词 |
| 过 | 拜访（古今异义，今多指经过、过错） | **古今异义** |
| 子 | 您（对男子的尊称） | 敬辞 |
| 无以易 | 没有什么可以替代/胜过（易：替代、交换） | 实词 |
| 至矣 | 到了极点（至：极点） | 实词 |
| 安从 | 从哪里（安：哪里，疑问代词） | 文言虚词 |
| 掩其上 | 超过他、盖过他（掩：盖过） | 实词 |
| 俦（chóu） | 同类、同辈（"严相国俦"=严嵩一类的人） | 实词（**关键讽刺词**） |
| 门卒 | 门下的差役、仆从 | 词组 |
| 朝房 | 百官上朝前候见休息的房屋 | 文化常识 |
| 聆（líng） | 倾听 | 实词 |
| 罗拜 | 环绕着行礼叩拜（罗：环绕排列） | 动作 |

**【叙事手法】**：
- ⭐⭐⭐ **倒叙揭秘**：先写"归"与"大胜"的结果，再借马伶自述揭秘三年苦学的经过，造成悬念又豁然开朗，是全文最精彩的叙事设计。
- ⭐⭐⭐ **细节描写**："李伶忽失声，匍匐前称弟子"——以李伶（原胜者）的极端反应反衬马伶技艺之精，胜过千言赞美。
- ⭐⭐⭐ **画龙点睛的议论**：马伶一句"顾秉谦者，严相国俦也"是全篇的**文眼与讽刺核心**——点明学的是顾秉谦，演的是严嵩，二者本是一类。这是"以伶喻政"的关键一笔。
- ⭐⭐ **首尾呼应的场面**：同样的对手、同样的剧目、同样的观众，前后两场演出形成强烈对照，结构严谨。

**【逻辑作用】**：揭示马伶取胜的秘诀——以顾秉谦为"师"；同时点出全篇的讽刺主旨。

### 第四段：补充——马伶的身世

> **原文**：马伶，名锦，字云将，其先西域人，当时犹称马回回云。

**【译文】**马伶，名叫锦，字云将，他的祖先是西域人，当时还称他们为"马回回"。

**【字词】**
| 字词 | 释义 | 类别 |
|------|------|------|
| 其先 | 他的祖先（先：祖先、前辈） | 实词 |
| 西域 | 玉门关以西的中亚地区 | 地理常识 |
| 回回 | 旧时对来自西域信奉伊斯兰教民族的称呼（马姓多出自回回） | 文化常识 |

**【叙事手法】**：补充交代传主的名、字、籍贯，符合传记体"叙其乡里族系"的体例（仿《史记》体例）。马伶为西域回族人，亦可暗示其演艺天赋有异域血脉。

**【逻辑作用】**：完善传记体例，补充人物背景。

### 第五段：作者论赞——点明主旨

> **原文**：侯方域曰：异哉，马伶之自得师也。夫其以李伶为绝技，无所干求，乃走事昆山，见昆山犹之见分宜也；以分宜教分宜，安得不工哉？（呜乎！耻其技之不若，而去数千里为卒三年，倘三年犹不得，即犹不归耳。其志如此，技之工又须问耶？）

**【译文】**侯方域说：奇异啊，马伶自己找到老师的方法。他把李伶当作绝技（不可超越），没有地方去求教，于是跑去侍奉昆山（顾秉谦），见到昆山（顾秉谦）就像见到了分宜（严嵩）一样；用分宜（顾秉谦这一类的"分宜"）来教（演绎）分宜（严嵩），怎么能不精妙呢？（唉！他以技艺不如人为耻，而跑到几千里外做了三年差役，如果三年还学不到，就索性不回来了。他的志向如此，技艺的精妙还需要问吗？）

**【字词】**
| 字词 | 释义 | 类别 |
|------|------|------|
| 异哉 | 奇异啊（"异"为形容词作谓语前置，感叹句） | 句式 |
| 自得师 | 自己求得老师 | 词组 |
| 绝技 | 无法企及的绝顶技艺 | 实词 |
| 干（gān）求 | 求取、求教（干：求取） | 实词 |
| 乃 | 于是、就 | 文言虚词 |
| 走事 | 跑去侍奉（走：前往；事：侍奉） | 词组 |
| 分宜 | 严嵩（江西分宜人，以籍贯代指） | **借代（关键讽刺）** |
| 工 | 精妙、精巧（形容词，指技艺高超） | 实词（"工"作"精巧"义） |
| 不若 | 不如、比不上（若：如、及） | 实词 |
| 倘 | 假使、如果 | 文言虚词 |

**【叙事手法/议论手法】**：
- ⭐⭐⭐ **论赞体（仿《史记》"太史公曰"）**：文末以作者直接议论收束，是史传体的标准格式，表明本文是史传性质的传记。
- ⭐⭐⭐ **一语双关、点破寓意**："**见昆山犹之见分宜也；以分宜教分宜，安得不工哉？**"——这是全文最锋利的讽刺：顾秉谦（昆山）和严嵩（分宜）本是一类奸臣，所以用活着的奸臣学死了的奸臣，自然惟妙惟肖。明面上夸马伶，暗地里骂当朝权臣，是**指桑骂槐、借古讽今**的极致。
- ⭐⭐ **双重评价**：先以"安得不工哉"论其技，再以"其志如此"赞其志，从"技"和"志"两方面立论，全面肯定马伶。

**【逻辑作用】**：点明主旨，完成从"叙事"到"议论"的过渡，揭示全篇的讽刺寓意。

## 三、叙事手法鉴赏（记叙文核心考点）

### 1. ⭐⭐⭐ 结构艺术：倒叙手法，悬念丛生

本文最突出的结构特色是**倒叙**：

```
按时间顺序本应是：
败走（金陵）→ 苦学（京城三年）→ 归来大胜（金陵）

实际写作顺序（倒叙）：
① 败走 → ②【略去三年苦学】→ ③ 归来大胜（先写结果）→ ④ 借马伶自述揭秘"三年苦学"
```

**倒叙的效果**：
1. **制造悬念**：读者好奇马伶败走后三年去了哪里、做了什么，为何能反败为胜。
2. **突出重点**：把最精彩的"揭秘"放在高潮，避免平铺直叙。
3. **凸显讽刺**：让"以顾秉谦为师演严嵩"这一关键情节在最后亮相，使讽刺效果更强烈、更震撼。
4. **结构紧凑**：避免冗长的过程叙述，使文章短小精悍、引人入胜。

---

### 2. ⭐⭐⭐ 对比手法：贯穿全篇的双线对照

| 对比维度 | 第一回合 | 第二回合 |
|----------|----------|----------|
| 马伶扮演严嵩 | 失败、败走 | 大胜、惟妙惟肖 |
| 李伶反应 | 观众追捧"西顾而叹" | "失声，匍匐前称弟子" |
| 兴化部地位 | "辍其技不奏" | "凌出华林部远甚" |
| 观众反应 | "首不复东" | 反向西拜服 |

**对比的效果**：前后两场同题竞演的对比，鲜明地表现了马伶三年苦学的巨大进步，使"勤学苦练、精益求精"的主题更加突出。

---

### 3. ⭐⭐⭐ 人物塑造：正面描写与侧面烘托结合

#### （1）马伶（正面+侧面）

**正面描写**：
- **心理（耻）**："马伶耻出李伶下"——一个"耻"字写出其强烈的自尊与好胜。
- **行动（学）**："走京师，求为其门卒三年，日侍昆山相国于朝房，察其举止，聆其语言"——"走、求、侍、察、聆"五个动词，写尽三年苦学的执着。
- **语言（述）**：自述学艺经过，冷静、坦诚、自信。
- **志向（决）**："倘三年犹不得，即犹不归耳"——破釜沉舟的决心。

**侧面烘托**：
- 借观众反应（首不复东）写其初败之惨。
- 借李伶"失声匍匐称弟子"写其再胜之精。
- 借华林部"罗拜而去"写其技艺之服众。

#### （2）李伶（侧面为主）

通过观众"西顾而叹""大呼命酒"的狂热反应，烘托李伶技艺之高，为马伶的失败做铺垫，也为后文的反败为胜设置难度（增强故事张力）。

---

### 4. ⭐⭐⭐ 细节描写：一字传神

| 细节 | 字眼 | 传神之处 |
|------|------|----------|
| 马伶败走 | "**耻**"出李伶下 | 强烈的自尊、好胜 |
| 马伶败走 | "**易衣遁**" | 无颜见人、仓皇离开 |
| 三年苦学 | "**察**其举止，**聆**其语言" | 用心观察、潜心揣摩 |
| 李伶折服 | "**失声**""**匍匐前**" | 极端惊叹、心服口服 |
| 华林部致敬 | "**罗拜**而去" | 集体折服 |

> **特别提示**："**耻**"字是马伶形象的核心，"耻"驱动了"遁"，驱动了"走京师"，驱动了"为卒三年"——一个"耻"字贯穿全篇人物行动逻辑，是**一字立骨**的典范。

---

### 5. ⭐⭐⭐ 寓言式讽刺（以伶喻政）：全文最深刻的艺术

本文表面写伶人技艺之争，**实则讽刺当朝权臣**：

| 表层（明） | 深层（暗） |
|------------|------------|
| 马伶学演严嵩 | 影射**奸臣可被模仿、可被复制** |
| "顾秉谦者，严相国俦也" | 当朝相国顾秉谦=严嵩，**皆是奸臣** |
| "以分宜教分宜" | 用活的"分宜"（顾秉谦）教死的"分宜"（严嵩），讽刺现实比历史更丑恶 |
| 马伶之"志"可赞 | 反衬权臣之"奸"可恨——**伶人尚知耻精进，权臣却祸国殃民** |

**这种"借古讽今、指桑骂槐"的手法**，与司马迁《史记·优孟传》、柳宗元《种树郭橐驼传》同属寓言式讽刺，但因直接点名"顾秉谦"，讽刺更为大胆犀利。

---

::: warning 记叙文"叙事手法鉴赏"三段式答题模板
**第一步（指法）**：指出手法名称（如"倒叙""对比""侧面烘托""一字立骨""寓言讽刺"等）。
**第二步（析例）**：结合原文具体语句分析该手法如何运用。
**第三步（明效）**：说明该手法在情节、人物、主题方面的作用。

例：本文运用了**倒叙手法**（指法）。文章先写马伶败走，再写三年后归来大胜，最后借马伶自述揭秘其"走京师为门卒三年"的苦学经过，打乱了事件的时间顺序（析例）。这种写法制造了强烈的悬念，避免了平铺直叙；同时把"以顾秉谦为师"的揭秘放在高潮，使讽刺效果更震撼，结构也更紧凑精巧（明效）。
:::

## 五、主旨情感

### 中心思想（双重主旨）

**明面主旨（叙事层面）**：通过马伶败而不馁、甘当三年门卒、以现实人物为师、精益求精终于反败为胜的故事，赞美了**勤学苦练、不耻下问、精益求精**的敬业精神与坚韧意志。这一点具有普遍的人生启迪意义。

**暗面主旨（讽刺层面，更重要）**：借马伶"以顾秉谦为师演严嵩"的故事，**讽刺当朝权臣顾秉谦之流与历史上的奸臣严嵩本是一丘之貉**——现实比历史更丑恶，活的奸臣比死的奸臣更像奸臣。这是对明末阉党专权的深刻批判，体现了作者的政治立场与文人风骨。

### 思想内涵
1. **敬业精神**：马伶"为卒三年""倘三年犹不得，即犹不归耳"的执着，是任何行业取得成就的根本。
2. **生活是艺术的源泉**：马伶不向同行学，而向生活学——到现实中去观察"严嵩一类人"的举止语言，说明艺术源于生活、高于生活。
3. **政治讽刺**：以伶人喻政客，以技艺喻权术，揭露权奸的丑恶本质。
4. **史学价值**：以伶人立传，体现"凡有一技可传者皆可立传"的进步史观，丰富了中国传记文学的题材。

### 情感态度
- **赞**：赞马伶的耻、勤、志、慧。
- **叹**：叹"安得不工哉"的精妙讽刺。
- **讽**：讽顾秉谦、严嵩之流的奸臣嘴脸。
- **敬**：敬业、求知、自强不息的人文精神。

## 六、考点对接

### 常考题型
1. 翻译重点语句（"见昆山犹之见分宜也；以分宜教分宜，安得不工哉"等）。
2. 分析倒叙手法的运用及作用。
3. 理解"以分宜教分宜"的讽刺寓意。
4. 分析人物塑造手法（正面描写+侧面烘托）。
5. 比较本文与《史记》传记体例的继承关系。

---

### 预测题及标准答案

**【预测题1·翻译题】**
将下列句子翻译成现代汉语。

（1）坐客乃西顾而叹，或大呼命酒，或移座更近之，首不复东。
（2）我闻今相国昆山顾秉谦者，严相国俦也。
（3）见昆山犹之见分宜也；以分宜教分宜，安得不工哉？

**【参考答案】**
（1）观众们于是都回头向西看并赞叹，有人大喊着叫上酒，有人挪动座位离得更近，头再也不转向东边。
（2）我听说当今的相国昆山人顾秉谦，和严嵩相国是同类人。
（3）见到昆山（顾秉谦）就像见到了分宜（严嵩）一样；用分宜（顾秉谦）来教（演绎）分宜（严嵩），怎么能不精妙呢？

---

**【预测题2·简答题】**
本文在叙事结构上最大的特色是什么？这种结构有何艺术效果？

**【参考答案】** 本文最大的结构特色是**倒叙**。
（1）文章先写马伶第一回合败于李伶、"易衣遁"离开金陵；接着跳过三年，直接写马伶归来与华林部再演《鸣凤》、大获全胜；最后借马伶自述，才揭秘这三年"走京师为门卒、师法顾秉谦"的苦学经过。
（2）**艺术效果**：①制造悬念，激发读者好奇心（马伶三年去了哪里？为何能反败为胜？）；②突出高潮，把"以顾秉谦为师演严嵩"这一最具讽刺性的揭秘放在最后，使讽刺效果震撼有力；③避免平铺直叙，使短小的文章波澜起伏、引人入胜；④结构紧凑，张弛有度。

---

**【预测题3·论述题】**
如何理解"见昆山犹之见分宜也；以分宜教分宜，安得不工哉？"这句话的深刻含意？它体现了本文怎样的写作特色？

**【参考答案】**
（1）**字面含意**：见到昆山相国顾秉谦，就像见到分宜人严嵩一样；用"活的分宜"（顾秉谦）来教马伶演绎"死的分宜"（严嵩），怎么可能不精妙呢？
（2）**深层讽刺**：这是全篇的文眼。作者借马伶之口暗示——**当朝相国顾秉谦与历史奸臣严嵩本是同类（"俦"）**，二者在举止、神态、奸诈本质上如出一辙。马伶用顾秉谦为原型去演严嵩，自然惟妙惟肖。这就把矛头直指明末阉党专权，讽刺现实比历史更丑恶。
（3）**写作特色——寓言式讽刺（借古讽今、指桑骂槐）**：本文表面是伶人技艺之争的传记，实则借题发挥、影射政治。这种"以伶喻政"的手法，继承了《史记·优孟传》的传统，但因直接点名当朝权臣，讽刺更为大胆犀利，是本文最深刻、最独特的艺术成就。

---

**【预测题4·分析题】**
本文在塑造马伶形象时，综合运用了多种手法。请结合文本具体分析。

**【参考答案】**
（1）**正面描写**：①心理——"耻出李伶下"，一个"耻"字写出其强烈的自尊与好胜，是行动的根本驱动力；②行动——"走京师，求为其门卒三年，日侍……察其举止，聆其语言"，五个动词写尽苦学；③语言——自述学艺经过，冷静坦诚；④志向——"倘三年犹不得，即犹不归耳"，破釜沉舟。
（2）**侧面烘托**：①借观众"首不复东"烘托其初败之惨；②借李伶"失声、匍匐前称弟子"烘托其再胜之精；③借华林部"罗拜而去"烘托其技艺之服众。
（3）**对比手法**：前后两场竞演、马伶与李伶的对比，使马伶由败到胜的进步更加鲜明。
（4）**作者论赞**：文末"侯方域曰"从"技"和"志"两方面直接评价，使人物形象更加完整、立体。
通过这些手法的综合运用，马伶勤学苦练、耻于落后、精益求精的敬业者形象跃然纸上。

---

**【预测题5·选择题】**
下列对本文的理解和分析，不正确的一项是（  ）

A. 文章以"马伶者，金陵梨园部也"开篇，仿《史记》纪传体例，点明传主身份。
B. "华林部过马伶"中的"过"是"拜访"之意，与今义"经过"不同，属古今异义。
C. 文章写马伶"走京师，求为其门卒三年"，主要表现其不耻下问、勤学苦练的精神，与政治讽刺无关。
D. 文末"侯方域曰"的论赞体，仿《史记》"太史公曰"，是史传体的标志。

**【参考答案】** C
**解析**：C项错误。马伶"走京师为门卒三年"固然表现了勤学苦练的精神，但其师法对象是当朝权臣顾秉谦（"严相国俦"），这就使技艺学习带上了"以伶喻政"的讽刺意味——用活的奸臣学死的奸臣，揭示现实政治的丑恶。因此说"与政治讽刺无关"是错误的。

**核心记忆三句话**：
1. **"倒叙见巧，悬念揭谜"**——本文以倒叙结构，先写败走与大胜，最后借马伶自述揭秘"走京师为门卒三年、师法顾秉谦"的经过，制造悬念、突出讽刺。
2. **"以分宜教分宜，借伶而喻政"**——全篇文眼是"见昆山犹之见分宜也"，借马伶学演严嵩的故事，讽刺当朝权臣顾秉谦与严嵩本是一丘之貉，是"借古讽今"的寓言式传记典范。
3. **"耻字立骨，勤学求精"**——马伶以"耻"驱动"遁、学、胜"，三年苦学、精益求精，体现敬业精神与生活是艺术源泉的真理，是史传中少见的以伶人为主角的励志篇章。

## 附录：原文

> 马伶者，金陵梨园部也。金陵为明之留都，社稷百官皆在，而又当太平盛时，人易为乐。其士女之问桃叶渡、游雨花台者，趾相错也。梨园以技鸣者，无虑数十辈，而其最著者二：曰兴化部，曰华林部。
>
> 一日，新安贾合两部为大会，遍征金陵之贵客文人，与夫妖姬静女，莫不毕集。列兴化于东肆，华林于西肆，两肆皆奏《鸣凤》，所谓椒山先生者。迨半奏，引商刻羽，抗坠疾徐，并称善也。当两相国论河套，而西肆之为严嵩相国者曰李伶，东肆则马伶。坐客乃西顾而叹，或大呼命酒，或移座更近之，首不复东。未几更进，则东肆不复能终曲。询其故，盖马伶耻出李伶下，已易衣遁矣。马伶者，金陵之善歌者也。既去，而兴化部又不肯辄以易之，乃竟辍其技不奏，而华林部独著。
>
> 去后且三年而马伶归，遍告其故侣，请于新安贾曰："今日幸为开宴，招前日宾客，愿与华林部更奏《鸣凤》，奉一日欢。"既奏，已而论河套，马伶复为严嵩相国以出，李伶忽失声，匍匐前称弟子。兴化部是日遂凌出华林部远甚。其夜，华林部过马伶："子，天下之善技也，然无以易李伶。李伶之为严相国至矣，子又安从授之而掩其上哉？"马伶曰："固然，天下无以易李伶；李伶即又不肯授我。我闻今相国昆山顾秉谦者，严相国俦也。我走京师，求为其门卒三年，日侍昆山相国于朝房，察其举止，聆其语言，久乃得之。此吾之所为师也。"华林部相与罗拜而去。
>
> 马伶，名锦，字云将，其先西域人，当时犹称马回回云。
>
> 侯方域曰：异哉，马伶之自得师也。夫其以李伶为绝技，无所干求，乃走事昆山，见昆山犹之见分宜也；以分宜教分宜，安得不工哉？（呜乎！耻其技之不若，而去数千里为卒三年，倘三年犹不得，即犹不归耳。其志如此，技之工又须问耶？）
