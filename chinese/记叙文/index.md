# 📜 记叙文

> 史传 / 传记 / 寓言 / 游记 / 抒情散文 / 文赋，共 **18 篇**精读课文。

::: tip 记叙文考点
- **六要素**：时间、地点、人物、起因、经过、结果
- **表达方式**：记叙、描写、抒情、议论的综合运用
