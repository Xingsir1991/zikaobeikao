# 《爱尔克的灯光》

> 巴金（现代） · 抒情叙事散文 · 选自散文集《龙·虎·狗》

::: tip 本篇定位
本文是巴金1941年初回成都老家后写成的抒情叙事散文名篇。"爱尔克的灯光"本是欧洲一个姐姐为航海弟弟点灯守候的悲情传说，作者借这一典故为题，以"灯光"为核心意象，通过对十八年后故园回望的所见所思，深沉悼念寂寞死去的姐姐，并有力批判"长宜子孙"的封建家庭观念，表达了对"狭小的家"的诀别与对"广大的世界"的坚定追求。它与《都江堰》《纪念傅雷》并列为现当代记叙散文的"标配三件套"，是象征手法与抒情艺术的典范。
:::

## 一、知人论世

### 作者其人——巴金

**巴金**（1904—2005），原名李尧棠，字芾甘，四川成都人，现代著名小说家、散文家、翻译家，20世纪中国文学巨匠之一。2005年逝世，寿至101岁，被誉为"二十世纪中国文学的良心"。

**笔名由来**：巴金晚年自述，"巴"来自留法时的一位自杀身亡的同学巴恩波，"金"是一位学哲学的同学詹剑峰为他随手取自克鲁泡特金（Kropotkin）。

**代表作品**：
- 长篇小说"激流三部曲"《家》《春》《秋》（代表作，《家》影响尤巨）
- "爱情三部曲"《雾》《雨》《电》
- "抗战三部曲"《火》
- 中篇小说《憩园》《第四病室》《寒夜》
- 散文集《海行杂记》《忆》《龙·虎·狗》《随想录》（晚年力作，被誉为"说真话的书"）
- 译作《往事与随想》（赫尔岑）、《处女地》（屠格涅夫）等

### 写作背景

1923年春，19岁的巴金冲破封建大家庭的束缚，离开成都李家公馆，奔赴上海、南京求学，后赴法国。此后十八年间，他接受了新思想，投身文学与进步事业，而家中的姐姐李尧桢等亲人则在旧式婚姻与封建家庭中憔悴而死。

1941年初，离家十八年的巴金第一次回到成都老家。故园面目全非：李家公馆早已易主，太平缸与石狮子不见了，照壁上祖父留下的"长宜子孙"四个字却依然如故；而当年送他出嫁才一个多月、答应他"回来看她"的姐姐，早已寂寞死去。巨大的悲恸与清醒的反思促使他写下本文，同年3月作于成都，后收入散文集《龙·虎·狗》。

### 题目典故——爱尔克的灯光

"爱尔克"（又译爱尔克、埃尔德）是流传于北欧/德国的一个古老传说：哈立希岛上的姐姐爱尔克，每夜在窗前点亮灯笼，为航海远行的兄弟照路，日复一日、年复一年，直到死去也未能等到兄弟归来，最终带着失望进入坟墓。这一悲情传说象征着"姐姐对兄弟无私却绝望的爱与等待"。巴金借用它，既照应自己与姐姐的悲剧，又以"灯"贯串全文，赋予其多重心象意义。

### 文体定位

本文属于**记叙文中的"抒情叙事散文"（抒情与叙事交融）**。其特点：
- 以回乡见闻为叙事线索（叙事性）
- 以灯光为核心意象层层渲染（象征性）
- 以悼亡、批判、追求为旨归（抒情性、思想性）

它是"叙事—抒情—议论"水乳交融的复合散文，叙事是基础，抒情是底色，议论（对"长宜子孙"的批判）是锋芒，象征（三种灯光）是灵魂。

## 二、文本梳理（按回乡见闻与情感层次分段）

全文以"傍晚回望—联想往事—黑暗见灯—议论批判—再次出走"为序，可分五个层次。

### 第一部分（第1—2段）：傍晚回望故园，照壁"长宜子孙"

**原文节选**：
> 傍晚，我靠着逐渐黯淡的最后的阳光的指引，走过十八年前的故居……照壁上"长宜子孙"四个字却是原样地嵌在那里，似乎连颜色也不曾被风雨剥蚀……我仿佛要在这里寻找十八年以前的辽远的旧梦。

> 街道仍是静静的。忽然一个熟悉的声音在我耳边轻轻地唱起了这个欧洲的古传说……这时候我想起了自己的事情。

**大意概述**：文章以"傍晚"的回望开篇。十八年未归的"我"走过故居，街市面目全非——太平缸、石狮子俱已不见，巍峨门墙代替了旧物；唯独照壁上祖父留下的"长宜子孙"四个字"原样地嵌在那里，似乎连颜色也不曾被风雨剥蚀"。这不变的四个字，瞬间攫住"我"的心，引出对十八年前旧梦的追寻。接着写街道静寂中"一个熟悉的声音"唱起欧洲古传说，自然过渡到对自己的事情的联想——这是全文叙事与抒情的转捩点，也悄然引出"爱尔克的灯"的典故。

| 字词 | 释义 |
|------|------|
| 故居 | 旧时的居所 |
| 巍峨 | 形容高大雄伟 |
| 太平缸 | 旧式大家族门前储水防火的大缸 |
| 照壁 | 大门外或门内挡视线、作装饰的墙壁 |
| 长宜子孙 | 长久地适宜、造福子孙（封建家训套语） |
| 剥蚀 | 风雨侵蚀脱落 |

**叙事手法**：⭐⭐⭐ **以景起兴、寓情于景**（"逐渐黯淡的阳光"暗示心境）；⭐⭐⭐ **细节特写与对比**（变了的"门墙/石狮子" vs 不变的"长宜子孙"，物是人非，触目惊心）；⭐⭐ **承上启下的过渡段**（古传说引出下文姐姐之事）。

**逻辑作用**：交代背景、奠定怅惘基调、引出核心意象"长宜子孙"与"爱尔克的灯"。

---

### 第二部分（第3—5段）：联想姐姐悲剧与十八年浪费

**原文节选**：
> 十八年前在一个春天的早晨……我也曾有一个姊姊，也曾答应过有一天回来看她……那时我的姊姊还是一个出阁才只一个多月的新嫁娘。

> 然而人的安排终于被"偶然"毁坏了……我离家不过一年半光景，就接到了姊姊的死讯……她寂寞地活着，寂寞地死去。死带走了她的一切，这就是在我们那地方的旧式女子的命运。

> 十九年，似乎一切全变了，又似乎都没有改变……浪费，浪费，还是那许多不必要的浪费——生命，精力，感情，财富，甚至欢笑和眼泪。

**大意概述**：由古传说自然联想到自己的姐姐——十八年前离家时，姐姐是出嫁才一个多月的新嫁娘，"我"曾答应有一天回来看她。然而"人的安排终于被'偶然'毁坏了"，离家仅一年半，"我"就接到姐姐的死讯：哥哥颤抖的笔写下她"悲惨的结局"和"死后的冷落"，所谓"温良"的丈夫则堕落至吸鸦片。姐姐"寂寞地活着，寂寞地死去"，这正是旧式女子的命运。由此，作者激愤地感叹十九年"似乎全变了，又似乎都没有改变"——生命、精力、感情、财富甚至欢笑眼泪，都在这个封建家庭的小圈子里被白白"浪费"。

| 字词 | 释义 |
|------|------|
| 出阁 | 女子出嫁 |
| 新嫁娘 | 新婚的女子 |
| 温良 | 温和善良（此处带反讽） |
| 寂寞 | 孤单冷落 |
| 憔悴 | 形容人瘦弱面色不好 |
| 装束 | 穿着打扮 |

**叙事手法**：⭐⭐⭐ **今昔对比/插叙**（十八年前的姐姐 vs 今日的死讯，以时间为经，以悲剧为纬）；⭐⭐⭐ **典型化概括**（"这就是在我们那地方的旧式女子的命运"，把个人悲剧上升为一代女性的命运）；⭐⭐ **反复（反复修辞）**（"寂寞地活着，寂寞地死去"；"浪费，浪费，还是那许多……浪费"），强化悲愤；⭐⭐ **反语/讽刺**（"所谓温良的人"）。

**逻辑作用**：把个人悲剧与封建家庭的罪恶联系起来，为下文对"长宜子孙"的批判蓄势。

---

### 第三部分（第6—8段）：黑暗中三见灯光，核心意象展开

**原文节选**：
> 黑暗来了……于是大门内亮起灯光。灯光并不曾照亮什么，反而在我心上添加了黑暗……我仿佛看见了哈立希岛上的灯光。那应该是姊姊爱尔克点的灯笼……她一直到死都在等待那个出远门的兄弟回来。最后她带着失望进入坟墓。

> 守门的武装兵士用疑惑的眼光看我……他却用眼光驱逐一个人的许多亲密的回忆。

**大意概述**：天黑后，故居大门内亮起的灯光（**第一种灯**），非但没照亮什么，反而"在我心上添加了黑暗"——这是令人失望的、属于旧家的灯。它使"我"联想到哈立希岛上爱尔克的灯（**第二种灯，即题目的灯**）：姐姐每夜为航海兄弟点灯照路，至死等待，终带着失望进入坟墓——这正是自己姐姐寂寞而死命运的写照。写完故居之灯与爱尔克之灯，作者又插入"守门的武装兵士用疑惑的眼光看我"一笔——兵士的眼光"驱逐了一个人的许多亲密的回忆"，暗示旧家如今连凭吊都不被允许，加剧了"家"的陌生与异己。

| 字词 | 释义 |
|------|------|
| 黯淡 | 昏暗不明亮 |
| 失望 | 感到没有希望 |
| 驱逐 | 赶走 |
| 回忆 | 回想往事 |

**叙事手法**：⭐⭐⭐ **象征（核心手法）**——"故居的灯"象征令人窒息的旧家；"爱尔克的灯"象征姐姐无私却绝望的爱与封建女子的悲剧命运；⭐⭐⭐ **类比联想**（由故居之灯自然联想到哈立希岛爱尔克之灯，把个人之悲与古老传说叠合）；⭐⭐ **虚实相生**（实写故居一线微光，虚写传说中爱尔克的灯笼）；⭐⭐ **细节白描**（兵士疑惑的眼光，侧面烘托"我"与旧家的隔阂）。

**逻辑作用**：核心意象"灯光"正式登场并展开双重含义，为后文的议论批判与心灵之灯的升华埋下伏笔。

### 第四部分（第9—11段）：议论批判"长宜子孙"

**原文节选**：
> "长宜子孙"，这四个字的年龄比我的还不知要大了多少……他用空空两手造就了一份家业……但是儿孙们回答他的还是同样的字：分和卖……财富并不"长宜子孙"，倘使不给他们一个生活技能，不向他们指示一条道路？"家"这个小圈子只能摧毁年青心灵的发育成长。

> "长宜子孙"，我恨不能削去这四个字！许多可爱的年青生命被摧残了，许多有为的年青心灵被囚禁了……这就是"家"！"甜蜜的家"！这不是我应该来的地方。爱尔克的灯光不会把我引到这里来的。

**大意概述**：作者由照壁上"长宜子孙"四字切入议论——这是祖父留下的家训，祖父"用空空两手造就了一份家业"，临终周到地为儿孙安排舒适生活，可儿孙们回报他的只有"分和卖"。作者由此点破一个"浅显道理"：财富并不"长宜子孙"，若不教给谋生技能、不指引人生道路，"家"这个小圈子只能摧毁青年心灵，财富只能毁灭崇高理想。议论推向高潮，作者激愤地喊出："'长宜子孙'，我恨不能削去这四个字！"那些被摧残的年青生命、被囚禁的年青心灵，正"在这个小圈子里面憔悴地捱着日子"——这就是所谓的"家"、"甜蜜的家"！最后以一句斩钉截铁的判断收束："爱尔克的灯光不会把我引到这里来的"——姐姐的灯指引的是爱与等待，而非把我引回这个扼杀人的旧家。

| 字词 | 释义 |
|------|------|
| 遗嘱 | 人生前对后事的嘱托 |
| 造就 | 创立、建成 |
| 周到 | 面面俱到、细致 |
| 削去 | 用刀削掉（极言痛恨） |
| 摧残 | 损害、使受折磨 |
| 囚禁 | 关押、禁锢 |
| 捱着日子 | 苦熬着度日（"捱"同"挨"） |

**叙事手法**：⭐⭐⭐ **夹叙夹议、叙议交融**（由照壁四字引出议论，议论紧扣姐姐之死，叙议互证）；⭐⭐⭐ **反复与呼告**（"长宜子孙"反复出现；"这就是'家'！'甜蜜的家'！"以呼告与反讽倾泻愤激）；⭐⭐ **反语**（"甜蜜的家"实为最不甜蜜的牢笼）；⭐⭐ **判断式警句**（"财富并不'长宜子孙'""爱尔克的灯光不会把我引到这里来的"），掷地有声。

**逻辑作用**：由情感升华为思想，由悼亡转为批判，是全文思想锋芒最锐利之处。

### 第五部分（第12—15段）：再次出走与心灵之灯的升华

**原文节选**：
> 我在外面一直跑了十八年……只有偶尔在梦里我看见了爱尔克的灯光……我望着那灯光，路是那么远，我又没有翅膀，我只有一个渴望：飞！飞！

> 但是我终于回来了……我走了很多的路。

> 于是在一个春天的早晨，依旧是十八年前的那些人把我送到门口……这次是我找不着她的坟墓。

> 忽然在前面田野里一片绿的蚕豆和黄的菜花中间，我仿佛又看见了一线光，一个亮……这一定是我的心灵的灯，它永远给我指示着我应该走的路。

> 我很高兴，自己又一次离开了狭小的家，走向广大的世界中去！

**大意概述**：作者回忆十八年漂泊中，常在梦里看见爱尔克的灯光，灯光在呼唤他，他"只有一个渴望：飞！飞！"——这体现了他对旧家的诀别冲动。可是"我终于回来了"，翻过"像山一样的十八年的长岁月"，却再也找不着姐姐的坟墓。于是，在一个春天的早晨，"我"像十八年前一样再次离家（同样是春天，同样的人送行，只是少了几个、多了几个，同样看不见姐姐的影子——只是这次不是"没有等待她"，而是"找不着她的坟墓"）。就在离去的田野上，"绿的蚕豆和黄的菜花中间"，"我"仿佛又看见"一线光，一个亮"——但这既不是故居令人失望的灯，也不是爱尔克悲剧的灯，而是**"我的心灵的灯"**（**第三种灯**），它"永远给我指示着我应该走的路"。文章以"我高兴地来，苦痛地去"收束：汽车离站时虽有留恋，但清晨微风、田野菜花"覆盖了离愁"，"我很高兴，自己又一次离开了狭小的家，走向广大的世界中去！"——一个全新的、坚定的"我"在灯光指引下走向新生。

| 字词 | 释义 |
|------|------|
| 熬煎 | 煎熬、折磨 |
| 梦魇 | 梦中惊恶；喻可怕的经历 |
| 刻印 | 深深地印记 |
| 留恋 | 不忍舍弃、离去 |
| 覆盖 | 此处指冲淡、掩盖 |

**叙事手法**：⭐⭐⭐ **象征意象的最终升华**（第三种灯"心灵的灯"登场，前两种灯的悲剧被新希望取代）；⭐⭐⭐ **首尾呼应与对称对照**（同样是春天的早晨、同样的人送行、同样看不见姐姐，构成精巧的环形结构，但"我"已由当年懵懂少年变为今日清醒的出走者）；⭐⭐⭐ **以景结情**（以"绿的蚕豆和黄的菜花""广大田野里一片盛开的菜子花"的明丽之景作结，色调由黯淡转向明亮，象征新生的方向）；⭐⭐ **反复**（"飞！飞！"），强烈抒情。

**逻辑作用**：完成由"悲—愤—悟—走"的情感弧线，主旨得到最终升华。

---

## 三、叙事手法鉴赏（记叙文核心考点）

### 1. ⭐⭐⭐ 象征手法——全文灵魂（三种灯光）

这是本文最重要、最核心、必考的手法。全文以"灯光"为贯穿意象，设计了三种含义递进的灯：

| 灯光 | 出处 | 象征意义 | 情感色彩 |
|------|------|----------|----------|
| **故居大门内的灯** | 第6段，黑暗中看到 | 虚幻、令人失望的旧家幻影；封建家庭的窒息 | 黯淡、压抑、失望 |
| **爱尔克的灯** | 第6段联想，题目点题 | 姐姐无私却绝望的爱；封建旧式女子寂寞而死的悲剧命运；旧家对人的等待与吞噬 | 悲伤、痛悼、同情 |
| **心灵的灯** | 第14段，离家田野上 | "我"内心的理想与信念；指引前进的新路；走向广大世界的希望 | 明亮、坚定、昂扬 |

三种灯由"旧家—悲剧—新生"递进，由外而内、由暗到明，完成全文情感与思想的升华。"心灵的灯"取代"爱尔克的灯"，正象征着"我"与旧家的彻底诀别与新路的开启。这是象征手法在记叙文中运用的典范。

---

### 2. ⭐⭐⭐ 今昔对比与环形结构

全文以"十八年前"与"十八年后"的对比为经，以两次离家的对照为纬：

| 对比维度 | 十八年前（离家） | 十八年后（归来再走） |
|----------|------------------|----------------------|
| 时间 | 春天的早晨 | 春天的早晨 |
| 送行人 | 同样的人 | "少了几个，也多了几个" |
| 姐姐 | "没有等待她"（还在，是新嫁娘） | "找不着她的坟墓"（已死） |
| "我" | 懵懂少年 | 走了很多路的清醒者 |

两次离家几乎完全对称，唯有姐姐由"在"变"不在"、"我"由"不等待"变为"找不着坟墓"——物是人非，悲剧深重。这种**环形对照结构**既使文章首尾圆合，又以"变与不变"凸显了封建家庭吞噬人的罪恶，结构匠心极其精巧。

---

### 3. ⭐⭐⭐ 夹叙夹议、叙议交融

本文叙事与议论水乳交融：
- 叙姐姐之死 → 议"旧式女子的命运"；
- 叙照壁"长宜子孙" → 议"财富并不长宜子孙"；
- 叙爱尔克的灯 → 议"家的小圈子只能摧毁青年心灵"；
- 叙心灵的灯 → 议"走向广大的世界"。

**特点**：议论从叙事中自然生发，不空泛；叙事因议论而深化，不停留于悲。这种写法使本文既有浓烈的抒情，又有锐利的思想锋芒。

---

### 4. ⭐⭐⭐ 反复、呼告与反语（抒情语言）

作者为倾泻浓烈情感，大量运用反复、呼告、反语等修辞：

| 修辞 | 例句 | 效果 |
|------|------|------|
| 反复 | "寂寞地活着，寂寞地死去"；"浪费，浪费，还是那许多……浪费"；"飞！飞！" | 强化悲愤/渴望 |
| 呼告 | "'长宜子孙'，我恨不能削去这四个字！" | 直接抒情，愤激之情喷薄而出 |
| 反语 | "这就是'家'！'甜蜜的家'！" | 以"甜蜜"反衬"残酷"，讽刺辛辣 |

---

### 5. ⭐⭐ 类比联想

由故居一线微光，自然联想到传说中哈立希岛上爱尔克的灯——这是全文最巧妙的一笔。**类比联想**把"我"姐姐的悲剧与古老传说叠合，使个人之悲获得普遍而悠远的悲剧意味，也使题目"爱尔克的灯光"获得具体而深沉的内涵。

---

### 6. ⭐⭐ 以景结情、寓情于景

文章开篇"逐渐黯淡的最后的阳光"定下怅惘基调，结尾"绿的蚕豆和黄的菜花""广大田野里一片盛开的菜子花"则以明丽之景象征新生的方向——色调由暗到明，情感由悲到壮，寓情于景，余味无穷。

::: warning 记叙文手法鉴赏三段式答题模板
**第一段（点）**：明确指出手法名称（如"本文运用了象征手法"）。
**第二段（析）**：结合原文具体分析——找出象征体与本体，说明二者之间的关联。
**第三段（效）**：点出该手法的表达效果——使形象更含蓄、使主旨更深刻、使情感更浓郁、给读者想象空间等。

例（象征手法）：
> 本文以"灯光"为核心象征意象，设计了三种含义递进的灯：故居大门内"不曾照亮什么、反添黑暗"的灯，象征虚幻而令人窒息的旧家；联想中哈立希岛上爱尔克的灯，象征姐姐无私却绝望的爱与封建女子寂寞而死的悲剧命运；最后田野中"我的心灵的灯"，象征"我"内心的理想与信念，指引"我"走向广大的世界。三种灯由旧家之灯到爱尔克之灯再到心灵之灯，由外而内、由暗到明，层层递进、相互映衬，把作者对姐姐的悼念、对封建家庭的批判、对新生活的追求含蓄而深沉地表达出来，使文章意象丰满、主旨深刻、情感浓郁，给读者以广阔的想象空间，是象征手法在记叙文中的典范运用。
:::

## 五、主旨情感

### 中心思想

本文以"爱尔克的灯光"为题，借回乡见闻与一个古老传说，表达了三层递进的思想感情：

1. **深沉悼念**：悼念寂寞死去的姐姐，同情一切旧式女子被封建家庭吞噬的悲剧命运；
2. **锐利批判**：批判"长宜子孙"的封建家庭观念与财富观念，揭露封建大家庭对青年生命与心灵的摧残；
3. **坚定追求**：在心灵的灯的指引下，与"狭小的家"彻底诀别，走向"广大的世界"，追求自由、进步的新生活。

### 思想内涵（更深一层）

本文是巴金"反封建、求新生"思想的集中体现。在他笔下，"家"不再是温馨的港湾，而是"许多可爱的年青生命被摧残""许多有为的年青心灵被囚禁"的牢笼——这与他的小说《家》一脉相承。作者借"灯光"意象告诉我们：真正指引人生的不应是旧家的虚幻之光，也不应是悲剧性的等待之光，而应是内心觉醒的"心灵之灯"。这种由"破"到"立"、由"悲"到"走"的精神走向，正是五四以来中国知识分子精神觉醒的缩影。

### 情感态度

- 对姐姐：深切痛悼、无限愧疚（"生前我没有好好地爱过她"）；
- 对旧家/封建家庭：厌恶、批判、决绝；
- 对姐姐式的悲剧：同情、悲愤；
- 对新路/广大世界：坚定、昂扬、充满希望。

### 文学价值

本文是现代抒情叙事散文的经典之作，其最大艺术成就在于：**以"灯光"为核心象征意象**，把悼亡、批判、追求三种情感统一于一个意象之下；又以**今昔对比的环形结构**与**叙议交融的笔法**，使文章既有浓烈的抒情性，又有锐利的思想锋芒。它与巴金的小说《家》并称为"破家"主题的双璧，对理解中国现代文学的反封建精神具有重要意义。

### 预测题与标准答案

**【预测题一·象征鉴赏】** 本文以"爱尔克的灯光"为题，文中先后写到三种"灯光"。请指出这三种灯光各是什么，并分析其象征意义。（8分）

> **【参考答案】**
> 文中先后写到三种灯光，含义递进：
> ①**第一种——故居大门内的灯**（"灯光并不曾照亮什么，反而在我心上添加了黑暗"）：象征虚幻、令人失望的旧家幻影，暗示封建家庭只能给人窒息与黑暗；（2分）
> ②**第二种——哈立希岛上爱尔克的灯**（题目点题）：象征姐姐无私却绝望的爱，以及封建旧式女子寂寞而死、被旧家吞噬的悲剧命运；（3分）
> ③**第三种——心灵的灯**（"这一定是我的心灵的灯，它永远给我指示着我应该走的路"）：象征"我"内心觉醒的理想与信念，是指引"我"走向广大世界、追求新生活的新希望。（3分）
> 三种灯由旧家之灯→爱尔克之灯→心灵之灯，由外而内、由暗到明，层层递进，最终以心灵之灯取代爱尔克之灯，象征"我"与旧家的彻底诀别与新路的开启。

---

**【预测题二·结构作用】** 文章两次写到"在一个春天的早晨"离家，送行的人也基本相同。这样写是否重复？有什么作用？请简要分析。（6分）

> **【参考答案】**
> 不重复，这是作者精心设计的**环形对照结构**，作用有三：
> ①**首尾呼应、结构圆合**：两次离家时间（春天的早晨）、地点、送行人都相同，使文章首尾圆合，结构严谨；（2分）
> ②**今昔对比、凸显悲剧**：两次离家中唯有姐姐由"在"（新嫁娘）变为"不在"（"找不着她的坟墓"），"我"由"没有等待她"变为"找不着她的坟墓"，以"变与不变"强烈凸显了十八年间姐姐被封建家庭吞噬的悲剧；（2分）
> ③**深化主旨、表现成长**：同样的离家，当年的"我"是懵懂少年，今日的"我"已是"走了很多的路"、在心灵之灯指引下坚定走向广大世界的清醒者，以对照表现了"我"的精神觉醒与成长，深化了反封建、求新生的主旨。（2分）

---

**【预测题三·主旨探究】** 作者为什么说"爱尔克的灯光不会把我引到这里来的"？请结合全文探究。（6分）

> **【参考答案】**
> 这句话意味深长，含意有三层：
> ①**就灯光的性质而言**：爱尔克的灯是姐姐为航海兄弟点灯照路、至死等待的灯，象征的是无私的爱与悲剧性的等待，它指引的是"爱与等待"，而非把人引回这个扼杀人的旧家；（2分）
> ②**就作者的态度而言**：作者已清醒认识到这个"长宜子孙"的旧家只是"摧残年青生命、囚禁年青心灵"的牢笼，是"甜蜜的家"反衬下的悲剧之源，因此他绝不愿被任何东西引回旧家；（2分）
> ③**就主旨而言**：这句话斩钉截铁地表明了作者与封建家庭彻底诀别的坚定立场，与下文"心灵的灯"指引"我"走向广大世界相呼应，集中体现了本文反封建、求新生的核心主旨。（2分）

---

**【预测题四·语言赏析】** 赏析下面语句的表达效果。（6分）
> "长宜子孙"，我恨不能削去这四个字！……这就是"家"！"甜蜜的家"！

> **【参考答案】**
> ①**呼告与反复**：作者直接对"长宜子孙"四字发出呼告，并以"我恨不能削去这四个字"的极端之语倾泻愤激之情，感情喷薄、掷地有声；（2分）
> ②**反语**：以"甜蜜的家"这一反语形容这个"摧残生命、囚禁心灵"的牢笼，以"甜蜜"反衬"残酷"，讽刺辛辣、力透纸背；（2分）
> ③**排比式的揭露**：上文"许多可爱的年青生命被摧残了，许多有为的年青心灵被囚禁了，许多人在这个小圈子里面憔悴地捱着日子"三个"许多"构成排比，层层揭露封建家庭的罪恶，为呼告与反语蓄足气势，使作者对封建家庭的厌恶、批判之情表达得淋漓尽致。（2分）

---

**【预测题五·选择】** 下列对本文的理解与分析，不正确的一项是（3分）
A. 文章以"傍晚""逐渐黯淡的最后的阳光"开篇，奠定怅惘压抑的感情基调，与结尾明丽的田野菜花形成色调对照。
B. "她寂寞地活着，寂寞地死去"运用反复修辞，强化了对姐姐悲剧命运的痛悼，并把个人悲剧上升为"旧式女子的命运"。
C. 文中写"守门的武装兵士用疑惑的眼光看我"，是为了直接批评兵士的态度，表达作者对社会现实的不满。
D. 结尾"我"再次离家时"很高兴"，是因为"我"在心灵之灯的指引下坚定走向广大世界，与开头的回望形成决绝的对照。

> **【参考答案】** **C**
> C项"是为了直接批评兵士的态度"错误。写兵士疑惑的眼光，是为了**侧面烘托"我"与旧家的隔阂与陌生**——兵士"不了解我的心情""不会认识十八年前的少年人"，"用眼光驱逐一个人的许多亲密的回忆"，暗示旧家如今连凭吊都不被允许，加剧了"家"的异己与陌生，并非"直接批评兵士的态度"。其余三项分析均正确。

### 核心记忆三句话

1. **文体与定位**：现代抒情叙事散文，巴金1941年回成都后作，收入《龙·虎·狗》，与小说《家》同为"破家"主题双璧。
2. **核心手法（必背）**：**象征（三种灯光递进）**——故居大门内的灯（旧家幻影/失望）→爱尔克的灯（姐姐悲剧/无私绝望的爱）→心灵的灯（理想/新路）；辅以**今昔对比的环形结构**、**夹叙夹议**、**反复呼告反语**。
3. **主旨三层**：悼念寂寞死去的姐姐；批判"长宜子孙"的封建家庭；在心灵之灯指引下与旧家诀别、走向广大世界，追求自由新生。

## 附录：原文（已清洗去标签去重，按原叙事逻辑还原）

> 　　傍晚，我靠着逐渐黯淡的最后的阳光的指引，走过十八年前的故居。这街的一切，这建筑的一切开始在我眼前隐藏起来，像在躲避一个久别的旧友。但是它们的改变了的面貌于我还是十分亲切。我认识它们，就像认识我自己。还是那样宽的街，宽的房屋。巍峨的门墙代替了太平缸和石狮子，那对常常做我们坐骑的背脊光滑的雄狮也不知逃进了哪座荒山。然而大门开着，照壁上"长宜子孙"四个字却是原样地嵌在那里，似乎连颜色也不曾被风雨剥蚀。我望着那同样的照壁，我被一种奇异的感情抓住了，我仿佛要在这里看出过去的十九个年头，不，我仿佛要在这里寻找十八年以前的辽远的旧梦。
>
> 　　街道仍是静静的。忽然一个熟悉的声音在我耳边轻轻地唱起了这个欧洲的古传说。在这里不会有人歌咏这样的故事。应该是书本在我心上留下的影响。但是这时候我想起了自己的事情。
>
> 　　十八年前在一个春天的早晨，我离开这同样城市，同样街道的时候，我也曾有一个姊姊，也曾答应过有一天回来看她，同她谈一些外面的事情。我相信着自己的约言。那时我的姊姊还是一个出阁才只一个多月的新嫁娘，都说她有一个性情温良的丈夫，因此也会有着长久的幸福的岁月。
>
> 　　然而人的安排终于被"偶然"毁坏了。这应该是一个"意外"。但是这"意外"却毫无怜悯地在年青的心上下着打击。我离家不过一年半光景，就接到了姊姊的死讯。我的哥哥用了颤抖的哭诉的笔叙说了一个善良的女性的悲惨的结局，还说到她死后所得着的冷落的待遇。从此那个作过她的丈夫的所谓温良的人改变了，他往一条丧失人性的路走去，他想往上爬，结果却不停地向下面落，终于到了用鸦片来延续生命的地步。对于姊姊，她生前我没有好好地爱过她，死后也不曾做过一件纪念她的事。她寂寞地活着，寂寞地死去。死带走了她的一切，这就是在我们那地方的旧式女子的命运。
>
> 　　十九年，似乎一切全变了，又似乎都没有改变。死了许多人，毁了许多家。许多可爱的生命葬入黄土。接着又有许多新的人继续来演那不必要的悲剧。浪费，浪费，还是那许多不必要的浪费——生命，精力，感情，财富，甚至欢笑和眼泪。我去的时候是这样，回来时看见的还是一样情形。关在这个小圈子里我禁不住几次问我自己：难道这十八年全是白费？难道在这许多年中间所改变的就只是装束和名词！我苦痛地搓自己的手，不敢给一个回答。
>
> 　　在这个我永不能忘记的城市里，我过了五十个傍晚。我花费了自己不少眼泪和欢笑，也消耗了别人不少眼泪和欢笑。我匆匆地来，也将匆匆地去。用留恋的眼光看我出生的房屋，这应该是最后的一次了。我的心似乎想在那里寻觅什么，但是我所要的东西不会在那里找到。我不会像我的一个姑母或嫂嫂，设法进到那所已经易了几个主人的公馆，对着园中花树垂泪，慨叹着一个家族的盛衰。摘吃自己栽种树上的苦果，这是一个人的本分。我没有跟着那些人走一条路，我当然在这里找不着自己的脚迹。几次走过这地方，我所看见的还只是那四个字"长宜子孙"。
>
> 　　黑暗来了。我的眼睛失掉了一切。于是大门内亮起灯光。灯光并不曾照亮什么，反而在我心上添加了黑暗。我只得失望地走了。我向着来时的路回去。已经走了四五步，我忽然不自主地掉回头，再看那建筑。依旧是阴暗中一丝微光。我好像看见一个盛满希望的水碗一下子就落在地上打碎了一般，我苦痛地在心里叫起来，在这被夜幕覆盖着的近代城市的静寂的街中，我仿佛看见了哈立希岛上的灯光。那应该是姊姊爱尔克点的灯笼，她用这灯来给她的航海的兄弟照路，每夜每夜灯光亮在她的窗前，她一直到死都在等待那个出远门的兄弟回来。最后她带着失望进入坟墓。
>
> 　　守门的武装兵士用疑惑的眼光看我。他不了解我的心情！他不会认识十八年前的少年人。他却用眼光驱逐一个人的许多亲密的回忆。
>
> 　　"长宜子孙"，这四个字的年龄比我的还不知要大了多少。这也该是我祖父留下的东西罢。最近在家里我还读到他的遗嘱。他用空空两手造就了一份家业。到临死还周到地为儿孙安排了舒适的生活。他叮属后人保留着他修建的房屋和他辛苦地搜集起来的书画。但是儿孙们回答他的还是同样的字：分和卖。我很奇怪，为什么这样聪明的老人还不明白一个浅显道理：财富并不"长宜子孙"，倘使不给他们一个生活技能，不向他们指示一条道路？"家"这个小圈子只能摧毁年青心灵的发育成长；倘使不同时让他们睁起眼睛去看广大世界，财富只能毁灭崇高的理想和善良的气质，要是它只消耗在个人的享乐上面。
>
> 　　"长宜子孙"，我恨不能削去这四个字！许多可爱的年青生命被摧残了，许多有为的年青心灵被囚禁了。许多人在这个小圈子里面憔悴地捱着日子。这就是"家"！"甜蜜的家"！这不是我应该来的地方。爱尔克的灯光不会把我引到这里来的。
>
> 　　我在外面一直跑了十八年。我从没有向人谈过我姊姊。只有偶尔在梦里我看见了爱尔克的灯光。一年前在上海我常常睁着眼睛做梦。我望着远远的在窗前发亮的灯，我面前横着一片大海，灯光在呼唤我，我恨不得腋下生出翅膀，即刻飞到那边去。沉重的梦压着我的心灵，我好像在同许多无形的魔手挣扎。我望着那灯光，路是那么远，我又没有翅膀，我只有一个渴望：飞！飞！那些熬煎着心的日子！那些可怕的梦魇！
>
> 　　但是我终于回来了。我越过那堆积着像山一样的十八年的长岁月，回到了生我养我而且让我刻印了无数儿时回忆的地方。我走了很多的路。
>
> 　　于是在一个春天的早晨，依旧是十八年前的那些人把我送到门口，这里面少了几个，也多了几个。还是和那次一样，看不见我姊姊的影子，那次是我没有等待她，这次是我找不着她的坟墓。一个叔父一个堂兄弟到车站送我，十八年前他们也送过我一段路程的。
>
> 　　忽然在前面田野里一片绿的蚕豆和黄的菜花中间，我仿佛又看见了一线光，一个亮，这还是我常常看见的灯光。这不会是爱尔克的灯里照出来的，我那可怜的姊姊已经死去了。这一定是我的心灵的灯，它永远给我指示着我应该走的路。
>
> 　　我高兴地来，苦痛地去。汽车离站时我心里的确充满了留恋。但是清晨的微风，路上的尘土，马达的叫吼，车轮的滚动，和广大田野里一片盛开的菜子花，这一切覆盖了我的离愁。我不顾同行者的劝告，把头伸到车窗外面，去呼吸广大天幕下的新鲜空气。我很高兴，自己又一次离开了狭小的家，走向广大的世界中去！
