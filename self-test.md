# ✍️ 默写自测模式（占位）

该页面为**默写自测模式**的占位入口，后续整理好题库后接入交互组件即可启用。

## 预期功能

- **名词默写**：给出术语（如「实践」），默写出定义/要点，点击展开对照答案。
- **年表填空**：给出历史事件，默写发生时间或反过来。
- **选择题**：单选/多选，提交后即时判分并统计正确率。
- **错题本**：自动收集答错的题目，便于针对性复习。

## 如何接入题库（规划）

把题目整理成结构化 md，放到各科目目录下，例如：

```
sixiu/
├── index.md
├── 第一章-坚定理想信念.md      # 学习笔记
└── 第一章-默写题.md            # 自测题库
```

题库 md 推荐采用以下约定（方便后续解析成交互题）：

```markdown
## Q1 【名词】实践

- 答案：人类能动地改造客观世界的物质活动
- 要点：客观物质性 / 自觉能动性 / 社会历史性

## Q2 【填空】《窦娥冤》作者

- 答案：关汉卿

## Q3 【选择】马克思主义哲学的核心是？

- A. 辩证唯物主义
- B. 历史唯物主义
- C. 实践观点 ✅
- D. 矛盾分析法
```
