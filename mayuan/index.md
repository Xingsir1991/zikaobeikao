# 🟡 15044 马克思主义基本原理

> 自考课程代码 **15044** · 本科层次公共政治课

## 课程概览

本课程是高等教育自学考试**思想政治理论课**必修课，系统讲授马克思主义的**三个组成部分**及其基本原理。

内容分三大板块：

| 板块 | 主要内容 | 重点 |
| --- | --- | --- |
| **马克思主义哲学** | 唯物论、辩证法、认识论、唯物史观 | 重点（题量最大） |
| **马克思主义政治经济学** | 商品理论、剩余价值、资本积累 | 难点 |
| **科学社会主义** | 社会主义从空想到科学、共产主义理想 | 易得分 |

## 备考策略

::: tip 备考要点
- **选择题**：关键术语定义要精确（如"物质""实践""剩余价值"），易考名词辨析
- **简答题**：原理答题讲究"三要素"（前提 + 核心 + 推论），逻辑链完整
- **论述题**：原理 + 联系实际 + 方法论意义
- 哲学部分是重中之重：唯物论、辩证法（三大规律）、认识论、唯物史观要能成体系
:::
