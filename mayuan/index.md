# 🟡 马原（马克思主义基本原理）

> 哲学、政治经济学、科学社会主义三大板块。

## 使用说明

- 左侧目录展示本科目所有章节
- 每篇笔记底部可点击 **「标记为已掌握」**，在 [进度看板](/progress) 汇总
- 顶部搜索框支持中文全文检索

## 添加新资料

把整理好的 md 文件直接放进 `mayuan/` 目录，例如：

```
mayuan/
├── index.md                    # 本页
├── 唯物论-物质与意识.md
├── 辩证法-三大规律.md
└── 认识论-实践与认识.md
```

并在 `.vitepress/config.mts` 的 `sidebarGroup` 中补充对应条目。

## 重点提示（示例）

::: tip 备考要点
- 哲学：唯物论、辩证法、认识论、唯物史观（重点）
- 政治经济学：商品理论、剩余价值、资本积累
- 科社：社会主义从空想到科学、共产主义理想
- 关键术语定义要精确（如「物质」「实践」），易考名词解释
:::
