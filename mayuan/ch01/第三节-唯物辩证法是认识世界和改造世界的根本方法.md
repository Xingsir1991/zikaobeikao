# 第三节 唯物辩证法是认识世界和改造世界的根本方法

## 一、唯物辩证法的本质特征和认识功能

### 唯物辩证法的本质（必背）

> 唯物辩证法是**客观辩证法与主观辩证法的统一**。

| 概念 | 含义 |
| --- | --- |
| **客观辩证法** | 客观事物本身的辩证运动（自然界、社会的辩证法） |
| **主观辩证法** | 人类思维活动的辩证法（概念的辩证运动） |

> 主观辩证法是客观辩证法的**反映**。

### 唯物辩证法的认识功能

- 唯物辩证法是**认识世界和改造世界的根本方法**
- 是**科学认识方法**的哲学基础

---

## 二、辩证思维的基本方法（必背五对）

::: danger 必背：辩证思维的五对方法
1. **归纳与演绎**
2. **分析与综合**
3. **抽象与具体**
4. **逻辑与历史相统一**
5. **从抽象上升到具体**
:::

### 归纳与演绎

| 方法 | 含义 |
| --- | --- |
| **归纳** | 从个别到一般 |
| **演绎** | 从一般到个别 |

> 两者相互依赖、相互促进。

### 分析与综合

| 方法 | 含义 |
| --- | --- |
| **分析** | 把整体分解为部分 |
| **综合** | 把部分联结为整体 |

## 三、不断增强思维能力（必背五种思维）

::: danger 必背：五种思维能力
1. **辩证思维能力**——用联系的、发展的、全面的方法思考

:::

### 预测题一（单选）

**唯物辩证法本质上是（　）**
A. 客观辩证法与主观辩证法的统一
B. 对立统一规律
C. 联系和发展的观点
D. 量变质变规律

> **答案：A**
> **解析**：唯物辩证法本质上是**客观辩证法与主观辩证法的统一**。主观辩证法是客观辩证法的反映。

### 易错点提醒

::: warning 本节三大易错点
1. **主观辩证法是客观辩证法的反映**（主观从属于客观）。
2. **归纳=个别到一般；演绎=一般到个别**。
3. **五种思维能力**：辩证、历史、系统、战略、底线。
:::
