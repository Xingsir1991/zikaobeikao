# 第二节 事物的普遍联系和变化发展

> 教材第 40–53 页 · **唯物辩证法**——联系发展、三大规律、五对基本环节。

本节是马原哲学的**核心中的核心**——辩证法三大规律几乎每年必考。

---

## 一、联系和发展的普遍性

### 联系的特征（必背）

::: danger 必背：联系的四个特征
1. **客观性**——联系是事物本身固有的
2. **普遍性**——任何事物都不能孤立存在
3. **多样性**——联系的形式多种多样
4. **条件性**——联系受条件制约
:::

### 发展的实质（必背金句）

::: danger 必背：发展的实质
> **发展的实质是新事物的产生和旧事物的灭亡。**
:::

---

## 二、三大规律（极高频考点，必背全部）

### ① 对立统一规律（核心规律，必背）

::: danger 必背：对立统一规律的地位
> **对立统一规律是唯物辩证法的实质和核心**，是事物发展的**根本规律**。
:::

#### 矛盾的同一性和斗争性（必背）

| 属性 | 含义 |
| --- | --- |
| **同一性** | 矛盾双方相互依存、相互贯通（一定条件下相互转化） |
| **斗争性** | 矛盾双方相互排斥、相互对立 |

> 同一性是**相对的**，斗争性是**绝对的**。斗争性寓于同一性之中。

#### 矛盾的普遍性和特殊性（必背，极高频考点）

::: danger 必背：矛盾普遍性和特殊性
- **普遍性**：矛盾无处不在、无时不有（共性、绝对性）
- **特殊性**：不同事物的矛盾各有特点（个性、相对性）

> 二者是**共性与个性、一般与个别**的关系。**矛盾普遍性寓于特殊性之中**。
:::

**方法论意义**：具体问题具体分析。

### ② 量变质变规律（必背）

::: danger 必背：量变质变规律
- **量变**——事物数量的增减和场所的变更（**渐进性、不显著**的变化）
- **质变**——事物根本性质的变化（**飞跃性、显著**的变化）

> **量变是质变的必要准备，质变是量变的必然结果。** 量变超过一定限度（"度"）就引起质变。质变又引起新的量变。
:::

**方法论意义**：适度原则；重视量的积累。

### ③ 否定之否定规律（必背）

::: danger 必背：否定之否定规律
> 事物的发展经过**肯定→否定→否定之否定**三个阶段，形成一个周期。

- **辩证的否定**是**自我的否定**（事物内部矛盾引起）
- **辩证的否定**是**发展和联系的环节**（扬弃——既克服又保留）
- 总趋势是**前进的、上升的**；道路是**曲折的**
:::

**方法论意义**：前途是光明的，道路是曲折的。

## 三、联系和发展的基本环节（五对范畴，必背）

::: danger 必背：五对基本范畴
1. **原因与结果**——揭示事物的前后相继、彼此制约
2. **必然性与偶然性**——揭示事物发生、发展和灭亡的不同趋势
3. **可能性与现实性**——揭示事物的过去、现在和将来
4. **现象与本质**——揭示事物的外部表现和内部联系
5. **内容与形式**——揭示事物的内在要素和结构方式
:::

## 四、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 对立统一是核心、矛盾的普遍性和特殊性、量变质变关系、辩证否定的实质 |
| **多选** | 三大规律、五对范畴、矛盾两种属性 |
| **简答** | 对立统一规律为什么是核心？量变和质变的关系？ |
| **论述** | 如何理解矛盾的普遍性和特殊性？ |

### 预测题一（单选）

**唯物辩证法的实质和核心是（　）**
A. 质量互变规律
B. 对立统一规律
C. 否定之否定规律
D. 联系和发展的观点

> **答案：B**
> **解析**：**对立统一规律是唯物辩证法的实质和核心**，是事物发展的根本规律。它揭示了事物发展的源泉和动力。A、C 是其他两大规律；D 是唯物辩证法的总特征。

### 预测题二（简答）

**简述量变和质变的辩证关系。**

> 量变和质变是事物变化的两种基本状态，辩证关系是：
> 1. **量变是质变的必要准备**——质变必须有量的积累
> 2. **质变是量变的必然结果**——量变达到一定程度必然引起质变
> 3. **质变引起新的量变**——质变后在新质基础上开始新的量变
>
> 量变和质变相互转化、相互渗透。方法论意义：重视量的积累，把握适度原则，不失时机促成质变。

### 易错点提醒

::: warning 本节四大易错点
1. **核心规律=对立统一规律**（不是量变质变或否定之否定）。
2. **矛盾的同一性是相对的，斗争性是绝对的**（不要搞反）。
3. **量变是质变的"必要准备"，质变是量变的"必然结果"**——表述固定。
4. **辩证否定的实质=扬弃**（既克服又保留，不是全盘否定）。
:::
