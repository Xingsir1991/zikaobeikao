# 第一节 人类社会的存在与发展

> 教材第 123–155 页 · **唯物史观的基石**——社会存在与社会意识、社会基本矛盾、社会形态更替。

---

## 一、社会存在与社会意识（必背，极高频考点）

### 社会存在（必背）

::: danger 必背：社会存在的含义
> **社会存在是指社会的物质生活条件**，是社会物质生活的总和，主要包括：

1. **自然地理环境**
2. **人口因素**
3. **物质生产方式**（最根本、决定性力量）

> 物质生产方式 = **生产力 + 生产关系**
:::

### 社会意识（必背）

> **社会意识是社会生活的精神方面**，是社会存在的反映。

社会意识分类：
1. **意识形态**（政治法律思想、道德、艺术、宗教、哲学）——有阶级性
2. **非意识形态**（自然科学、逻辑学、语言学）——无阶级性

### 社会存在与社会意识的辩证关系（必背金句）

::: danger 必背：社会存在与社会意识的关系
> **社会存在决定社会意识，社会意识对社会存在具有能动的反作用。**

1. **社会存在决定社会意识**——社会存在是社会意识内容的来源；社会存在的发展决定社会意识的变化
2. **社会意识具有相对独立性**——这是社会意识的突出特点
3. **社会意识对社会存在具有能动的反作用**——先进的社会意识促进社会发展，落后的社会意识阻碍社会发展
:::

### 社会意识的相对独立性（必背，高频考点）

::: danger 必背：社会意识相对独立性的表现
1. **与社会存在变化发展的不完全同步性**（可能落后或超前于社会存在）
2. **在发展上具有历史继承性**
3. **社会意识各种形式之间相互影响和相互作用**
4. **对社会存在具有能动的反作用**（最突出表现）
:::

## 二、生产力与生产关系（必背，极高频考点）

### 生产力（必背）

::: danger 必背：生产力的含义
> **生产力是人类在生产实践中形成的改造和影响自然以使其适合社会需要的物质力量，标志着人类改造自然的实际程度。**
:::

生产力三要素（必背）：

| 要素 | 内容 |
| --- | --- |
| **劳动者**（主体要素） | 具有一定生产经验、劳动技能和科学知识的人 |
| **劳动资料**（特别是生产工具） | 改造自然的物质手段，**生产工具是生产力发展水平的客观尺度** |
| **劳动对象** | 劳动所指向的物质客体 |

> **劳动者是生产力中最活跃的因素**。生产工具是衡量生产力发展水平的客观尺度。
>
> **科学技术是先进生产力的集中体现和主要标志，是第一生产力。**

### 生产关系（必背）

::: danger 必背：生产关系的含义
> **生产关系是人们在物质生产过程中形成的不以人的意志为转移的经济关系**。
:::

生产关系三方面（必背）：

1. **生产资料所有制关系**（最基本、决定性的）
2. 生产中人与人的关系
3. 产品分配关系

### 生产力与生产关系的辩证关系（必背金句，极高频考点）

::: danger 必背：生产力与生产关系的关系
> **生产力决定生产关系，生产关系反作用于生产力。**

1. **生产力决定生产关系**——生产力的状况决定生产关系的性质；生产力的发展决定生产关系的变革
2. **生产关系对生产力具有反作用**——适合时促进，不适合时阻碍
3. **生产关系一定要适合生产力状况的规律**——这是人类社会发展的基本规律
:::

---

## 三、经济基础与上层建筑（必背，极高频考点）

### 经济基础

::: danger 必背：经济基础的含义
> **经济基础是指由社会一定发展阶段的生产力所决定的生产关系的总和。**
:::

### 上层建筑

::: danger 必背：上层建筑的含义
> **上层建筑是指建立在一定经济基础之上的意识形态以及相应的制度、组织和设施。**
:::

上层建筑两大部分：

| 部分 | 内容 |
| --- | --- |
| **观念上层建筑（意识形态）** | 政治法律思想、道德、艺术、宗教、哲学 |
| **政治上层建筑** | 国家政治制度、立法司法制度和行政制度；国家政权机构、政党、军队、警察、法庭、监狱等政治组织形态和设施 |

> **国家政权是政治上层建筑的核心**。**政治上层建筑在整个上层建筑中居主导地位**。

### 经济基础与上层建筑的辩证关系（必背金句）

::: danger 必背：经济基础与上层建筑的关系
> **经济基础决定上层建筑，上层建筑对经济基础具有反作用。**

1. **经济基础决定上层建筑**——决定其产生、性质、变化发展
2. **上层建筑对经济基础具有反作用**——为其经济基础服务（保卫自己经济基础）
3. **经济基础与上层建筑的相互作用构成两者的矛盾运动**
4. **上层建筑一定要适合经济基础状况的规律**——人类社会发展的又一基本规律
:::

### 概念辨析

| 概念 | 含义 |
| --- | --- |
| 社会基本矛盾 | **生产力—生产关系、经济基础—上层建筑**这两对矛盾 |
| 社会主要矛盾 | 不同时期不同，由社会特定阶段决定 |

## 四、社会形态更替的一般规律（必背）

### 社会形态的含义

> **社会形态是同生产力发展一定阶段相适应的经济基础与上层建筑的统一体。**

### 社会形态更替的统一性与多样性

- **五种社会形态**：原始社会、奴隶社会、封建社会、资本主义社会、共产主义社会（社会主义是其初级阶段）
- **更替的统一性**：五种社会形态依次更替
- **多样性**：不同民族、国家发展道路有差异，可能跨越某些社会形态

### 社会形态更替的必然性与人们的历史选择性

- **必然性**：社会形态更替遵循基本规律，是必然的
- **历史选择性**：具体发展道路由人民选择

## 五、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 社会存在含义、生产力三要素、生产关系最基本方面、上层建筑核心 |
| **多选** | 社会存在构成、社会意识分类、社会意识相对独立性表现、上层建筑两部分 |
| **简答** | 社会存在与社会意识的关系？生产力与生产关系的辩证关系？ |
| **论述** | 如何理解社会基本矛盾运动规律？ |

### 预测题一（单选）

**生产关系中起决定作用的方面是（　）**
A. 人们在生产中的地位
B. 产品分配方式
C. 生产资料所有制关系
D. 人与人的相互关系

> **答案：C**
> **解析**：**生产资料所有制关系是生产关系中最基本的、决定性的方面**。它决定着生产中人与人的关系、产品分配关系。

### 预测题二（简答）

**简述生产力与生产关系的辩证关系。**

> 生产力与生产关系是**相互作用的辩证关系**：
> 1. **生产力决定生产关系**：生产力的状况决定生产关系的性质；生产力的发展变化决定生产关系的变革。
> 2. **生产关系对生产力具有反作用**：当生产关系适合生产力发展要求时，就推动生产力发展；当生产关系不适合生产力发展要求时，就阻碍生产力发展。
> 3. **生产关系一定要适合生产力状况的规律**是推动人类社会发展的基本规律。

### 易错点提醒

::: warning 本节五大易错点
1. **生产力的"最活跃因素"是劳动者**；**衡量生产力水平的"客观尺度"是生产工具**——不要混淆。
2. **生产关系中最基本、决定性的方面是生产资料所有制关系**。
3. **上层建筑的核心是国家政权**；**政治上层建筑在整体上层建筑中居主导地位**。
4. **社会存在决定社会意识，社会意识具有相对独立性**——不能只说一面。
5. **社会基本矛盾是两对**：生产力—生产关系 + 经济基础—上层建筑。
:::
