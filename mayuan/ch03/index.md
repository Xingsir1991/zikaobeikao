# 第三章 人类社会及其发展规律

> 教材第 123–181 页 · **历史唯物主义（唯物史观）**——马克思主义关于社会历史发展一般规律的科学。

## 本章框架

本章是马克思主义哲学的**唯物史观**部分，是马克思的**第一大发现**。本章考点密集，**社会存在与社会意识、社会基本矛盾（生产力—生产关系、经济基础—上层建筑）、人民群众是历史创造者**是高频考点。

## 本章的特殊地位

::: tip 第三章是"唯物史观核心"
唯物史观是马克思**两大发现之一**（另一为剩余价值学说）。它揭示了社会历史发展的**一般规律**，是科学社会主义的理论基石。唯物史观与剩余价值学说共同使社会主义从空想变为科学。
:::
