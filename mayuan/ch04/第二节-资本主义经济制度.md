# 第二节 资本主义经济制度

> 教材第 201–222 页 · **马克思剩余价值学说的核心**——揭示资本主义剥削的秘密。

---

## 一、劳动力成为商品与货币转化为资本（必背，极高频考点）

### 货币与资本的区别

| 范畴 | 流通公式 | 目的 |
| --- | --- | --- |
| **货币（简单商品流通）** | W（商品）—G（货币）—W（商品） | 获得所需商品（使用价值） |
| **资本（资本流通）** | G（货币）—W（商品）—G'（更多货币） | 获得更多货币（价值增殖） |

> **G' = G + ΔG**，其中 ΔG 是**剩余价值**。

### 劳动力成为商品是货币转化为资本的前提（必背金句）

::: danger 必背：劳动力成为商品是货币转化为资本的前提
> **劳动力成为商品，是货币转化为资本的前提。**
:::

### 劳动力成为商品的两个条件（必背）

1. **劳动者有人身自由**——可以自由出卖劳动力
2. **劳动者丧失生产资料**——只能靠出卖劳动力为生

### 劳动力商品的价值和使用价值（必背，与普通商品不同）

::: danger 必背：劳动力商品的特殊性
> 劳动力商品的特殊性在于其**使用价值**：

- **劳动力商品的价值**——由生产和再生产劳动力所必需的生活资料的价值决定（含三部分：维持劳动者本人生存、养家糊口、教育培训费用）
- **劳动力商品的使用价值（最特殊！）**——**它的使用价值是价值的源泉**！它在被使用的过程中**能创造出比自身价值更大的价值**，即创造**剩余价值**

> 这就是**资本主义剥削的秘密所在**。
:::

## 二、剩余价值的生产（极高频考点，必背全部）

### 剩余价值的含义（必背金句）

::: danger 必背：剩余价值
> **剩余价值（m）是雇佣工人创造的、被资本家无偿占有的、超过劳动力价值以上的那部分价值**。

> **剩余价值规律是资本主义的基本经济规律。**
:::

### 资本主义生产过程的两重性

| 过程 | 内容 |
| --- | --- |
| **劳动过程** | 工人具体劳动创造使用价值 |
| **价值增殖过程** | 工人抽象劳动创造价值，**其中超过劳动力价值部分是剩余价值**（资本主义生产过程的本质） |

### 不变资本与可变资本（必背，极高频考点）

::: danger 必背：不变资本与可变资本
> 以**是否增殖价值**为标准，资本分为：

1. **不变资本（c）**——以**生产资料**形态存在的资本，**在生产过程中不改变自己的价值量**（价值转移，不增殖）
2. **可变资本（v）**——以**劳动力**形态存在的资本，**在生产过程中价值量发生变化（增殖）**

> **可变资本是剩余价值的真正源泉**。剩余价值（m）只是由可变资本（v）创造的，与不变资本（c）无关。
:::

### 剩余价值率（必背）

::: danger 必背：剩余价值率
> **剩余价值率（m'）= 剩余价值（m） / 可变资本（v）**

> 它**准确地反映了资本家对工人的剥削程度**。
:::

### 绝对剩余价值与相对剩余价值（必背，极高频考点）

::: danger 必背：剩余价值的两种生产方法
1. **绝对剩余价值的生产**——在**必要劳动时间不变的条件下**，通过**绝对延长工作日长度**（或提高劳动强度）来生产剩余价值

2. **相对剩余价值的生产**——在**工作日长度不变的条件下**，通过**缩短必要劳动时间**（从而相应延长剩余劳动时间）来生产剩余价值

> **相对剩余价值的生产以整个社会劳动生产率的提高为条件**。
:::

### 超额剩余价值（必背，与相对剩余价值区分）

> **超额剩余价值**是单个资本家由于**率先提高劳动生产率**，使商品的个别价值低于社会价值而获得的额外剩余价值。

### 三者关系（高频考点）

| 类别 | 方法 | 条件 |
| --- | --- | --- |
| 绝对剩余价值 | 延长工作日长度（必要劳动时间不变） | 工作日长度 |
| 相对剩余价值 | 缩短必要劳动时间（工作日不变） | **整个社会劳动生产率提高** |
| 超额剩余价值 | 个别资本家率先提高劳动生产率 | 个别资本家率先改进技术 |

> **追逐超额剩余价值是推动资本主义技术进步的直接动因；相对剩余价值是无数资本家追逐超额剩余价值的结果**。

## 三、资本积累（必背）

### 资本积累的含义

::: danger 必背：资本积累
> **资本积累是剩余价值的资本化**——资本家把剩余价值的一部分转化为资本。
:::

### 资本积累的必然性

- 内在动力：**对剩余价值的追求**
- 外在压力：**资本主义竞争**的强制作用

### 资本的技术构成、价值构成、有机构成

::: danger 必背：资本有机构成
| 概念 | 含义 |
| --- | --- |
| **资本技术构成** | 不变资本与可变资本在物质形态上的比例（生产资料与劳动力的比例）|
| **资本价值构成** | 不变资本价值与可变资本价值的比例 |
| **资本有机构成** | 由技术构成决定并反映技术构成变化的价值构成（**c:v**）|

> **资本主义发展趋势：资本有机构成不断提高。**
:::

## 四、资本主义基本矛盾与经济危机（必背，极高频考点）

### 资本主义基本矛盾（必背金句）

::: danger 必背：资本主义基本矛盾
> **生产社会化与生产资料资本主义私人占有之间的矛盾**是**资本主义的基本矛盾**。
:::

### 资本主义经济危机（必背，极高频考点）

::: danger 必背：经济危机的实质、根源、特征
1. **实质**——**生产相对过剩**（不是绝对过剩，是相对于劳动人民有支付能力的需求过剩）
2. **根本原因/根源**——**资本主义的基本矛盾**（生产社会化与生产资料私人占有之间的矛盾）
3. **抽象形态上可能性**——由货币执行**流通手段**和**支付手段**职能产生
4. **基本特征**——**生产相对过剩**

> **经济危机实际上使资本主义一切矛盾尖锐化**。
:::

## 五、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 劳动力商品特殊性、剩余价值定义、不变资本可变资本划分、剩余价值率、相对剩余价值、经济危机实质 |
| **多选** | 劳动力商品价值三部分、剩余价值两种生产方法、资本有机构成 |
| **简答** | 为什么说劳动力成为商品是货币转化为资本的前提？不变资本和可变资本的划分及其意义？相对剩余价值和绝对剩余价值的区别？ |
| **论述** | 如何理解资本主义经济危机的实质和根源？ |

### 预测题一（单选）

**资本主义经济危机的实质是（　）**
A. 生产绝对过剩
B. 生产相对过剩
C. 需求不足
D. 货币危机

> **答案：B**
> **解析**：资本主义经济危机的**实质是生产相对过剩**——不是绝对过剩，而是相对于劳动人民有支付能力的需求来说过剩。其根源在于资本主义基本矛盾（生产社会化与生产资料私人占有之间的矛盾）。这是区分资本主义经济危机与其他社会危机的关键。

### 预测题二（简答）

**简述不变资本和可变资本的划分及其意义。**

> 马克思以**资本的不同部分在价值增殖过程中所起的作用不同**为标准，把资本区分为**不变资本**和**可变资本**：
> 1. **不变资本（c）**——以生产资料形态存在的资本，在生产过程中价值量不发生变化（只是转移价值）。
> 2. **可变资本（v）**——以劳动力形态存在的资本，在生产过程中价值量发生变化（增殖），创造剩余价值。
>
> **划分意义**：
> 1. 揭示了剩余价值的真正源泉——可变资本，**揭露了资本主义剥削的秘密**
> 2. 为**剩余价值率（m'=m/v）**的计算提供了基础，准确反映资本家对工人的剥削程度

### 易错点提醒

::: warning 本节六大易错点
1. **货币转化为资本的前提=劳动力成为商品**（固定表述）。
2. **劳动力商品的"使用价值"是价值的源泉**——这是它与其他商品最根本的区别。
3. **不变资本不增殖，可变资本增殖**——剩余价值率 m'=m/v。
4. **绝对剩余价值（延长工作日）vs 相对剩余价值（缩短必要劳动时间）vs 超额剩余价值（个别率先改进）**——三概念易混。
5. **相对剩余价值的前提=整个社会劳动生产率提高**；**超额剩余价值的前提=个别资本家率先提高**——二者区分。
6. **经济危机实质=生产相对过剩**（不是绝对过剩）；**根源=资本主义基本矛盾**。
:::
