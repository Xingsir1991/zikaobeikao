# 第一节 商品经济和价值规律

> 教材第 182–200 页 · **政治经济学的起点**——商品理论。

---

## 一、商品经济（必背）

### 商品经济产生的历史条件（必背）

::: danger 必背：商品经济产生的两个条件
1. **社会分工**——是商品经济产生的一般前提
2. **生产资料和劳动产品属于不同的所有者**——是商品经济产生和存在的决定性条件
:::

### 商品经济的两种类型

- **简单商品经济**（小商品经济）——以个体私有制和个体劳动为基础
- **市场经济**（发达商品经济）——以社会化大生产为基础

---

## 二、商品的二因素（极高频考点，必背全部）

::: danger 必背：商品二因素
> **商品是使用价值和价值的矛盾统一体**：

1. **使用价值**——物品的**有用性**，是商品的**自然属性**，体现人与自然的关系
2. **价值**——凝结在商品中的**无差别的人类劳动**，是商品的**社会属性**，体现商品生产者之间的社会关系

> **交换价值**是价值的表现形式，**价值是交换价值的基础**。
:::

### 使用价值与价值的关系（必背）

| 关系 | 含义 |
| --- | --- |
| **统一性** | 商品是使用价值和价值的统一体，二者缺一不可 |
| **矛盾性** | 一个商品对于生产者只具有价值（让别人使用），对于消费者只具有使用价值（让别人实现价值）|

> **使用价值是价值的物质承担者**，价值是商品的本质属性。

---

## 三、生产商品的劳动二重性（极高频考点，必背全部）

::: danger 必背：劳动二重性
> **生产商品的劳动具有二重性**：

1. **具体劳动**——在一定的具体形式下进行的劳动，**创造使用价值**，**反映人与自然的关系**
2. **抽象劳动**——撇开具体形式的、无差别的一般人类劳动，**形成价值**，**反映商品生产者之间的社会关系**
:::

### 二因素与二重性的对应关系（必背，高频考点）

| 劳动 | 创造/形成 | 体现关系 |
| --- | --- | --- |
| **具体劳动** | **使用价值** | 人与自然的关系 |
| **抽象劳动** | **价值** | 人与人之间的社会关系 |

> **劳动二重性理论是理解马克思主义政治经济学的枢纽。**——马克思

### 商品内在矛盾的根源（必背）

::: danger 必背：私人劳动和社会劳动的矛盾
> **私人劳动和社会劳动的矛盾**是**简单商品经济的基本矛盾**，是商品内在一系列矛盾的**根源**。

- **私人劳动**——生产资料私有制决定
- **社会劳动**——社会分工决定
:::

---

## 四、价值规律（极高频考点，必背全部）

### 价值规律的内容（必背金句）

::: danger 必背：价值规律的内容
> **商品的价值量由生产商品的社会必要劳动时间决定；商品交换以价值量为基础，按照等价交换的原则进行。**

简记：**价值量由社会必要劳动时间决定，等价交换。**
:::

### 价值规律的表现形式（必背）

::: danger 必背：价值规律的表现形式
> **商品价格围绕价值上下波动**——这是价值规律的**唯一表现形式**。
:::

### 价值规律的作用（必背）

::: danger 必背：价值规律的三方面作用
1. **自发调节生产资料和劳动力在各生产部门之间的分配**（调节作用）
2. **自发刺激生产者改进技术、提高劳动生产率**（刺激作用）
3. **促使商品生产者优胜劣汰**（分化作用）
:::

---

## 五、货币（必背）

### 货币的产生

货币是商品交换发展到一定阶段的**自发产物**。它经历了：简单的价值形式→扩大的价值形式→一般价值形式→**货币形式**。

### 货币的本质（必背金句）

::: danger 必背：货币的本质
> **货币是从商品世界中分离出来的，固定地充当一般等价物的商品**。
:::

### 货币的五大职能（必背，极高频考点）

::: danger 必背：货币五大职能
1. **价值尺度**——衡量商品价值大小的尺度（**基本职能**）
2. **流通手段**——充当商品交换的媒介（**基本职能**）
3. **贮藏手段**——退出流通领域，作为社会财富贮藏
4. **支付手段**——延期支付、赊销赊购、纳税、还债等
5. **世界货币**——在世界市场上充当一般等价物
:::

### 概念辨析（高频考点）

| 职能 | 一句话区分 |
| --- | --- |
| 价值尺度 | 标价、定价（观念上的货币） |
| 流通手段 | 现钱买卖、一手交钱一手交货 |
| 贮藏手段 | 金银贮藏（必须是现实的足值金属货币） |
| 支付手段 | 赊销赊购、租金、工资、还债、税收 |
| 世界货币 | 国际贸易、外汇储备 |

## 六、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 商品二因素、抽象劳动形成什么、货币本质、货币五大职能、价值规律表现形式 |
| **多选** | 商品经济产生条件、价值规律三方面作用、货币五大职能 |
| **简答** | 商品二因素及其关系？劳动二重性及其与商品二因素的关系？价值规律的内容和作用？ |
| **论述** | 如何理解私人劳动和社会劳动的矛盾是简单商品经济的基本矛盾？ |

### 预测题一（单选）

**凝结在商品中的无差别的一般人类劳动是（　）**
A. 使用价值
B. 价值
C. 交换价值
D. 价格

> **答案：B**
> **解析**：**价值是凝结在商品中的无差别的人类劳动**。使用价值是物品的有用性；交换价值是价值的表现形式；价格是价值的货币表现。价值是商品的社会属性，体现商品生产者之间的社会关系。

### 预测题二（简答）

**简述劳动二重性及其与商品二因素的关系。**

> 生产商品的劳动具有**二重性**：
> 1. **具体劳动**——在一定的具体形式下进行的劳动，**创造使用价值**，反映人与自然的关系
> 2. **抽象劳动**——撇开具体形式的、无差别的一般人类劳动，**形成价值**，反映商品生产者之间的社会关系
>
> **具体劳动和抽象劳动是同一劳动过程的两个方面**，不是两次劳动。
>
> 与商品二因素的对应：**具体劳动创造使用价值，抽象劳动形成价值**。劳动二重性理论是理解马克思主义政治经济学的枢纽。

### 易错点提醒

::: warning 本节六大易错点
1. **使用价值是"自然属性"，价值是"社会属性"**——不要混。
2. **具体劳动创造使用价值，抽象劳动形成价值**——固定对应。
3. **货币的本质是一般等价物**，不是"价值尺度"或"流通手段"（那是职能）。
4. **货币五大职能中，价值尺度和流通手段是基本职能**。
5. **价值规律的表现形式是"价格围绕价值上下波动"**——这是唯一表现形式。
6. **简单商品经济的基本矛盾=私人劳动和社会劳动的矛盾**。
:::
