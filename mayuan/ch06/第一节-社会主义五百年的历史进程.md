# 第一节 社会主义五百年的历史进程

> 教材第 278–298 页 · **社会主义的"四次飞跃"**——空想→科学→现实→多国→中国发展。

---

## 一、社会主义从空想到科学（必背，极高频考点）

### 空想社会主义

::: danger 必背：空想社会主义
> **空想社会主义是科学社会主义的直接思想来源**。

代表人物：圣西门、傅立叶、欧文（19世纪初三大空想社会主义者）。
:::

空想社会主义的历史功绩和局限：

| 功绩 | 局限 |
| --- | --- |
| 对资本主义进行了深刻批判 | 未能揭示资本主义灭亡的客观规律 |
| 对未来社会提出许多积极设想 | 找不到实现社会变革的阶级力量 |
| | 拒绝无产阶级革命，主张和平改良 |

### 社会主义从空想到科学的飞跃：两大发现（必背金句）

::: danger 必背：马克思恩格斯两大发现
> **马克思和恩格斯创立的两大发现使社会主义从空想变为科学**：

1. **唯物史观**——揭示了人类社会历史发展的客观规律
2. **剩余价值学说**——揭示了资本主义剥削的秘密

> **《共产党宣言》（1848年）的发表，标志着科学社会主义的诞生。**
:::

### 科学社会主义创立的理论基础

- 唯物史观和剩余价值学说是**两大发现**
- 这两大发现揭示了：
  - 人类社会发展的**一般规律**（唯物史观）
  - 资本主义**必然灭亡、社会主义必然胜利**的客观根据（剩余价值学说）

## 二、社会主义从理论到现实：十月革命（必背，极高频考点）

### 巴黎公社（必背）

::: danger 必背：巴黎公社
> **1871年巴黎公社是无产阶级建立政权的第一次伟大尝试**。

> 它丰富了马克思主义关于无产阶级革命和无产阶级专政的学说。
:::

### 十月革命（必背金句，极高频考点）

::: danger 必背：十月革命
> **1917年俄国十月革命的胜利，实现了社会主义从理论到现实的伟大飞跃**。

> **建立了世界上第一个社会主义国家**——苏维埃政权（俄罗斯苏维埃联邦社会主义共和国）。
:::

十月革命的意义：
1. 开辟了人类历史**新纪元**——世界从此进入从资本主义向社会主义过渡的时代
2. 实现了社会主义从理论到现实的飞跃
3. 鼓舞了世界各国无产阶级革命
4. 推动了民族解放运动

### 列宁的贡献

::: danger 必背：列宁对科学社会主义的发展
> 列宁在帝国主义时代发展了科学社会主义：

1. **帝国主义理论**——揭示了帝国主义时代的特征
2. **社会主义革命可以首先在一国或多国取得胜利**——突破了马克思恩格斯"多国同时革命"的论断
3. **新经济政策**——探索了向社会主义过渡的道路
4. 创立了**第三国际（共产国际）**
:::

## 三、社会主义从一国到多国（必背）

### 社会主义阵营的形成

::: danger 必背：从一国到多国
> **第二次世界大战后，社会主义从一国发展到多国**——形成以苏联为首的**13个社会主义国家**，世界形成"两大阵营"对立格局。
:::

### 苏联模式（必背，客观评价）

::: danger 必背：苏联模式
> **苏联模式（斯大林模式）是特定的历史条件下的产物**：

- **历史作用**：短期内迅速推进工业化，奠定反法西斯战争胜利的物质基础
- **历史局限**：计划经济过度集中；党政不分；忽视轻工业和农业

> 苏联模式是在特定历史条件下形成的社会主义建设的**第一个具体模式**，**不是社会主义的唯一模式**。
:::

### 20世纪80年代末90年代初的剧变

> 东欧剧变、苏联解体是社会主义在实践中的**重大挫折**——但不是社会主义的失败，而是**苏联模式的失败**。

---

## 四、社会主义在中国焕发出蓬勃生机（必背）

### 中国特色社会主义

::: danger 必背：社会主义在中国的生机
> **中国共产党领导中国人民开辟了中国特色社会主义道路**：

1. 中国新民主主义革命胜利、新中国成立（1949年）——建立社会主义制度的前提
2. 三大改造完成（1956年）——确立社会主义基本制度
3. 改革开放（1978年）——开创中国特色社会主义
4. **进入新时代**（党的十八大以来）——中国特色社会主义进入新时代

> **社会主义在中国焕发出蓬勃生机，使世界社会主义走出低谷**。
:::

### 预测题一（单选）

**使社会主义从空想变为科学的两大发现是（　）**
A. 辩证法和唯物主义
B. 唯物史观和剩余价值学说
C. 唯物史观和阶级斗争理论
D. 剩余价值学说和国家学说

> **答案：B**
> **解析**：**马克思恩格斯创立的唯物史观和剩余价值学说两大发现使社会主义从空想变为科学**。唯物史观揭示了人类社会发展的一般规律；剩余价值学说揭示了资本主义剥削的秘密和资本主义必然灭亡的根源。

### 预测题二（简答）

**为什么说马克思恩格斯使社会主义从空想变为科学？**

> 马克思恩格斯创立的**两大发现——唯物史观和剩余价值学说**，使社会主义从空想变为科学：
> 1. **唯物史观揭示了人类社会发展的一般规律**——生产力与生产关系、经济基础与上层建筑的矛盾运动是社会发展的根本动力，揭示了资本主义必然为社会主义代替的客观必然性
> 2. **剩余价值学说揭示了资本主义剥削的秘密**——揭示了无产阶级的历史使命
> 3. **找到了实现社会变革的阶级力量**——无产阶级
> 4. **揭示了实现社会变革的途径**——无产阶级革命和无产阶级专政
>
> 两大发现克服了空想社会主义的根本局限，使社会主义建立在科学理论的基础上。1848年《共产党宣言》发表，标志着科学社会主义的诞生。

### 易错点提醒

::: warning 本节五大易错点
1. **两大发现=唯物史观+剩余价值学说**（不是辩证法+唯物主义，也不是阶级斗争理论）。
2. **巴黎公社=第一次伟大尝试**；**十月革命=第一个社会主义国家**——两者不要混淆。
3. **空想社会主义的直接思想来源——三大空想社会主义者：圣西门、傅立叶、欧文**。
4. **科学社会主义诞生的标志=1848年《共产党宣言》发表**。
5. **东欧剧变、苏联解体是苏联模式的失败，不是社会主义的失败**。
:::
