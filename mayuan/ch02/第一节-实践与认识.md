# 第一节 实践与认识

> 教材第 69–90 页 · **认识论基石**——实践观与认识的本质、认识运动基本规律。

---

## 一、实践是认识的基础（极高频考点，必背）

### 实践的科学含义（必背金句）

::: danger 必背：实践的含义
> **实践是人类能动地改造客观世界的社会性的物质活动。**
:::

实践具有三个基本特征：

| 特征 | 含义 |
| --- | --- |
| **客观物质性** | 实践的主体、客体、手段、结果都是客观的 |
| **自觉能动性** | 实践是人有意识、有目的的活动 |
| **社会历史性** | 实践是社会性的活动，并随历史发展而发展 |

### 实践的基本形式

1. **物质生产实践**（最基本、决定其他形式）
2. **社会政治实践**
3. **科学文化实践**

### 实践决定认识的四个方面（极高频考点，必背金句）

::: danger 必背：实践是认识的基础（实践决定认识）
1. **实践是认识的来源**——认识在实践中产生
2. **实践是认识发展的动力**——实践不断提出新问题、提供新经验
3. **实践是认识的目的**——认识世界是为了改造世界
4. **实践是检验认识真理性的唯一标准**

> **实践决定认识，是认识的基础。**
:::

### 概念辨析

| 提法 | 正误 |
| --- | --- |
| 实践是认识的**来源** | ✔ |
| 实践是认识发展的**动力** | ✔ |
| 实践是认识的**目的** | ✔ |
| 实践是检验认识真理性的**唯一标准** | ✔ |
| 直接经验是认识的**唯一**来源 | ✘（直接经验和间接经验都是认识的"来源"，归根到底源于实践）|

## 二、认识的本质（必背，极高频考点）

::: danger 必背：认识的本质
> **认识是主体在实践基础上对客体的能动反映。**
:::

### 认识本质的三个要点

1. **认识是主体对客体的反映**——反映论（不是先验论）
2. **这种反映建立在实践基础上**——以实践为基础
3. **这种反映是能动的、创造性的反映**——能动反映论（不是直观被动反映）

### 马克思主义认识论与其他哲学认识论的区别（必背）

| 哲学流派 | 认识论性质 | 根本特点 |
| --- | --- | --- |
| **辩证唯物主义（马原）** | **能动的革命的反映论** | 以实践为基础，是能动创造性的反映 |
| **旧唯物主义** | **直观被动反映论** | 离开实践、离开辩证法 |
| **唯心主义** | **先验论** | 认识是主观自生或神启的 |

## 三、认识运动的基本规律（必背，极高频考点）

### 两次飞跃（必背金句）

::: danger 必背：认识运动的两次飞跃
1. **第一次飞跃**：从**实践到认识**（感性认识→理性认识）
2. **第二次飞跃**：从**认识到实践**（理性认识回到实践）

> **认识运动的基本规律**：实践→认识→实践，循环往复以至无穷。
:::

### 第一次飞跃：感性认识到理性认识（必背）

| 阶段 | 形式 | 特点 |
| --- | --- | --- |
| **感性认识** | 感觉、知觉、表象 | 直接性、具体性、表面性（初级阶段） |
| **理性认识** | 概念、判断、推理 | 间接性、抽象性、深刻性（高级阶段） |

实现第一次飞跃必须具备两个条件：
1. **占有丰富真实的感性材料**（基础）
2. **运用科学思维方法进行加工制作**（思考）

### 第二次飞跃：理性认识到实践（必背，更重要）

::: danger 必背：第二次飞跃更重要
> **从认识到实践（第二次飞跃）比从实践到认识（第一次飞跃）意义更加重大。**
:::

第二次飞跃更重要的原因：
1. 理性认识只有回到实践才能**检验其真理性**
2. 理性认识只有回到实践才能**指导实践、实现认识的目的**

### 认识运动的反复性和无限性

> 实践→认识→实践，**循环往复、以至无穷**——这是认识发展的**总规律**。

公式：**实践、认识、再实践、再认识，循环往复以至无穷，逐步深入。**

## 四、考点对接

### 高频考点分布

| 题型 | 高频考点 |
| --- | --- |
| **单选** | 实践含义、实践决定认识四方面、认识本质、两次飞跃 |
| **多选** | 实践三个特征、实践三种基本形式、感性认识与理性认识三种形式 |
| **简答** | 为什么说实践是认识的基础？认识运动的两次飞跃是什么？ |
| **论述** | 如何理解认识运动的基本规律？ |

### 预测题一（单选）

**认识的本质是（　）**
A. 主体对客体的直观反映
B. 主体在实践基础上对客体的能动反映
C. 主体对客体的主观建构
D. 客体对主体的决定作用

> **答案：B**
> **解析**：马克思主义认识论认为，**认识是主体在实践基础上对客体的能动反映**。A 是旧唯物主义观点；C 是主观唯心主义；D 颠倒了主客体关系。

### 预测题二（简答）

**简述实践是认识的基础（实践决定认识的表现）。**

> 实践决定认识，是认识的基础。具体表现在四个方面：
> 1. **实践是认识的来源**——认识在实践中产生
> 2. **实践是认识发展的动力**——实践提出新问题，推动认识发展
> 3. **实践是认识的目的**——认识世界的目的在于改造世界
> 4. **实践是检验认识真理性的唯一标准**

### 易错点提醒

::: warning 本节四大易错点
1. **实践是认识的"基础"**——不要只答"来源"，四个方面缺一不可。
2. **第二次飞跃比第一次飞跃更重要**——这是固定表述。
3. **认识是"能动反映"**——不是直观反映、不是主观建构。
4. **直接经验和间接经验都是认识来源**——但归根到底来源于实践。
:::
