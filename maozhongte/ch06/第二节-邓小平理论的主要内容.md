# 第二节 邓小平理论的主要内容

## 一、社会主义初级阶段理论和党的基本路线（必背）

### 1. 社会主义初级阶段理论

::: danger 必背：初级阶段两层含义
1. 我国社会**已经是社会主义社会**——必须坚持而不能离开社会主义
2. 我国的社会主义社会**还处在初级阶段**——必须从这个实际出发，不能超越这个阶段
:::

- 系统阐述：党的十三大
- 特指：我国在生产力落后、商品经济不发达条件下建设社会主义必然要经历的**特定阶段**
- 是制定路线方针政策的**出发点和根本依据**

### 2. 党在社会主义初级阶段的基本路线（必背金句）

::: danger 必背：党在社会主义初级阶段的基本路线
> **领导和团结全国各族人民，以经济建设为中心，坚持四项基本原则，坚持改革开放，自力更生，艰苦创业，为把我国建设成为富强、民主、文明的社会主义现代化国家而奋斗。**

简明概括：**"一个中心、两个基本点"**
:::

| 部分 | 内容 | 回答的问题 |
| --- | --- | --- |
| 奋斗目标 | 富强、民主、文明的社会主义现代化国家 | —— |
| 基本途径 | 以经济建设为中心 | 社会主义根本任务 |
|  | 坚持四项基本原则 | 政治保证（**立国之本**） |
|  | 坚持改革开放 | 发展动力和外部条件（**强国之路**） |
| 领导力量依靠力量 | 领导和团结全国各族人民 | —— |
| 根本立足点 | 自力更生，艰苦创业 | —— |

### 3. 四项基本原则（必背）

::: danger 必背：四项基本原则
1. 坚持社会主义道路
2. 坚持人民民主专政
3. 坚持共产党的领导
4. 坚持马列主义、毛泽东思想
:::

> 四项基本原则是**立国之本**，是党和国家生存发展的**政治基石**；改革开放是**强国之路**。

## 二、社会主义根本任务和发展战略理论

### 1. 社会主义的根本任务（必背）

::: danger 必背：社会主义根本任务
**社会主义的根本任务是发展生产力**。党和国家的工作重点是经济建设。

邓小平："**发展才是硬道理**"。
:::

- **科学技术是第一生产力**（邓小平 1988 年）
- 发展生产力离不开科学技术——"四个现代化，关键是科学技术的现代化"

### 2."三步走"发展战略（必背）

::: danger 必背："三步走"战略
| 步骤 | 时间 | 目标 |
| --- | --- | --- |
| 第一步 | 到 1990 年 | 国民生产总值比 1980 年翻一番，**解决温饱问题** |
| 第二步 | 到 20 世纪末 | 国民生产总值再增长一倍，**人民生活达到小康水平** |
| 第三步 | 到 21 世纪中叶 | 人均 GNP 达到中等发达国家水平，**基本实现现代化** |
:::

### 3. 共同富裕思想

> 让一部分人、一部分地区先富起来，大原则是共同富裕。

- "两个大局"思想：沿海先发展→带动内地发展
- 共同富裕≠完全平均，是**有先有后**的历史过程

## 三、社会主义改革开放和社会主义市场经济理论

### 1. 改革开放理论

::: danger 必背：改革与开放的核心论断
- **改革是中国的第二次革命**——是社会主义制度的自我完善和发展
- **改革是社会主义社会发展的直接动力**
- **对外开放是基本国策**——"现在的世界是开放的世界，中国的发展离不开世界"
:::

### 2."三个有利于"标准（必背）

::: danger 必背："三个有利于"标准
判断改革和各方面是非得失，归根到底主要看：
1. 是否有利于**发展社会主义社会的生产力**
2. 是否有利于**增强社会主义国家的综合国力**
3. 是否有利于**提高人民生活水平**
:::

### 3. 社会主义市场经济理论（必背）

::: danger 必背：邓小平南方谈话
> **计划多一点还是市场多一点，不是社会主义与资本主义的本质区别。计划经济不等于社会主义，资本主义也有计划；市场经济不等于资本主义，社会主义也有市场。计划和市场经济都是经济手段。**
:::

社会主义市场经济理论的内涵：
1. 计划经济和市场经济**不是划分社会制度的标志**
2. 计划和市场都是**经济手段**
3. 市场经济作为资源配置手段**本身不具有制度属性**

> 党的十四大正式把建立**社会主义市场经济体制**确立为我国经济体制改革的目标。

## 四、"两手抓，两手都要硬"

::: danger 必背："两手抓"系列
- 一手抓**物质文明**，一手抓**精神文明**
- 一手抓**建设**，一手抓**法制**
- 一手抓**改革开放**，一手抓**惩治腐败**
:::

> "我们要建设的社会主义国家，不但要有高度的物质文明，而且要有高度的精神文明。"

---

## 五、"一国两制"与祖国统一（必背）

::: danger 必背："一国两制"科学内涵
**基本内容**：
1. 坚持**一个中国**——核心，是发展两岸关系和实现和平统一的**基础**
2. **两制并存**——国家主体坚持社会主义制度，港澳台保持原有制度长期不变
3. **高度自治**——港澳台作为特别行政区享有高度自治权
4. 尽最大努力争取**和平统一**，但不承诺放弃使用武力
5. 解决台湾问题，实现祖国完全统一，**寄希望于台湾人民**
:::

::: tip 核心论断
"一国两制"是邓小平运用辩证唯物主义和历史唯物主义，把**和平共处原则**用于解决一个国家的统一问题，是对马克思主义国家学说的**创造性发展**。
:::

## 六、其他内容（简要）

- **外交和国际战略**：和平与发展是时代主题；独立自主的和平外交政策；和平共处五项原则；"冷静观察、稳住阵脚、沉着应付"
- **党的建设理论**：关键在于坚持、加强和改善党的领导；坚持民主集中制；干部队伍"四化"方针（革命化、年轻化、知识化、专业化）

### 预测题一（单选）

**党在社会主义初级阶段基本路线的"两个基本点"是（　）**
A. 以经济建设为中心和坚持改革开放
B. 坚持四项基本原则和坚持改革开放
C. 坚持改革开放和坚持自力更生
D. 坚持党的领导和坚持人民民主专政

> **答案：B**
> **解析**：基本路线简明概括为"**一个中心、两个基本点**"。一个中心=以经济建设为中心；**两个基本点=坚持四项基本原则（立国之本）、坚持改革开放（强国之路）**。

### 预测题二（简答）

**简述邓小平"三个有利于"标准。**

> 判断改革和各方面工作是非得失，归根到底要以"三个有利于"为标准：
> 1. 是否有利于**发展社会主义社会的生产力**（首要、最根本的一条）
> 2. 是否有利于**增强社会主义国家的综合国力**
> 3. 是否有利于**提高人民生活水平**
>
> 三者内在统一——发展生产力是前提，增强综合国力和提高人民生活水平是发展生产力的目的。

### 易错点提醒

::: warning 本节五大易错点
1. **"一个中心、两个基本点"**：一个中心=经济建设；两个基本点=四项基本原则+改革开放。
2. **立国之本=四项基本原则**；**强国之路=改革开放**；**兴国之要=以经济建设为中心**。
3. **根本任务=发展生产力**（不是"共同富裕"，共同富裕是目标）。
4. **改革的性质=第二次革命+社会主义制度的自我完善和发展**（不是"一个阶级推翻另一个阶级"的革命）。
5. **"一国两制"的核心=一个中国**（不是两制并存或高度自治）。
:::
