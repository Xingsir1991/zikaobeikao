# 第一节 从新民主主义到社会主义的转变

> 教材第 67–75 页 · 过渡时期总路线——"一化三改"。

---

## 一、新民主主义社会是一个过渡性的社会（必背）

### 核心论断

::: danger 必背
> 新民主主义社会**不是一个独立的形态**，而是由新民主主义向社会主义转变的**过渡性社会形态**。
:::

### 五种经济成分（必背）

| 经济成分 | 性质 |
| --- | --- |
| **国营经济** | 社会主义性质（领导地位） |
| **合作社经济** | 半社会主义性质 |
| **个体经济** | 农民和手工业者 |
| **私人资本主义经济** | 资本主义性质 |
| **国家资本主义经济** | 过渡形式 |

### 主要矛盾（必背）

> 随着土地改革基本完成，**工人阶级和资产阶级的矛盾**逐步成为我国社会的**主要矛盾**。

---

## 二、党在过渡时期的总路线（必背，极高频考点）

### 总路线全文（必背）

::: danger 必背：过渡时期总路线
> **从中华人民共和国成立，到社会主义改造基本完成，这是一个过渡时期。党在这个过渡时期的总路线和总任务，是要在一个相当长的时期内，逐步实现国家的社会主义工业化，并逐步实现国家对农业、对手工业和对资本主义工商业的社会主义改造。**
:::

### "一化三改"（必背，极高频考点）

::: danger 必背：一化三改
- **"一化"**——社会主义**工业化**（主体）
- **"三改"**——对**农业**、**手工业**和**资本主义工商业**的社会主义改造（两翼）

> 比喻为鸟的**"主体"和"两翼"**，两者相辅相成、相互促进。
:::

### 总路线的特点

> 社会主义**建设**和社会主义**改造**同时并举——体现了解放生产力与发展生产力、变革生产关系与发展生产力的有机统一。

### 总路线提出的依据

1. 已有相对强大的国营经济（1952年工业产值占56%）
2. 农民有互助合作的要求
3. 国家资本主义形式已出现
4. 有利的国际环境

---

## 三、考点对接

### 预测题（单选）

**党在过渡时期总路线的主体是（　）**
A. 对农业的社会主义改造
B. 对资本主义工商业的社会主义改造
C. **逐步实现国家的社会主义工业化**
D. 对手工业的社会主义改造

> **答案：C**——"一化"（工业化）是主体，"三改"是两翼。

### 易错点

::: warning
1. **一化=工业化**（主体）；**三改=农业+手工业+资工商**（两翼）。
2. 新民主主义社会是**过渡性**社会形态，不是独立的。
3. 过渡时期**主要矛盾=工人阶级与资产阶级的矛盾**。
:::
