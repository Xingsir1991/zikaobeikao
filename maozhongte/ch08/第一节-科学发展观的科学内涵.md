## 二、第一要义：发展（推动经济社会发展）

::: danger 必背：第一要义
**推动经济社会发展是科学发展观的第一要义。**

在当代中国，**坚持发展是硬道理的本质要求就是坚持科学发展**。
:::

### 核心论断

> "发展是解决中国一切问题的'总钥匙'。"

::: tip 坚持科学发展必须做到
1. **加快转变经济发展方式**——把经济结构战略性调整作为主攻方向
2. **推动科学技术的跨越式发展**——建设创新型国家
3. **培养高素质创新型人才**——实施人才强国战略
4. **善于抓住和用好机遇**——21 世纪头二十年是重要战略机遇期
:::

## 三、核心立场：以人为本（必背）

::: danger 必背：核心立场
**以人为本是科学发展观的核心立场。**

- 集中体现了马克思主义的基本原理
- 体现了我们党全心全意为人民服务的根本宗旨
- 体现了推动经济社会发展的根本目的
- 是社会主义的本质特征
:::

### "以人为本"的科学内涵

> 以人为本的"人"，是指**人民群众**——以工人、农民、知识分子等劳动者为主体，包括社会各阶层人民在内的中国最广大人民。

::: danger 必背：以人为本的根本含义
**发展为了人民、发展依靠人民、发展成果由人民共享。**

做到"**权为民所用、情为民所系、利为民所谋**"。
:::

### "以人为本"与相关概念的区别

| 概念 | 区别 |
| --- | --- |
| 中国古代"民本思想" | 本质是为维护封建统治者地位（唯心史观） |
| 西方"人本主义" | 离开具体历史条件、人的社会性（为资产阶级服务） |
| **马克思主义"以人为本"** | 坚持历史唯物主义立场，人民利益放首位（**根本区别**） |

## 四、基本要求：全面协调可持续（必背）

::: danger 必背：基本要求
**全面协调可持续是科学发展观的基本要求。**
:::

### 三个关键词的内涵

| 关键词 | 含义 |
| --- | --- |
| **全面** | 发展要有全面性、整体性——经济、政治、文化、社会、生态等各个方面都要发展 |
| **协调** | 发展要有协调性、均衡性——各个方面、各个环节的发展要相互适应、相互促进 |
| **可持续** | 发展要有持久性、连续性——不仅当前要发展，还要保证长远发展 |

### 五位一体总体布局（必背）

::: danger 必背：五大建设定位
| 建设 | 定位 |
| --- | --- |
| 经济建设 | **中心和基础** |
| 政治建设 | 方向和保障 |
| 文化建设 | 灵魂和血脉 |
| 社会建设 | 支撑和归宿 |
| 生态文明建设 | 根基和条件 |
:::

### 文明发展道路（必背）

::: danger 必背：文明发展道路
坚持走**生产发展、生活富裕、生态良好**的文明发展道路。
:::

### 五个统筹（必背）

::: danger 必背：五个统筹
1. 统筹**城乡发展**
2. 统筹**区域发展**
3. 统筹**经济社会发展**
4. 统筹**人与自然和谐发展**
5. 统筹**国内发展和对外开放**

党的十七大进一步强调：统筹中央和地方关系、统筹个人利益和集体利益/局部利益和整体利益/当前利益和长远利益、统筹国内国际两个大局。

### 预测题一（单选）

**科学发展观的核心立场是（　）**
A. 推动经济社会发展
B. 以人为本
C. 全面协调可持续
D. 统筹兼顾

> **答案：B**
> **解析**：科学发展观——**第一要义是发展，核心立场是以人为本，基本要求是全面协调可持续，根本方法是统筹兼顾**。A 是第一要义，C 是基本要求，D 是根本方法。

### 预测题二（简答）

**简述科学发展观的科学内涵。**

> 科学发展观的科学内涵包括四个方面：
>
> 1. **第一要义是发展**（推动经济社会发展）——发展是解决中国一切问题的"总钥匙"，坚持发展是硬道理的本质要求就是坚持科学发展。
> 2. **核心立场是以人为本**——以最广大人民的根本利益为出发点和落脚点，做到发展为了人民、发展依靠人民、发展成果由人民共享。
> 3. **基本要求是全面协调可持续**——全面推进经济、政治、文化、社会、生态文明建设，促进现代化建设各个环节相协调，坚持走生产发展、生活富裕、生态良好的文明发展道路。
> 4. **根本方法是统筹兼顾**——正确认识和妥善处理中特事业中的重大关系，做到"五个统筹"。

### 易错点提醒

::: warning 本节五大易错点
1. **四要素必须精准对应**：第一要义=发展，核心立场=以人为本，基本要求=全面协调可持续，根本方法=统筹兼顾（四对一，绝不混淆）。
2. **注意是"核心立场"**（不是单纯的"核心"）——教材表述为"核心立场"。
3. **精神实质十六字**：解放思想、实事求是、与时俱进、求真务实（不能少"求真务实"）。
4. **文明发展道路**：生产发展、生活富裕、生态良好（顺序不能错）。
5. **五个统筹**要全部记住，是高频多选题考点。
:::
