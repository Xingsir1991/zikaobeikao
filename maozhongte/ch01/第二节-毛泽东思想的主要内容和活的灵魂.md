# 第二节 毛泽东思想的主要内容和活的灵魂

> 教材第 20–27 页 · 毛泽东思想的六大内容和活的灵魂三个方面。

---

## 一、毛泽东思想的主要内容（六大方面，必背）

::: danger 必背：六大主要内容
1. **新民主主义革命理论**
2. **社会主义革命和社会主义建设理论**
3. **革命军队建设和军事战略的理论**
4. **政策和策略的理论**
5. **思想政治工作和文化工作的理论**
6. **党的建设理论**
:::

### 六大内容的要点

| 内容 | 关键点 |
| --- | --- |
| 新民主主义革命理论 | 三大法宝：统一战线、武装斗争、党的建设 |
| 社会主义革命建设理论 | 人民民主专政、社会主义改造、两类矛盾 |
| 军事战略 | 人民军队、人民战争思想 |
| 政策策略 | "政策和策略是党的生命"；有理有利有节 |
| 思想政治文化 | 百花齐放、百家争鸣；古为今用、洋为中用 |
| 党的建设 | **思想上建党**；三大作风 |

### 党的三大作风（必背）

::: danger 必背：三大作风
毛泽东指出，中国共产党区别于其他政党的显著标志：
1. **理论和实践相结合**的作风
2. **和人民群众紧密地联系在一起**的作风
3. **自我批评**的作风
:::

## 二、毛泽东思想活的灵魂（极高频考点，必背）

### 核心论断（必背金句）

::: danger 必背：活的灵魂三个方面
> **实事求是、群众路线、独立自主**，是毛泽东思想活的灵魂。

这三个方面是毛泽东把辩证唯物主义和历史唯物主义运用于中国革命和建设实践中所形成的**立场、观点、方法**。
:::

### ① 实事求是（必背）

> **实事求是是毛泽东思想的基本点，是毛泽东思想的精髓**。

- 就是**一切从实际出发，理论联系实际**，坚持在实践中检验真理和发展真理
- 中国共产党的**思想路线**
- 毛泽东《反对本本主义》提出"**没有调查，没有发言权**"
- 《实践论》《矛盾论》深刻阐述

### ② 群众路线（必背）

> **群众路线**就是**一切为了群众，一切依靠群众，从群众中来，到群众中去**，把党的正确主张变为群众的自觉行动。

- 党的**根本工作路线**、党的**生命线**
- 本质上体现的是马克思主义关于**人民群众是历史的创造者**这一基本原理
- 毛泽东："人民，只有人民，才是创造世界历史的动力"

### ③ 独立自主（必背）

> **独立自主**就是坚持独立思考，走自己的路，坚定不移地维护民族独立、捍卫国家主权。

- 立足点放在**依靠自己力量**的基础上
- 中华民族精神之魂
- 立党立国的重要原则
- 毛泽东1930年《反对本本主义》："中国革命斗争的胜利要靠中国同志了解中国情况"

### 预测题一（单选）

**毛泽东思想的精髓是（　）**
A. 群众路线
B. 实事求是
C. 独立自主
D. 武装斗争

> **答案：B**
> **解析**：教材原文——**实事求是是毛泽东思想的基本点，是毛泽东思想的精髓**。群众路线是根本工作路线；独立自主是立党立国的重要原则；武装斗争是新民主主义革命三大法宝之一。

### 预测题二（简答）

**简述毛泽东思想活的灵魂。**

> 毛泽东思想活的灵魂是贯串于毛泽东思想各个组成部分的**立场、观点、方法**，包括**三个方面**：
> 1. **实事求是**——毛泽东思想的基本点和精髓；一切从实际出发，理论联系实际
> 2. **群众路线**——一切为了群众，一切依靠群众，从群众中来，到群众中去；党的生命线和根本工作路线
> 3. **独立自主**——坚持独立思考，走自己的路；立党立国的重要原则

### 易错点提醒

::: warning 本节三大易错点
1. **活的灵魂三个方面**：实事求是、群众路线、独立自主——缺一不可。
2. **精髓=实事求是**（不是群众路线或独立自主）。
3. **三大作风**：理论联系实际、密切联系群众、自我批评——不是活的灵魂。
:::
