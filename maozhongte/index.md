# 🔵 毛中特（毛泽东思想与中国特色社会主义理论体系）

> 毛泽东思想、邓小平理论、"三个代表"、科学发展观、习近平新时代中国特色社会主义思想。

## 使用说明

- 左侧目录展示本科目所有章节
- 每篇笔记底部可点击 **「标记为已掌握」**，在 [进度看板](/progress) 汇总
- 顶部搜索框支持中文全文检索

## 添加新资料

把整理好的 md 文件直接放进 `maozhongte/` 目录，例如：

```
maozhongte/
├── index.md                    # 本页
├── 毛泽东思想-新民主主义革命.md
├── 邓小平理论.md
└── 习近平新时代中国特色社会主义思想.md
```

并在 `.vitepress/config.mts` 的 `sidebarGroup` 中补充对应条目。
